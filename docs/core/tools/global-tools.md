<span data-ttu-id="bba3a-153">グローバル ツールは、[フレームワークに依存するアプリケーション](../deploying/index.md#framework-dependent-deployments-fdd)です。つまり、お使いのマシンにインストールされている .NET Core ランタイムに依存します。</span><span class="sxs-lookup"><span data-stu-id="bba3a-153">Global Tools are [framework-dependent applications](../deploying/index.md#framework-dependent-deployments-fdd), which means they rely on a .NET Core runtime installed on your machine.</span></span> <span data-ttu-id="bba3a-154">予定していたランタイムが見つからない場合、ツールは次のような通常の .NET Core ランタイムのロール フォワード ルールに従います。</span><span class="sxs-lookup"><span data-stu-id="bba3a-154">If the expected runtime is not found, they follow normal .NET Core runtime roll-forward rules such as:</span></span>
