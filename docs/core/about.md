## <a name="comparisons-to-other-net-implementations"></a><span data-ttu-id="67909-172">その他の .NET 実装との比較</span><span class="sxs-lookup"><span data-stu-id="67909-172">Comparisons to other .NET implementations</span></span>

### <a name="comparison-with-net-framework"></a><span data-ttu-id="67909-174">.NET Framework との比較</span><span class="sxs-lookup"><span data-stu-id="67909-174">Comparison with .NET Framework</span></span>

<span data-ttu-id="67909-177">.NET Core と .NET Framework の主な違いは、次のとおりです。</span><span class="sxs-lookup"><span data-stu-id="67909-177">The major differences between .NET Core and the .NET Framework:</span></span>

- <span data-ttu-id="67909-178">**アプリ モデル** -- .NET Core は一部の .NET Framework アプリ モデルをサポートしていません。</span><span class="sxs-lookup"><span data-stu-id="67909-178">**App-models** -- .NET Core does not support all the .NET Framework app-models.</span></span> <span data-ttu-id="67909-179">具体的には、ASP.NET Web フォームと MVC をサポートしていません。</span><span class="sxs-lookup"><span data-stu-id="67909-179">In particular, it doesn't support ASP.NET Web Forms and MVC.</span></span> <span data-ttu-id="67909-180">[.NET Core 3 が WPF と Windows フォームをサポートする](https://blogs.msdn.microsoft.com/dotnet/2018/05/07/net-core-3-and-support-for-windows-desktop-applications/)ことが発表されています。</span><span class="sxs-lookup"><span data-stu-id="67909-180">It was announced that [.NET Core 3 will support WPF and Windows Forms](https://blogs.msdn.microsoft.com/dotnet/2018/05/07/net-core-3-and-support-for-windows-desktop-applications/).</span></span>
- <span data-ttu-id="67909-181">**API** -- .NET Core には、ファクタリングが異なる (アセンブリ名が異なる、型に対して公開されているメンバーが主要なケースで異なる) .NET Framework 基本クラス ライブラリの大規模なサブセットが含まれています。</span><span class="sxs-lookup"><span data-stu-id="67909-181">**APIs** -- .NET Core contains a large subset of .NET Framework Base Class Library, with a different factoring (assembly names are different; members exposed on types differ in key cases).</span></span> <span data-ttu-id="67909-182">この相違によって、ポートの発信元を .NET Core に変更しなければならない場合があります ([microsoft/dotnet-apiport](https://github.com/microsoft/dotnet-apiport) を参照)。</span><span class="sxs-lookup"><span data-stu-id="67909-182">These differences require changes to port source to .NET Core in some cases (see [microsoft/dotnet-apiport](https://github.com/microsoft/dotnet-apiport)).</span></span> <span data-ttu-id="67909-183">.NET Core は [.NET Standard](../standard/net-standard.md) API 仕様を実装します。</span><span class="sxs-lookup"><span data-stu-id="67909-183">.NET Core implements the [.NET Standard](../standard/net-standard.md) API specification.</span></span>
