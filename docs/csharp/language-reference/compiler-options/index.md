# C# Compiler Options
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

コンパイラは、実行可能 \(.exe\) ファイル、ダイナミック リンク ライブラリ \(.dll\)、またはコード モジュール \("\) を生成します。  
  
 各コンパイラ オプションには、それぞれ **\-option** と **\/option** という 2 つの形式があります。  ドキュメントでは **\/option** の形式だけを示しています。  
  
 Visual Web Developer 2008 では、web.config ファイルでコンパイラ オプションを設定します。  詳細については、「[\<compiler\> 要素](../Topic/%3Ccompiler%3E%20Element.md)」を参照してください。
