# /nologo (C# Compiler Options)
**\/nologo** オプションは、コンパイラ起動時の著作権情報や、コンパイル中の情報メッセージを表示しないようにします。  
  
## 構文  
  
```  
/nologo  
```  
  
## 解説  
 このオプションは開発環境内では利用できません。このオプションを利用できるのは、コマンド ラインからコンパイルするときだけです。  
  
 このコンパイラ オプションは、Visual Studio で利用できず、プログラムで変更することもできません。
