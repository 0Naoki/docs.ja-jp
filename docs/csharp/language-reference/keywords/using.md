# <a name="using-c-reference"></a>using (C# リファレンス)
`using` キーワードは、主に次の 2 つの場合に使用します。  
  
-   ディレクティブとして使用: 名前空間のエイリアスを作成すとき、または他の名前空間で定義されている型をインポートするとき。 「[using ディレクティブ](../../../csharp/language-reference/keywords/using-directive.md)」を参照してください。  
  
-   ステートメントとして使用: 破棄されるオブジェクトの末尾でスコープを定義するとき。 「[using ステートメント](../../../csharp/language-reference/keywords/using-statement.md)」を参照してください。  
  
さらに、[using static](using-static.md) ディレクティブを使用すると、型名を指定せずにアクセスできる静的メンバーの型を指定することができます。
