# global (C# リファレンス)
`global` コンテキスト キーワードは、[:: 演算子](../../../csharp/language-reference/operators/namespace-alias-qualifer.md)の前に置かれ、C\# プログラムの既定の名前空間であるグローバル名前空間を参照します。このキーワードで参照しない限り、グローバル名前空間は無名の名前空間です。  詳細については、「[方法: グローバル名前空間エイリアスを使用する](../../../csharp/programming-guide/namespaces/how-to-use-the-global-namespace-alias.md)」を参照してください。
