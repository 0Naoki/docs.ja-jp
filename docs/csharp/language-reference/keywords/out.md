# <a name="out-c-reference"></a>out (C# リファレンス)
`out` コンテキスト キーワードは、2 つのコンテキストで使うことができます。

- [パラメーター修飾子](../../../csharp/language-reference/keywords/out-parameter-modifier.md)として。この場合、値ではなく参照で、メソッドに引数を渡すことができます。

- インターフェイスおよびデリゲートの[ジェネリック型パラメーターの宣言](../../../csharp/language-reference/keywords/out-generic-modifier.md)で。この場合は、型パラメーターが共変であることを指定します。
