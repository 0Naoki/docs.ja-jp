# メソッドのパラメーター (C# リファレンス)
[ref](../../../csharp/language-reference/keywords/ref.md) または [out](../../../csharp/language-reference/keywords/out.md) を指定せずにメソッドのパラメーターを宣言した場合、このパラメーターには値を関連付けることができます。  この値はメソッド内で変更できますが、変更された値は、制御が呼び出しプロシージャに戻されたときには保持されません。  メソッド パラメーターのキーワードを使用すると、この動作を変更できます。  
  
 ここでは、メソッド パラメーターの宣言に使用できるキーワードについて説明します。  
  
-   [params](../../../csharp/language-reference/keywords/params.md)  
  
-   [ref](../../../csharp/language-reference/keywords/ref.md)  
  
-   [&#91;out&#93;](../../../csharp/language-reference/keywords/out.md)
