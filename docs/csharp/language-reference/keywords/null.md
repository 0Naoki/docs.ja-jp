# null (C# リファレンス)
`null` キーワードは、null 参照を表すリテラル キーワードです。null 参照はオブジェクトを一切参照しません。  `null` は参照型変数の既定値です。  値型には、通常、null を設定できません。  しかし、C\# 2.0 には、値型に null を設定できるようにする方法が用意されています。  [null 許容型](../../../csharp/programming-guide/nullable-types/index.md) を参照してください。
