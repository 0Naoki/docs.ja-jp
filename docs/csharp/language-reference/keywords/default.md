# default (C# リファレンス)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

`default` キーワード。  
  
 `default` キーワードは `switch` ステートメントまたはジェネリック コードで使用できます。  
  
-   [switch \(C\# リファレンス\)](../../../csharp/language-reference/keywords/switch.md) : 既定のラベルを指定します。  
  
-   [ジェネリック コードの default キーワード \(C\# プログラミング ガイド\)](../../../csharp/programming-guide/generics/default-keyword-in-generic-code.md) : 型パラメーターの既定値を指定します。  参照型の場合は null、値型の場合はゼロになります。
