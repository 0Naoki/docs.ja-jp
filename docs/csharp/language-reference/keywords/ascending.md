# ascending (C# リファレンス)
`ascending` コンテキスト キーワードは、クエリ式の [orderby 句](../../../csharp/language-reference/keywords/orderby-clause.md)で使用され、並べ替え順序を昇順にすることを指定します。  `ascending` は既定の並べ替え順序なので、指定する必要はありません。
