# <a name="operator-keywords-c-reference"></a>演算子キーワード (C# リファレンス)
オブジェクトの作成、オブジェクトのランタイム型のチェック、型のサイズの取得といった、さまざまなアクションを実行するために使用します。 このセクションでは、次のキーワードについて説明します。  
  
-   [as](../../../csharp/language-reference/keywords/as.md) オブジェクトを互換性のある型に変換します。  
  
-   [await](../../../csharp/language-reference/keywords/await.md) 待機中のタスクが完了するまで非同期のメソッドを中断します。  
  
-   [is](../../../csharp/language-reference/keywords/is.md) オブジェクトのランタイム型を確認します。  
  
-   [new](../../../csharp/language-reference/keywords/new.md)  
  
    -   [new 演算子](../../../csharp/language-reference/keywords/new-operator.md) オブジェクトを作成します。  
  
    -   [new 修飾子](../../../csharp/language-reference/keywords/new-modifier.md) 継承されたメンバーを非表示にします。  
  
    -   [new 制約](../../../csharp/language-reference/keywords/new-constraint.md) 型パラメーターを修飾します。  
  
-   [nameof](nameof.md) 変数、型、またはメンバーの単純な (修飾されていない) 文字列名を取得します。
 
-   [sizeof](../../../csharp/language-reference/keywords/sizeof.md) 型のサイズを取得します。  
  
-   [typeof](../../../csharp/language-reference/keywords/typeof.md) 型の **System.Type** オブジェクトを取得します。  
  
-   [true](../../../csharp/language-reference/keywords/true.md)  
  
    -   [true 演算子](../../../csharp/language-reference/keywords/true-operator.md) true を示す場合はブール値 true を返し、それ以外の場合は false を返します。  
  
    -   [true リテラル](../../../csharp/language-reference/keywords/true-literal.md) ブール値 true を表します。  
  
-   [false](../../../csharp/language-reference/keywords/false.md)  
  
    -   [false 演算子](../../../csharp/language-reference/keywords/false-operator.md) false を示す場合はブール値 true を返し、それ以外の場合は false を返します。  
  
    -   [false リテラル](../../../csharp/language-reference/keywords/false-literal.md) ブール値 false を表します。  
  
-   [stackalloc](../../../csharp/language-reference/keywords/stackalloc.md) スタックにメモリ ブロックを割り当てます。  
  
 演算子およびステートメントとして使用できる次のキーワードは、「[ステートメント](../../../csharp/language-reference/keywords/statement-keywords.md)」セクションで説明します。  
  
-   [checked](../../../csharp/language-reference/keywords/checked.md) checked コンテキストを指定します。  
  
-   [unchecked](../../../csharp/language-reference/keywords/unchecked.md) unchecked コンテキストを指定します。
