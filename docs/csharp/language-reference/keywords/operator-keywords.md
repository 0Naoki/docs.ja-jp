# 演算子キーワード (C# リファレンス)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

型のサイズをチェックする作成などそのほかの操作を実行するためにオブジェクトを使用すると他のアクションを派生オブジェクトのランタイム型。  ここでは、以下のキーワードについて説明します。  
  
-   [as](../../../csharp/language-reference/keywords/as.md) オブジェクトの型を互換性のある型に変換します。  
  
-   [待機します。](../../../csharp/language-reference/keywords/await.md) は必要なタスクが完了するまで非同期のメソッドを中断します。  
  
-   [is](../../../csharp/language-reference/keywords/is.md) オブジェクトのランタイム型を検証します。  
  
-   [new](../../../csharp/language-reference/keywords/new.md)  
  
    -   [new 演算子 \(C\# リファレンス\)](../../../visual-basic/language-reference/operators/new-operator.md) オブジェクトを作成します。  
  
    -   [new 修飾子](../../../csharp/language-reference/keywords/new-modifier.md) 継承されたメンバーを隠ぺいします。  
  
    -   [new 制約 \(C\# リファレンス\)](../../../csharp/language-reference/keywords/new-constraint.md) 型パラメーターを修飾します。  
  
-   [sizeof](../../../csharp/language-reference/keywords/sizeof.md) 型のサイズを取得します。  
  
-   [typeof](../../../csharp/language-reference/keywords/typeof.md) 型の **System.Type** オブジェクトを取得します。  
  
-   [true](../../../csharp/language-reference/keywords/true.md)  
  
    -   [true 演算子 \(C\# リファレンス\)](../../../csharp/language-reference/keywords/true-operator.md) true を示す場合にはブール値 true を返し、それ以外の場合は false を返します。  
  
    -   [true リテラル \(C\# リファレンス\)](../../../csharp/language-reference/keywords/true-literal.md) ブール値 true を表します。  
  
-   [false](../../../csharp/language-reference/keywords/false.md)  
  
    -   [false 演算子 \(C\# リファレンス\)](../../../csharp/language-reference/keywords/false-operator.md) false を示す場合にはブール値 true を返し、それ以外の場合は false を返します。  
  
    -   [false リテラル \(C\# リファレンス\)](../../../csharp/language-reference/keywords/false-literal.md) ブール値 false を表します。  
  
-   [stackalloc](../../../csharp/language-reference/keywords/stackalloc.md) スタックにメモリ ブロックを割り当てます。  
  
 次のキーワードは、演算子やステートメントとして使用できます。それぞれの詳細については、「[ステートメント](../../../csharp/language-reference/keywords/statement-keywords.md)」を参照してください。  
  
-   [checked](../../../csharp/language-reference/keywords/checked.md) checked コンテキストを指定します。  
  
-   [unchecked](../../../csharp/language-reference/keywords/unchecked.md) unchecked コンテキストを指定します。
