# <a name="operator-keywords-c-reference"></a>演算子キーワード (C# リファレンス)

オブジェクトの作成、オブジェクトのランタイム型のチェック、型のサイズの取得といった、さまざまなアクションを実行するために使用します。 このセクションでは、次のキーワードについて説明します。

- [as](as.md) オブジェクトを互換性のある型に変換します。

- [await](await.md) 待機中のタスクが完了するまで[非同期](async.md)メソッドを中断します。

- [is](is.md) オブジェクトのランタイム型をチェックするか、(C# 7.0 以降では) パターンとの比較によって式をテストします。

- [new](new.md)

  - [new 演算子](new-operator.md) オブジェクトを作成します。

  - [new 修飾子](new-modifier.md) 継承されたメンバーを非表示にします。

  - [new 制約](new-constraint.md) 型パラメーターを修飾します。

- [nameof](nameof.md) 変数、型、またはメンバーの単純な (修飾されていない) 文字列名を取得します。

- [sizeof](sizeof.md) アンマネージド型のサイズを取得します。  

- [typeof](typeof.md) 型の <xref:System.Type?displayProperty=nameWithType> オブジェクトを取得します。  

- [true](true.md)  

  - [true 演算子](true-false-operators.md) オペランドが確実に true であることを示す[ブール](bool.md)値 `true` を返します。

  - [true リテラル](true-literal.md) [ブール](bool.md)値 `true` を表します。

- [false](false.md)  

  - [false 演算子](true-false-operators.md) オペランドが確実に false であることを示す[ブール](bool.md)値 `true` を返します。

  - [false リテラル](false-literal.md) [ブール](bool.md)値 `false` を表します。

- [stackalloc](stackalloc.md) スタックにメモリ ブロックを割り当てます。  

演算子およびステートメントとして使用できる次のキーワードは、「[ステートメント](statement-keywords.md)」セクションで説明します。

- [checked](checked.md) checked コンテキストを指定します。  

- [unchecked](unchecked.md) unchecked コンテキストを指定します。
