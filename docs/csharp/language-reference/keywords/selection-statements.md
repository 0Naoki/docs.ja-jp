# <a name="selection-statements-c-reference"></a>選択ステートメント (C# リファレンス)

選択ステートメントは、特定の条件が `true` かどうかに基づいて、プログラムの制御を特定のフローに移します。

選択ステートメントでは次のキーワードが使われます。

- [if](if-else.md)

- [else](if-else.md)

- [switch](switch.md)

- [case](switch.md)

- [default](switch.md)
