# <a name="compiler-error-cs0507"></a><span data-ttu-id="bea49-102">コンパイラ エラー CS0507</span><span class="sxs-lookup"><span data-stu-id="bea49-102">Compiler Error CS0507</span></span>
<span data-ttu-id="bea49-103">'function1': 'access' の継承メンバー 'function2' をオーバーライドするときに、アクセス修飾子を変更できません</span><span class="sxs-lookup"><span data-stu-id="bea49-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>  
  
 <span data-ttu-id="bea49-104">メソッド オーバーライドで、アクセス指定を変更しようとしています。</span><span class="sxs-lookup"><span data-stu-id="bea49-104">An attempt was made to change the access specification in a method override.</span></span>  
  
## <a name="example"></a><span data-ttu-id="bea49-105">例</span><span class="sxs-lookup"><span data-stu-id="bea49-105">Example</span></span>  
 <span data-ttu-id="bea49-106">次の例では CS0507 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="bea49-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="bea49-107">例</span><span class="sxs-lookup"><span data-stu-id="bea49-107">Example</span></span>  
 <span data-ttu-id="bea49-108">CS0507 は、参照されているメタデータで定義されている `protected internal` とマークされたメソッドを、クラスがオーバーライドしようとした場合にも発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="bea49-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="bea49-109">この場合、オーバーライドするメソッドは `protected` とマークする必要があります。</span><span class="sxs-lookup"><span data-stu-id="bea49-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="bea49-110">例</span><span class="sxs-lookup"><span data-stu-id="bea49-110">Example</span></span>  
 <span data-ttu-id="bea49-111">次の例では CS0507 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="bea49-111">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507_c.cs

public class cly : clx  
{  
   protected internal override void f() {}   // CS0507  
   // try the following line instead

}  
```
