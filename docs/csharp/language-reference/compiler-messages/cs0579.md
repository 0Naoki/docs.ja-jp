# <a name="compiler-error-cs0579"></a><span data-ttu-id="1a0be-102">コンパイラ エラー CS0579</span><span class="sxs-lookup"><span data-stu-id="1a0be-102">Compiler Error CS0579</span></span>
<span data-ttu-id="1a0be-103">'<属性>' 属性が重複しています</span><span class="sxs-lookup"><span data-stu-id="1a0be-103">Duplicate 'attribute' attribute</span></span>  
  
 <span data-ttu-id="1a0be-104">[AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md) で **AllowMultiple=true** に指定されていない場合は、同じ属性を複数回指定できません。</span><span class="sxs-lookup"><span data-stu-id="1a0be-104">It is not possible to specify the same attribute more than once unless the attribute specifies **AllowMultiple=true** in its [AttributeUsage](../../programming-guide/concepts/attributes/attributeusage.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1a0be-105">例</span><span class="sxs-lookup"><span data-stu-id="1a0be-105">Example</span></span>  
 <span data-ttu-id="1a0be-106">次の例では CS0579 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1a0be-106">The following example generates CS0579.</span></span>  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
