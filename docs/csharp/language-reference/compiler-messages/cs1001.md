# <a name="compiler-error-cs1001"></a><span data-ttu-id="63250-102">コンパイラ エラー CS1001</span><span class="sxs-lookup"><span data-stu-id="63250-102">Compiler Error CS1001</span></span>
<span data-ttu-id="63250-103">識別子が必要です。</span><span class="sxs-lookup"><span data-stu-id="63250-103">Identifier expected</span></span>  
  
 <span data-ttu-id="63250-104">識別子が指定されていません。</span><span class="sxs-lookup"><span data-stu-id="63250-104">You did not supply an identifier.</span></span> <span data-ttu-id="63250-105">識別子は、ユーザーが指定するクラス、構造体、名前空間、メソッド、変数などの名前です。</span><span class="sxs-lookup"><span data-stu-id="63250-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="63250-106">次の例では、単純なクラスを宣言していますが、クラスの名前が指定されていません。</span><span class="sxs-lookup"><span data-stu-id="63250-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="63250-107">次の例では、列挙型を宣言するときにメンバーを指定する必要があるため、CS1001 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="63250-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="63250-108">パラメーター名は、インターフェイス定義など、コンパイラによって使用されない場合でも指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="63250-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="63250-109">これらのパラメーターは、そのインターフェイスを利用するプログラマーが、パラメーターの意味を理解するのに必要です。</span><span class="sxs-lookup"><span data-stu-id="63250-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
