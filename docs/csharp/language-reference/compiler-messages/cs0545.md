# <a name="compiler-error-cs0545"></a><span data-ttu-id="7e316-102">コンパイラ エラー CS0545</span><span class="sxs-lookup"><span data-stu-id="7e316-102">Compiler Error CS0545</span></span>

 <span data-ttu-id="7e316-104">基底クラスにこのようなオーバーライドする定義がない場合に、プロパティ アクセサーのオーバーライドを定義する処理が試行されました。</span><span class="sxs-lookup"><span data-stu-id="7e316-104">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="7e316-105">このエラーは、次の方法で解決できます。</span><span class="sxs-lookup"><span data-stu-id="7e316-105">You can resolve this error by:</span></span>

## <a name="example"></a><span data-ttu-id="7e316-111">例</span><span class="sxs-lookup"><span data-stu-id="7e316-111">Example</span></span>  
 <span data-ttu-id="7e316-112">次の例では CS0545 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7e316-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
