## <a name="example"></a><span data-ttu-id="07ee6-106">例</span><span class="sxs-lookup"><span data-stu-id="07ee6-106">Example</span></span>  
 <span data-ttu-id="07ee6-107">次の C++ プログラムは、他の言語では使用できない可能性がある RequiredAttributeAttribute 属性を使用しています。</span><span class="sxs-lookup"><span data-stu-id="07ee6-107">The following C++ program uses an attribute, RequiredAttributeAttribute, which may not be consumed by other languages.</span></span>  
  
```cpp
// CPP0570.cpp  
// compile with: /clr /LD  
  
using namespace System;  
using namespace System::Runtime::CompilerServices;  
  
namespace CS0570_Server {  
   [RequiredAttributeAttribute(Int32::typeid)]    
   public ref struct Scenario1 {  
      int intVar;  
   };  
  
   public ref struct CS0570Class {  
      Scenario1 ^ sc1_field;  
  
      property virtual Scenario1 ^ sc1_prop {  
         Scenario1 ^ get() { return sc1_field; }  
      }  
  
      Scenario1 ^ sc1_method() { return sc1_field; }  
   };  
};  
```  
  
## <a name="example"></a><span data-ttu-id="07ee6-108">例</span><span class="sxs-lookup"><span data-stu-id="07ee6-108">Example</span></span>  
 <span data-ttu-id="07ee6-109">次の例では CS0570 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="07ee6-109">The following sample generates CS0570.</span></span>

```csharp  
   }

```
