# <a name="compiler-error-cs1946"></a><span data-ttu-id="719c6-102">コンパイラ エラー CS1946</span><span class="sxs-lookup"><span data-stu-id="719c6-102">Compiler Error CS1946</span></span>

<span data-ttu-id="719c6-103">匿名メソッド式を式のツリーに変換することはできません。</span><span class="sxs-lookup"><span data-stu-id="719c6-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

## <a name="example"></a><span data-ttu-id="719c6-108">例</span><span class="sxs-lookup"><span data-stu-id="719c6-108">Example</span></span>

<span data-ttu-id="719c6-109">次の例では CS1946 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="719c6-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
