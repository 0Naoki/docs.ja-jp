<span data-ttu-id="8d586-103">'Finalize' メソッドを導入すると、デストラクターの呼び出しに影響する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="8d586-103">Introducing a 'Finalize' method can interfere with destructor invocation.</span></span> <span data-ttu-id="8d586-104">デストラクターを宣言しようとしましたか?</span><span class="sxs-lookup"><span data-stu-id="8d586-104">Did you intend to declare a destructor?</span></span>  
  
 <span data-ttu-id="8d586-105">この警告は、シグネチャが `public virtual void Finalize` のメソッドを使用するクラスを作成すると発生します。</span><span class="sxs-lookup"><span data-stu-id="8d586-105">This warning occurs when you create a class with a method whose signature is `public virtual void Finalize`.</span></span>  
  
 <span data-ttu-id="8d586-106">このようなクラスを基底クラスとして使用し、派生クラスにデストラクターが定義されている場合、デストラクターは <xref:System.Object.Finalize%2A> ではなく基底クラスの `Finalize` メソッドをオーバーライドします。</span><span class="sxs-lookup"><span data-stu-id="8d586-106">If such a class is used as a base class and if the deriving class defines a destructor, the destructor will override the base class `Finalize` method, not <xref:System.Object.Finalize%2A>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8d586-107">例</span><span class="sxs-lookup"><span data-stu-id="8d586-107">Example</span></span>  
 <span data-ttu-id="8d586-108">次の例では CS0465 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8d586-108">The following sample generates CS0465.</span></span>  
  
```csharp  
// CS0465.cs  
// compile with: /target:library  
class A  
{  
   public virtual void Finalize() {}   // CS0465  
}  
  
// OK  
class B  
{  
   ~B() {}  
}  
```
