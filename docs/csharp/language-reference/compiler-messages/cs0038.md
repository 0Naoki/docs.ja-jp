# <a name="compiler-error-cs0038"></a><span data-ttu-id="4649f-102">コンパイラ エラー CS0038</span><span class="sxs-lookup"><span data-stu-id="4649f-102">Compiler Error CS0038</span></span>
<span data-ttu-id="4649f-103">入れ子になっている型 'type2' を経由して、外の型 'type1' の静的でないメンバーにアクセスすることはできません</span><span class="sxs-lookup"><span data-stu-id="4649f-103">Cannot access a nonstatic member of outer type 'type1' via nested type 'type2'</span></span>  
  
 <span data-ttu-id="4649f-104">クラスのフィールドは、入れ子になったクラスに自動的に使用できません。</span><span class="sxs-lookup"><span data-stu-id="4649f-104">A field in a class is not automatically available to a nested class.</span></span> <span data-ttu-id="4649f-105">入れ子になったクラスに使用できるようにするには、フィールドを[静的](../../../csharp/language-reference/keywords/static.md)にする必要があります。</span><span class="sxs-lookup"><span data-stu-id="4649f-105">To be available to a nested class, the field must be [static](../../../csharp/language-reference/keywords/static.md).</span></span> <span data-ttu-id="4649f-106">静的ではない場合、外側のクラスのインスタンスを作成する必要があります。</span><span class="sxs-lookup"><span data-stu-id="4649f-106">Otherwise, you must create an instance of the outer class.</span></span> <span data-ttu-id="4649f-107">詳細については、「[入れ子にされた型](../../../csharp/programming-guide/classes-and-structs/nested-types.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="4649f-107">For more information, see [Nested Types](../../../csharp/programming-guide/classes-and-structs/nested-types.md).</span></span>  
  
 <span data-ttu-id="4649f-108">次の例では CS0038 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4649f-108">The following sample generates CS0038:</span></span>  
  
```csharp  
// CS0038.cs  
class OuterClass  
{  
   public int count;  
   // try the following line instead  
   // public static int count;  
  
   class InnerClass  
   {  
      void func()  
      {  
         // or, create an instance  
         // OuterClass class_inst = new OuterClass();  
         // int count2 = class_inst.count;  
         int count2 = count;   // CS0038  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
