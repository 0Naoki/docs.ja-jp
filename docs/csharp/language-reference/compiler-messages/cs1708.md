# <a name="compiler-error-cs1708"></a><span data-ttu-id="6c6a8-102">コンパイラ エラー CS1708</span><span class="sxs-lookup"><span data-stu-id="6c6a8-102">Compiler Error CS1708</span></span>
<span data-ttu-id="6c6a8-103">固定バッファーには、ローカルまたはフィールドをとおしてのみアクセスできます</span><span class="sxs-lookup"><span data-stu-id="6c6a8-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="6c6a8-104">C# 2.0 の新しい機能として、`struct` 内でインライン配列を定義できることが挙げられます。</span><span class="sxs-lookup"><span data-stu-id="6c6a8-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="6c6a8-105">これらの配列は、ローカル変数またはフィールドを介してのみアクセスでき、式の左辺で中間値として参照することはできません。</span><span class="sxs-lookup"><span data-stu-id="6c6a8-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="6c6a8-106">また、`static` または `readonly` として宣言されたフィールドで、この配列にアクセスすることもできません。</span><span class="sxs-lookup"><span data-stu-id="6c6a8-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="6c6a8-107">このエラーを解決するには、配列変数を定義し、この変数にインライン配列を代入します。</span><span class="sxs-lookup"><span data-stu-id="6c6a8-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="6c6a8-108">または、インライン配列を表すフィールドから、`static` 修飾子か `readonly` 修飾子を削除します。</span><span class="sxs-lookup"><span data-stu-id="6c6a8-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6c6a8-109">例</span><span class="sxs-lookup"><span data-stu-id="6c6a8-109">Example</span></span>  
 <span data-ttu-id="6c6a8-110">次の例では CS1708 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6c6a8-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
