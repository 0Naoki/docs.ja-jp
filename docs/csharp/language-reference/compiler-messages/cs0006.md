# Compiler Error CS0006
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

'dll\_name' メタデータが見つかりませんでした。  
  
 プログラムがコンパイルされ、メタデータを格納したファイル名が明示的に渡されましたが、.dll が見つかりませんでした。  詳細については、「[\/reference \(Import Metadata\)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md)」を参照してください。
