# <a name="compiler-error-cs0006"></a>コンパイラ エラー CS0006
メタデータ ファイル 'dll_name' が見つかりませんでした。  
  
 プログラムがコンパイルされ、メタデータを含むファイルの名前が明示的に渡されましたが、.dll ファイルが見つかりませんでした。 詳しくは、「[/reference (C# コンパイラ オプション)](../../../csharp/language-reference/compiler-options/reference-compiler-option.md)」をご覧ください。
