## <a name="example"></a><span data-ttu-id="0612d-115">例</span><span class="sxs-lookup"><span data-stu-id="0612d-115">Example</span></span>

```csharp

{  
   interface I  
   {  
      void m();  
      static public void f();   // CS0106

   {

   }
```
