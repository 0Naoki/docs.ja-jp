## <a name="example"></a><span data-ttu-id="6e3ed-115">例</span><span class="sxs-lookup"><span data-stu-id="6e3ed-115">Example</span></span>
 <span data-ttu-id="6e3ed-116">次の例では CS0106 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6e3ed-116">The following sample generates CS0106.</span></span>

```csharp
// CS0106.cs
namespace MyNamespace
{
   interface I
   {
      void m();
      static public void f();   // CS0106
   }

   public class MyClass
   {
      public void I.m() {}   // CS0106
      public static void Main() {}
   }
}
```
