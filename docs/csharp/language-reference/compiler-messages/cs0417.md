# <a name="compiler-error-cs0417"></a><span data-ttu-id="827ae-102">コンパイラ エラー CS0417</span><span class="sxs-lookup"><span data-stu-id="827ae-102">Compiler Error CS0417</span></span>

<span data-ttu-id="827ae-103">'<識別子>': 変数型のインスタンスを作成するときに、引数を指定できません</span><span class="sxs-lookup"><span data-stu-id="827ae-103">'identifier': cannot provide arguments when creating an instance of a variable type</span></span>  
  
 <span data-ttu-id="827ae-104">このエラーは、型パラメーターの `new` 演算子の呼び出しで、引数を指定した場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="827ae-104">This error occurs if a call to the `new` operator on a type parameter has arguments.</span></span> <span data-ttu-id="827ae-105">未知のパラメーター型の `new` 演算子を使用して呼び出すことができる唯一のコンストラクターは、引数を持たないコンストラクターです。</span><span class="sxs-lookup"><span data-stu-id="827ae-105">The only constructor that can be called by using the `new` operator on an unknown parameter type is a constructor that has no arguments.</span></span> <span data-ttu-id="827ae-106">他のコンストラクターを呼び出す場合は、クラス型制約またはインターフェイス制約を使用してください。</span><span class="sxs-lookup"><span data-stu-id="827ae-106">If you need to call another constructor, consider using a class type constraint or interface constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="827ae-107">例</span><span class="sxs-lookup"><span data-stu-id="827ae-107">Example</span></span>  

 <span data-ttu-id="827ae-108">次の例では CS0417 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="827ae-108">The following example generates CS0417:</span></span>  

```csharp
// CS0417  
class ExampleClass<T> where T : new()  
{  
    // The following line causes CS0417.  
    T instance1 = new T(1);
  
    // The following line doesn't cause the error.  
    T instance2 = new T();  
}  
```
