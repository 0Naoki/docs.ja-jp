# <a name="compiler-error-cs1721"></a><span data-ttu-id="464dc-102">コンパイラ エラー CS1721</span><span class="sxs-lookup"><span data-stu-id="464dc-102">Compiler Error CS1721</span></span>
<span data-ttu-id="464dc-103">クラス 'class' に複数の基本クラスを指定することはできません: 'class_1' および 'class_2'</span><span class="sxs-lookup"><span data-stu-id="464dc-103">Class 'class' cannot have multiple base classes: 'class_1' and 'class_2'</span></span>  
  
 <span data-ttu-id="464dc-104">このエラー メッセージの最も一般的な原因としては、多重継承の使用があります。</span><span class="sxs-lookup"><span data-stu-id="464dc-104">The most common cause of this error message is attempting to use multiple inheritance.</span></span> <span data-ttu-id="464dc-105">C# のクラスは 1 つのクラスから直接継承することしかできません。</span><span class="sxs-lookup"><span data-stu-id="464dc-105">A class in C# may only inherit directly from one class.</span></span> <span data-ttu-id="464dc-106">ただし、1 つのクラスは任意の数のインターフェイスを実装できます。</span><span class="sxs-lookup"><span data-stu-id="464dc-106">However, a class can implement any number of interfaces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="464dc-107">例</span><span class="sxs-lookup"><span data-stu-id="464dc-107">Example</span></span>  
 <span data-ttu-id="464dc-108">次の例は、1 つの方法で CS1721 が生成され、エラーを回避する方法は 2 つを示します。</span><span class="sxs-lookup"><span data-stu-id="464dc-108">The following example shows one way in which CS1721 is generated, and then shows two possible ways to avoid the error.</span></span>  
  
```csharp  
// CS1721.cs  
public class A {}  
public class B {}  
public class MyClass : A, B {}   // CS1721  
  
// One possible fix is to use the following approach instead:  
public class A {}  
public class B : A {}  
public class C : B {}  
  
// Another possible fix is to use interfaces instead of base classes:  
public class A {}  
public interface B {}  
public class C : A, B {}  
```
