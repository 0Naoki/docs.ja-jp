<span data-ttu-id="5c4d2-103">整数型の値が必要です</span><span class="sxs-lookup"><span data-stu-id="5c4d2-103">A value of an integral type expected</span></span>

## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="5c4d2-106">あいまいな変換の例</span><span class="sxs-lookup"><span data-stu-id="5c4d2-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="5c4d2-107">このエラーは、変換がなかった場合、または使用できる暗黙的な変換結果があいまいな状況になった場合に発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="5c4d2-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="5c4d2-108">次の例では CS0151 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5c4d2-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```

## <a name="example-of-void-method"></a><span data-ttu-id="5c4d2-109">void メソッドの例</span><span class="sxs-lookup"><span data-stu-id="5c4d2-109">Example of void method</span></span>

<span data-ttu-id="5c4d2-110">[switch](../keywords/switch.md) match 式で [void](../../../csharp/language-reference/keywords/void.md) メソッドを呼び出すと、CS0151 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5c4d2-110">A [void](../../../csharp/language-reference/keywords/void.md) method invocation in a [switch](../keywords/switch.md) match expression generates CS0151.</span></span> <span data-ttu-id="5c4d2-111">このエラーを修正するには、代わりに [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md) や [long](../../../csharp/language-reference/builtin-types/integral-numeric-types.md) などの整数値を返すメソッドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="5c4d2-111">You can fix the error by calling a method that returns an integral type such as [int](../../../csharp/language-reference/builtin-types/integral-numeric-types.md) or [long](../../../csharp/language-reference/builtin-types/integral-numeric-types.md) instead.</span></span>

```csharp
class C
{
    static void Main()
    {
        switch (M()) // CS0151
        {
            default:
                break;
        }
    }

    static void M()
    {
    }
}
```
