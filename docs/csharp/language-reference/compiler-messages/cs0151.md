# <a name="compiler-error-cs0151"></a><span data-ttu-id="24ab3-102">コンパイラ エラー CS0151</span><span class="sxs-lookup"><span data-stu-id="24ab3-102">Compiler Error CS0151</span></span>
<span data-ttu-id="24ab3-103">整数型の値が必要です</span><span class="sxs-lookup"><span data-stu-id="24ab3-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="24ab3-104">整数データ型が必要な状況で変数が使用されました。</span><span class="sxs-lookup"><span data-stu-id="24ab3-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="24ab3-105">詳細については、「[Types](../../../csharp/programming-guide/types/index.md)」(型) を参照してください。</span><span class="sxs-lookup"><span data-stu-id="24ab3-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="24ab3-106">例</span><span class="sxs-lookup"><span data-stu-id="24ab3-106">Example</span></span>  
 <span data-ttu-id="24ab3-107">このエラーは、変換がなかった場合、または使用できる暗黙的な変換結果があいまいな状況になった場合に発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="24ab3-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="24ab3-108">次の例では CS0151 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="24ab3-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="24ab3-109">例</span><span class="sxs-lookup"><span data-stu-id="24ab3-109">Example</span></span>  
 <span data-ttu-id="24ab3-110">Visual Studio 2008 以降では、[void](../../../csharp/language-reference/keywords/void.md) メソッドの呼び出しで CS0151 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="24ab3-110">In Visual Studio 2008 and later, a [void](../../../csharp/language-reference/keywords/void.md) method invocation generates CS0151.</span></span> <span data-ttu-id="24ab3-111">このエラーを修正するには、[int](../../../csharp/language-reference/keywords/int.md) や [long](../../../csharp/language-reference/keywords/long.md) などの整数値を返すメソッドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="24ab3-111">You can fix the error by calling a method that returns an integral type such as [int](../../../csharp/language-reference/keywords/int.md) or [long](../../../csharp/language-reference/keywords/long.md).</span></span>  
  
```csharp  
class C  
{  
    static void Main()  
    {  
  
        switch (M()) // CS0151  
        {  
            default:  
                break;  
        }  
    }  
  
    static void M()  
    {  
    }  
}  
```
