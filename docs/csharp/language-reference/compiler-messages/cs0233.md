# <a name="compiler-error-cs0233"></a><span data-ttu-id="aa20e-102">コンパイラ エラー CS0233</span><span class="sxs-lookup"><span data-stu-id="aa20e-102">Compiler Error CS0233</span></span>
<span data-ttu-id="aa20e-103">'identifier' には定義済みのサイズが指定されていないため、sizeof は unsafe コンテキストでのみ使用できます (System.Runtime.InteropServices.Marshal.SizeOf の使用をお勧めします)</span><span class="sxs-lookup"><span data-stu-id="aa20e-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context (consider using System.Runtime.InteropServices.Marshal.SizeOf)</span></span>  
  
 <span data-ttu-id="aa20e-104">[sizeof](../../../csharp/language-reference/keywords/sizeof.md) 演算子は、コンパイル時定数である型にのみ使用できます。</span><span class="sxs-lookup"><span data-stu-id="aa20e-104">The [sizeof](../../../csharp/language-reference/keywords/sizeof.md) operator can only be used for types that are compile-time constants.</span></span> <span data-ttu-id="aa20e-105">このエラーが発生する場合は、識別子のサイズをコンパイル時に決定できることを確認してください。</span><span class="sxs-lookup"><span data-stu-id="aa20e-105">If you are getting this error, make sure that the size of the identifier can be determined at compile time.</span></span> <span data-ttu-id="aa20e-106">決定できない場合は、`sizeof` ではなく <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> を使用してください。</span><span class="sxs-lookup"><span data-stu-id="aa20e-106">If it cannot, then use <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> instead of `sizeof`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="aa20e-107">例</span><span class="sxs-lookup"><span data-stu-id="aa20e-107">Example</span></span>  
 <span data-ttu-id="aa20e-108">次の例では CS0233 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="aa20e-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
