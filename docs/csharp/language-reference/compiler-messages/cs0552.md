 <span data-ttu-id="d034c-106">次の例では CS0552 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d034c-106">The following sample generates CS0552:</span></span>  
  
```csharp  
// CS0552.cs

{  
}

{  
   // delete the routine to resolve CS0552  
   public static implicit operator ii(a aa) // CS0552  
   {  
      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
