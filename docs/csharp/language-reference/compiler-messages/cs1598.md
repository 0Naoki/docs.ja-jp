# Compiler Warning (level 1) CS1598
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

XML パーサーを以下の理由で読み込めませんでした: '理由'。XML ドキュメント ファイル 'file' は生成されません。  
  
 [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) オプションが指定されましたが、コンパイラでは msxml3.dll を見つけて読み込むことができませんでした。  msxml3.dll ファイルがインストールおよび登録されているかどうかを確認してください。
