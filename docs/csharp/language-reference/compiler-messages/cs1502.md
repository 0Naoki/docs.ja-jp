# <a name="compiler-error-cs1502"></a><span data-ttu-id="15f7d-102">コンパイラ エラー CS1502</span><span class="sxs-lookup"><span data-stu-id="15f7d-102">Compiler Error CS1502</span></span>

 <span data-ttu-id="15f7d-104">このエラーは、メソッドに渡された引数の型が、そのメソッドのパラメーターの型と一致しない場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="15f7d-104">This error occurs when the argument types being passed to the method do not match the parameter types of that method.</span></span> <span data-ttu-id="15f7d-105">呼び出されたメソッドがオーバーロードされている場合は、オーバーロードされたバージョンのどのシグネチャも、渡された引数の型と一致しません。</span><span class="sxs-lookup"><span data-stu-id="15f7d-105">If the called method is overloaded, then none of the overloaded versions has a signature that matches the argument types being passed.</span></span>  
  
 <span data-ttu-id="15f7d-106">この問題を解決するには、次のいずれかのようにします。</span><span class="sxs-lookup"><span data-stu-id="15f7d-106">To resolve this problem, do one of the following:</span></span>

-   <span data-ttu-id="15f7d-111">必要に応じて、送信されるパラメーター型と一致するように、メソッドの別のオーバーロード バージョンを定義します。</span><span class="sxs-lookup"><span data-stu-id="15f7d-111">If appropriate, define another overloaded version of the method to match the parameter types that are being sent.</span></span>
