# Compiler Error CS0001
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

内部コンパイル エラー  
  
 エラーの発生原因が、コンパイラで予期しない構文を解析できなかったためかどうかを調べてください。  このエラーが繰り返し発生する場合は、Microsoft にご連絡ください。
