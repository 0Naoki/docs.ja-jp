# <a name="compiler-error-cs1943"></a><span data-ttu-id="0a118-102">コンパイラ エラー CS1943</span><span class="sxs-lookup"><span data-stu-id="0a118-102">Compiler Error CS1943</span></span>
<span data-ttu-id="0a118-103">ソース型 'type' のクエリ式では後ろに続く from 句で型 'type' が許可されていません。</span><span class="sxs-lookup"><span data-stu-id="0a118-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="0a118-104">'method' の呼び出しで型を推論できませんでした。</span><span class="sxs-lookup"><span data-stu-id="0a118-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="0a118-105">すべての範囲変数が、クエリ可能型である必要があります。</span><span class="sxs-lookup"><span data-stu-id="0a118-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0a118-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="0a118-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="0a118-107">型が、`IEnumerable`、`IEnumerable<T>` か派生インターフェイス、またはクエリ パターンが定義された他の型を実装する、クエリ可能型であることを確認します。</span><span class="sxs-lookup"><span data-stu-id="0a118-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2.  <span data-ttu-id="0a118-108">型が非ジェネリック `IEnumerable` である場合、範囲変数に明示的な型を指定します。</span><span class="sxs-lookup"><span data-stu-id="0a118-108">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0a118-109">例</span><span class="sxs-lookup"><span data-stu-id="0a118-109">Example</span></span>  
 <span data-ttu-id="0a118-110">次のコードでは CS1943 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0a118-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
