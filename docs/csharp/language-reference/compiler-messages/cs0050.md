## <a name="example"></a><span data-ttu-id="7f78e-106">例</span><span class="sxs-lookup"><span data-stu-id="7f78e-106">Example</span></span>  
 <span data-ttu-id="7f78e-107">次の例では、`MyClass` にアクセス修飾子が指定されていないため、アクセシビリティは既定の `private` なので、CS0050 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7f78e-107">The following sample generates CS0050 because no accessiblity modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>

```csharp  
{  
}

{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
