 <span data-ttu-id="0a648-105">ソース型をクエリするには、クエリで呼び出している標準のクエリ演算子メソッドをその型が実装する必要があります。</span><span class="sxs-lookup"><span data-stu-id="0a648-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="0a648-106">実装は、適切な `using` ディレクティブによりスコープに取り込まれるクラス メンバーまたは拡張メソッドのどちらかの形式で行うことができます。</span><span class="sxs-lookup"><span data-stu-id="0a648-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>

## <a name="example"></a><span data-ttu-id="0a648-110">例</span><span class="sxs-lookup"><span data-stu-id="0a648-110">Example</span></span>  
 <span data-ttu-id="0a648-111">次の例では CS1936 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0a648-111">The following example produces CS1936:</span></span>  
  
```csharp  
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```  
  
 <span data-ttu-id="0a648-112">このエラーは通常、これらのオブジェクトのコレクションではなく、一部の型のオブジェクトを誤ってクエリしようとした場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="0a648-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
