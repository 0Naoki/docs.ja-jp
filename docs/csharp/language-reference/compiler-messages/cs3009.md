# <a name="compiler-warning-level-1-cs3009"></a><span data-ttu-id="eab09-102">コンパイラの警告 (レベル 1) CS3009</span><span class="sxs-lookup"><span data-stu-id="eab09-102">Compiler Warning (level 1) CS3009</span></span>
<span data-ttu-id="eab09-103">'type': 基本型 'type' は CLS に準拠していません</span><span class="sxs-lookup"><span data-stu-id="eab09-103">'type': base type 'type' is not CLS-compliant</span></span>  
  
 <span data-ttu-id="eab09-104">共通言語仕様 (CLS) 準拠としてマークされているアセンブリで、基本型に CLS 準拠不要のマークが付けられました。</span><span class="sxs-lookup"><span data-stu-id="eab09-104">A base type was marked as not having to be compliant with the Common Language Specification (CLS) in an assembly that was marked as being CLS compliant.</span></span> <span data-ttu-id="eab09-105">アセンブリが CLS 準拠であることを指定する属性を削除するか、または型が CLS 準拠でないことを示す属性を削除してください。</span><span class="sxs-lookup"><span data-stu-id="eab09-105">Either remove the attribute that specifies the assembly is CLS compliant or remove the attribute that indicates the type is not CLS compliant.</span></span> <span data-ttu-id="eab09-106">CLS 準拠について詳しくは、「[CLS 準拠コードの記述](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)」および「[言語への非依存性、および言語非依存コンポーネント](https://msdn.microsoft.com/library/12a7a7h3)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="eab09-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3) and [Language Independence and Language-Independent Components](https://msdn.microsoft.com/library/12a7a7h3).</span></span>  
  
## <a name="example"></a><span data-ttu-id="eab09-107">例</span><span class="sxs-lookup"><span data-stu-id="eab09-107">Example</span></span>  
 <span data-ttu-id="eab09-108">次の例では CS3009 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="eab09-108">The following example generates CS3009:</span></span>  
  
```csharp  
// CS3009.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
[CLSCompliant(false)]  
public class B  
{  
}  
  
public class C : B   // CS3009  
{  
    public static void Main () {}  
}  
```
