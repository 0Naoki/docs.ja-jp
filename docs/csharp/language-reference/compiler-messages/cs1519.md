# <a name="compiler-error-cs1519"></a><span data-ttu-id="f5ec7-102">コンパイラ エラー CS1519</span><span class="sxs-lookup"><span data-stu-id="f5ec7-102">Compiler Error CS1519</span></span>
<span data-ttu-id="f5ec7-103">クラス、構造体またはインターフェイスのメンバー宣言でトークン '<トークン>' が無効です</span><span class="sxs-lookup"><span data-stu-id="f5ec7-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="f5ec7-104">このエラーは、トークンが属していない場所でトークンが見つかるたびに生成されます。</span><span class="sxs-lookup"><span data-stu-id="f5ec7-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="f5ec7-105">*トークン*とは、キーワード、識別子 (クラス、構造体、メソッドなどの名前)、"Hello" などの文字列、'A' などの文字、108 などの数値リテラル値、`==` などの演算子、`;` などの区切り記号です。</span><span class="sxs-lookup"><span data-stu-id="f5ec7-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="f5ec7-106">型より前に、無効な修飾子を含む[クラス](../../../csharp/language-reference/keywords/class.md)、構造体、またはインターフェイス メンバー宣言があると、このエラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="f5ec7-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="f5ec7-107">このエラーを解決するには、無効な修飾子を削除します。</span><span class="sxs-lookup"><span data-stu-id="f5ec7-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="f5ec7-108">次の例では、トークンが無効な場所に配置されていることが原因で、5 か所で CS1519 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="f5ec7-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
