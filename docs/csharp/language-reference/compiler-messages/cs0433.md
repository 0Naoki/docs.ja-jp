# <a name="compiler-error-cs0433"></a><span data-ttu-id="e2bbb-102">コンパイラ エラー CS0433</span><span class="sxs-lookup"><span data-stu-id="e2bbb-102">Compiler Error CS0433</span></span>
<span data-ttu-id="e2bbb-103">型 'TypeName1' が 'TypeName2' と 'TypeName3' の両方に存在します</span><span class="sxs-lookup"><span data-stu-id="e2bbb-103">The type TypeName1 exists in both TypeName2 and TypeName3</span></span>  
  
 <span data-ttu-id="e2bbb-104">アプリケーション内で参照される 2 つの異なるアセンブリに同じ名前空間と型が含まれているため、あいまいさが生じています。</span><span class="sxs-lookup"><span data-stu-id="e2bbb-104">Two different assemblies referenced in your application contain the same namespace and type, which produces ambiguity.</span></span>

## <a name="example"></a><span data-ttu-id="e2bbb-106">例</span><span class="sxs-lookup"><span data-stu-id="e2bbb-106">Example</span></span>

```csharp  
   public class AggPubImpAggPubImp {}  
}  
```

```csharp  
namespace TypeBindConflicts

   public class AggPubImpAggPubImp {}
```  

## <a name="example"></a><span data-ttu-id="e2bbb-110">例</span><span class="sxs-lookup"><span data-stu-id="e2bbb-110">Example</span></span>  
 <span data-ttu-id="e2bbb-111">次の例では CS0433 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e2bbb-111">The following example generates CS0433.</span></span>  
  
```csharp  
// CS0433_3.cs  
// compile with: /reference:cs0433_1.dll /reference:cs0433_2.dll  
using TypeBindConflicts;  
public class Test   
{  
   public static void Main()   
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="e2bbb-112">例</span><span class="sxs-lookup"><span data-stu-id="e2bbb-112">Example</span></span>

```csharp  
// compile with: /reference:cs0433_1.dll /reference:TypeBindConflicts=cs0433_2.dll

   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();  
   }  
}  
```
