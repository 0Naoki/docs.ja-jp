# <a name="compiler-error-cs1674"></a><span data-ttu-id="178cc-102">コンパイラ エラー CS1674</span><span class="sxs-lookup"><span data-stu-id="178cc-102">Compiler Error CS1674</span></span>
<span data-ttu-id="178cc-103">'T': using ステートメントで使用される型は、暗黙的に 'System.IDisposable' への変換が可能でなければなりません</span><span class="sxs-lookup"><span data-stu-id="178cc-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="178cc-104">[using ステートメント](../../../csharp/language-reference/keywords/using-statement.md)は、`using` ブロックの最後に、オブジェクトを確実に破棄することを目的としています。そのため、このステートメントでは、破棄可能な型のみ使うことができます。</span><span class="sxs-lookup"><span data-stu-id="178cc-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="178cc-105">たとえば、値型は破棄できません。また、クラスに制約されない型パラメーターも、破棄可能な型とは見なされません。</span><span class="sxs-lookup"><span data-stu-id="178cc-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="178cc-106">例</span><span class="sxs-lookup"><span data-stu-id="178cc-106">Example</span></span>  
 <span data-ttu-id="178cc-107">次の例では CS1674 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="178cc-107">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674.cs  
class C  
{  
   public static void Main()  
   {  
      int a = 0;  
      a++;  
  
      using (a) {}   // CS1674  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="178cc-108">例</span><span class="sxs-lookup"><span data-stu-id="178cc-108">Example</span></span>  
 <span data-ttu-id="178cc-109">次の例では CS1674 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="178cc-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
