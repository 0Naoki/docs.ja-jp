# <a name="compiler-error-cs0266"></a><span data-ttu-id="eac38-102">コンパイラ エラー CS0266</span><span class="sxs-lookup"><span data-stu-id="eac38-102">Compiler Error CS0266</span></span>
<span data-ttu-id="eac38-103">型 'type1' を型 'type2' に暗黙的に変換できません。</span><span class="sxs-lookup"><span data-stu-id="eac38-103">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="eac38-104">明示的な変換が存在します (型変換を忘れていませんか)。</span><span class="sxs-lookup"><span data-stu-id="eac38-104">An explicit conversion exists (are you missing a cast?)</span></span>  
  
 <span data-ttu-id="eac38-105">このエラーは、暗黙的に変換できない 2 つの型間の変換をコードが試行したときに (ただし、明示的に変換できるときに) 発生します。</span><span class="sxs-lookup"><span data-stu-id="eac38-105">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="eac38-106">詳細については、「[キャストと型変換](../../../csharp/programming-guide/types/casting-and-type-conversions.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="eac38-106">For more information, see [Casting and Type Conversions](../../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="eac38-107">次のコードは、CS0266 を生成する列を示しています。</span><span class="sxs-lookup"><span data-stu-id="eac38-107">The following code shows examples that generate CS0266.</span></span>  
  
```csharp  
// CS0266.cs  
class MyClass  
{  
    public static void Main()  
    {  
        // You cannot implicitly convert a double to an integer.  
        double d = 3.2;  
  
        // The following line causes compiler error CS0266.  
        int i1 = d;  
  
        // However, you can resolve the error by using an explicit conversion.  
        int i2 = (int)d;  
  
        // You cannot implicitly convert an object to a class type.  
        object obj = "MyString";  
  
        // The following assignment statement causes error CS0266.  
        MyClass myClass = obj;    
  
        // You can resolve the error by using an explicit conversion.  
        MyClass c = ( MyClass )obj;  
  
        // You cannot implicitly convert a base class object to a derived class type.  
        MyClass mc = new MyClass();  
        DerivedClass dc = new DerivedClass();  
  
        // The following line causes compiler error CS0266.  
        dc = mc;  
  
        // You can resolve the error by using an explicit conversion.  
        dc = (DerivedClass)mc;  
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
