## <a name="example"></a><span data-ttu-id="ef98b-108">例</span><span class="sxs-lookup"><span data-stu-id="ef98b-108">Example</span></span>

 <span data-ttu-id="ef98b-109">次の例では CS0122 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ef98b-109">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
