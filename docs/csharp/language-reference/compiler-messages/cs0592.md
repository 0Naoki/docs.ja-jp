# <a name="compiler-error-cs0592"></a><span data-ttu-id="f91b0-102">コンパイラ エラー CS0592</span><span class="sxs-lookup"><span data-stu-id="f91b0-102">Compiler Error CS0592</span></span>
<span data-ttu-id="f91b0-103">属性 'attribute' はこの宣言型では無効です。</span><span class="sxs-lookup"><span data-stu-id="f91b0-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="f91b0-104">'type' 宣言でのみ有効です。</span><span class="sxs-lookup"><span data-stu-id="f91b0-104">It is valid on 'type' declarations only.</span></span>  
  
 <span data-ttu-id="f91b0-105">属性を定義する場合、`AttributeTargets` 値を指定してコンストラクトに適用できるものを定義します。</span><span class="sxs-lookup"><span data-stu-id="f91b0-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="f91b0-106">次の例では、`AttributeUsage` 属性に `AttributeTargets.Iterface` が指定されているため、`MyAttribute` 属性はインターフェイスにのみ適用できます。</span><span class="sxs-lookup"><span data-stu-id="f91b0-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Iterface`.</span></span> <span data-ttu-id="f91b0-107">属性はクラス (クラス `A`) に適用されているため、エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="f91b0-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="f91b0-108">例</span><span class="sxs-lookup"><span data-stu-id="f91b0-108">Example</span></span>  
 <span data-ttu-id="f91b0-109">次の例では CS0592 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f91b0-109">The following sample generates CS0592:</span></span>  
  
```csharp  
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute   
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.   
public class A    
{  
    public static void Main()  
    {  
    }  
}  
```
