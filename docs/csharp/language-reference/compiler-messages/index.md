# C# Compiler Errors
一部の C\# コンパイラ エラーには、エラーが生成された理由について説明するトピックが用意されています。また、エラーの解決方法が解説されている場合もあります。  特定のエラー メッセージのヘルプが存在するかどうかを調べるには、次のいずれかの手順に従います。  
  
-   [\[出力\] ウィンドウ](/visual-studio/ide/reference/output-window)でエラー番号 \(たとえば、CS0029\) し、MSDN の検索します。  
  
-   [\[出力\] ウィンドウ](/visual-studio/ide/reference/output-window)でエラー番号 \(たとえば、CS0029\) をクリックし、F1 キーを押します。  
  
-   インデックスでは、**\[文字の指定\]** ボックスにエラー番号を入力します。  
  
 これらの手順では、エラーに関する情報の原因がない場合、この最後のページに移動し、エラーの数またはテキストを含むフィードバックを送信します。  
  
 C\# でエラーまたは警告のオプションを構成する方法については、「[\[ビルド\] ページ \(プロジェクト デザイナー\) \(C\#\)](/visual-studio/ide/reference/build-page-project-designer-csharp)」を参照してください。
