# C# Compiler Errors
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

一部の C\# コンパイラ エラーには、エラーが生成された理由について説明するトピックが用意されています。また、エラーの解決方法が解説されている場合もあります。  特定のエラー メッセージのヘルプが存在するかどうかを調べるには、次のいずれかの手順に従います。  
  
-   [\[出力\] ウィンドウ](../Topic/Output%20Window.md)でエラー番号 \(たとえば、CS0029\) し、MSDN の検索します。  
  
-   [\[出力\] ウィンドウ](../Topic/Output%20Window.md)でエラー番号 \(たとえば、CS0029\) をクリックし、F1 キーを押します。  
  
-   インデックスでは、**\[文字の指定\]** ボックスにエラー番号を入力します。  
  
 これらの手順では、エラーに関する情報の原因がない場合、この最後のページに移動し、エラーの数またはテキストを含むフィードバックを送信します。  
  
 C\# でエラーまたは警告のオプションを構成する方法については、「[\[ビルド\] ページ \(プロジェクト デザイナー\) \(C\#\)](../Topic/Build%20Page,%20Project%20Designer%20\(C%23\).md)」を参照してください。
