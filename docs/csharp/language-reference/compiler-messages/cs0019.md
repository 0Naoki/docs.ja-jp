 <span data-ttu-id="9f072-106">クラス型でこのエラーが発生した場合は、クラスがその演算子をオーバーロードしていないためです。</span><span class="sxs-lookup"><span data-stu-id="9f072-106">If you encounter this error with a class type, it is because the class does not overload the operator.</span></span> <span data-ttu-id="9f072-107">詳細については、[operator](../../../csharp/language-reference/keywords/operator.md) キーワードに関する記事および[オーバーロード可能な演算子](../../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md)に関する記事を参照してください。</span><span class="sxs-lookup"><span data-stu-id="9f072-107">For more information, see the [operator](../../../csharp/language-reference/keywords/operator.md) keyword article and [Overloadable Operators](../../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="9f072-108">例</span><span class="sxs-lookup"><span data-stu-id="9f072-108">Example</span></span>

 <span data-ttu-id="9f072-109">C# の [bool](../../../csharp/language-reference/keywords/bool.md) は [int](../../../csharp/language-reference/keywords/int.md) に変更できないため、次の例の CS0019 は 2 か所で生成されています。また、減算演算子が文字列に適用された場合にも CS0019 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="9f072-109">In the following example, CS0019 is generated in two places because [bool](../../../csharp/language-reference/keywords/bool.md) in C# is not convertible to [int](../../../csharp/language-reference/keywords/int.md). CS0019 also is generated when the subtraction operator is applied to a string.</span></span> <span data-ttu-id="9f072-110">加算演算子 (+) は文字列オペランドにも使用できます。これは、文字列の連結を実行する `String` クラスによってオーバーロードされるためです。</span><span class="sxs-lookup"><span data-stu-id="9f072-110">The addition operator (+) can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>  

```csharp
static void Main()  
{  
    bool result = true;  
    if (result > 0) //CS0019  
    {  
        // Do something.  
    }  
  
    int i = 1;  
    // You cannot compare an integer and a boolean value.  
    if (i == true) //CS0019  
    {  
        //Do something...  
    }  
  
    // The following use of == causes no error. It is the comparison of
    // an integer and a boolean value that causes the error in the
    // previous if statement.  
    if (result == true)  
    {  
        //Do something...  
    }  
  
    string s = "Just try to subtract me.";  
    float f = 100 - s; // CS0019  
}  
```

 <span data-ttu-id="9f072-114">次の例では CS0019 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="9f072-114">The following sample generates CS0019.</span></span>  

```csharp

{

```
