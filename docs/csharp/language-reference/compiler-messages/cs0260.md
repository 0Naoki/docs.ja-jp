# <a name="compiler-error-cs0260"></a><span data-ttu-id="55658-102">コンパイラ エラー CS0260</span><span class="sxs-lookup"><span data-stu-id="55658-102">Compiler Error CS0260</span></span>
<span data-ttu-id="55658-103">partial 修飾子が型 'type' にありません。この型の別の部分宣言が存在します</span><span class="sxs-lookup"><span data-stu-id="55658-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="55658-104">このエラーは、同じ名前のクラスを複数回宣言していることを示します。</span><span class="sxs-lookup"><span data-stu-id="55658-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="55658-105">また、全部ではなく、少なくとも 1 つの宣言に `partial` 修飾子が含まれています。</span><span class="sxs-lookup"><span data-stu-id="55658-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="55658-106">いくつかの部分にクラスを定義する場合、キーワード `partial` を利用して各部分を宣言する必要があります。</span><span class="sxs-lookup"><span data-stu-id="55658-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="55658-107">このエラーは、クラスを宣言したとき、同じ名前空間のどこかで宣言されている部分クラスと同じ名前を間違って与えてしまった場合にも発生します。</span><span class="sxs-lookup"><span data-stu-id="55658-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="55658-108">次の例では CS0260 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="55658-108">The following sample generates CS0260:</span></span>  
  
```csharp  
// CS0260.cs  
// You must mark both parts of the definition of class C   
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C    
{  
}  
  
partial class C  
{  
}  
```
