# <a name="endregion-c-reference"></a>#endregion (C# リファレンス)
`#endregion` は [#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) ブロックの終了を示します。 例:  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
