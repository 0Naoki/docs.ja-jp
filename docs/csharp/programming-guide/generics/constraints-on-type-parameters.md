 <span data-ttu-id="a7364-142">パブリック クラス `SampleClass<T>{}` の T など、制約がない型パラメーターは、非バインド型パラメーターと呼ばれます。</span><span class="sxs-lookup"><span data-stu-id="a7364-142">Type parameters that have no constraints, such as T in public class `SampleClass<T>{}`, are called unbounded type parameters.</span></span> <span data-ttu-id="a7364-143">非バインド型パラメーターには次の規則があります。</span><span class="sxs-lookup"><span data-stu-id="a7364-143">Unbounded type parameters have the following rules:</span></span>
