# &lt;code&gt; (C# プログラミング ガイド)
## 構文  
  
```  
<code>content</code>  
```  
  
#### パラメーター  
 `content`  
 コードとして指定するテキスト。  
  
## 解説  
 \<code\> タグは、複数行をコードとして指定する場合に使用します。  説明内のテキストをコードとして指定する場合は、[\<c\>](../../../csharp/programming-guide/xmldoc/code-inline.md) タグを使用します。  
  
 コンパイル時に [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) を指定してドキュメント コメントをファイルに出力します。  
  
## 使用例  
 \<code\> タグの使用例については、「[\<example\>](../../../csharp/programming-guide/xmldoc/example.md)」を参照してください。
