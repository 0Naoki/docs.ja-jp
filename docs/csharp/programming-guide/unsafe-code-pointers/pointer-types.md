type* identifier;

<span data-ttu-id="b4218-105">ポインター型の `*` の前に指定された型は、**参照型**と呼ばれます。</span><span class="sxs-lookup"><span data-stu-id="b4218-105">The type specified before the `*` in a pointer type is called the **referrent type**.</span></span> <span data-ttu-id="b4218-106">次の型はいずれも参照型になります。</span><span class="sxs-lookup"><span data-stu-id="b4218-106">Any of the following types may be a referrent type:</span></span>

- <span data-ttu-id="b4218-107">任意の整数型: [sbyte](../../language-reference/keywords/sbyte.md)、[byte](../../language-reference/keywords/byte.md)、[short](../../language-reference/keywords/short.md)、[ushort](../../language-reference/keywords/ushort.md)、[int](../../language-reference/keywords/int.md)、[uint](../../language-reference/keywords/uint.md)、[long](../../language-reference/keywords/long.md)、[ulong](../../language-reference/keywords/ulong.md)。</span><span class="sxs-lookup"><span data-stu-id="b4218-107">Any integral type: [sbyte](../../language-reference/keywords/sbyte.md), [byte](../../language-reference/keywords/byte.md), [short](../../language-reference/keywords/short.md), [ushort](../../language-reference/keywords/ushort.md), [int](../../language-reference/keywords/int.md), [uint](../../language-reference/keywords/uint.md), [long](../../language-reference/keywords/long.md), [ulong](../../language-reference/keywords/ulong.md).</span></span>
- <span data-ttu-id="b4218-108">任意の浮動小数点型: [float](../../language-reference/keywords/float.md)、[double](../../language-reference/keywords/double.md)。</span><span class="sxs-lookup"><span data-stu-id="b4218-108">Any floating point type: [float](../../language-reference/keywords/float.md), [double](../../language-reference/keywords/double.md).</span></span>
- <span data-ttu-id="b4218-109">[char](../../language-reference/keywords/char.md)。</span><span class="sxs-lookup"><span data-stu-id="b4218-109">[char](../../language-reference/keywords/char.md).</span></span>
- <span data-ttu-id="b4218-110">[bool](../../language-reference/keywords/bool.md)。</span><span class="sxs-lookup"><span data-stu-id="b4218-110">[bool](../../language-reference/keywords/bool.md).</span></span>
- <span data-ttu-id="b4218-111">[decimal](../../language-reference/keywords/decimal.md)。</span><span class="sxs-lookup"><span data-stu-id="b4218-111">[decimal](../../language-reference/keywords/decimal.md).</span></span>
- <span data-ttu-id="b4218-112">任意の[列挙](../../language-reference/keywords/enum.md)型。</span><span class="sxs-lookup"><span data-stu-id="b4218-112">Any [enum](../../language-reference/keywords/enum.md) type.</span></span>
- <span data-ttu-id="b4218-113">任意のポインター型。</span><span class="sxs-lookup"><span data-stu-id="b4218-113">Any pointer type.</span></span> <span data-ttu-id="b4218-114">`void**` などの式を使用できます。</span><span class="sxs-lookup"><span data-stu-id="b4218-114">This allows expressions such as `void**`.</span></span>
- <span data-ttu-id="b4218-115">アンマネージ型のフィールドのみを含むユーザー定義の struct 型。</span><span class="sxs-lookup"><span data-stu-id="b4218-115">Any user-defined struct type that contains fields of unmanaged types only.</span></span>

<span data-ttu-id="b4218-116">ポインター型は [object](../../language-reference/keywords/object.md) を継承せず、ポインター型と `object` の間で変換を行う方法はありません。</span><span class="sxs-lookup"><span data-stu-id="b4218-116">Pointer types do not inherit from [object](../../language-reference/keywords/object.md) and no conversions exist between pointer types and `object`.</span></span> <span data-ttu-id="b4218-117">また、ボックス化とボックス化解除もポインターをサポートしません。</span><span class="sxs-lookup"><span data-stu-id="b4218-117">Also, boxing and unboxing do not support pointers.</span></span> <span data-ttu-id="b4218-118">ただし、異なるポインター型の間で変換したり、ポインター型と整数型の間で変換したりすることはできます。</span><span class="sxs-lookup"><span data-stu-id="b4218-118">However, you can convert between different pointer types and between pointer types and integral types.</span></span>

<span data-ttu-id="b4218-119">同じ 1 つの宣言で複数のポインターを宣言する場合、アスタリスク (\*) は基底の型だけに記述します。各ポインター名のプレフィックスとしては使用しません。</span><span class="sxs-lookup"><span data-stu-id="b4218-119">When you declare multiple pointers in the same declaration, the asterisk (\*) is written together with the underlying type only; it is not used as a prefix to each pointer name.</span></span> <span data-ttu-id="b4218-120">例:</span><span class="sxs-lookup"><span data-stu-id="b4218-120">For example:</span></span>
