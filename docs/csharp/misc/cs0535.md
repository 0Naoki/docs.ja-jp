# <a name="compiler-error-cs0535"></a><span data-ttu-id="cc300-102">コンパイラ エラー CS0535</span><span class="sxs-lookup"><span data-stu-id="cc300-102">Compiler Error CS0535</span></span>
<span data-ttu-id="cc300-103">'class' はインターフェイス メンバー 'member' を実装しません</span><span class="sxs-lookup"><span data-stu-id="cc300-103">'class' does not implement interface member 'member'</span></span>  
  
 <span data-ttu-id="cc300-104">[クラス](../../csharp/language-reference/keywords/class.md) は [インターフェイス](../../csharp/language-reference/keywords/interface.md)から派生しましたが、クラスは 1 つ以上のインターフェイスのメンバーを実装しませんでした。</span><span class="sxs-lookup"><span data-stu-id="cc300-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="cc300-105">クラスは、派生元のインターフェイスのすべてのメンバーを実装する必要があります。そうでない場合は `abstract`として宣言される必要があります。</span><span class="sxs-lookup"><span data-stu-id="cc300-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cc300-106">例</span><span class="sxs-lookup"><span data-stu-id="cc300-106">Example</span></span>  
 <span data-ttu-id="cc300-107">次の例では CS0535 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cc300-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
