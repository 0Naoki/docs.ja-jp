 <span data-ttu-id="8df9a-105">この警告は、ソース ファイル (file_2) 内の名前空間が、file_1 のインポートされた型と競合している場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="8df9a-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="8df9a-106">この場合、コンパイラは、ソース ファイル内のものを使用します。</span><span class="sxs-lookup"><span data-stu-id="8df9a-106">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="8df9a-107">次の例では、CS0435 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8df9a-107">The following example generates CS0435:</span></span>

   public class A { }  
}  
```  
  
 <span data-ttu-id="8df9a-109">続けて、次のファイルをコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="8df9a-109">Then, compile this file:</span></span>

{  
   public class A { }  
}

{  
   public static void Main()   
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
