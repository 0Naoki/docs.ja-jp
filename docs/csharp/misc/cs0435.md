 <span data-ttu-id="847cd-105">この警告は、ソース ファイル (file_2) 内の名前空間が、file_1 のインポートされた型と競合している場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="847cd-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="847cd-106">この場合、コンパイラは、ソース ファイル内のものを使用します。</span><span class="sxs-lookup"><span data-stu-id="847cd-106">The compiler uses the one in the source file.</span></span>

```csharp

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="847cd-109">続けて、次のファイルをコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="847cd-109">Then, compile this file:</span></span>  
  
```csharp

namespace Util   
{  
   public class A { }  
}  
  
public class Test   
{  
   public static void Main()   
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
