<span data-ttu-id="85804-103">'construct' に関する XML コメントの cref 属性 'name' の構文が正しくありません</span><span class="sxs-lookup"><span data-stu-id="85804-103">XML comment on 'construct' has syntactically incorrect cref attribute 'name'</span></span>  
  
 <span data-ttu-id="85804-104">Cref タグに渡される文字列内など、\<例外 > タグには、現在のビルド環境で使用できないメンバーを参照します。</span><span class="sxs-lookup"><span data-stu-id="85804-104">A string passed to a cref tag, for example, within an \<exception> tag, referred to a member that is not available within the current build environment.</span></span> <span data-ttu-id="85804-105">cref タグに渡す文字列は、メンバーまたはフィールドの構文的に正しい名前である必要があります。</span><span class="sxs-lookup"><span data-stu-id="85804-105">The string that you pass to a cref tag must be the syntactically correct name of a member or field.</span></span>  
  
 <span data-ttu-id="85804-106">詳細については、「 [ドキュメント コメント用の推奨タグ](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="85804-106">For more information, see [Recommended Tags for Documentation Comments](../../csharp/programming-guide/xmldoc/recommended-tags-for-documentation-comments.md).</span></span>  
  
 <span data-ttu-id="85804-107">次の例では CS1574 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="85804-107">The following sample generates CS1574:</span></span>  
  
```csharp  
// CS1574.cs

using System;  
  
/// <exception cref="System.Console.WriteLin">An exception class.</exception>   // CS1574  
// instead, uncomment and try the following line  
// /// <exception cref="System.Console.WriteLine">An exception class.</exception>

{  
}  
  
class TestClass  
{  
   public static void Main()  
   {  
      try  
      {  
      }  
      catch(EClass)  
      {  
      }  
   }  
}  
```
