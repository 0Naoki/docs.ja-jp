# <a name="compiler-error-cs0529"></a><span data-ttu-id="b063b-102">コンパイラ エラー CS0529</span><span class="sxs-lookup"><span data-stu-id="b063b-102">Compiler Error CS0529</span></span>
<span data-ttu-id="b063b-103">継承インターフェイス 'interface1' により、'interface2' のインターフェイス階層内で循環参照が発生します</span><span class="sxs-lookup"><span data-stu-id="b063b-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="b063b-104">[interface](../../csharp/language-reference/keywords/interface.md) の継承リストに、自分自身への直接または間接参照が含まれています。</span><span class="sxs-lookup"><span data-stu-id="b063b-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="b063b-105">インターフェイスは、それ自体からは継承できません。</span><span class="sxs-lookup"><span data-stu-id="b063b-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="b063b-106">次の例では CS0529 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b063b-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
