## <a name="example"></a><span data-ttu-id="7a7b3-106">例</span><span class="sxs-lookup"><span data-stu-id="7a7b3-106">Example</span></span>  
 <span data-ttu-id="7a7b3-107">次の例では CS0551 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7a7b3-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
