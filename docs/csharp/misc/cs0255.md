# <a name="compiler-error-cs0255"></a><span data-ttu-id="81219-102">コンパイラ エラー CS0255</span><span class="sxs-lookup"><span data-stu-id="81219-102">Compiler Error CS0255</span></span>
<span data-ttu-id="81219-103">stackalloc は catch または finally ブロックで使用されない可能性があります。</span><span class="sxs-lookup"><span data-stu-id="81219-103">stackalloc may not be used in a catch or finally block</span></span>  
  
 <span data-ttu-id="81219-104">[Stackalloc](../../csharp/language-reference/keywords/stackalloc.md)でキーワードを使用しない場合があります、[キャッチ](../../csharp/language-reference/keywords/try-catch.md)または[最後に](../../csharp/language-reference/keywords/try-catch-finally.md)ブロックします。</span><span class="sxs-lookup"><span data-stu-id="81219-104">The [stackalloc](../../csharp/language-reference/keywords/stackalloc.md) keyword may not be used in a [catch](../../csharp/language-reference/keywords/try-catch.md) or [finally](../../csharp/language-reference/keywords/try-catch-finally.md) block.</span></span> <span data-ttu-id="81219-105">詳細については、次を参照してください。[例外と例外処理](../../csharp/programming-guide/exceptions/index.md)します。</span><span class="sxs-lookup"><span data-stu-id="81219-105">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="81219-106">次の例では CS0255 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="81219-106">The following sample generates CS0255:</span></span>  
  
```  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
