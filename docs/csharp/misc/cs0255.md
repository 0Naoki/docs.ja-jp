# <a name="compiler-error-cs0255"></a><span data-ttu-id="428b7-102">コンパイラ エラー CS0255</span><span class="sxs-lookup"><span data-stu-id="428b7-102">Compiler Error CS0255</span></span>
<span data-ttu-id="428b7-103">stackalloc は catch または finally ブロックで使用されない可能性があります。</span><span class="sxs-lookup"><span data-stu-id="428b7-103">stackalloc may not be used in a catch or finally block</span></span>  
  
<span data-ttu-id="428b7-104">使用することはできません、 [stackalloc 演算子](../language-reference/operators/stackalloc.md)で、[catch](../language-reference/keywords/try-catch.md)または[finally](../language-reference/keywords/try-catch-finally.md)ブロックします。</span><span class="sxs-lookup"><span data-stu-id="428b7-104">You cannot use the [stackalloc operator](../language-reference/operators/stackalloc.md) in a [catch](../language-reference/keywords/try-catch.md) or [finally](../language-reference/keywords/try-catch-finally.md) block.</span></span> <span data-ttu-id="428b7-105">詳細については、[例外と例外処理](../programming-guide/exceptions/index.md)を参照してください。</span><span class="sxs-lookup"><span data-stu-id="428b7-105">For more information, see [Exceptions and Exception Handling](../programming-guide/exceptions/index.md).</span></span>  
  
<span data-ttu-id="428b7-106">次の例では CS0255 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="428b7-106">The following sample generates CS0255:</span></span>  
  
```csharp  
// CS0255.cs  
// compile with: /unsafe  
using System;  
  
public class TestTryFinally  
{  
   public static unsafe void Test()  
   {  
      int i = 123;  
      string s = "Some string";  
      object o = s;  
  
      try  
      {  
         // Conversion is not valid; o contains a string not an int  
         i = (int) o;  
      }  
      finally  
      {  
         Console.Write("i = {0}", i);  
         int* fib = stackalloc int[100];   // CS0255  
      }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
