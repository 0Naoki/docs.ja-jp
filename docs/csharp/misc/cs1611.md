# <a name="compiler-error-cs1611"></a>コンパイラ エラー CS1611
Out または ref と同様に、params パラメーターを宣言することはできません。  
  
 キーワード[で](../../csharp/language-reference/keywords/in-parameter-modifier.md)、 [ref](../../csharp/language-reference/keywords/ref.md)または[out](../../csharp/language-reference/keywords/out-parameter-modifier.md)では使用できません、 [params](../../csharp/language-reference/keywords/params.md)キーワード。  
  
 次の例では CS1611 が生成されます。  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
