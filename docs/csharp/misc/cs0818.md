# コンパイラ エラー CS0818
暗黙的に型指定されたローカル変数を初期化しなければなりません  
  
 暗黙的に型指定されたローカル変数は、宣言と同時に初期化して、値を設定する必要があります。  
  
### このエラーを解決するには  
  
1.  変数に値を割り当てるか、明示的な型を指定します。  
  
## 使用例  
 次のコードでは CS0818 が生成されます。  
  
```  
// cs0818.cs class A { public static int Main() { var a; // CS0818 return -1; } }  
```
