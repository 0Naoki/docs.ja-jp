# <a name="compiler-error-cs1660"></a><span data-ttu-id="4d750-102">コンパイラ エラー CS1660</span><span class="sxs-lookup"><span data-stu-id="4d750-102">Compiler Error CS1660</span></span>
<span data-ttu-id="4d750-103">デリゲート型ではないため、匿名メソッド ブロックを型 'type' に変換できません</span><span class="sxs-lookup"><span data-stu-id="4d750-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="4d750-104">このエラーは、匿名メソッド ブロックをデリゲート型以外の型に代入または変換しようとした場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="4d750-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="4d750-105">次の例では CS1660 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4d750-105">The following sample generates CS1660:</span></span>  
  
```  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
