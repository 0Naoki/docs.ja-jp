 <span data-ttu-id="8abce-104">ユーザー定義の [演算子](../../csharp/language-reference/keywords/operator.md) の戻り値の型は、キーワード operator の前に記述する必要があります。</span><span class="sxs-lookup"><span data-stu-id="8abce-104">The return type for a user-defined [operator](../../csharp/language-reference/keywords/operator.md) must appear before the keyword operator.</span></span>
