## <a name="example"></a><span data-ttu-id="a0e3e-109">例</span><span class="sxs-lookup"><span data-stu-id="a0e3e-109">Example</span></span>  
 <span data-ttu-id="a0e3e-110">次のコードでは CS1947 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a0e3e-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
