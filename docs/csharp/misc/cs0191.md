 <span data-ttu-id="30223-104">[readonly](../../csharp/language-reference/keywords/readonly.md) フィールドでは、コンストラクターまたは宣言時の割り当てだけを行うことができます。</span><span class="sxs-lookup"><span data-stu-id="30223-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) field can only take an assignment in a constructor or at declaration.</span></span> <span data-ttu-id="30223-105">詳細については、「[コンストラクター](../../csharp/programming-guide/classes-and-structs/constructors.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="30223-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>

## <a name="example"></a><span data-ttu-id="30223-107">例</span><span class="sxs-lookup"><span data-stu-id="30223-107">Example</span></span>  
 <span data-ttu-id="30223-108">次の例では CS0191 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="30223-108">The following sample generates CS0191.</span></span>  
  
```csharp  
// CS0191.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  // OK to assign to readonly field in declaration  
  
    MyClass()  
    {  
        TestInt = 11; // OK to assign to readonly field in constructor  
    }  
  
    public void TestReadOnly()  
    {  
        TestInt = 19;                  // CS0191  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
