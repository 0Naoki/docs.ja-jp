# <a name="compiler-error-cs0060"></a><span data-ttu-id="3522d-102">コンパイラ エラー CS0060</span><span class="sxs-lookup"><span data-stu-id="3522d-102">Compiler Error CS0060</span></span>

 <span data-ttu-id="3522d-104">クラスのアクセシビリティは、基底クラスと継承クラスの間で一貫している必要があります。</span><span class="sxs-lookup"><span data-stu-id="3522d-104">Class accessibility should be consistent between the base class and inherited class.</span></span>

```csharp  
// public class MyClass  
{  
}

   {  
   }  
}  
```
