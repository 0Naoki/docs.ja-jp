# <a name="compiler-warning-level-3-cs0419"></a><span data-ttu-id="b53a5-102">コンパイラの警告 (レベル 3) CS0419</span><span class="sxs-lookup"><span data-stu-id="b53a5-102">Compiler Warning (level 3) CS0419</span></span>
<span data-ttu-id="b53a5-103">Cref 属性の参照があいまいです: 'Method Name1'。</span><span class="sxs-lookup"><span data-stu-id="b53a5-103">Ambiguous reference in cref attribute: 'Method Name1'.</span></span>  <span data-ttu-id="b53a5-104">'Method Name2' を仮定しますが、'Method Name3' を含む別のオーバーロードに一致した可能性もあります。</span><span class="sxs-lookup"><span data-stu-id="b53a5-104">Assuming 'Method Name2', but could have also matched other overloads including 'Method Name3'.</span></span>  
  
 <span data-ttu-id="b53a5-105">コード内の XML ドキュメント コメントで、参照を解決できませんでした。</span><span class="sxs-lookup"><span data-stu-id="b53a5-105">In an XML documentation comment in the code, a reference could not be resolved.</span></span> <span data-ttu-id="b53a5-106">このエラーは、メソッドがオーバーロードされている場合や、同じ名前に 2 つの異なる識別子が見つかった場合に発生することがあります。</span><span class="sxs-lookup"><span data-stu-id="b53a5-106">This could occur if the method is overloaded, or if two different identifiers with the same name are found.</span></span> <span data-ttu-id="b53a5-107">警告を解決するには、修飾名を使用して参照を明確にするか、特定のオーバーロードをかっこで囲みます。</span><span class="sxs-lookup"><span data-stu-id="b53a5-107">To resolve the warning, use a qualified name to disambiguate the reference, or include the specific overload in parentheses.</span></span>  
  
 <span data-ttu-id="b53a5-108">次の例では CS0419 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b53a5-108">The following sample generates CS0419.</span></span>  
  
```csharp  
// cs0419.cs  
// compile with: /doc:x.xml /W:3  
interface I  
{  
   /// text for F(void)  
   void F();  
   /// text for F(int)  
   void F(int i);  
}  
/// text for class MyClass  
public class MyClass  
{  
   /// <see cref="I.F"/>  
   public static void MyMethod(int i)  
   {  
   }  
/* Try this instead:  
   /// <see cref="I.F(int)"/>  
   public static void MyMethod(int i)  
   {  
   }  
*/  
   /// text for Main  
   public static void Main ()  
   {  
   }  
}  
```
