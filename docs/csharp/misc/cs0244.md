 <span data-ttu-id="b070c-104">キーワードの [is](../../csharp/language-reference/keywords/is.md) と [as](../../csharp/language-reference/keywords/as.md) は、ポインター型では使用できません。</span><span class="sxs-lookup"><span data-stu-id="b070c-104">The [is](../../csharp/language-reference/keywords/is.md) and [as](../../csharp/language-reference/keywords/as.md) keywords are not valid for use on pointer types.</span></span> <span data-ttu-id="b070c-105">詳細については、「[アンセーフ コードとポインター](../../csharp/programming-guide/unsafe-code-pointers/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="b070c-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="b070c-106">次の例では CS0244 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b070c-106">The following sample generates CS0244:</span></span>  
  
```  
// CS0244.cs  
// compile with: /unsafe  
  
class UnsafeTest  
{  
   unsafe static void SquarePtrParam (int* p)  
   {  
      bool b = p is object;   // CS0244 p is pointer  
   }  
  
   unsafe public static void Main()  
   {  
      int i = 5;  
      SquarePtrParam (&i);  
   }  
}  
```
