 <span data-ttu-id="fd82d-104">使用する場合、このエラーが発生した、 `in`、 `ref`、または`out`匿名メソッド ブロックの内部でパラメーター。</span><span class="sxs-lookup"><span data-stu-id="fd82d-104">This error occurs if you use an `in`, `ref`, or `out` parameter inside an anonymous method block.</span></span> <span data-ttu-id="fd82d-105">このエラーを回避するには、ローカル変数を使用するか、別のコンストラクトを使用します。</span><span class="sxs-lookup"><span data-stu-id="fd82d-105">To avoid this error, use a local variable or some other construct.</span></span>

```csharp  
      // int tmp = i;  
      // MyDelegate d = delegate { return tmp; };  
  }

  {  
  
  }  
}  
```
