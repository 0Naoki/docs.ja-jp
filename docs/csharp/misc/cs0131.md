## <a name="example"></a><span data-ttu-id="707d4-108">例</span><span class="sxs-lookup"><span data-stu-id="707d4-108">Example</span></span>  
 <span data-ttu-id="707d4-109">次の例では CS0131 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="707d4-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```
