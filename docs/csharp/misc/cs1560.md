# <a name="compiler-error-cs1560"></a>コンパイラ エラー CS1560
プリプロセッサ ディレクティブに対して無効なファイル名が指定されました。 ファイル名が長すぎるか、有効なファイル名ではありません  
  
 指定されたファイル名[#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) _MAX_PATH (256 文字) またはを行を超えて`#line`超過 2000 文字が見つかりました。  
  
## <a name="example"></a>例  
 次の例では CS1560 が生成されます。  
  
```csharp  
// cs1560.cs   
using System;   
class MyClass   
{     
   public static void Main()      
   {        
      Console.WriteLine("Normal line #1.");     
      #line 21 "MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890MyFile1234567890.txt"   // CS1560  
    }  
}  
```
