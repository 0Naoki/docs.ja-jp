# <a name="compiler-error-cs0756"></a><span data-ttu-id="952af-102">コンパイラ エラー CS0756</span><span class="sxs-lookup"><span data-stu-id="952af-102">Compiler Error CS0756</span></span>
<span data-ttu-id="952af-103">部分メソッドには、複数の定義宣言を指定することはできません。</span><span class="sxs-lookup"><span data-stu-id="952af-103">A partial method may not have multiple defining declarations.</span></span>  
  
 <span data-ttu-id="952af-104">部分メソッドの定義宣言は、メソッドのシグネチャを指定する部分であり、実装 (メソッド本体) を指定する部分ではありません。</span><span class="sxs-lookup"><span data-stu-id="952af-104">The defining declaration of a partial method is the part that specifies the method signature, but not the implementation (method body).</span></span> <span data-ttu-id="952af-105">部分メソッドには、一意のシグネチャごとに 1 つの定義宣言が必要です。</span><span class="sxs-lookup"><span data-stu-id="952af-105">A partial method must have exactly one defining declaration for each unique signature.</span></span> <span data-ttu-id="952af-106">部分メソッドのオーバーロードされた各バージョンには、固有の定義宣言が必要です。</span><span class="sxs-lookup"><span data-stu-id="952af-106">Each overloaded version of a partial method must have its own defining declaration.</span></span>
