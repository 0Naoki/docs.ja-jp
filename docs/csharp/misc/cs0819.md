## <a name="example-1"></a><span data-ttu-id="e9eca-109">例 1</span><span class="sxs-lookup"><span data-stu-id="e9eca-109">Example 1</span></span>

```csharp
// cs0819.cs
class Program
{

        var a = 3, b = 2; // CS0819

        // First correction option.
        //int a = 3, b = 2;

        //var a = 3;
        //var b = 2;
```
