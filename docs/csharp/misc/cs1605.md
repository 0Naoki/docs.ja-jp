# <a name="compiler-error-cs1605"></a>コンパイラ エラー CS1605
読み取り専用なので 'var' は ref または out 引数として渡せません  
  
 変数として渡さ、 [ref](../../csharp/language-reference/keywords/ref.md)または[out](../../csharp/language-reference/keywords/out-parameter-modifier.md)パラメーターは、呼び出されたメソッドで変更する必要があります。 したがって、読み取り専用パラメーターを `ref` または `out`として渡すことはできません。
