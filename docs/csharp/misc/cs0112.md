# <a name="compiler-error-cs0112"></a><span data-ttu-id="d9b29-102">コンパイラ エラー CS0112</span><span class="sxs-lookup"><span data-stu-id="d9b29-102">Compiler Error CS0112</span></span>
<span data-ttu-id="d9b29-103">静的メンバー 'function' を override、virtual、または abstract とすることはできません。</span><span class="sxs-lookup"><span data-stu-id="d9b29-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="d9b29-104">`override`、 **virtual**、または **abstract** キーワードを使用するメソッド宣言で、 **static** キーワードを同時に使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="d9b29-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="d9b29-105">詳細については、「[メソッド](../../csharp/programming-guide/classes-and-structs/methods.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="d9b29-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="d9b29-106">次の例では CS0112 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d9b29-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
