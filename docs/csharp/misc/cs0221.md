# <a name="compiler-error-cs0221"></a>コンパイラ エラー CS0221
定数値 'value' は 'type' に変換できません (unchecked 構文を使ってオーバーライドしてください)。  
  
 データが失われる可能性のある代入演算が、既定でオンになっている [checked](../../csharp/language-reference/keywords/checked.md)によって検出されました。 このエラーを解決するには、代入を修正するか、または [unchecked](../../csharp/language-reference/keywords/unchecked.md) を使用します。 詳細については、[オンとオフ](../../csharp/language-reference/keywords/checked-and-unchecked.md)を参照してください。  
  
 次の例では CS0221 が生成されます。  
  
```csharp  
// CS0221.cs  
public class MyClass  
{  
   public static void Main()  
   {  
      // unchecked  
      // {  
         int a = (int)0xFFFFFFFF;   // CS0221  
         a++;  
      // }  
   }  
}  
```
