<span data-ttu-id="acfe4-103">エイリアスが型を参照しているため、エイリアス '識別子' を '::' と使用できません。</span><span class="sxs-lookup"><span data-stu-id="acfe4-103">Cannot use alias 'identifier' with '::' since the alias references a type.</span></span> <span data-ttu-id="acfe4-104">代わりに、'.' を使用してください。</span><span class="sxs-lookup"><span data-stu-id="acfe4-104">Use '.' instead.</span></span>  
  
 <span data-ttu-id="acfe4-105">"::" を、型を参照するエイリアスと使用しました。</span><span class="sxs-lookup"><span data-stu-id="acfe4-105">You used "::" with an alias that references a type.</span></span> <span data-ttu-id="acfe4-106">このエラーを解決するには、"." 演算子を使用します。</span><span class="sxs-lookup"><span data-stu-id="acfe4-106">To resolve this error, use the "." operator.</span></span>  
  
 <span data-ttu-id="acfe4-107">次の例では CS0431 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="acfe4-107">The following sample generates CS0431:</span></span>  
  
```  
// CS0431.cs  
using A = Outer;  
  
public class Outer   
{  
   public class Inner   
   {  
      public static void Meth() {}  
   }  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      A::Inner.Meth();   // CS0431  
      A.Inner.Meth();   // OK  
   }  
}  
```
