# <a name="compiler-error-cs0210"></a><span data-ttu-id="f190d-102">コンパイラ エラー CS0210</span><span class="sxs-lookup"><span data-stu-id="f190d-102">Compiler Error CS0210</span></span>
<span data-ttu-id="f190d-103">fixed または using ステートメントの宣言の中に、初期化子を指定してください。</span><span class="sxs-lookup"><span data-stu-id="f190d-103">You must provide an initializer in a fixed or using statement declaration</span></span>  
  
 <span data-ttu-id="f190d-104">[fixed ステートメント](../language-reference/keywords/fixed-statement.md)で変数を宣言し、初期化する必要があります。</span><span class="sxs-lookup"><span data-stu-id="f190d-104">You must declare and initialize the variable in a [fixed statement](../language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="f190d-105">詳細については、「[アンセーフ コードとポインター](../programming-guide/unsafe-code-pointers/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="f190d-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="f190d-106">次の例では CS0210 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f190d-106">The following sample generates CS0210:</span></span>  
  
```csharp  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="f190d-107">[using ステートメント](../language-reference/keywords/using-statement.md) に初期化子がないため、CS0210 も生成されます。</span><span class="sxs-lookup"><span data-stu-id="f190d-107">The following sample also generates CS0210 because the [using statement](../language-reference/keywords/using-statement.md) has no initializer.</span></span>  
  
```csharp  
// CS0210b.cs  
  
using System.IO;  
class Test   
{  
   static void Main()   
   {  
      using (StreamWriter w) // CS0210  
      // Try this line instead:  
      // using (StreamWriter w = new StreamWriter("TestFile.txt"))   
      {  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
