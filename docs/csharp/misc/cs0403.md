## <a name="example"></a><span data-ttu-id="dba1c-108">例</span><span class="sxs-lookup"><span data-stu-id="dba1c-108">Example</span></span>  
 <span data-ttu-id="dba1c-109">次の例では CS0403 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="dba1c-109">The following sample generates CS0403.</span></span>  
  
```  
// CS0403.cs  
// compile with: /target:library  
class C<T>  
{  
   public void f()  
   {  
      T t = null;  // CS0403  
      T t2 = default(T);   // OK  
    }  
}  
  
class D<T> where T : class   
{  
   public void f()  
   {  
      T t = null;  // OK  
    }  
}  
```
