# <a name="compiler-error-cs0685"></a><span data-ttu-id="290c2-102">コンパイラ エラー CS0685</span><span class="sxs-lookup"><span data-stu-id="290c2-102">Compiler Error CS0685</span></span>
<span data-ttu-id="290c2-103">条件付きメンバー 'member' には out パラメーターを指定できません</span><span class="sxs-lookup"><span data-stu-id="290c2-103">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="290c2-104">メソッドで <xref:System.Diagnostics.ConditionalAttribute> 属性を使用する場合、そのメソッドに out パラメーターがない可能性があります。</span><span class="sxs-lookup"><span data-stu-id="290c2-104">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="290c2-105">これは、メソッド呼び出しが nothing にコンパイルされる場合、out パラメーターに使用される変数の値が定義されないためです。</span><span class="sxs-lookup"><span data-stu-id="290c2-105">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="290c2-106">このエラーを避けるためには、条件付きのメソッド宣言から out パラメーターを削除するか、条件付きの属性を使用しないでください。</span><span class="sxs-lookup"><span data-stu-id="290c2-106">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="290c2-107">例</span><span class="sxs-lookup"><span data-stu-id="290c2-107">Example</span></span>  
 <span data-ttu-id="290c2-108">次の例では CS0685 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="290c2-108">The following sample generates CS0685:</span></span>  
  
```csharp

{  
    [Conditional("DEBUG")]  
    void trace(out int i)  // CS0685

        i = 1;  
    }  
}  
```
