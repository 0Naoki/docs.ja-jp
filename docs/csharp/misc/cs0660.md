<span data-ttu-id="30d39-104">ユーザー定義の等値または非等値演算子のオーバーライドがありません、コンパイラが検出された、<xref:System.Object.Equals%2A?displayProperty=nameWithType>メソッド。</span><span class="sxs-lookup"><span data-stu-id="30d39-104">The compiler detected the user-defined equality or inequality operator, but no override for the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="30d39-105">ユーザー定義の等値演算子または非等値演算子の意味をオーバーライドすることも、<xref:System.Object.Equals%2A>メソッド。</span><span class="sxs-lookup"><span data-stu-id="30d39-105">A user-defined equality or inequality operator implies that you also want to override the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="30d39-106">詳細については、「[方法 :型の値の等価性を定義する](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="30d39-106">For more information, see [How to: Define Value Equality for a Type](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span></span>

```csharp
   // }

      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
