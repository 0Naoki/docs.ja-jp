# <a name="compiler-error-cs0663"></a><span data-ttu-id="34bb5-102">コンパイラ エラー CS0663</span><span class="sxs-lookup"><span data-stu-id="34bb5-102">Compiler Error CS0663</span></span>
<span data-ttu-id="34bb5-103">ref と out でのみ異なるオーバーロードされたメソッドを定義することはできません。</span><span class="sxs-lookup"><span data-stu-id="34bb5-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="34bb5-104">メソッドの使用のみ異なる[で](../../csharp/language-reference/keywords/in-parameter-modifier.md)、 [ref](../../csharp/language-reference/keywords/ref.md)と[アウト](../../csharp/language-reference/keywords/out-parameter-modifier.md)パラメーターでは許可されていません。</span><span class="sxs-lookup"><span data-stu-id="34bb5-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="34bb5-105">次の例では CS0663 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="34bb5-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
