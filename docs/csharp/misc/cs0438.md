 <span data-ttu-id="f4eb6-104">このエラーは、ソース ファイル内の型が別のソース ファイル内の名前空間と競合するときに発生します。</span><span class="sxs-lookup"><span data-stu-id="f4eb6-104">This error occurs when a type in a source file conflicts with a namespace in another source file.</span></span> <span data-ttu-id="f4eb6-105">通常は、いずれかまたは両方が、追加されたモジュールによるものである場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="f4eb6-105">This typically happens when one or both come from an added module.</span></span> <span data-ttu-id="f4eb6-106">解決するには、競合の原因となった型または名前空間の名前を変更します。</span><span class="sxs-lookup"><span data-stu-id="f4eb6-106">To resolve, rename the type or the namespace that caused the conflict.</span></span>

   public class A { }

 <span data-ttu-id="f4eb6-110">それから、このファイルをコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="f4eb6-110">And then compile this file:</span></span>
