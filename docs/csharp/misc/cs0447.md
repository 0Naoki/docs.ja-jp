 <span data-ttu-id="e6964-106">次のコード行ではこのエラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="e6964-106">The following line of code will generate this error.</span></span> <span data-ttu-id="e6964-107">前のコード行に定義されているクラス `C`に、 `MyStaticMethod`という静的メソッドがあることを前提にしています。</span><span class="sxs-lookup"><span data-stu-id="e6964-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>
