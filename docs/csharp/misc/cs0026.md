## <a name="example"></a><span data-ttu-id="c762e-107">例</span><span class="sxs-lookup"><span data-stu-id="c762e-107">Example</span></span>  
 <span data-ttu-id="c762e-108">次の例では、CS0026 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c762e-108">The following example generates CS0026:</span></span>  
  
```csharp  
// CS0026.cs

{  
   public static int i = 0;  
  
   public static void Main()  
   {  
// CS0026  
      this.i = this.i + 1;     
      // Try the following line instead:  
      // i = i + 1;  
   }  
}  
```
