# <a name="compiler-error-cs0452"></a><span data-ttu-id="b62c6-102">コンパイラ エラー CS0452</span><span class="sxs-lookup"><span data-stu-id="b62c6-102">Compiler Error CS0452</span></span>
<span data-ttu-id="b62c6-103">型 'type name' は、ジェネリック型のパラメーター 'parameter name'、またはメソッド 'identifier of generic' として使用するために、参照型でなければなりません</span><span class="sxs-lookup"><span data-stu-id="b62c6-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="b62c6-104">このエラーは、 `struct` や `int` などの値型を、ジェネリック型または参照型制約のあるジェネリック メソッドのパラメーターとして渡した場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="b62c6-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b62c6-105">例</span><span class="sxs-lookup"><span data-stu-id="b62c6-105">Example</span></span>  
 <span data-ttu-id="b62c6-106">次のコードではエラー CS0452 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b62c6-106">The following code generates error CS0452.</span></span>

public class Derived1 : BaseClass<int> { } // CS0452
