## <a name="example"></a><span data-ttu-id="e4925-106">例</span><span class="sxs-lookup"><span data-stu-id="e4925-106">Example</span></span>

```csharp

{
    delegate void D(double d );
    static void F(int i) { }

    static void Main()

        D d = new D(F);  // CS0410
```
