<span data-ttu-id="1ab59-103">using 句は、extern エイリアス宣言以外の、名前空間で定義された他のすべての要素の前に使用しなければなりません。</span><span class="sxs-lookup"><span data-stu-id="1ab59-103">A using clause must precede all other elements defined in the namespace except extern alias declarations</span></span>

## <a name="example"></a><span data-ttu-id="1ab59-105">例</span><span class="sxs-lookup"><span data-stu-id="1ab59-105">Example</span></span>  
 <span data-ttu-id="1ab59-106">次の例では CS1529 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1ab59-106">The following sample generates CS1529:</span></span>  
  
```csharp  
// CS1529.cs  
namespace X  
{  
    namespace Subspace  
    {  
        using Microsoft;  
  
        class SomeClass  
        {  
        };  
  
        using Microsoft;      // CS1529, place before class definition  
    }  
  
    using System.Reflection;  // CS1529, place before namespace 'Subspace'  
}  
  
using System;                 // CS1529, place at the beginning of the file  
```
