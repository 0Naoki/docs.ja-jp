 <span data-ttu-id="470d6-104">条件式で、 [== operator](../../csharp/language-reference/operators/assignment-operator.md) ではなく [= operator](../../csharp/language-reference/operators/equality-comparison-operator.md)が使用されています。</span><span class="sxs-lookup"><span data-stu-id="470d6-104">A conditional expression used the [= operator](../../csharp/language-reference/operators/assignment-operator.md) and not the [== operator](../../csharp/language-reference/operators/equality-comparison-operator.md).</span></span>

```csharp  
   }  
}  
```
