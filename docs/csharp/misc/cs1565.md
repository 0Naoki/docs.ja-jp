# <a name="compiler-error-cs1565"></a>コンパイラ エラー CS1565
競合するオプションが指定されました: Win32 リソース ファイル、Win32 アイコン  
  
 [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) コンパイラ オプションと [/win32ico](../../csharp/language-reference/compiler-options/win32icon-compiler-option.md) コンパイラ オプションを同じコンパイルで両方とも指定することは正しくありません。
