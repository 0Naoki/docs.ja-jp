## <a name="example"></a><span data-ttu-id="2d52a-105">例</span><span class="sxs-lookup"><span data-stu-id="2d52a-105">Example</span></span>  
 <span data-ttu-id="2d52a-106">次の例では CS1909 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2d52a-106">The following sample generates CS1909.</span></span>  
  
```  
// CS1909.cs

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
