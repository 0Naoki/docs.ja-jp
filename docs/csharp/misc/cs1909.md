## <a name="example"></a><span data-ttu-id="a96b9-105">例</span><span class="sxs-lookup"><span data-stu-id="a96b9-105">Example</span></span>  
 <span data-ttu-id="a96b9-106">次の例では CS1909 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a96b9-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs

using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
