# <a name="compiler-error-cs0619"></a><span data-ttu-id="fdbe2-102">コンパイラ エラー CS0619</span><span class="sxs-lookup"><span data-stu-id="fdbe2-102">Compiler Error CS0619</span></span>
<span data-ttu-id="fdbe2-103">'member' は使用されなくなりました: 'text'</span><span class="sxs-lookup"><span data-stu-id="fdbe2-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="fdbe2-104">クラスのメンバーは、[現在不使用] でマークされました ((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) 属性は、クラスのメンバーが参照されているときにエラーが発行するようにします。</span><span class="sxs-lookup"><span data-stu-id="fdbe2-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="fdbe2-105">次の例では CS0619 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="fdbe2-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
