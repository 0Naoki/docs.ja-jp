# <a name="compiler-error-cs0619"></a><span data-ttu-id="8799a-102">コンパイラ エラー CS0619</span><span class="sxs-lookup"><span data-stu-id="8799a-102">Compiler Error CS0619</span></span>
<span data-ttu-id="8799a-103">'member' は使用されなくなりました: 'text'</span><span class="sxs-lookup"><span data-stu-id="8799a-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="8799a-104">クラスメンバーは、[Obsolete] (../programming-guide/concepts/attributes/common-attributes.md # Obsolete) 属性。クラスメンバーが参照されるときにエラーが発行されます。</span><span class="sxs-lookup"><span data-stu-id="8799a-104">A class member was marked with the [Obsolete]((../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="8799a-105">次の例では CS0619 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8799a-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
