# <a name="compiler-error-cs1631"></a><span data-ttu-id="4eb2e-102">コンパイラ エラー CS1631</span><span class="sxs-lookup"><span data-stu-id="4eb2e-102">Compiler Error CS1631</span></span>
<span data-ttu-id="4eb2e-103">catch 句の本体で値を生成することはできません。</span><span class="sxs-lookup"><span data-stu-id="4eb2e-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="4eb2e-104">yield ステートメントは、catch 句の本体内からは使用できません。</span><span class="sxs-lookup"><span data-stu-id="4eb2e-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="4eb2e-105">このエラーを回避するには、yield ステートメントを catch 句の本体の外に移動します。</span><span class="sxs-lookup"><span data-stu-id="4eb2e-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="4eb2e-106">次の例では CS1631 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4eb2e-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
