# <a name="compiler-error-cs1678"></a><span data-ttu-id="112a8-102">コンパイラ エラー CS1678</span><span class="sxs-lookup"><span data-stu-id="112a8-102">Compiler Error CS1678</span></span>
<span data-ttu-id="112a8-103">パラメーター 'number' が型 'type1' として宣言されていますが、'type2' にする必要があります</span><span class="sxs-lookup"><span data-stu-id="112a8-103">Parameter 'number' is declared as type 'type1' but should be 'type2'</span></span>  
  
 <span data-ttu-id="112a8-104">このエラーは、匿名メソッドのパラメーター型が、そのメソッドのキャスト先となるデリゲートの宣言と異なる場合に生じます。</span><span class="sxs-lookup"><span data-stu-id="112a8-104">This error occurs when the parameter type in an anonymous method is different from the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="112a8-105">次の例では CS1678 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="112a8-105">The following sample generates CS1678:</span></span>  
  
```csharp  
// CS1678  
delegate void D(int i);  
class Errors   
{  
   static void Main()   
   {  
      D d = delegate(string s) { };   // CS1678  
      // To resolve, use the following line instead:  
      // D d = delegate(int s) { };  
   }  
}  
```
