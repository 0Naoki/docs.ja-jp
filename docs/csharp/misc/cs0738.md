## <a name="example"></a><span data-ttu-id="b713d-108">例</span><span class="sxs-lookup"><span data-stu-id="b713d-108">Example</span></span>  
 <span data-ttu-id="b713d-109">クラス メソッドは `void` を返し、同じ名前のインターフェイス メンバーは `int`を返すため、次のコードでは、CS0738 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b713d-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
{  
    int TestMethod();  
}

{

    // public int TestMethod();  
}  
```
