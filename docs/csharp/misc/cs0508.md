## <a name="example"></a><span data-ttu-id="ec9dc-106">例</span><span class="sxs-lookup"><span data-stu-id="ec9dc-106">Example</span></span>  
 <span data-ttu-id="ec9dc-107">次の例では CS0508 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ec9dc-107">The following sample generates CS0508.</span></span>  
  
```  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
