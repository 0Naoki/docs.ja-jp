<span data-ttu-id="45141-103">'Type 1': オーバーライドされたメンバー 'Member Name' に対応するために、戻り値の型は 'Type 2' でなければなりません</span><span class="sxs-lookup"><span data-stu-id="45141-103">'Type 1': return type must be 'Type 2' to match overridden member 'Member Name'</span></span>

## <a name="example"></a><span data-ttu-id="45141-106">例</span><span class="sxs-lookup"><span data-stu-id="45141-106">Example</span></span>  
 <span data-ttu-id="45141-107">次の例では CS0508 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="45141-107">The following sample generates CS0508.</span></span>  
  
```csharp  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
