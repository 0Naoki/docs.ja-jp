# <a name="compiler-error-cs1101"></a><span data-ttu-id="cf932-102">コンパイラ エラー CS1101</span><span class="sxs-lookup"><span data-stu-id="cf932-102">Compiler Error CS1101</span></span>
<span data-ttu-id="cf932-103">パラメーター修飾子 'ref' は 'this' と共に使用することはできません</span><span class="sxs-lookup"><span data-stu-id="cf932-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="cf932-104">`this` キーワードは、静的メソッドの最初のパラメーターを修飾するとき、そのメソッドが拡張メソッドであることをコンパイラに通知します。</span><span class="sxs-lookup"><span data-stu-id="cf932-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="cf932-105">拡張メソッドの最初のパラメーターでは、その他の修飾子は必要ないか、または許可されません。</span><span class="sxs-lookup"><span data-stu-id="cf932-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cf932-106">例</span><span class="sxs-lookup"><span data-stu-id="cf932-106">Example</span></span>  
 <span data-ttu-id="cf932-107">次の例では CS1101 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cf932-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
