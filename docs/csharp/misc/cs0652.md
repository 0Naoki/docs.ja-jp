# <a name="compiler-warning-level-2-cs0652"></a><span data-ttu-id="c6b31-102">コンパイラの警告 (レベル 2) CS0652</span><span class="sxs-lookup"><span data-stu-id="c6b31-102">Compiler Warning (level 2) CS0652</span></span>

 <span data-ttu-id="c6b31-105">次の例では CS0652 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c6b31-105">The following sample generates CS0652:</span></span>  
  
```csharp  
// CS0652.cs  
// compile with: /W:2  
public class Class1  
{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```
