<span data-ttu-id="c4a4c-103">キャッチ、または スローされた型は System.Exception から派生したものでなければなりません。</span><span class="sxs-lookup"><span data-stu-id="c4a4c-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="c4a4c-104">**System.Exception** から派生していないデータ型を [catch](../../csharp/language-reference/keywords/try-catch.md) ブロックに渡そうとしました。</span><span class="sxs-lookup"><span data-stu-id="c4a4c-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="c4a4c-105">**System.Exception** から派生したデータ型のみ **catch** ブロックに渡すことができます。</span><span class="sxs-lookup"><span data-stu-id="c4a4c-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="c4a4c-106">詳細については、次を参照してください。[例外と例外処理](../../csharp/programming-guide/exceptions/index.md)します。</span><span class="sxs-lookup"><span data-stu-id="c4a4c-106">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="c4a4c-107">次の例では CS0155 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c4a4c-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
