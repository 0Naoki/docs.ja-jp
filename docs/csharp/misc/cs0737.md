## <a name="example"></a><span data-ttu-id="ecb6d-109">例</span><span class="sxs-lookup"><span data-stu-id="ecb6d-109">Example</span></span>  
 <span data-ttu-id="ecb6d-110">次のコードでは CS0737 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ecb6d-110">The following code generates CS0737:</span></span>  
  
```csharp  
// cs0737.cs  
interface ITest  
{  
    // Default access of private with no modifier.  
    int Return42();  
    // Try the following line instead.  
    // public int Return42();  
}  
  
struct Struct1 : ITest // CS0737  
{  
    int Return42() { return (42); }  
}  
  
public class Test  
{  
    public static int Main(string[] args)  
    {  
        Struct1 s1 = new Struct1();  
  
        return (1);  
    }  
  
}  
```
