# <a name="compiler-error-cs0423"></a><span data-ttu-id="c5121-102">コンパイラ エラー CS0423</span><span class="sxs-lookup"><span data-stu-id="c5121-102">Compiler Error CS0423</span></span>
<span data-ttu-id="c5121-103">'class' は ComImport 属性を含むため、'method' は extern または abstract にしなければなりません。</span><span class="sxs-lookup"><span data-stu-id="c5121-103">Since 'class' has the ComImport attribute, 'method' must be extern or abstract</span></span>  
  
 <span data-ttu-id="c5121-104">ComImport 属性を指定することは、クラスの実装が COM モジュールからインポートされることを意味します。</span><span class="sxs-lookup"><span data-stu-id="c5121-104">Specifying the ComImport attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="c5121-105">追加のメソッドは、定義されない可能性があります。</span><span class="sxs-lookup"><span data-stu-id="c5121-105">Additional methods may not be defined.</span></span>  
  
 <span data-ttu-id="c5121-106">次の例では CS0423 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c5121-106">The following sample generates CS0423:</span></span>  
  
```csharp  
// CS0423.cs  
  
using System.Runtime.InteropServices;  
  
[  
  ComImport,  
  Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829")  
]  
class ImageProperties  
{  
  public static void Main()  // CS0423  
  {  
    ImageProperties i = new ImageProperties();  
  }  
}  
```
