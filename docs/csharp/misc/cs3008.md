# <a name="compiler-warning-level-1-cs3008"></a>コンパイラの警告 (レベル 1) CS3008
大文字、小文字の違いのみの識別子 'identifier' は CLS に準拠していません。  
  
 A[パブリック](../../csharp/language-reference/keywords/public.md)、[保護](../../csharp/language-reference/keywords/protected.md)、または`protected internal`識別子がアンダー スコア文字 (_) で始まる場合、共通言語仕様 (CLS) に対応を中断します。CLS 準拠の詳細については、次を参照してください。 [CLS 準拠コードの記述](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)と[言語非依存および言語非依存コンポーネント](https://msdn.microsoft.com/library/12a7a7h3)です。  
  
## <a name="example"></a>例  
 次の例では、CS3008 が生成されます。  
  
```  
// CS3008.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int _a = 0;  // CS3008  
    // OK, private  
    // private static int _a1 = 0;  
  
    public static void Main()  
    {  
    }  
}  
```
