<span data-ttu-id="fb7ac-103">ラベル 'label' は既にこの switch ステートメントで使用されています</span><span class="sxs-lookup"><span data-stu-id="fb7ac-103">The label 'label' already occurs in this switch statement</span></span>  
  
 <span data-ttu-id="fb7ac-104">[switch](../language-reference/keywords/switch.md) ステートメントでラベルを繰り返しました。</span><span class="sxs-lookup"><span data-stu-id="fb7ac-104">A label was repeated in a [switch](../language-reference/keywords/switch.md) statement.</span></span> <span data-ttu-id="fb7ac-105">詳細については、「 [switch](../language-reference/keywords/switch.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="fb7ac-105">For more information, see [switch](../language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="fb7ac-106">次の例では CS0152 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="fb7ac-106">The following sample generates CS0152:</span></span>  
  
```csharp  
// CS0152.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
  
         switch (i)  
         {  
            case 1:  
               i++;  
               return;  
  
            case 1:   // CS0152, two case 1 statements  
               i++;  
               return;  
         }  
      }  
   }  
}  
```
