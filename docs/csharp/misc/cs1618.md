# <a name="compiler-error-cs1618"></a><span data-ttu-id="f41fa-102">コンパイラ エラー CS1618</span><span class="sxs-lookup"><span data-stu-id="f41fa-102">Compiler Error CS1618</span></span>
<span data-ttu-id="f41fa-103">'method' は条件付き属性なので、この属性でデリゲートを作成できません</span><span class="sxs-lookup"><span data-stu-id="f41fa-103">Cannot create delegate with 'method' because it has a Conditional attribute</span></span>  
  
 <span data-ttu-id="f41fa-104">一部のビルドにメソッドが存在しない可能性があるため、条件付きメソッドを使用してデリゲートを作成することはできません。</span><span class="sxs-lookup"><span data-stu-id="f41fa-104">You cannot create a delegate with a conditional method because the method might not exist in some builds.</span></span>  
  
 <span data-ttu-id="f41fa-105">次の例では CS1618 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f41fa-105">The following sample generates CS1618:</span></span>  
  
```csharp  
// CS1618.cs  
using System;  
using System.Diagnostics;  
  
delegate void del();  
  
class MakeAnError {  
   public static void Main() {  
      del d = new del(ConditionalMethod);   // CS1618  
      // Invalid because on builds where DEBUG is not set,   
      // there will be no "ConditionalMethod".  
   }  
   // To fix the error, remove the next line:  
   [Conditional("DEBUG")]  
   public static void ConditionalMethod()   
   {  
      Console.WriteLine("Do something only in debug");  
   }  
}  
```
