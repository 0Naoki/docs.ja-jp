 <span data-ttu-id="7d89a-105">C# コンパイラがの使用を許可していない<xref:System.ParamArrayAttribute?displayProperty=nameWithType>; を使用して[params](../../csharp/language-reference/keywords/params.md)代わりにします。</span><span class="sxs-lookup"><span data-stu-id="7d89a-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="7d89a-106">次の例では CS0674 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7d89a-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs

public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
