# <a name="compiler-error-cs0674"></a><span data-ttu-id="d896f-102">コンパイラ エラー CS0674</span><span class="sxs-lookup"><span data-stu-id="d896f-102">Compiler Error CS0674</span></span>
<span data-ttu-id="d896f-103">'System.ParamArrayAttribute' は使用しないでください。</span><span class="sxs-lookup"><span data-stu-id="d896f-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="d896f-104">代わりに 'params' キーワードを使用します。</span><span class="sxs-lookup"><span data-stu-id="d896f-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="d896f-105">C# コンパイラがの使用を許可していない<xref:System.ParamArrayAttribute?displayProperty=nameWithType>; を使用して[params](../../csharp/language-reference/keywords/params.md)代わりにします。</span><span class="sxs-lookup"><span data-stu-id="d896f-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="d896f-106">次の例では CS0674 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d896f-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
