# コンパイラ エラー CS1563
出力 'output file' にソース ファイルがありません。  
  
 [\/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) コンパイラ オプションが指定されましたが、ソース コード ファイルがその後にありません。**\/out** を使用しないか、または **\/out** の後にソース コード ファイルを指定する必要があります。
