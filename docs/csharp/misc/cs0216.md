# <a name="compiler-error-cs0216"></a><span data-ttu-id="2407b-102">コンパイラ エラー CS0216</span><span class="sxs-lookup"><span data-stu-id="2407b-102">Compiler Error CS0216</span></span>
<span data-ttu-id="2407b-103">演算子 'operator' を定義するには、合致する演算子 'missing_operator' が必要です。</span><span class="sxs-lookup"><span data-stu-id="2407b-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="2407b-104">ユーザー定義の [true](../../csharp/language-reference/keywords/true.md) 演算子にはユーザー定義の [false](../../csharp/language-reference/keywords/false.md) 演算子が必要であり、その逆もまた同様です。</span><span class="sxs-lookup"><span data-stu-id="2407b-104">A user-defined [true](../../csharp/language-reference/keywords/true.md) operator requires a user-defined [false](../../csharp/language-reference/keywords/false.md) operator, and vice versa.</span></span> <span data-ttu-id="2407b-105">詳細については、「[演算子](../../csharp/programming-guide/statements-expressions-operators/operators.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="2407b-105">For more information, see [Operators](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span></span>  
  
 <span data-ttu-id="2407b-106">次の例では CS0216 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2407b-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
