<span data-ttu-id="699e3-103">部分メソッドにはアクセス修飾子や、virtual、abstract、override、new、sealed、または extern 修飾子を指定できません。</span><span class="sxs-lookup"><span data-stu-id="699e3-103">A partial method cannot have access modifiers or the virtual, abstract, override, new, sealed, or extern modifiers.</span></span>  
  
 <span data-ttu-id="699e3-104">部分メソッドは特別な動作をします。そのため、部分メソッドは使用できる修飾子に制限される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="699e3-104">Because of their special behavior, partial methods are subject to restrictions as to the modifiers they can accept.</span></span>

1. <span data-ttu-id="699e3-106">承認されていない修飾子を、メソッドの宣言から削除します。</span><span class="sxs-lookup"><span data-stu-id="699e3-106">Remove the unauthorized modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="699e3-107">例</span><span class="sxs-lookup"><span data-stu-id="699e3-107">Example</span></span>  
 <span data-ttu-id="699e3-108">次の例では CS0750 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="699e3-108">The following example generates CS0750:</span></span>  
  
```csharp  
// cs0750.cs  
using System;  
  
public class Base  
{  
    protected virtual void PartG()  
    {  
    }  
  
    protected void PartH()  
    {  
    }  
    protected virtual void PartI()  
    {  
    }  
}  
  
public partial class C:Base  
{  
    // All these partial method declarations  
    // will generate CS0750.  
    public partial void PartA();  
    private partial void PartB();  
    protected partial void PartC();  
    internal partial void PartD();  
    virtual partial void PartE();  
    abstract partial void PartF();  
    override partial void PartG();  
    new partial void PartH();  
    sealed override partial void PartI();  
    extern partial void PartJ();  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
