# コンパイラ エラー CS2003
応答ファイル 'file' が複数回含まれていました  
  
 応答ファイルがコンパイラに複数回渡されました。 応答ファイルをコンパイラに渡せるのは、出力ファイルごとに 1 回だけです。  
  
 応答ファイルについて詳しくは、「[@ \(応答ファイルの指定\)](../../csharp/language-reference/compiler-options/response-file-compiler-option.md)」をご覧ください。
