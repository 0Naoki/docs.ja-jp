# コンパイラ エラー CS1664
長さ 'length'、型 'type' の固定サイズ バッファーは大きすぎます  
  
 固定サイズ バッファーの最大サイズ \(長さに要素サイズを乗算して決定される\) は、2^31 \= 268435455 です。
