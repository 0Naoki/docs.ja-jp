 <span data-ttu-id="5c576-104">正しくないステートメントがコンパイラによって検出されました。</span><span class="sxs-lookup"><span data-stu-id="5c576-104">The compiler found an invalid statement.</span></span>

```csharp  
// CS1044.cs

{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");

      {  
      }  
   }  
}  
```
