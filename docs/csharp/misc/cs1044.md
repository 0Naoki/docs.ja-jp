<span data-ttu-id="79c58-103">for、using、fixed または declaration ステートメント に 1 つ以上の型を使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="79c58-103">Cannot use more than one type in a for, using, fixed, or declaration statement</span></span>  
  
 <span data-ttu-id="79c58-104">正しくないステートメントがコンパイラによって検出されました。</span><span class="sxs-lookup"><span data-stu-id="79c58-104">The compiler found an invalid statement.</span></span>  
  
 <span data-ttu-id="79c58-105">次の例では CS1044 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="79c58-105">The following sample generates CS1044:</span></span>  
  
```csharp  
// CS1044.cs  
using System;  
  
public class MyClass : IDisposable  
{  
   public void Dispose()  
   {  
      Console.WriteLine("Res1.Dispose()");  
   }  
  
   public static void Main()  
   {  
      using (MyClass mc1 = new MyClass(),  
             MyClass mc2 = new MyClass())   // CS1044, remove an instantiation  
      {  
      }  
   }  
}  
```
