## <a name="example"></a><span data-ttu-id="72286-106">例</span><span class="sxs-lookup"><span data-stu-id="72286-106">Example</span></span>  
 <span data-ttu-id="72286-107">次のコード例では、警告 CS3023 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="72286-107">The following example generates warning CS3023:</span></span>

```csharp  
{  
    [return:System.CLSCompliant(true)]  // CS3023

    {  
        return 0;  
    }  
}  
```
