# <a name="compiler-error-cs0502"></a><span data-ttu-id="80320-102">コンパイラ エラー CS0502</span><span class="sxs-lookup"><span data-stu-id="80320-102">Compiler Error CS0502</span></span>
<span data-ttu-id="80320-103">'member' を abstract および sealed に同時に指定することはできません</span><span class="sxs-lookup"><span data-stu-id="80320-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="80320-104">クラス メンバーを [abstract](../language-reference/keywords/abstract.md) および [sealed](../language-reference/keywords/sealed.md)に同時に指定することはできません。</span><span class="sxs-lookup"><span data-stu-id="80320-104">A class member cannot be both [abstract](../language-reference/keywords/abstract.md) and [sealed](../language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="80320-105">次の例では CS0502 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="80320-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
