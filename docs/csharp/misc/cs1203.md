# コンパイラの警告 (レベル 1) CS1203
機能 'feature' は推奨されていません。 代わりに 'feature' を使用してください。  
  
 機能 'invalid feature' は推奨されていません。 代わりに 'valid feature' を使用してください。  
  
 使用しようとしている機能は、現在推奨されていません。 有効な機能を使用するように、コードを更新してください。
