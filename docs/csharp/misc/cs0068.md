# <a name="compiler-error-cs0068"></a><span data-ttu-id="97319-102">コンパイラ エラー CS0068</span><span class="sxs-lookup"><span data-stu-id="97319-102">Compiler Error CS0068</span></span>
<span data-ttu-id="97319-103">'event': インターフェイスのイベントに初期化子を含めることができません</span><span class="sxs-lookup"><span data-stu-id="97319-103">'event': event in interface cannot have initializer</span></span>  
  
 <span data-ttu-id="97319-104">インターフェイスのイベントに初期化子を含めることはできません。</span><span class="sxs-lookup"><span data-stu-id="97319-104">An event in an interface cannot have an initializer.</span></span> <span data-ttu-id="97319-105">詳細については、「[インターフェイス](../../csharp/programming-guide/interfaces/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="97319-105">For more information, see [Interfaces](../../csharp/programming-guide/interfaces/index.md).</span></span>  
  
 <span data-ttu-id="97319-106">次の例では CS0068 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="97319-106">The following sample generates CS0068:</span></span>  
  
```csharp  
// CS0068.cs  
  
delegate void MyDelegate();  
  
interface I  
{  
   event MyDelegate d = new MyDelegate(M.f);   // CS0068  
   // try the following line instead  
   // event MyDelegate d2;  
}  
  
class M  
{  
   event MyDelegate d = new MyDelegate(M.f);  
  
   public static void f()  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
