# <a name="compiler-error-cs0617"></a><span data-ttu-id="95aa4-102">コンパイラ エラー CS0617</span><span class="sxs-lookup"><span data-stu-id="95aa4-102">Compiler Error CS0617</span></span>
<span data-ttu-id="95aa4-103">'reference' は有効な属性パラメーター型ではないため、有効な名前付き属性引数ではありません</span><span class="sxs-lookup"><span data-stu-id="95aa4-103">'reference' is not a valid named attribute argument because it is not a valid attribute parameter type</span></span>  
  
 <span data-ttu-id="95aa4-104">属性クラスの [private](../language-reference/keywords/private.md) メンバーにアクセスしようとしました。</span><span class="sxs-lookup"><span data-stu-id="95aa4-104">An attempt was made to access a [private](../language-reference/keywords/private.md) member of an attribute class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="95aa4-105">例</span><span class="sxs-lookup"><span data-stu-id="95aa4-105">Example</span></span>  
 <span data-ttu-id="95aa4-106">次の例では CS0617 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="95aa4-106">The following sample generates CS0617.</span></span>  
  
```csharp  
// CS0617.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Struct |   
                AttributeTargets.Class |  
                AttributeTargets.Interface)]  
public class MyClass : Attribute  
{  
   public int Name;  
  
   public MyClass (int sName)  
   {  
      Name = sName;  
      Bad = -1;  
      Bad2 = -1;  
   }  
  
   public readonly int Bad;  
   public int Bad2;  
}  
  
[MyClass(5, Bad=0)] class Class1 {}   // CS0617  
[MyClass(5, Bad2=0)] class Class2 {}  
```
