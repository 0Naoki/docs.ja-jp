# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="5726b-102">コンパイラの警告 (レベル 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="5726b-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="5726b-103">'interface' インターフェイスは、'CoClassAttribute' でマークされていますが、'ComImportAttribute' ではマークされていません</span><span class="sxs-lookup"><span data-stu-id="5726b-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="5726b-104">インターフェイスに **CoClassAttribute** を指定した場合は、 **ComImportAttribute**も指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="5726b-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="5726b-105">次の例では CS0684 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5726b-105">The following sample generates CS0684:</span></span>  
  
```csharp  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
