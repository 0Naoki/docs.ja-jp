# <a name="compiler-error-cs0546"></a><span data-ttu-id="8cc32-102">コンパイラ エラー CS0546</span><span class="sxs-lookup"><span data-stu-id="8cc32-102">Compiler Error CS0546</span></span>
<span data-ttu-id="8cc32-103">'accessor': 'property' に、オーバーライド可能な set アクセサーがないため、オーバーライドできません。</span><span class="sxs-lookup"><span data-stu-id="8cc32-103">'accessor' : cannot override because 'property' does not have an overridable set accessor</span></span>  
  
 <span data-ttu-id="8cc32-104">アクセサーをオーバーライドできないために、プロパティのアクセサー メソッドのオーバーライドに失敗しました。</span><span class="sxs-lookup"><span data-stu-id="8cc32-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="8cc32-105">このエラーは、次の場合に発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="8cc32-105">This error can occur if:</span></span>  
  
-   <span data-ttu-id="8cc32-106">基底クラス プロパティが [virtual](../../csharp/language-reference/keywords/virtual.md)として宣言されていません。</span><span class="sxs-lookup"><span data-stu-id="8cc32-106">the base class property is not declared as [virtual](../../csharp/language-reference/keywords/virtual.md).</span></span>  
  
-   <span data-ttu-id="8cc32-107">基底クラスのプロパティが、オーバーライドする [get](../../csharp/language-reference/keywords/get.md) または [set](../../csharp/language-reference/keywords/set.md) アクセサーを宣言していません。</span><span class="sxs-lookup"><span data-stu-id="8cc32-107">the base class property does not declare the [get](../../csharp/language-reference/keywords/get.md) or [set](../../csharp/language-reference/keywords/set.md) accessor you are trying to override.</span></span>  
  
 <span data-ttu-id="8cc32-108">基底クラス プロパティをオーバーライドしない場合、派生クラスでプロパティの前に [new](../../csharp/language-reference/keywords/new.md) キーワードを使用することができます。</span><span class="sxs-lookup"><span data-stu-id="8cc32-108">If you do not want to override the base class property, you can use the [new](../../csharp/language-reference/keywords/new.md) keyword before the property in derived class.</span></span>  
  
 <span data-ttu-id="8cc32-109">詳細については、「[プロパティの使用](../../csharp/programming-guide/classes-and-structs/using-properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="8cc32-109">For more information, see [Using Properties](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="8cc32-110">例</span><span class="sxs-lookup"><span data-stu-id="8cc32-110">Example</span></span>  
 <span data-ttu-id="8cc32-111">次の例では、基底クラスでプロパティの set アクセサーが宣言されていないため、CS0546 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="8cc32-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
