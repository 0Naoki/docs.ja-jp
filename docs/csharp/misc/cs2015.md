# <a name="compiler-error-cs2015"></a>コンパイラ エラー CS2015
'file' はテキスト ファイルではなく、バイナリ ファイルです  
  
 コンパイラに渡されたファイルは、バイナリ ファイルでした。 コンパイラでは、ソース コード ファイルが必要です。
