# <a name="compiler-error-cs1621"></a><span data-ttu-id="a59f5-102">コンパイラ エラー CS1621</span><span class="sxs-lookup"><span data-stu-id="a59f5-102">Compiler Error CS1621</span></span>
<span data-ttu-id="a59f5-103">yield ステートメントは、匿名メソッドまたはラムダ式の内部では使用できません</span><span class="sxs-lookup"><span data-stu-id="a59f5-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="a59f5-104">yield ステートメントを反復子の匿名メソッド ブロックに記述することはできません。</span><span class="sxs-lookup"><span data-stu-id="a59f5-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a59f5-105">例</span><span class="sxs-lookup"><span data-stu-id="a59f5-105">Example</span></span>  
 <span data-ttu-id="a59f5-106">次の例では CS1621 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a59f5-106">The following sample generates CS1621:</span></span>  
  
```  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
