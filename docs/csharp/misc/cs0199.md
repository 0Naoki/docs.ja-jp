<span data-ttu-id="69f43-103">静的読み取り専用フィールド 'name' には、静的コンストラクター内を除き、ref または out を渡すことはできません</span><span class="sxs-lookup"><span data-stu-id="69f43-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="69f43-104">[readonly](../../csharp/language-reference/keywords/readonly.md) 変数では、それが [ref](../../csharp/language-reference/keywords/static.md) または [out](../../csharp/language-reference/keywords/ref.md) パラメーターとして渡されるコンストラクターと同じように [static](../../csharp/language-reference/keywords/out-parameter-modifier.md) を使用する必要があります。</span><span class="sxs-lookup"><span data-stu-id="69f43-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="69f43-105">詳細については、「[パラメーターの引き渡し](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="69f43-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="69f43-106">例</span><span class="sxs-lookup"><span data-stu-id="69f43-106">Example</span></span>  
 <span data-ttu-id="69f43-107">次の例では CS0199 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="69f43-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
