## <a name="example"></a><span data-ttu-id="4ebe8-106">例</span><span class="sxs-lookup"><span data-stu-id="4ebe8-106">Example</span></span>  
 <span data-ttu-id="4ebe8-107">次の例では CS0199 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4ebe8-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
