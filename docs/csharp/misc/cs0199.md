<span data-ttu-id="7e061-103">静的読み取り専用フィールド 'name' には、静的コンストラクター内を除き、ref または out を渡すことはできません</span><span class="sxs-lookup"><span data-stu-id="7e061-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="7e061-104">[readonly](../language-reference/keywords/readonly.md) 変数では、それが [ref](../language-reference/keywords/static.md) または [out](../language-reference/keywords/ref.md) パラメーターとして渡されるコンストラクターと同じように [static](../language-reference/keywords/out-parameter-modifier.md) を使用する必要があります。</span><span class="sxs-lookup"><span data-stu-id="7e061-104">A [readonly](../language-reference/keywords/readonly.md) variable must have the same [static](../language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../language-reference/keywords/ref.md) or [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="7e061-105">詳細については、「[パラメーターの引き渡し](../programming-guide/classes-and-structs/passing-parameters.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="7e061-105">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="7e061-106">例</span><span class="sxs-lookup"><span data-stu-id="7e061-106">Example</span></span>  
 <span data-ttu-id="7e061-107">次の例では CS0199 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7e061-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
