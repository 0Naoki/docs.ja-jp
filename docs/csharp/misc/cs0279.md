## <a name="example"></a><span data-ttu-id="2b95e-109">例</span><span class="sxs-lookup"><span data-stu-id="2b95e-109">Example</span></span>  
 <span data-ttu-id="2b95e-110">次の例では、CS0279 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="2b95e-110">The following example generates CS0279:</span></span>

```csharp  
using System.Collections;  
  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }

        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
