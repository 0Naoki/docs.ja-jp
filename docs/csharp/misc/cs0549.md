# <a name="compiler-error-cs0549"></a><span data-ttu-id="a361c-102">コンパイラ エラー CS0549</span><span class="sxs-lookup"><span data-stu-id="a361c-102">Compiler Error CS0549</span></span>
<span data-ttu-id="a361c-103">'function' はシール クラス 'class' の新しい仮想メンバーです。</span><span class="sxs-lookup"><span data-stu-id="a361c-103">'function' is a new virtual member in sealed class 'class'</span></span>  
  
 <span data-ttu-id="a361c-104">A[シール](../../csharp/language-reference/keywords/sealed.md)[クラス](../../csharp/language-reference/keywords/class.md)基底クラスとして使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="a361c-104">A [sealed](../../csharp/language-reference/keywords/sealed.md)[class](../../csharp/language-reference/keywords/class.md) cannot be used as a base class.</span></span>  <span data-ttu-id="a361c-105">したがって、シール クラスに仮想メソッドを使用しても意味がありません。</span><span class="sxs-lookup"><span data-stu-id="a361c-105">Therefore, it is useless to have a virtual method in sealed class.</span></span>  
  
 <span data-ttu-id="a361c-106">次の例では CS0549 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a361c-106">The following sample generates CS0549:</span></span>

```csharp  
{  
   virtual public void TestMethod() {}   // CS0549

}  
```
