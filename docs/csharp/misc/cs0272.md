## <a name="example"></a><span data-ttu-id="3918d-107">例</span><span class="sxs-lookup"><span data-stu-id="3918d-107">Example</span></span>  
 <span data-ttu-id="3918d-108">次の例では、CS0272 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3918d-108">The following example generates CS0272:</span></span>  
  
```  
// CS0272.cs  
public class MyClass  
{  
    public int Property  
    {  
        get { return 0; }  
        private set { }  
    }  
}  
  
public class Test  
{  
    static void Main()  
    {  
        MyClass c = new MyClass();  
        c.Property = 10;      // CS0272  
        // To resolve, remove the previous line   
        // or use an appropriate modifier on the set accessor.  
    }  
}  
```
