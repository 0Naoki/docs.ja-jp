# <a name="compiler-error-cs1713"></a>コンパイラ エラー CS1713
型 Typename1 のメタデータ名のビルド中に予期しないエラーが発生しました—'Reason'  
  
 多くの場合、このエラーの原因は内部コンパイラ エラーです。 名前の長さを短くするなど、コードに若干の変更を加えてから再コンパイルを行うと、このエラーが解決する場合があります。
