# <a name="compiler-error-cs0724"></a>コンパイラ エラー CS0724
アセンブリには CLSCompliant 属性がないため、CLSCompliant 属性は不要です。  
  
 次の例では、 `throw` 句ブロック内の `finally` ステートメントが原因で CS0724 が生成されます。  
  
## <a name="example"></a>例  
 次の例では CS0724 が生成されます。  
  
```csharp  
// CS0724.cs  
using System;  
  
class X  
{  
    static void Test()  
    {  
        try  
        {  
            throw new Exception();  
        }  
        catch  
        {  
            try  
            {  
            }  
            finally  
            {  
                throw; // CS0724  
            }  
        }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
