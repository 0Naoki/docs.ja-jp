# <a name="compiler-error-cs1715"></a><span data-ttu-id="70703-102">コンパイラ エラー CS1715</span><span class="sxs-lookup"><span data-stu-id="70703-102">Compiler Error CS1715</span></span>
<span data-ttu-id="70703-103">'Type1': オーバーライドされたメンバー 'MemberName' に対応するために、型は 'Type2' でなければなりません</span><span class="sxs-lookup"><span data-stu-id="70703-103">'Type1': type must be 'Type2' to match overridden member 'MemberName'</span></span>  
  
 <span data-ttu-id="70703-104">このエラーは [Compiler Error CS0508](../../csharp/misc/cs0508.md)と同じですが、CS0508 が戻り値の型を持つメソッドにのみ適用されるようになったのに対し、CS1715 は '戻り値の型' ではなく '型' のみを持つプロパティとインデクサーに適用される点が異なります。</span><span class="sxs-lookup"><span data-stu-id="70703-104">This error is the same as [Compiler Error CS0508](../../csharp/misc/cs0508.md), except that CS0508 now only applies to methods that have return types, while CS1715 applies to properties and indexers that only have 'types' instead of 'return types'.</span></span>  
  
## <a name="example"></a><span data-ttu-id="70703-105">例</span><span class="sxs-lookup"><span data-stu-id="70703-105">Example</span></span>  
 <span data-ttu-id="70703-106">次のコードでは CS1715 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="70703-106">The following code generates CS1715.</span></span>  
  
```csharp  
// CS1715.cs  
abstract public class Base  
{  
    abstract public int myProperty  
    {  
        get;  
        set;  
    }  
}  
  
public class Derived : Base  
{  
    int myField;  
    public override double myProperty  // CS1715  
    // try the following line instead  
    // public override int myProperty  
    {  
        get { return myField; }  
        set { myField;= value; }  
    }  
  
    public static void Main()  
    {  
        Derived d = new Derived();  
        d.myProperty = 5;  
    }  
}  
```
