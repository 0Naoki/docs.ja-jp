# <a name="compiler-error-cs0157"></a><span data-ttu-id="8cc38-102">コンパイラ エラー CS0157</span><span class="sxs-lookup"><span data-stu-id="8cc38-102">Compiler Error CS0157</span></span>
<span data-ttu-id="8cc38-103">コントロールが finally 句の本体から出られません。</span><span class="sxs-lookup"><span data-stu-id="8cc38-103">Control cannot leave the body of a finally clause</span></span>  
  
 <span data-ttu-id="8cc38-104">[finally](../../csharp/language-reference/keywords/try-catch-finally.md) 句のすべてのステートメントを実行する必要があります。</span><span class="sxs-lookup"><span data-stu-id="8cc38-104">All of the statements in a [finally](../../csharp/language-reference/keywords/try-catch-finally.md) clause must execute.</span></span> <span data-ttu-id="8cc38-105">詳細については、次を参照してください。[例外と例外処理](../../csharp/programming-guide/exceptions/index.md)します。</span><span class="sxs-lookup"><span data-stu-id="8cc38-105">For more information, see [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="8cc38-106">次の例では CS0157 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8cc38-106">The following sample generates CS0157:</span></span>  
  
```csharp  
// CS0157.cs  
using System;  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
         }  
  
         finally  
         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
