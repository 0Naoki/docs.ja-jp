 <span data-ttu-id="82ec2-104">クラスまたは構造体のメンバーは、メンバーがコンストラクターでない限り、クラスまたは構造体と同じ名前にすることはできません。</span><span class="sxs-lookup"><span data-stu-id="82ec2-104">The members of a class or struct cannot have the same name as the class or struct, unless the member is a constructor.</span></span>

 <span data-ttu-id="82ec2-106">このエラーは、コンストラクターに誤って戻り値の型を指定したことにより、コンストラクターが事実上、通常のメソッドとなる場合に発生する可能性があります。</span><span class="sxs-lookup"><span data-stu-id="82ec2-106">This error might be caused if you inadvertently put a return type on a constructor, which in effect makes it into an ordinary method.</span></span> <span data-ttu-id="82ec2-107">次の例では、 `F` が戻り値の型を含むことから、コンストラクターではなくメソッドとなるため、CS0542 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="82ec2-107">The following example generates CS0542 because `F` is a method, not a constructor, because it has a return type:</span></span>

   // Remove void from F() to resolve the problem.

   public static void Main()  
   {  
   }
