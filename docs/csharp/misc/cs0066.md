# <a name="compiler-error-cs0066"></a><span data-ttu-id="e041a-102">コンパイラ エラー CS0066</span><span class="sxs-lookup"><span data-stu-id="e041a-102">Compiler Error CS0066</span></span>
<span data-ttu-id="e041a-103">'event': イベントはデリゲート型でなければなりません。</span><span class="sxs-lookup"><span data-stu-id="e041a-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="e041a-104">event キーワードには、 [デリゲート](../../csharp/language-reference/keywords/delegate.md) 型が必要です。</span><span class="sxs-lookup"><span data-stu-id="e041a-104">The event keyword requires a [delegate](../../csharp/language-reference/keywords/delegate.md) type.</span></span> <span data-ttu-id="e041a-105">詳細については、次を参照してください。[イベント](../../csharp/programming-guide/events/index.md)と[デリゲート](../../csharp/programming-guide/delegates/index.md)です。</span><span class="sxs-lookup"><span data-stu-id="e041a-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="e041a-106">次の例では CS0066 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e041a-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
