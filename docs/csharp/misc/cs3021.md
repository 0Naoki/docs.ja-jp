 <span data-ttu-id="4b7c3-104">この警告が発生`[CLSCompliant(false)]`を true に、アセンブリ レベル CLSCompliant 属性が設定されていないアセンブリ内のクラスに表示されます (つまり、行`[assembly: CLSCompliant(true)]`)。</span><span class="sxs-lookup"><span data-stu-id="4b7c3-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="4b7c3-105">アセンブリは自らを CLS 準拠と宣言していないため、非準拠であると見なされます。そのため、アセンブリ内では非準拠を宣言する必要はありません。</span><span class="sxs-lookup"><span data-stu-id="4b7c3-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="4b7c3-106">CLS 準拠の詳細については、次を参照してください。 [CLS 準拠コードの記述](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3)です。</span><span class="sxs-lookup"><span data-stu-id="4b7c3-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>  
  
 <span data-ttu-id="4b7c3-107">この警告を消去するには、対象属性を削除するか、アセンブリ レベルの属性を追加します。</span><span class="sxs-lookup"><span data-stu-id="4b7c3-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4b7c3-108">例</span><span class="sxs-lookup"><span data-stu-id="4b7c3-108">Example</span></span>  
 <span data-ttu-id="4b7c3-109">次の例では、CS3021 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="4b7c3-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
