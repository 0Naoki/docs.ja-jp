# <a name="compiler-warning-level-2-cs3021"></a>コンパイラの警告 (レベル 2) CS3021
アセンブリに CLSCompliant 属性が含まれていないため、'type' には CLSCompliant 属性が必要ありません  
  
 この警告が発生`[CLSCompliant(false)]`を true に、アセンブリ レベル CLSCompliant 属性が設定されていないアセンブリ内のクラスに表示されます (つまり、行`[assembly: CLSCompliant(true)]`)。 アセンブリは自らを CLS 準拠と宣言していないため、非準拠であると見なされます。そのため、アセンブリ内では非準拠を宣言する必要はありません。 CLS 準拠について詳しくは、「 [CLS 準拠コードの記述](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)」をご覧ください。  
  
 この警告を消去するには、対象属性を削除するか、アセンブリ レベルの属性を追加します。  
  
## <a name="example"></a>例  
 次の例では、CS3021 エラーが生成されます。  
  
```  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
