# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="7d4a4-102">コンパイラの警告 (レベル 2) CS3021</span><span class="sxs-lookup"><span data-stu-id="7d4a4-102">Compiler Warning (level 2) CS3021</span></span>
<span data-ttu-id="7d4a4-103">アセンブリに CLSCompliant 属性が含まれていないため、'type' には CLSCompliant 属性が必要ありません</span><span class="sxs-lookup"><span data-stu-id="7d4a4-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="7d4a4-104">この警告は、場合に発生します。`[CLSCompliant(false)]`を true に、アセンブリ レベルの CLSCompliant 属性セットがないアセンブリ内のクラスに表示されます (行は、 `[assembly: CLSCompliant(true)]`)。</span><span class="sxs-lookup"><span data-stu-id="7d4a4-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="7d4a4-105">アセンブリは自らを CLS 準拠と宣言していないため、非準拠であると見なされます。そのため、アセンブリ内では非準拠を宣言する必要はありません。</span><span class="sxs-lookup"><span data-stu-id="7d4a4-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="7d4a4-106">CLS 準拠の詳細については、[言語の独立性および言語非依存コンポーネント](../../standard/language-independence.md)を参照してください。</span><span class="sxs-lookup"><span data-stu-id="7d4a4-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
 <span data-ttu-id="7d4a4-107">この警告を消去するには、対象属性を削除するか、アセンブリ レベルの属性を追加します。</span><span class="sxs-lookup"><span data-stu-id="7d4a4-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7d4a4-108">例</span><span class="sxs-lookup"><span data-stu-id="7d4a4-108">Example</span></span>  
 <span data-ttu-id="7d4a4-109">次の例では、CS3021 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="7d4a4-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
