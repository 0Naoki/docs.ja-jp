 <span data-ttu-id="a2a31-104">[Indexers](../../csharp/programming-guide/indexers/index.md)をサポートしないデータ型に対し、インデクサーによって値にアクセスしようとしました。</span><span class="sxs-lookup"><span data-stu-id="a2a31-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../../csharp/programming-guide/indexers/index.md).</span></span>

## <a name="example"></a><span data-ttu-id="a2a31-108">例</span><span class="sxs-lookup"><span data-stu-id="a2a31-108">Example</span></span>  
 <span data-ttu-id="a2a31-109">エラーを生成するには、このファイルをコンパイルして ( `DefaultMember` 属性をコメント アウトして) .dll ファイルを作成します。</span><span class="sxs-lookup"><span data-stu-id="a2a31-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="a2a31-110">例</span><span class="sxs-lookup"><span data-stu-id="a2a31-110">Example</span></span>

```  
// compile with: /reference:CPP0021.dll

{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();

    }  
}  
```
