 <span data-ttu-id="b46f2-104">[Indexers](../programming-guide/indexers/index.md)をサポートしないデータ型に対し、インデクサーによって値にアクセスしようとしました。</span><span class="sxs-lookup"><span data-stu-id="b46f2-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../programming-guide/indexers/index.md).</span></span>  
  
 <span data-ttu-id="b46f2-105">C++ アセンブリでインデクサーの使用を試行すると、CS0021 エラーが発生することがあります。</span><span class="sxs-lookup"><span data-stu-id="b46f2-105">You may get CS0021 when trying to use an indexer in a C++ assembly.</span></span> <span data-ttu-id="b46f2-106">この場合は、既定のインデクサーを C# コンパイラが判別できるように、C++ クラスを `DefaultMember` 属性で修飾します。</span><span class="sxs-lookup"><span data-stu-id="b46f2-106">In this case, decorate the C++ class with the `DefaultMember` attribute so the C# compiler knows which indexer is the default.</span></span> <span data-ttu-id="b46f2-107">次の例では CS0021 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="b46f2-107">The following sample generates CS0021.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b46f2-108">例</span><span class="sxs-lookup"><span data-stu-id="b46f2-108">Example</span></span>  
 <span data-ttu-id="b46f2-109">エラーを生成するには、このファイルをコンパイルして ( `DefaultMember` 属性をコメント アウトして) .dll ファイルを作成します。</span><span class="sxs-lookup"><span data-stu-id="b46f2-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="b46f2-110">例</span><span class="sxs-lookup"><span data-stu-id="b46f2-110">Example</span></span>

```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
