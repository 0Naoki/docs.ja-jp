<span data-ttu-id="82ff3-104">[Indexers](../programming-guide/indexers/index.md)をサポートしないデータ型に対し、インデクサーによって値にアクセスしようとしました。</span><span class="sxs-lookup"><span data-stu-id="82ff3-104">An attempt was made to access a value through an indexer on a data type that does not support [Indexers](../programming-guide/indexers/index.md).</span></span>

## <a name="example"></a><span data-ttu-id="82ff3-108">例</span><span class="sxs-lookup"><span data-stu-id="82ff3-108">Example</span></span>  

<span data-ttu-id="82ff3-109">次C++の例では、.dll ファイルにコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="82ff3-109">The following C++ example compiles to a .dll file.</span></span> <span data-ttu-id="82ff3-110">エラーを生成するために、 [DefaultMember](xref:System.Reflection.DefaultMemberAttribute)属性がコメントアウトされていることに注意してください。</span><span class="sxs-lookup"><span data-stu-id="82ff3-110">Note that the the [DefaultMember](xref:System.Reflection.DefaultMemberAttribute) attribute is commented out in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```  

<span data-ttu-id="82ff3-111">次C#の例では、.dll ファイルを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="82ff3-111">The following C# example calls the .dll file.</span></span> <span data-ttu-id="82ff3-112">インデクサーを使用してクラスにアクセスしようとしますが、既定のインデクサーとして宣言されたメンバーがないため、エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="82ff3-112">It attempts to access the class via an indexer, but because no member was declared as the default indexer, the error is generated.</span></span> <span data-ttu-id="82ff3-113">このエラーを解決するには `[DefaultMember("myItem")]` という行をコメント解除します。</span><span class="sxs-lookup"><span data-stu-id="82ff3-113">You can address the error by uncommenting the line `[DefaultMember("myItem")]`.</span></span>
  
```csharp  
// CS0021.cs  
// compile with: /reference:CPP0021.dll  
public class MyClass  
{  
    public static void Main()  
    {  
        MyClassMC myMC = new MyClassMC();  
        int j = myMC[1]; // CS0021  
    }  
}  
```
