# <a name="compiler-warning-level-1-cs3004"></a>コンパイラの警告 (レベル 1) CS3004
混合、分解された Unicode 文字は CLS に準拠していません。  
  
 のみで構成の UNICODE 文字は使用[パブリック](../../csharp/language-reference/keywords/public.md)、[保護](../../csharp/language-reference/keywords/protected.md)、または`protected internal`共通言語仕様 (CLS) に準拠するために識別子。 CLS 準拠について詳しくは、「[CLS 準拠コードの記述](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)」および「[言語への非依存性、および言語非依存コンポーネント](https://msdn.microsoft.com/library/12a7a7h3)」をご覧ください。
