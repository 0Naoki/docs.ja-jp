# <a name="compiler-error-cs0631"></a>コンパイラ エラー CS0631
ref および out はこのコンテキストでは有効ではありません。  
  
 宣言、[インデクサー](../../csharp/programming-guide/indexers/index.md)の使用を含めることはできません[ref](../../csharp/language-reference/keywords/ref.md)または[out](../../csharp/language-reference/keywords/out-parameter-modifier.md)パラメーター。  
  
## <a name="example"></a>例  
 次の例では CS0631 が生成されます。  
  
```csharp  
// CS0631.cs  
public class MyClass  
{  
    public int this[ref int i]   // CS0631  
    // try the following line instead  
    // public int this[int i]  
    {  
        get  
        {  
            return 0;  
        }  
    }  
}  
  
public class MyClass2  
{  
    public static void Main()  
    {  
    }  
}  
```
