# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="0df0e-102">コンパイラの警告 (レベル 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="0df0e-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="0df0e-103">メンバー 'member1' が古い形式のメンバー 'member2' をオーバーライドします。</span><span class="sxs-lookup"><span data-stu-id="0df0e-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="0df0e-104">Obsolete 属性を 'member1' に追加してください</span><span class="sxs-lookup"><span data-stu-id="0df0e-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="0df0e-105">コンパイラが `override` としてマークされたメソッドへの `obsolete`を検出しました。</span><span class="sxs-lookup"><span data-stu-id="0df0e-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="0df0e-106">ただし、オーバーライド元のメソッド自体は obsolete とマークされていませんでした。</span><span class="sxs-lookup"><span data-stu-id="0df0e-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="0df0e-107">オーバーライド元のメソッドを呼び出すと、引き続き [CS0612](./cs0612.md)が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0df0e-107">The overriding method will still generate [CS0612](./cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="0df0e-108">メソッド宣言を確認し、メソッド (およびそのすべてのオーバーライド) を `obsolete`とマークするかどうかを明示的に指定します。</span><span class="sxs-lookup"><span data-stu-id="0df0e-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="0df0e-109">次の例では CS0672 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0df0e-109">The following sample generates CS0672:</span></span>  
  
```csharp  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
