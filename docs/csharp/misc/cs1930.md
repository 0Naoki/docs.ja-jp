## <a name="example"></a><span data-ttu-id="f984b-108">例</span><span class="sxs-lookup"><span data-stu-id="f984b-108">Example</span></span>  
 <span data-ttu-id="f984b-109">次の例では、識別子 `num` が、 `from` 句の範囲変数および `let` 句によって導入された範囲変数に使用されるため、CS1930 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f984b-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```csharp  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
