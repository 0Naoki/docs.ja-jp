# <a name="compiler-error-cs0308"></a><span data-ttu-id="788b0-102">コンパイラ エラー CS0308</span><span class="sxs-lookup"><span data-stu-id="788b0-102">Compiler Error CS0308</span></span>
<span data-ttu-id="788b0-103">型引数を持つ、非ジェネリック型または非ジェネリック メソッド 'identifier' を使用できません。</span><span class="sxs-lookup"><span data-stu-id="788b0-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="788b0-104">メソッドまたは型はジェネリックではありませんが、型引数とともに使用されました。</span><span class="sxs-lookup"><span data-stu-id="788b0-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="788b0-105">このエラーを回避するには、山かっこと型引数を削除するか、メソッドまたは型をジェネリックのメソッドまたは型として再宣言します。</span><span class="sxs-lookup"><span data-stu-id="788b0-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="788b0-106">次の例では CS0308 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="788b0-106">The following example generates CS0308:</span></span>  
  
```  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="788b0-107">次の例でも CS0308 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="788b0-107">The following example also generates CS0308.</span></span> <span data-ttu-id="788b0-108">エラーを解決するには、ディレクティブ "using System.Collections.Generic" を使用します。</span><span class="sxs-lookup"><span data-stu-id="788b0-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
