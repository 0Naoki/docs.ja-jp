## <a name="example"></a><span data-ttu-id="e06d5-109">例</span><span class="sxs-lookup"><span data-stu-id="e06d5-109">Example</span></span>  
 <span data-ttu-id="e06d5-110">次の例では CS0755 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e06d5-110">The following example generates CS0755:</span></span>  
  
```csharp  
// cs0755.cs  
    public static partial class Ext  
    {  
        static partial void Part(this C c); //Extension method  
  
        // Typically the implementing declaration is in a separate file.  
        static partial void Part(C c) //CS0755  
        {  
        }  
    }

    {  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
