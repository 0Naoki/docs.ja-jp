# <a name="compiler-error-cs0143"></a><span data-ttu-id="119ed-102">コンパイラ エラー CS0143</span><span class="sxs-lookup"><span data-stu-id="119ed-102">Compiler Error CS0143</span></span>

 <span data-ttu-id="119ed-104">使用できる適切なコンストラクターがありません。</span><span class="sxs-lookup"><span data-stu-id="119ed-104">There is no appropriate constructor available.</span></span> <span data-ttu-id="119ed-105">これが該当するのは、値を代入するだけで初期化される、組み込みの数値型の場合です。</span><span class="sxs-lookup"><span data-stu-id="119ed-105">This is the case for built-in numeric value types, which are initialized simply by assigning a value to them.</span></span>  
  
 <span data-ttu-id="119ed-106">次の例では CS0143 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="119ed-106">The following sample generates CS0143:</span></span>  
  
```csharp  
// CS0143.cs

{  
   static public void Main ()  
   {  
      double d = new double(4.5);   // CS0143  
      // Try this line instead:  
      // double d = 4.5;  
   }  
}  
```
