# <a name="compiler-error-cs1623"></a><span data-ttu-id="7e0cb-102">コンパイラ エラー CS1623</span><span class="sxs-lookup"><span data-stu-id="7e0cb-102">Compiler Error CS1623</span></span>
<span data-ttu-id="7e0cb-103">反復子は、ref または out パラメーターを含めることはできません。</span><span class="sxs-lookup"><span data-stu-id="7e0cb-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="7e0cb-104">このエラーは、反復子メソッドが受け取る場合に発生します。、 `in`、 `ref`、または`out`パラメーター。</span><span class="sxs-lookup"><span data-stu-id="7e0cb-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="7e0cb-105">このエラーを回避するには、削除、 `in`、 `ref`、または`out`メソッド シグネチャからキーワード。</span><span class="sxs-lookup"><span data-stu-id="7e0cb-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7e0cb-106">例</span><span class="sxs-lookup"><span data-stu-id="7e0cb-106">Example</span></span>  
 <span data-ttu-id="7e0cb-107">次の例では CS1623 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7e0cb-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
