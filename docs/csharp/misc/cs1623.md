# <a name="compiler-error-cs1623"></a><span data-ttu-id="24c31-102">コンパイラ エラー CS1623</span><span class="sxs-lookup"><span data-stu-id="24c31-102">Compiler Error CS1623</span></span>
<span data-ttu-id="24c31-103">反復子は、ref または out パラメーターを持つことはできません。</span><span class="sxs-lookup"><span data-stu-id="24c31-103">Iterators cannot have in ref or out parameters</span></span>  
  
 <span data-ttu-id="24c31-104">反復子メソッドは、このエラーが発生、 `in`、 `ref`、または`out`パラメーター。</span><span class="sxs-lookup"><span data-stu-id="24c31-104">This error occurs if an iterator method takes an `in`, `ref`, or `out` parameter.</span></span> <span data-ttu-id="24c31-105">このエラーを回避するには、削除、 `in`、 `ref`、または`out`メソッド シグネチャからキーワード。</span><span class="sxs-lookup"><span data-stu-id="24c31-105">To avoid this error, remove the `in`, `ref`, or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="24c31-106">例</span><span class="sxs-lookup"><span data-stu-id="24c31-106">Example</span></span>  
 <span data-ttu-id="24c31-107">次の例では CS1623 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="24c31-107">The following sample generates CS1623:</span></span>  
  
```csharp  
// CS1623.cs  
using System.Collections;

class C : IEnumerable
{
    public IEnumerator GetEnumerator()
    {
        yield return 0;
    }

    // To resolve the error, remove in  
    public IEnumerator GetEnumerator(in short i)  // CS1623  
    {
        yield return i;
    }
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {
        yield return i;
    }

    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {
        f = 0.0F;
        yield return f;
    }
}
```
