# コンパイラ エラー CS0682
'type2' はこの言語でサポートされていないため、'type1' で実装できません  
  
 このエラーは、別の言語で記述されたインターフェイスを実装しようとし、そのインターフェイスがコンパイラでサポートされていない場合に発生します。
