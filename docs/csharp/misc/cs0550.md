# <a name="compiler-error-cs0550"></a><span data-ttu-id="1afa4-102">コンパイラ エラー CS0550</span><span class="sxs-lookup"><span data-stu-id="1afa4-102">Compiler Error CS0550</span></span>

 <span data-ttu-id="1afa4-104">派生クラスにおけるプロパティの実装に、基本インターフェイスで指定されていないアクセサーが含まれています。</span><span class="sxs-lookup"><span data-stu-id="1afa4-104">The implementation of a property in a derived class contains an accessor that was not specified in the base interface.</span></span>  
  
 <span data-ttu-id="1afa4-105">詳細については、「[プロパティの使用](../../csharp/programming-guide/classes-and-structs/using-properties.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="1afa4-105">For more information, see [Using Properties](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1afa4-106">例</span><span class="sxs-lookup"><span data-stu-id="1afa4-106">Example</span></span>  
 <span data-ttu-id="1afa4-107">次の例では CS0550 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1afa4-107">The following sample generates CS0550.</span></span>  
  
```csharp  
// CS0550.cs  
namespace x  
{  
   interface ii  
   {  
      int i  
      {  
         get;  
         // add the following accessor to resolve this CS0550  
         // set;  
      }  
   }  
  
   public class a : ii  
   {  
      int ii.i  
      {  
         get  
         {  
            return 0;  
         }  
         set {}   // CS0550  no set in interface  
      }  
  
      public static void Main() {}  
   }  
}  
```
