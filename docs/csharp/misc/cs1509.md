# <a name="compiler-error-cs1509"></a>コンパイラ エラー CS1509
参照したファイル 'file' はアセンブリではありません。/addmodule オプションを使用してください  
  
 [/Target: module](../language-reference/compiler-options/target-module-compiler-option.md) (アセンブリマニフェストを持たない) を使用したコンパイルで生成された出力ファイル (出力ファイル 1) が[参照](../language-reference/compiler-options/reference-compiler-option.md)に指定されました。 そのため、現在のプログラムのアセンブリには、アセンブリが追加されずに、出力ファイル 1 に含まれるメタデータ情報が追加されます。
