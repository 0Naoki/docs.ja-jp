## <a name="example"></a><span data-ttu-id="c590c-106">例</span><span class="sxs-lookup"><span data-stu-id="c590c-106">Example</span></span>  
 <span data-ttu-id="c590c-107">次の例では CS3002 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c590c-107">The following example generates CS3002:</span></span>  
  
```csharp  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
