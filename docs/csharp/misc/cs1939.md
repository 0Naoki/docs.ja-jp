# コンパイラ エラー CS1939
範囲変数 'name' は out または ref パラメーターとして渡すことはできません  
  
 範囲変数は、クエリ式で導入される読み取り専用の変数であり、ソース シーケンス内の連続する各要素の識別子として機能します。 この変数を変更することはできないため、これを `ref` または `out` によって渡しても意味はありません。 したがって、どちらの操作も無効です。  
  
### このエラーを解決するには  
  
1.  範囲変数を値で渡します。  
  
## 使用例  
 次の例では CS1939 が生成されます。  
  
```  
// cs1939.cs using System.Linq; class Test { public static void F(ref int i) { } public static void Main() { var list = new int[] { 0, 1, 2, 3, 4, 5 }; var q = from x in list let k = x select Test.F(ref x); // CS1939 } }  
```
