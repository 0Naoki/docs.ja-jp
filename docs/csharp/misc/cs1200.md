# コンパイラの警告 (レベル 1) CS1200
機能 'invalid feature' は推奨されません。 代わりに 'valid feature' を使用してください。  
  
 使用しようとしている機能が使用されなくなりました。 代わりに、有効な機能を使用するようにコードを更新します。
