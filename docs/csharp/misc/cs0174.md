# コンパイラ エラー CS0174
'base' 参照には基底クラスが必要です。  
  
 このエラーは、.NET Framework 共通言語ランタイムが、基底クラスを持たない唯一のクラスである、`System.Object` クラスのソース コードをコンパイルする場合にのみ発生します。
