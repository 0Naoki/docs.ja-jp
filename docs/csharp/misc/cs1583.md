# <a name="compiler-error-cs1583"></a>コンパイラ エラー CS1583
'file' は有効な Win32 のリソース ファイルではありません  
  
 このエラーは、無効なまたは正しく書式設定されていない Win32 リソース ファイルを、 [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) コンパイラ オプションのファイル名で指定したときに発生します。 Visual Studio では、ファイル名はプロジェクト デザイナーのアプリケーション ウィンドウで指定されます。
