# <a name="compiler-error-cs1583"></a>コンパイラ エラー CS1583
'file' は有効な Win32 のリソース ファイルではありません  
  
 このエラーを持つファイル名を指定するときに発生、 [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md)コンパイラ オプションを無効または正しく書式設定された Win32 リソース ファイルではありません。 Visual Studio では、ファイル名はプロジェクト デザイナーのアプリケーション ウィンドウで指定されます。
