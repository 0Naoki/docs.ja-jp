# <a name="compiler-error-cs0557"></a><span data-ttu-id="4de59-102">コンパイラ エラー CS0557</span><span class="sxs-lookup"><span data-stu-id="4de59-102">Compiler Error CS0557</span></span>
<span data-ttu-id="4de59-103">型 'class' でユーザー定義の変換が重複しています</span><span class="sxs-lookup"><span data-stu-id="4de59-103">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="4de59-104">重複する変換ルーチンをクラスで使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="4de59-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="4de59-105">次の例では CS0557 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4de59-105">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
