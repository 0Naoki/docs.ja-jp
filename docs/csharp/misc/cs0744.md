# <a name="compiler-error-cs0744"></a><span data-ttu-id="5dddd-102">コンパイラ エラー CS0744</span><span class="sxs-lookup"><span data-stu-id="5dddd-102">Compiler Error CS0744</span></span>
<span data-ttu-id="5dddd-103">コンテキスト キーワード 'equals' が必要です。</span><span class="sxs-lookup"><span data-stu-id="5dddd-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="5dddd-104">`join` 句のパターンは、この例のように、 `join`...`in`...`on`...`equals`です。</span><span class="sxs-lookup"><span data-stu-id="5dddd-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5dddd-105">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="5dddd-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="5dddd-106">`equals` キーワードを `join` 句に追加します。</span><span class="sxs-lookup"><span data-stu-id="5dddd-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5dddd-107">例</span><span class="sxs-lookup"><span data-stu-id="5dddd-107">Example</span></span>  
 <span data-ttu-id="5dddd-108">次のコードでは CS0744 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5dddd-108">The following code generates CS0744:</span></span>  
  
```  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
