# <a name="compiler-error-cs0012"></a><span data-ttu-id="eca1e-102">コンパイラ エラー CS0012</span><span class="sxs-lookup"><span data-stu-id="eca1e-102">Compiler Error CS0012</span></span>
<span data-ttu-id="eca1e-103">型 'type' は、参照されていないアセンブリに定義されています。</span><span class="sxs-lookup"><span data-stu-id="eca1e-103">The type 'type' is defined in an assembly that is not referenced.</span></span> <span data-ttu-id="eca1e-104">アセンブリ 'assembly' に参照を追加する必要があります。</span><span class="sxs-lookup"><span data-stu-id="eca1e-104">You must add a reference to assembly 'assembly'.</span></span>  
  
 <span data-ttu-id="eca1e-105">参照される型の定義が見つかりませんでした。</span><span class="sxs-lookup"><span data-stu-id="eca1e-105">The definition for a referenced type was not found.</span></span> <span data-ttu-id="eca1e-106">このエラーは、必要な .DLL ファイルがコンパイル時にインクルードされていない場合に発生することがあります。</span><span class="sxs-lookup"><span data-stu-id="eca1e-106">This could occur if a required .DLL file is not included in the compilation.</span></span> <span data-ttu-id="eca1e-107">詳細については、次を参照してください。[参照の追加 ダイアログ ボックス](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7)と[/reference (c# コンパイラ オプション)](../../csharp/language-reference/compiler-options/reference-compiler-option.md)です。</span><span class="sxs-lookup"><span data-stu-id="eca1e-107">For more information, see [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) and [/reference (C# Compiler Options)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
 <span data-ttu-id="eca1e-108">次のコンパイル シーケンスは CS0012 になります。</span><span class="sxs-lookup"><span data-stu-id="eca1e-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 <span data-ttu-id="eca1e-109">次のコードも同様の結果になります。</span><span class="sxs-lookup"><span data-stu-id="eca1e-109">Then:</span></span>  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="eca1e-110">次のコードも同様の結果になります。</span><span class="sxs-lookup"><span data-stu-id="eca1e-110">Then:</span></span>  
  
```csharp  
// cs0012c.cs  
// compile with: /reference:cs0012b.dll  
class C  
{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 <span data-ttu-id="eca1e-111">コンパイルすると、この CS0012 を解決する可能性があります`/reference:cs0012b.dll;cs0012a.dll`、または Visual Studio を使用して、[参照の追加 ダイアログ ボックス](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7)cs0012b.dll に加えて cs0012a.dll への参照を追加します。</span><span class="sxs-lookup"><span data-stu-id="eca1e-111">You could resolve this CS0012 by compiling with `/reference:cs0012b.dll;cs0012a.dll`, or in Visual Studio by using the [Add Reference Dialog Box](http://msdn.microsoft.com/library/2feb0fe2-0805-4cc9-8cba-b0315849dfb7) to add a reference to cs0012a.dll in addition to cs0012b.dll.</span></span>
