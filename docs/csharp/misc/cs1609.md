## <a name="example"></a><span data-ttu-id="46cc3-106">例</span><span class="sxs-lookup"><span data-stu-id="46cc3-106">Example</span></span>  
 <span data-ttu-id="46cc3-107">次の例では CS1609 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="46cc3-107">The following sample generates CS1609.</span></span>  
  
```csharp  
// CS1609.cs

delegate int Del();  
class A  
{  
   public event Del MyEvent   
   {  
      private add {}   // CS1609  
      // try the following line instead  
      // add {}  
      remove {}  
   }  
}  
```
