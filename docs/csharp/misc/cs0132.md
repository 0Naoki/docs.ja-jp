# <a name="compiler-error-cs0132"></a><span data-ttu-id="cb000-102">コンパイラ エラー CS0132</span><span class="sxs-lookup"><span data-stu-id="cb000-102">Compiler Error CS0132</span></span>
<span data-ttu-id="cb000-103">'constructor' : 静的コンストラクターにパラメーターを指定することはできません</span><span class="sxs-lookup"><span data-stu-id="cb000-103">'constructor' : a static constructor must be parameterless</span></span>  
  
 <span data-ttu-id="cb000-104">パラメーターを指定して、 [静的](../../csharp/language-reference/keywords/static.md) コンストラクターを宣言することはできません。</span><span class="sxs-lookup"><span data-stu-id="cb000-104">A [static](../../csharp/language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="cb000-105">詳細については、「[コンストラクター](../../csharp/programming-guide/classes-and-structs/constructors.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="cb000-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="cb000-106">次の例では CS0132 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cb000-106">The following sample generates CS0132:</span></span>  
  
```  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
