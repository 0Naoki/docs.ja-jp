# <a name="compiler-error-cs0065"></a><span data-ttu-id="7d912-102">コンパイラ エラー CS0065</span><span class="sxs-lookup"><span data-stu-id="7d912-102">Compiler Error CS0065</span></span>
<span data-ttu-id="7d912-103">'event': イベント プロパティには、add および remove アクセサーの両方を指定する必要があります</span><span class="sxs-lookup"><span data-stu-id="7d912-103">'event' : event property must have both add and remove accessors</span></span>  
  
 <span data-ttu-id="7d912-104">フィールドではないイベントには、両方のアクセス方法が必要です。</span><span class="sxs-lookup"><span data-stu-id="7d912-104">An event that is not a field must have both access methods.</span></span>  
  
 <span data-ttu-id="7d912-105">次の例では CS0065 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7d912-105">The following sample generates CS0065:</span></span>  
  
```csharp  
// CS0065.cs  
using System;  
public delegate void Eventhandler(object sender, int e);  
public class MyClass  
{  
   public event EventHandler Click   // CS0065,  
   {  
      // to fix, uncomment the add and remove definitions  
      /*  
      add  
      {  
         Click += value;  
      }  
      remove  
      {  
         Click -= value;  
      }  
      */  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
