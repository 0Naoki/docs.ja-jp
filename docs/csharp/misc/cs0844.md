## <a name="example"></a><span data-ttu-id="122fc-111">例</span><span class="sxs-lookup"><span data-stu-id="122fc-111">Example</span></span>  
 <span data-ttu-id="122fc-112">次のコードでは CS0844 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="122fc-112">The following code generates CS0844:</span></span>  
  
```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            num = 5; // CS0844  
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

## <a name="correct-the-error-by-using-thisnum-to-refer-to-the-class-field"></a><span data-ttu-id="122fc-113">使用して、エラーを修正`this.num`クラス フィールドを参照するには</span><span class="sxs-lookup"><span data-stu-id="122fc-113">Correct the error by using `this.num` to refer to the class field</span></span>

```csharp  
class Test  
    {  
        int num;  
        public void TestMethod()  
        {  
            this.num = 5; // Error fixed.
            int num = 6;
        }  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```

        {

        }

    }
