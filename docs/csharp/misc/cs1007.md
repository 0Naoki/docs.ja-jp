## <a name="example"></a><span data-ttu-id="c0fd2-106">例</span><span class="sxs-lookup"><span data-stu-id="c0fd2-106">Example</span></span>  
 <span data-ttu-id="c0fd2-107">次の例では CS1007 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c0fd2-107">The following sample generates CS1007:</span></span>

```csharp  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
