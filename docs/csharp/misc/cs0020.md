# コンパイラ エラー CS0020
定数 0 による除算です。  
  
 式では、除算の分母にリテラル \(変数ではなく\) 値 0 を使用しています。 0 による除算は未定義であるため、有効ではありません。  
  
 次の例では CS0020 が生成されます。  
  
```  
// CS0020.cs namespace x { public class b { public static void Main() { 1 / 0;   // CS0020 } } }  
```
