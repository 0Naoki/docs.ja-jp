# <a name="compiler-error-cs0637"></a><span data-ttu-id="ab29a-102">コンパイラ エラー CS0637</span><span class="sxs-lookup"><span data-stu-id="ab29a-102">Compiler Error CS0637</span></span>

<span data-ttu-id="ab29a-103">FieldOffset 属性は、static または const フィールドでは許可されません。</span><span class="sxs-lookup"><span data-stu-id="ab29a-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="ab29a-104">[FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 属性は、 [static](../../csharp/language-reference/keywords/static.md) または [const](../../csharp/language-reference/keywords/const.md)でマークされたフィールドで使用できません。</span><span class="sxs-lookup"><span data-stu-id="ab29a-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="ab29a-105">次の例では CS0637 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ab29a-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
