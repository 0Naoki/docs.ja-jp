# コンパイラの警告 (レベル 1) CS0602
機能 'old\_feature' は使用できません。 'new\_feature' を使用してください。  
  
 コードで使用される言語機能 \(*old\_feature*\) は引き続きサポートされていますが、将来のリリースでサポートが削除される可能性があります。 代わりに、構文 \(*new\_feature*\) を使用することをお勧めします。
