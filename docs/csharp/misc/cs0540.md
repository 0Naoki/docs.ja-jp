# <a name="compiler-error-cs0540"></a><span data-ttu-id="30a9b-102">コンパイラ エラー CS0540</span><span class="sxs-lookup"><span data-stu-id="30a9b-102">Compiler Error CS0540</span></span>
<span data-ttu-id="30a9b-103">'interface member' : 含む型は、インターフェイス 'interface' を実装しません</span><span class="sxs-lookup"><span data-stu-id="30a9b-103">'interface member' : containing type does not implement interface 'interface'</span></span>  
  
 <span data-ttu-id="30a9b-104">インターフェイス メンバーを、 [インターフェイス](../language-reference/keywords/class.md) から派生していない [クラス](../language-reference/keywords/interface.md)に実装しようとしました。</span><span class="sxs-lookup"><span data-stu-id="30a9b-104">You attempted to implement an interface member in a [class](../language-reference/keywords/class.md) that does not derive from the [interface](../language-reference/keywords/interface.md).</span></span> <span data-ttu-id="30a9b-105">インターフェイス メンバーの実装を削除するか、インターフェイスをクラスの基底クラス リストに追加する必要があります。</span><span class="sxs-lookup"><span data-stu-id="30a9b-105">You should either delete the implementation of the interface member or add the interface to the base-class list of the class.</span></span>  
  
## <a name="example"></a><span data-ttu-id="30a9b-106">例</span><span class="sxs-lookup"><span data-stu-id="30a9b-106">Example</span></span>  
 <span data-ttu-id="30a9b-107">次の例では CS0540 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="30a9b-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="30a9b-108">例</span><span class="sxs-lookup"><span data-stu-id="30a9b-108">Example</span></span>
