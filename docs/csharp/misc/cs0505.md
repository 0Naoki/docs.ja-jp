# <a name="compiler-error-cs0505"></a><span data-ttu-id="fd460-102">コンパイラ エラー CS0505</span><span class="sxs-lookup"><span data-stu-id="fd460-102">Compiler Error CS0505</span></span>
<span data-ttu-id="fd460-103">'member1': 'member2' は関数ではないためオーバーライドできません</span><span class="sxs-lookup"><span data-stu-id="fd460-103">'member1': cannot override because 'member2' is not a function</span></span>  
  
 <span data-ttu-id="fd460-104">クラス宣言で、基底クラスのメソッド以外のものをオーバーライドしようとしました。</span><span class="sxs-lookup"><span data-stu-id="fd460-104">A class declaration attempted to override a non-method in a base class.</span></span> <span data-ttu-id="fd460-105">オーバーライドでは、メンバーの型が一致しなければなりません。</span><span class="sxs-lookup"><span data-stu-id="fd460-105">Overrides must match the member type.</span></span> <span data-ttu-id="fd460-106">基底クラスのメソッドと同じ名前のメソッドが必要な場合、基底クラスのメソッド宣言で [new](../../csharp/language-reference/keywords/new-modifier.md) をご使用ください ( [override](../../csharp/language-reference/keywords/override.md)は使用しない)。</span><span class="sxs-lookup"><span data-stu-id="fd460-106">If a method with the same name as a method in a base class is desired, use [new](../../csharp/language-reference/keywords/new-modifier.md) (and not [override](../../csharp/language-reference/keywords/override.md)) on the method declaration in the base class.</span></span>  
  
 <span data-ttu-id="fd460-107">次の例では CS0505 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="fd460-107">The following sample generates CS0505:</span></span>  
  
```csharp  
// CS0505.cs  
// compile with: /target:library  
public class clx  
{  
   public int i;  
}  
  
public class cly : clx  
{  
   public override int i() { return 0; }   // CS0505  
}  
```
