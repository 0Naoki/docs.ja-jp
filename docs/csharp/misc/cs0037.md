# <a name="compiler-error-cs0037"></a>コンパイラ エラー CS0037
null 非許容の値型であるため、null を 'type' に変換できません  
  
 コンパイラは値型に null を割り当てることはできません。null は [参照型](../../csharp/language-reference/keywords/reference-types.md) または null 許容型にのみ割り当てることができます。 [構造体](../../csharp/language-reference/keywords/struct.md) は値型です。 詳細については、「[ull 許容型](../../csharp/programming-guide/nullable-types/index.md)」を参照してください。  
  
 次の例では CS0037 が生成されます。  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
