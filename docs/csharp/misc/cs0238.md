# <a name="compiler-error-cs0238"></a><span data-ttu-id="7320d-102">コンパイラ エラー CS0238</span><span class="sxs-lookup"><span data-stu-id="7320d-102">Compiler Error CS0238</span></span>
<span data-ttu-id="7320d-103">override ではないため、'member' をシールすることはできません。</span><span class="sxs-lookup"><span data-stu-id="7320d-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="7320d-104">[override](../../csharp/language-reference/keywords/sealed.md) でもマークされていないメンバーで [sealed](../../csharp/language-reference/keywords/override.md)が使用されました。</span><span class="sxs-lookup"><span data-stu-id="7320d-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="7320d-105">詳細については、「[継承](../../csharp/programming-guide/classes-and-structs/inheritance.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="7320d-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="7320d-106">次の例では CS0238 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7320d-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
