# <a name="compiler-error-cs1601"></a><span data-ttu-id="9e325-102">コンパイラ エラー CS1601</span><span class="sxs-lookup"><span data-stu-id="9e325-102">Compiler Error CS1601</span></span>
<span data-ttu-id="9e325-103">メソッドまたはデリゲート パラメーターを型 'type' にすることはできません</span><span class="sxs-lookup"><span data-stu-id="9e325-103">Method or delegate parameter cannot be of type 'type'</span></span>  
  
 <span data-ttu-id="9e325-104">.NET Framework クラス ライブラリの一部の型 ( <xref:System.TypedReference>、 <xref:System.RuntimeArgumentHandle> 、 <xref:System.ArgIterator> など) は、安全でない操作の実行に使用される可能性があるため、 [ref](../../csharp/language-reference/keywords/ref.md) パラメーターや [out](../../csharp/language-reference/keywords/out.md) パラメーターとしては使用できません。</span><span class="sxs-lookup"><span data-stu-id="9e325-104">Some types in the .NET Framework class library, for example, <xref:System.TypedReference>, <xref:System.RuntimeArgumentHandle> and <xref:System.ArgIterator> cannot be used as [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) parameters because they could potentially be used to perform unsafe operations.</span></span>  
  
 <span data-ttu-id="9e325-105">次の例では CS1601 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="9e325-105">The following sample generates CS1601:</span></span>  
  
```  
// CS1601.cs  
using System;  
  
class MyClass  
{  
   public void Test1 (ref TypedReference t)   // CS1601  
   {  
   }  
  
   public void Test2 (out ArgIterator t)   // CS1601  
   {  
   }  
}  
```
