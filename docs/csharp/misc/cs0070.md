# <a name="compiler-error-cs0070"></a><span data-ttu-id="5a068-102">コンパイラ エラー CS0070</span><span class="sxs-lookup"><span data-stu-id="5a068-102">Compiler Error CS0070</span></span>
<span data-ttu-id="5a068-103">イベント 'event' は +=、-= の左辺にのみ使用できます。ただし、'type' 型内から使用されている場合を除きます</span><span class="sxs-lookup"><span data-stu-id="5a068-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="5a068-104">定義されているクラスの外部では、 [event](../../csharp/language-reference/keywords/event.md) は参照の追加または削除のみできます。</span><span class="sxs-lookup"><span data-stu-id="5a068-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="5a068-105">詳細については、「[イベント](../../csharp/programming-guide/events/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="5a068-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="5a068-106">次の例では CS0070 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="5a068-106">The following sample generates CS0070:</span></span>  
  
```csharp  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
