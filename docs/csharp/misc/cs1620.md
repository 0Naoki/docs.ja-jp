# <a name="compiler-error-cs1620"></a><span data-ttu-id="86aaa-102">コンパイラ エラー CS1620</span><span class="sxs-lookup"><span data-stu-id="86aaa-102">Compiler Error CS1620</span></span>
<span data-ttu-id="86aaa-103">引数 'number' は 'keyword' キーワードと共に渡されなければなりません</span><span class="sxs-lookup"><span data-stu-id="86aaa-103">Argument 'number' must be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="86aaa-104">受け取る関数に引数を渡している場合、このエラーが発生した、 [ref](../../csharp/language-reference/keywords/ref.md)または[アウト](../../csharp/language-reference/keywords/out-parameter-modifier.md)パラメーターを指定しなかった、`ref`または`out`呼び出し、またはする時点でキーワードを含める正しくないキーワードです。</span><span class="sxs-lookup"><span data-stu-id="86aaa-104">This error occurs if you are passing an argument to a function that takes a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter and you don't include the `ref` or `out` keyword at the point of call, or you include the wrong keyword.</span></span> <span data-ttu-id="86aaa-105">エラー テキストには、使用するべき適切なキーワードと、エラーを引き起こした引数が示されています。</span><span class="sxs-lookup"><span data-stu-id="86aaa-105">The error text indicates the appropriate keyword to use and which argument caused the failure.</span></span>  
  
 <span data-ttu-id="86aaa-106">次の例では CS1620 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="86aaa-106">The following sample generates CS1620:</span></span>  
  
```csharp  
// CS1620.cs  
class C  
{  
    void f(ref int i) {}  
    public static void Main()  
    {  
        int x = 1;  
        f(out x);  // CS1620 – f takes a ref parameter, not an out parameter  
        // Try this line instead:  
        // f(ref x);  
    }  
}  
```
