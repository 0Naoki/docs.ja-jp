## <a name="example"></a><span data-ttu-id="e645f-107">例</span><span class="sxs-lookup"><span data-stu-id="e645f-107">Example</span></span>

```csharp  
// CS0514.cs

    {  
    }  
  
    public A(object o)  
    {  
    }

{  
    static B() : this(null) // CS0514  
    {  
    }  
  
    public B(object o)  
    {  
    }  
}  
```
