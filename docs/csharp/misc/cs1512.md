# <a name="compiler-error-cs1512"></a><span data-ttu-id="c02c8-102">コンパイラ エラー CS1512</span><span class="sxs-lookup"><span data-stu-id="c02c8-102">Compiler Error CS1512</span></span>
<span data-ttu-id="c02c8-103">キーワード 'base' は現在のコンテキストでは使用できません。</span><span class="sxs-lookup"><span data-stu-id="c02c8-103">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="c02c8-104">メソッド、プロパティ、またはコンストラクターの外部で、 [base](../language-reference/keywords/base.md) キーワードが使用されました。</span><span class="sxs-lookup"><span data-stu-id="c02c8-104">The [base](../language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="c02c8-105">次の例では、CS1512 エラーが生成されます。</span><span class="sxs-lookup"><span data-stu-id="c02c8-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
