## <a name="example"></a><span data-ttu-id="b5602-108">例</span><span class="sxs-lookup"><span data-stu-id="b5602-108">Example</span></span>  
 <span data-ttu-id="b5602-109">次の例では、CS3017 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b5602-109">The following example generates CS3017:</span></span>  
  
```

// compile with: /target:module

[module: CLSCompliant(true)]  
[assembly: CLSCompliant(false)]  // CS3017  
// Try this line instead:  
// [assembly: CLSCompliant(true)]

{

}  
```
