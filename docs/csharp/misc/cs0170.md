 <span data-ttu-id="5fb43-104">構造内のフィールドが、最初に初期化しないで使用されています。</span><span class="sxs-lookup"><span data-stu-id="5fb43-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="5fb43-105">この問題を解決するには、初期化されていないフィールドをまず確認し、そのフィールドにアクセスする前に初期化します。</span><span class="sxs-lookup"><span data-stu-id="5fb43-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="5fb43-106">構造体の初期化の詳細については、次を参照してください。[構造体](../../csharp/programming-guide/classes-and-structs/structs.md)と[を使用して構造体](../../csharp/programming-guide/classes-and-structs/using-structs.md)です。</span><span class="sxs-lookup"><span data-stu-id="5fb43-106">For more information about initializing structs, see [Structs](../../csharp/programming-guide/classes-and-structs/structs.md) and [Using Structs](../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="5fb43-107">次の例では CS0170 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5fb43-107">The following sample generates CS0170:</span></span>  
  
```  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
