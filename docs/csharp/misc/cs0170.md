# <a name="compiler-error-cs0170"></a><span data-ttu-id="35255-102">コンパイラ エラー CS0170</span><span class="sxs-lookup"><span data-stu-id="35255-102">Compiler Error CS0170</span></span>
<span data-ttu-id="35255-103">フィールド 'field' は、割り当てられていない可能性があります</span><span class="sxs-lookup"><span data-stu-id="35255-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="35255-104">構造内のフィールドが、最初に初期化しないで使用されています。</span><span class="sxs-lookup"><span data-stu-id="35255-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="35255-105">この問題を解決するには、初期化されていないフィールドをまず確認し、そのフィールドにアクセスする前に初期化します。</span><span class="sxs-lookup"><span data-stu-id="35255-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="35255-106">構造体の初期化の詳細につい[ては、「構造](../programming-guide/classes-and-structs/structs.md)体と[構造体の使用](../programming-guide/classes-and-structs/using-structs.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="35255-106">For more information about initializing structs, see [Structs](../programming-guide/classes-and-structs/structs.md) and [Using Structs](../programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="35255-107">次の例では CS0170 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="35255-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
