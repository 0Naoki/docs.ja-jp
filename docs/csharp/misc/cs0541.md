 <span data-ttu-id="3950d-104">明示的な [interface](../../csharp/language-reference/keywords/interface.md) 宣言が、 [クラス](../../csharp/language-reference/keywords/class.md) または [構造体](../../csharp/language-reference/keywords/struct.md)の外側で見つかりました。</span><span class="sxs-lookup"><span data-stu-id="3950d-104">An explicit [interface](../../csharp/language-reference/keywords/interface.md) declaration was found outside a [class](../../csharp/language-reference/keywords/class.md) or [struct](../../csharp/language-reference/keywords/struct.md).</span></span>  
  
 <span data-ttu-id="3950d-105">次の例では CS0541 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3950d-105">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
