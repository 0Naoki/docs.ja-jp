# <a name="compiler-error-cs0541"></a><span data-ttu-id="00931-102">コンパイラ エラー CS0541</span><span class="sxs-lookup"><span data-stu-id="00931-102">Compiler Error CS0541</span></span>
<span data-ttu-id="00931-103">'declaration' : 明示的インターフェイスはクラス、または構造体の中でのみ宣言できます。</span><span class="sxs-lookup"><span data-stu-id="00931-103">'declaration' : explicit interface declaration can only be declared in a class or struct</span></span>  
  
 <span data-ttu-id="00931-104">明示的な [interface](../language-reference/keywords/interface.md) 宣言が、 [クラス](../language-reference/keywords/class.md) または [構造体](../language-reference/keywords/struct.md)の外側で見つかりました。</span><span class="sxs-lookup"><span data-stu-id="00931-104">An explicit [interface](../language-reference/keywords/interface.md) declaration was found outside a [class](../language-reference/keywords/class.md) or [struct](../language-reference/keywords/struct.md).</span></span>  
  
 <span data-ttu-id="00931-105">次の例では CS0541 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="00931-105">The following sample generates CS0541:</span></span>  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
