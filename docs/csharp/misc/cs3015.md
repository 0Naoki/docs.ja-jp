## <a name="example"></a><span data-ttu-id="cbcec-106">例</span><span class="sxs-lookup"><span data-stu-id="cbcec-106">Example</span></span>  
 <span data-ttu-id="cbcec-107">次の例では C3015 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cbcec-107">The following sample generates C3015.</span></span>  
  
```csharp  
// CS3015.cs  
// compile with: /target:library  
using System;  
  
[assembly:CLSCompliant(true)]  
public class MyAttribute : Attribute  
{  
   public MyAttribute(int[] ai) {}   // CS3015  
   // try the following line instead  
   // public MyAttribute(int ai) {}  
}  
```
