# <a name="compiler-error-cs0611"></a><span data-ttu-id="94f39-102">コンパイラ エラー CS0611</span><span class="sxs-lookup"><span data-stu-id="94f39-102">Compiler Error CS0611</span></span>

 <span data-ttu-id="94f39-104">配列の型として使用できない型がいくつかあります。</span><span class="sxs-lookup"><span data-stu-id="94f39-104">There are some types that cannot be used as the type of an array.</span></span> <span data-ttu-id="94f39-105">これらの型には、 **System.TypedReference** 、 **System.ArgIterator**などがあります。</span><span class="sxs-lookup"><span data-stu-id="94f39-105">These types include **System.TypedReference** and **System.ArgIterator**.</span></span>  
  
 <span data-ttu-id="94f39-106">次の例では、配列要素として **System.TypedReference** を使用した結果、CS0611 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="94f39-106">The following sample generates CS0611 as a result of using **System.TypedReference** as an array element:</span></span>  
  
```csharp

public class a  
{  
   public static void Main()  
   {  
      System.TypedReference[] ao = new System.TypedReference [10];   // CS0611  
      // try the following line instead  
      // int[] ao = new int[10];  
   }  
}  
```
