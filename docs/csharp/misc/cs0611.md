 <span data-ttu-id="5410f-104">配列の型として使用できない型がいくつかあります。</span><span class="sxs-lookup"><span data-stu-id="5410f-104">There are some types that cannot be used as the type of an array.</span></span> <span data-ttu-id="5410f-105">これらの型には、 **System.TypedReference** 、 **System.ArgIterator**などがあります。</span><span class="sxs-lookup"><span data-stu-id="5410f-105">These types include **System.TypedReference** and **System.ArgIterator**.</span></span>

```

{  
   public static void Main()  
   {  
      System.TypedReference[] ao = new System.TypedReference [10];   // CS0611  
      // try the following line instead  
      // int[] ao = new int[10];  
   }  
}  
```
