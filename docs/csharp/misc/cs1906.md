# <a name="compiler-error-cs1906"></a>コンパイラ エラー CS1906
無効なオプション 'option' です。リソースの参照可能範囲は 'public' または 'private' でなければなりません  
  
 このエラーは、無効な[/resource (出力するためのリソースファイルの埋め込み)](../language-reference/compiler-options/resource-compiler-option.md)または[-Linkresource (.NET Framework リソースへのリンク)](../language-reference/compiler-options/linkresource-compiler-option.md)コマンドラインオプションが無効であることを示しています。 **/Resource**または **/linkresource**コマンドラインオプションの構文を確認し、使用されているアクセシビリティ修飾子が**public**または `private` であることを確認してください。
