# <a name="compiler-error-cs0568"></a><span data-ttu-id="211da-102">コンパイラ エラー CS0568</span><span class="sxs-lookup"><span data-stu-id="211da-102">Compiler Error CS0568</span></span>
<span data-ttu-id="211da-103">構造体に明示的なパラメーターのないコンストラクターを含めることはできません。</span><span class="sxs-lookup"><span data-stu-id="211da-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="211da-104">各[構造体](../../csharp/language-reference/keywords/struct.md)既にオブジェクトをゼロに初期化するパラメーターなしのコンス トラクターがあります。</span><span class="sxs-lookup"><span data-stu-id="211da-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="211da-105">そのため、構造体に対して作成するコンストラクターは、1 つ以上のパラメーターを受け取る必要があります。</span><span class="sxs-lookup"><span data-stu-id="211da-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="211da-106">次の例では CS0568 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="211da-106">The following sample generates CS0568:</span></span>  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
