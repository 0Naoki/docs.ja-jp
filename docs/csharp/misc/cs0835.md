## <a name="example"></a><span data-ttu-id="f3264-108">例</span><span class="sxs-lookup"><span data-stu-id="f3264-108">Example</span></span>  
 <span data-ttu-id="f3264-109">次の例では CS0835 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="f3264-109">The following example generates CS0835:</span></span>  
  
```csharp  
// cs0835.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
public class C  
{  
    public static int Main()  
    {  
        Expression<int> e = x => x + 1; // CS0835  
  
        // Try the following line instead.  
       // Expression<Func<int,int>> e2 = x => x + 1;  
  
        return 1;  
    }  
}  
```
