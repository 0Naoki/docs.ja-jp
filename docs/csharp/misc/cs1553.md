# <a name="compiler-error-cs1553"></a>コンパイラ エラー CS1553
宣言が無効です。使用して、' modifier 演算子\<dest 型 > (...'その代わりに  
  
 戻り値の型を[変換演算子](../../csharp/language-reference/keywords/implicit.md)、パラメーター リストの直前にする必要がありますと*修飾子*か`implicit`または**明示的な**します。  
  
 次の例では CS1553 が生成されます。  
  
```csharp  
// CS1553.cs  
class MyClass  
{  
   public static int implicit operator (MyClass f)   // CS1553  
   // try the following line instead  
   // public static implicit operator int (MyClass f)  
   {  
      return 6;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
