# <a name="compiler-warning-level-3-cs0168"></a><span data-ttu-id="cace3-102">コンパイラの警告 (レベル 3) CS0168</span><span class="sxs-lookup"><span data-stu-id="cace3-102">Compiler Warning (level 3) CS0168</span></span>
<span data-ttu-id="cace3-103">変数 'var' が宣言されているが使用されていません</span><span class="sxs-lookup"><span data-stu-id="cace3-103">The variable 'var' is declared but never used</span></span>

<span data-ttu-id="cace3-104">コンパイラは、変数を宣言するときは使用しない場合に、レベル&3; の警告を発行します。</span><span class="sxs-lookup"><span data-stu-id="cace3-104">The compiler issues a level-three warning when you declare a variable, but do not use it.</span></span>
  
<span data-ttu-id="cace3-105">次の例では、CS0168 警告が生成されます。</span><span class="sxs-lookup"><span data-stu-id="cace3-105">The following sample generates a CS0168 warning:</span></span>  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
