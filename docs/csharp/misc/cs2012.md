# <a name="compiler-error-cs2012"></a>コンパイラ エラー CS2012
'file' を書き込むために開くことができません  
  
 [/bugreport:](../../csharp/language-reference/compiler-options/bugreport-compiler-option.md)`file` コンパイラ オプションを使用しているときに、ファイルを書き込み用に開くことができませんでした。 有効なファイル名を指定していること、およびファイルが読み取り専用でないことを確認してください。
