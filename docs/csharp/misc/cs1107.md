## <a name="example"></a><span data-ttu-id="afb9c-105">例</span><span class="sxs-lookup"><span data-stu-id="afb9c-105">Example</span></span>  
 <span data-ttu-id="afb9c-106">次の例では CS1107 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="afb9c-106">The following example generates CS1107:</span></span>  
  
```csharp  
// cs1107.cs  
public static class Test
{
    // Extension methods.  
    public static void TestMethod(this this t) { } // CS1107  

}

public class TestTwo
{ 
    // Regular Instance Method  
    public void TestMethod(ref ref int i) { } // CS1107  

    // Regular Instance Method  
    public void TestMethod(in in double d) { } // CS1107  
}
```
