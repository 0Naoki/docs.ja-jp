# <a name="compiler-error-cs0313"></a><span data-ttu-id="77d89-102">コンパイラ エラー CS0313</span><span class="sxs-lookup"><span data-stu-id="77d89-102">Compiler Error CS0313</span></span>
<span data-ttu-id="77d89-103">型 'type1' をジェネリック型またはメソッド 'type2' で型パラメーター 'parameter name' として使用できません。</span><span class="sxs-lookup"><span data-stu-id="77d89-103">The type 'type1' cannot be used as type parameter 'parameter name' in the generic type or method 'type2'.</span></span> <span data-ttu-id="77d89-104">Null 許容型 'type1' が 'type2' の制約を満たしていません。</span><span class="sxs-lookup"><span data-stu-id="77d89-104">The nullable type 'type1' does not satisfy the constraint of 'type2'.</span></span> <span data-ttu-id="77d89-105">Null 許容型では、インターフェイス制約を満たすことはできません。</span><span class="sxs-lookup"><span data-stu-id="77d89-105">Nullable types cannot satisfy any interface constraints.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="77d89-108">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="77d89-108">To correct this error</span></span>  
  
1.  <span data-ttu-id="77d89-109">次の例のコードを使用する場合、1 つの解決方法は、 `ImplStruct` の呼び出しの最初の型引数として通常の `TestMethod`を指定することです。</span><span class="sxs-lookup"><span data-stu-id="77d89-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="77d89-110">次に、 `TestMethod` を変更して、その return ステートメントで `Implstruct` の Null 許容バージョンを作成します。</span><span class="sxs-lookup"><span data-stu-id="77d89-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>  
  
    ```  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="77d89-111">例</span><span class="sxs-lookup"><span data-stu-id="77d89-111">Example</span></span>  
 <span data-ttu-id="77d89-112">次のコードでは CS0313 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="77d89-112">The following code generates CS0313:</span></span>  
  
```  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
