# <a name="compiler-error-cs0313"></a><span data-ttu-id="4ea6e-102">コンパイラ エラー CS0313</span><span class="sxs-lookup"><span data-stu-id="4ea6e-102">Compiler Error CS0313</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="4ea6e-108">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="4ea6e-108">To correct this error</span></span>  
  
1. <span data-ttu-id="4ea6e-109">次の例のコードを使用する場合、1 つの解決方法は、 `ImplStruct` の呼び出しの最初の型引数として通常の `TestMethod`を指定することです。</span><span class="sxs-lookup"><span data-stu-id="4ea6e-109">Using the code that follows as an example, one solution is to specify an ordinary `ImplStruct` as the first type argument in the call to `TestMethod`.</span></span> <span data-ttu-id="4ea6e-110">次に、 `TestMethod` を変更して、その return ステートメントで `Implstruct` の Null 許容バージョンを作成します。</span><span class="sxs-lookup"><span data-stu-id="4ea6e-110">Then modify `TestMethod` to create a nullable version of `Implstruct` in its return statement:</span></span>  
  
    ```csharp  
    return new Nullable<T>(t);  
    ```  
  
## <a name="example"></a><span data-ttu-id="4ea6e-111">例</span><span class="sxs-lookup"><span data-stu-id="4ea6e-111">Example</span></span>  
 <span data-ttu-id="4ea6e-112">次のコードでは CS0313 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="4ea6e-112">The following code generates CS0313:</span></span>  
  
```csharp  
// cs0313.cs  
public interface BaseInterface { }  
public struct ImplStruct : BaseInterface { }  
  
public class TestClass  
{  
    public T? TestMethod<T, U>(T t) where T : struct, U  
    {  
        return t;  
    }  
}  
  
public class NullableTest  
{  
    public static void Run()  
    {  
  
        TestClass tc = new TestClass();  
        tc.TestMethod<ImplStruct?, BaseInterface>(new ImplStruct?()); // CS0313  
    }  
    public static void Main()  
    { }  
}  
```
