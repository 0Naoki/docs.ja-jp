# <a name="compiler-error-cs0077"></a><span data-ttu-id="94432-102">コンパイラ エラー CS0077</span><span class="sxs-lookup"><span data-stu-id="94432-102">Compiler Error CS0077</span></span>
<span data-ttu-id="94432-103">as 演算子は参照型または null 許容型で使用してください ('int' は null 非許容の値型です)。</span><span class="sxs-lookup"><span data-stu-id="94432-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="94432-104">[as](../../csharp/language-reference/keywords/as.md) 演算子に [値型](../../csharp/language-reference/keywords/value-types.md)が渡されました。</span><span class="sxs-lookup"><span data-stu-id="94432-104">The [as](../../csharp/language-reference/keywords/as.md) operator was passed a [value type](../../csharp/language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="94432-105">`as` は [null](../../csharp/language-reference/keywords/null.md)を返すことができます。そのため、 [参照型](../../csharp/language-reference/keywords/reference-types.md) または null 許容型以外は渡せません。</span><span class="sxs-lookup"><span data-stu-id="94432-105">Because `as` can return [null](../../csharp/language-reference/keywords/null.md), it can only be passed [reference types](../../csharp/language-reference/keywords/reference-types.md) or nullable type.</span></span> <span data-ttu-id="94432-106">Null 許容型の詳細については、次を参照してください。 [null 許容型](../../csharp/programming-guide/nullable-types/index.md)です。</span><span class="sxs-lookup"><span data-stu-id="94432-106">For more information about nullable types, see [Nullable Types](../../csharp/programming-guide/nullable-types/index.md).</span></span>  
  
 <span data-ttu-id="94432-107">次の例では CS0077 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="94432-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
