# <a name="compiler-error-cs0077"></a><span data-ttu-id="ae5d5-102">コンパイラ エラー CS0077</span><span class="sxs-lookup"><span data-stu-id="ae5d5-102">Compiler Error CS0077</span></span>
<span data-ttu-id="ae5d5-103">as 演算子は参照型または null 許容型で使用してください ('int' は null 非許容の値型です)。</span><span class="sxs-lookup"><span data-stu-id="ae5d5-103">The as operator must be used with a reference type or nullable type ('int' is a non-nullable value type).</span></span>  
  
 <span data-ttu-id="ae5d5-104">[as](../language-reference/operators/type-testing-and-cast.md#as-operator) 演算子に [値型](../language-reference/keywords/value-types.md)が渡されました。</span><span class="sxs-lookup"><span data-stu-id="ae5d5-104">The [as](../language-reference/operators/type-testing-and-cast.md#as-operator) operator was passed a [value type](../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="ae5d5-105">@No__t-0 は[null](../language-reference/keywords/null.md)を返すことができるため、[参照型](../language-reference/keywords/reference-types.md)または null 許容値型のみを渡すことができます。</span><span class="sxs-lookup"><span data-stu-id="ae5d5-105">Because `as` can return [null](../language-reference/keywords/null.md), it can only be passed [reference types](../language-reference/keywords/reference-types.md) or nullable value types.</span></span> <span data-ttu-id="ae5d5-106">Null 許容値型の詳細については、「 [Null 許容値型](../programming-guide/nullable-types/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="ae5d5-106">For more information about nullable value types, see [Nullable value types](../programming-guide/nullable-types/index.md).</span></span>
  
 <span data-ttu-id="ae5d5-107">次の例では CS0077 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ae5d5-107">The following sample generates CS0077:</span></span>  
  
```csharp  
// CS0077.cs  
using System;  
  
class C  
{  
}  
  
struct S  
{  
}  
  
class M  
{  
   public static void Main()  
   {  
      object o1, o2;  
      C c;  
      S s;  
  
      o1 = new C();  
      o2 = new S();  
  
      s = o2 as S;  // CS0077, S is not a reference type.  
      // try the following line instead  
      // c = o1 as C;  
   }  
}  
```
