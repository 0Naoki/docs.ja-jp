# <a name="compiler-error-cs0073"></a><span data-ttu-id="6e112-102">コンパイラ エラー CS0073</span><span class="sxs-lookup"><span data-stu-id="6e112-102">Compiler Error CS0073</span></span>
<span data-ttu-id="6e112-103">add または remove アクセサーには本体が必要です</span><span class="sxs-lookup"><span data-stu-id="6e112-103">An add or remove accessor must have a body</span></span>  
  
 <span data-ttu-id="6e112-104">**event** 定義に含まれる **add** キーワードまたは [remove](../language-reference/keywords/event.md) キーワードには本体が必要です。</span><span class="sxs-lookup"><span data-stu-id="6e112-104">An **add** or **remove** keyword in an [event](../language-reference/keywords/event.md) definition must have a body.</span></span> <span data-ttu-id="6e112-105">詳細については、「[イベント](../programming-guide/events/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="6e112-105">For more information, see [Events](../programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="6e112-106">次の例では CS0073 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6e112-106">The following sample generates CS0073:</span></span>  
  
```csharp  
// CS0073.cs  
delegate void del();  
  
class Test  
{  
   public event del MyEvent  
   {  
      add;   // CS0073  
      // try the following lines instead  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      remove  
      {  
         MyEvent -= value;  
      }  
  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
