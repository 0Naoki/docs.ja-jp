# <a name="compiler-error-cs0218"></a><span data-ttu-id="0d31c-102">コンパイラ エラー CS0218</span><span class="sxs-lookup"><span data-stu-id="0d31c-102">Compiler Error CS0218</span></span>
<span data-ttu-id="0d31c-103">型 ('type') に演算子 true および演算子 false の宣言が含まれている必要があります</span><span class="sxs-lookup"><span data-stu-id="0d31c-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="0d31c-104">場合は、ユーザー定義型のオーバー ロード、 [& 演算子](../language-reference/operators/and-operator.md)または[&#124;演算子](../language-reference/operators/or-operator.md)、それを定義する必要がありますも[true および false](../language-reference/keywords/true-false-operators.md) をショートサーキットするために、演算子[& & 演算子](../language-reference/operators/conditional-and-operator.md)または[ &#124; &#124;演算子](../language-reference/operators/conditional-or-operator.md)定義します。</span><span class="sxs-lookup"><span data-stu-id="0d31c-104">If a user-defined type overloads the [& operator](../language-reference/operators/and-operator.md) or [&#124; operator](../language-reference/operators/or-operator.md), it must also define [true and false](../language-reference/keywords/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/conditional-and-operator.md) or [&#124;&#124; operator](../language-reference/operators/conditional-or-operator.md) defined.</span></span>
  
 <span data-ttu-id="0d31c-105">次の例では CS0218 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0d31c-105">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
