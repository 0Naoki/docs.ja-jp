# <a name="compiler-error-cs0218"></a><span data-ttu-id="ae80a-102">コンパイラ エラー CS0218</span><span class="sxs-lookup"><span data-stu-id="ae80a-102">Compiler Error CS0218</span></span>
<span data-ttu-id="ae80a-103">型 ('type') に演算子 true および演算子 false の宣言が含まれている必要があります</span><span class="sxs-lookup"><span data-stu-id="ae80a-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
<span data-ttu-id="ae80a-104">ユーザー定義[ &#124; &#124; ](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-)型が[& 演算子](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-)または[ &#124;演算子](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-)をオーバーロードする場合は、演算子または演算子 & をショートサーキット[&](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-)するために、true 演算子[と false](../language-reference/operators/true-false-operators.md)演算子も定義する必要があります。れ.</span><span class="sxs-lookup"><span data-stu-id="ae80a-104">If a user-defined type overloads the [& operator](../language-reference/operators/boolean-logical-operators.md#logical-and-operator-) or [&#124; operator](../language-reference/operators/boolean-logical-operators.md#logical-or-operator-), it must also define [true and false](../language-reference/operators/true-false-operators.md) operators, in order to make short-circuiting [&& operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-and-operator-) or [&#124;&#124; operator](../language-reference/operators/boolean-logical-operators.md#conditional-logical-or-operator-) defined.</span></span>
  
 <span data-ttu-id="ae80a-105">次の例では CS0218 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ae80a-105">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
