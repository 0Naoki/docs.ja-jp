# <a name="compiler-warning-level-3-cs1717"></a><span data-ttu-id="2cac8-102">コンパイラの警告 (レベル 3) CS1717</span><span class="sxs-lookup"><span data-stu-id="2cac8-102">Compiler Warning (level 3) CS1717</span></span>
<span data-ttu-id="2cac8-103">同じ変数に割り当てられました。他の変数に割り当てますか?</span><span class="sxs-lookup"><span data-stu-id="2cac8-103">Assignment made to same variable; did you mean to assign something else?</span></span>  
  
 <span data-ttu-id="2cac8-104">この警告は、 `a = a`のように変数をそれ自体に代入したときに発生します。</span><span class="sxs-lookup"><span data-stu-id="2cac8-104">This warning occurs when you assign a variable to itself, such as `a = a`.</span></span>  
  
 <span data-ttu-id="2cac8-105">いくつかの一般的なミスにより、この警告が生成されることがあります。</span><span class="sxs-lookup"><span data-stu-id="2cac8-105">Several common mistakes can generate this warning:</span></span>  
  
-   <span data-ttu-id="2cac8-106">書き込み`a = a`の条件として、**場合**ステートメントなど`if (a = a)`です。</span><span class="sxs-lookup"><span data-stu-id="2cac8-106">Writing `a = a` as the condition of an **if** statement, such as `if (a = a)`.</span></span> <span data-ttu-id="2cac8-107">本来は `if (a == a)`と記述するつもりであったケースで、この式は常に true になるため、より簡潔に `if (true)`と記述することもできます。</span><span class="sxs-lookup"><span data-stu-id="2cac8-107">You probably meant to say `if (a == a)`, which is always true, so you could write this more concisely as `if (true)`.</span></span>  
  
-   <span data-ttu-id="2cac8-108">入力ミス。</span><span class="sxs-lookup"><span data-stu-id="2cac8-108">Mistyping.</span></span> <span data-ttu-id="2cac8-109">本来は `a = b`と記述するつもりであったケース。</span><span class="sxs-lookup"><span data-stu-id="2cac8-109">You probably meant to say `a = b`.</span></span>  
  
-   <span data-ttu-id="2cac8-110">パラメーターがフィールドと同じ名前を持ち、 **this** キーワードを使用していないコンストラクター。本来は `this.a = a`のように変数をそれ自体に代入したときに発生します。</span><span class="sxs-lookup"><span data-stu-id="2cac8-110">In a constructor where the parameter has the same name as the field, not using the **this** keyword: you probably meant to say `this.a = a`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="2cac8-111">例</span><span class="sxs-lookup"><span data-stu-id="2cac8-111">Example</span></span>

   public static void Main()
