<span data-ttu-id="d2217-103">部分的なクラスまたは構造体 'type' の複数の宣言内にあるフィールド間に定義された順序がありません。</span><span class="sxs-lookup"><span data-stu-id="d2217-103">There is no defined ordering between fields in multiple declarations of partial class or struct 'type'.</span></span> <span data-ttu-id="d2217-104">順序付けを指定するには、すべてのインスタンス フィールドが、同じ宣言になければなりません。</span><span class="sxs-lookup"><span data-stu-id="d2217-104">To specify an ordering, all instance fields must be in the same declaration.</span></span>
