<span data-ttu-id="7c11c-103">プログラム 'output file name' に、複数のエントリ ポイントが定義されています。</span><span class="sxs-lookup"><span data-stu-id="7c11c-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="7c11c-104">エントリ ポイントを含む型を指定するために、/main を使用してコンパイルします。</span><span class="sxs-lookup"><span data-stu-id="7c11c-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="7c11c-105">プログラムには、 [Main](../../csharp/programming-guide/main-and-command-args/index.md) メソッドを 1 つのみ指定できます。</span><span class="sxs-lookup"><span data-stu-id="7c11c-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="7c11c-106">このエラーを解決するには、Main メソッドを 1 つのみコードに残して残りをすべてコードから削除するか、 [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) コンパイラ オプションを使用してどの Main メソッドを使用するか指定できます。</span><span class="sxs-lookup"><span data-stu-id="7c11c-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="7c11c-107">次の例では CS0017 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="7c11c-107">The following sample generates CS0017:</span></span>  
  
```  
// CS0017.cs

public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
