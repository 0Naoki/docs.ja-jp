 <span data-ttu-id="0c8f3-104">[override](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) キーワードでマークされているメソッドでは、[Conditional](../../csharp/language-reference/keywords/override.md) 属性は許可されていません。</span><span class="sxs-lookup"><span data-stu-id="0c8f3-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="0c8f3-105">詳細については、次を参照してください。[と New キーワードを使用してオーバーライドする場合](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)します。</span><span class="sxs-lookup"><span data-stu-id="0c8f3-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>

 <span data-ttu-id="0c8f3-107">次の例では CS0243 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="0c8f3-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
