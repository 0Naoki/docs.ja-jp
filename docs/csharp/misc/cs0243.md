 <span data-ttu-id="34ddb-104">[条件付き](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb)でマークされているメソッドで属性は使用できません、[オーバーライド](../../csharp/language-reference/keywords/override.md)キーワード。</span><span class="sxs-lookup"><span data-stu-id="34ddb-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="34ddb-105">詳細については、次を参照してください。[がわからないときに使用するオーバーライド キーワードと New キーワードを](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)です。</span><span class="sxs-lookup"><span data-stu-id="34ddb-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="34ddb-106">コンパイラがオーバーライド メソッドにバインドすることはありません。バインドするのは基本メソッドのみであり、共通言語ランタイムが、必要に応じて、オーバーライドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="34ddb-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="34ddb-107">次の例では CS0243 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="34ddb-107">The following sample generates CS0243:</span></span>  
  
```csharp  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
