# <a name="compiler-error-cs0243"></a><span data-ttu-id="d3da9-102">コンパイラ エラー CS0243</span><span class="sxs-lookup"><span data-stu-id="d3da9-102">Compiler Error CS0243</span></span>

<span data-ttu-id="d3da9-103">オーバーライド メソッドであるため、条件付き属性は 'method' では無効です</span><span class="sxs-lookup"><span data-stu-id="d3da9-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="d3da9-104">[override](../programming-guide/concepts/attributes/common-attributes.md#Conditional) キーワードでマークされているメソッドでは、[Conditional](../language-reference/keywords/override.md) 属性は許可されていません。</span><span class="sxs-lookup"><span data-stu-id="d3da9-104">The [Conditional](../programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute is not allowed on a method that is marked with the [override](../language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="d3da9-105">詳細については、「 [Override キーワードと New キーワードを使用する場合](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md)について」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="d3da9-105">For more information, see [Knowing When to Use Override and New Keywords](../programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="d3da9-106">コンパイラがオーバーライド メソッドにバインドすることはありません。バインドするのは基本メソッドのみであり、共通言語ランタイムが、必要に応じて、オーバーライドを呼び出します。</span><span class="sxs-lookup"><span data-stu-id="d3da9-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="d3da9-107">次の例では CS0243 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d3da9-107">The following sample generates CS0243:</span></span>  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
