# コンパイラ エラー CS0013
メタデータをファイル 'ModelStore.dll' に書き込み中に予期しないエラーが発生しました \-\- 'ユーザー文字列をこれ以上作成する論理空間が残っていません。'  
  
 .NET Framework 共通言語ランタイムはメタデータを生成しませんでした。 パスが正しく指定されていることと、ディスクの空き容量が十分に残っていることを確認します。 それでも問題が解決されない場合は、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] と .NET Framework の修復または再インストールが必要です。
