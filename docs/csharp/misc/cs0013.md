# <a name="compiler-error-cs0013"></a>コンパイラ エラー CS0013
メタデータをファイル 'ModelStore.dll' に書き込み中に予期しないエラーが発生しました -- 'ユーザー文字列をこれ以上作成する論理空間が残っていません。'  
  
 .NET Framework 共通言語ランタイムはメタデータを生成しませんでした。 パスが正しく指定されていることと、ディスクの空き容量が十分に残っていることを確認します。 問題が解決しない場合は、修復または Visual Studio と .NET Framework を再インストールする必要があります。
