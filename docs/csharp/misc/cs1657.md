# <a name="compiler-error-cs1657"></a><span data-ttu-id="114fa-102">コンパイラ エラー CS1657</span><span class="sxs-lookup"><span data-stu-id="114fa-102">Compiler Error CS1657</span></span>
<span data-ttu-id="114fa-103">'reason' であるため、'parameter' を ref または out 引数として渡せません</span><span class="sxs-lookup"><span data-stu-id="114fa-103">Cannot pass 'parameter' as a ref or out argument because 'reason''</span></span>  
  
 <span data-ttu-id="114fa-104">このエラーは、変数が読み取り専用であるコンテキストでその変数が [ref](../../csharp/language-reference/keywords/ref.md) または [out](../../csharp/language-reference/keywords/out.md) 引数として渡される場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="114fa-104">This error occurs when a variable is passed as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out.md) argument in a context in which that variable is readonly.</span></span> <span data-ttu-id="114fa-105">読み取り専用コンテキストには、 [foreach](../../csharp/language-reference/keywords/foreach-in.md) 繰り返し変数、 [using](../../csharp/language-reference/keywords/using-statement.md) 変数、 `fixed` 変数などがあります。</span><span class="sxs-lookup"><span data-stu-id="114fa-105">Readonly contexts include [foreach](../../csharp/language-reference/keywords/foreach-in.md) iteration variables, [using](../../csharp/language-reference/keywords/using-statement.md) variables, and `fixed` variables.</span></span> <span data-ttu-id="114fa-106">このエラーを解決するには、 `foreach`ブロック、 `using` ステートメント、および `fixed` ステートメントで `ref` または `out` パラメーターとして `using` 、 `foreach` 、または `fixed` 変数を受け取る関数を呼び出さないでください。</span><span class="sxs-lookup"><span data-stu-id="114fa-106">To resolve this error, do not call functions that take the `foreach`, `using` or `fixed` variable as a `ref` or `out` parameter in `using` blocks, `foreach` statements, and `fixed` statements.</span></span>  
  
## <a name="example"></a><span data-ttu-id="114fa-107">例</span><span class="sxs-lookup"><span data-stu-id="114fa-107">Example</span></span>  
 <span data-ttu-id="114fa-108">次の例では CS1657 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="114fa-108">The following sample generates CS1657:</span></span>  
  
```  
// CS1657.cs  
using System;  
class C : IDisposable  
{  
    public int i;  
    public void Dispose() {}  
}  
  
class CMain  
{  
    static void f(ref C c)  
    {  
    }  
    static void Main()  
    {  
        using (C c = new C())  
        {  
            f(ref c);  // CS1657  
        }  
    }  
}  
```
