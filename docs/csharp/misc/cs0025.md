# コンパイラ エラー CS0025
標準ライブラリ ファイル 'file' が見つかりません  
  
 コンパイラで必要なファイルが見つかりませんでした。 パスが正しいことと、ファイルが存在することを確認してください。  
  
 ファイルが [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] システム ファイルの場合は、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] インストールの修復や、完全な再インストールが必要になる場合があります。
