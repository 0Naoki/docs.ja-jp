## <a name="example"></a><span data-ttu-id="e5ccb-107">例</span><span class="sxs-lookup"><span data-stu-id="e5ccb-107">Example</span></span>  
 <span data-ttu-id="e5ccb-108">次のコードでは CS0754 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="e5ccb-108">The following code generates CS0754:</span></span>  
  
```csharp  
// cs0754.cs  
using System;  
  
    public interface IF  
    {  
        void Part();  
    }  
  
    public partial class C : IF

        partial void IF.Part(); //CS0754

        {  
            return 1;  
        }  
    }  
```
