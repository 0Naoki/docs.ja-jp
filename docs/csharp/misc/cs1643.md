# <a name="compiler-error-cs1643"></a><span data-ttu-id="c490b-102">コンパイラ エラー CS1643</span><span class="sxs-lookup"><span data-stu-id="c490b-102">Compiler Error CS1643</span></span>
<span data-ttu-id="c490b-103">型 'type!' のメソッドに値を返さないコード パスがあります</span><span class="sxs-lookup"><span data-stu-id="c490b-103">Not all code paths return a value in method of type 'type!'</span></span>  
  
 <span data-ttu-id="c490b-104">このエラーは、デリゲートの本体に return ステートメントが存在しない、または存在したとしても、その return ステートメントまで到達可能なことをコンパイラが確認できない場合に発生します。</span><span class="sxs-lookup"><span data-stu-id="c490b-104">This error occurs if a delegate body does not have a return statement, or has a return statement that the compiler is unable to verify will be reached.</span></span> <span data-ttu-id="c490b-105">次の例では、コンパイラは分岐条件の結果を予測することで、匿名メソッドのブロックから常に値が戻されるかどうかを検査しません。</span><span class="sxs-lookup"><span data-stu-id="c490b-105">In the example below, the compiler does not attempt to predict the result of the branching condition in order to verify that the anonymous method block always returns a value.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c490b-106">例</span><span class="sxs-lookup"><span data-stu-id="c490b-106">Example</span></span>  
 <span data-ttu-id="c490b-107">次の例では CS1643 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="c490b-107">The following sample generates CS1643:</span></span>  
  
```csharp  
// CS1643.cs  
delegate int MyDelegate();  
  
class C  
{  
    static void Main()  
    {  
        MyDelegate d = delegate  
        {                 // CS1643  
            int i = 0;  
            if (i == 0)  
                return 1;  
        };  
    }  
}  
```
