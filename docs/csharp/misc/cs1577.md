# <a name="compiler-error-cs1577"></a>コンパイラ エラー CS1577
アセンブリの生成に失敗しました -- 理由  
  
 C# コンパイラがソース コードを正常にコンパイルした後、コンパイルのアセンブリ生成部分が失敗しました。 ALink ユーティリティのエラーに関するドキュメントを参照してください ([Al.exe ツールのエラーと警告](http://msdn.microsoft.com/library/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) 詳細についてはします。
