 <span data-ttu-id="a2c71-105">マークされたメソッドを`extern`必要がありますも属性でマークされますが、たとえば、 [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute)属性です。</span><span class="sxs-lookup"><span data-stu-id="a2c71-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>

   }
