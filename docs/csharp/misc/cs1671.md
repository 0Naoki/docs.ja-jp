# コンパイラ エラー CS1671
名前空間の宣言に、修飾子または属性を指定することはできません。  
  
 修飾子は名前空間に適用された場合意味がないため、使用できません。  
  
 次の例では CS1671 が生成されます。  
  
```  
// CS1671.cs public namespace NS // CS1671 { }  
```
