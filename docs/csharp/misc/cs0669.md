# <a name="compiler-error-cs0669"></a><span data-ttu-id="d0178-102">コンパイラ エラー CS0669</span><span class="sxs-lookup"><span data-stu-id="d0178-102">Compiler Error CS0669</span></span>

<span data-ttu-id="d0178-103">ComImport 属性を持つクラスでは、ユーザー定義のコンス トラクターを持つことはできません。</span><span class="sxs-lookup"><span data-stu-id="d0178-103">A class with the ComImport attribute cannot have a user-defined constructor.</span></span>

<span data-ttu-id="d0178-104">[ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) クラスのコンストラクターは、共通言語ランタイムの COM 相互運用層に用意されています。</span><span class="sxs-lookup"><span data-stu-id="d0178-104">The COM interop layer in the common language runtime supplies the constructor for [ComImport](xref:System.Runtime.InteropServices.ComImportAttribute) classes.</span></span> <span data-ttu-id="d0178-105">そのため、COM オブジェクトは、実行時にマネージ オブジェクトとして使用できます。</span><span class="sxs-lookup"><span data-stu-id="d0178-105">Consequently, a COM object can be used as a managed object in the runtime.</span></span>

<span data-ttu-id="d0178-106">次の例では CS0669 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="d0178-106">The following sample generates CS0669:</span></span>

```csharp
// CS0669.cs
using System.Runtime.InteropServices;
[ComImport, Guid("00000000-0000-0000-0000-000000000001")]
class TestClass
{
   TestClass()   // CS0669, delete constructor to resolve
   {
   }

   public static void Main()
   {
   }
}
```
