# <a name="compiler-error-cs0761"></a><span data-ttu-id="78622-102">コンパイラ エラー CS0761</span><span class="sxs-lookup"><span data-stu-id="78622-102">Compiler Error CS0761</span></span>

 <span data-ttu-id="78622-104">部分メソッドに実装がある場合、ジェネリック型制約はメソッド シグネチャで定義されている制約と同じである必要があります。</span><span class="sxs-lookup"><span data-stu-id="78622-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
