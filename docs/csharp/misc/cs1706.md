## <a name="example"></a><span data-ttu-id="1c472-107">例</span><span class="sxs-lookup"><span data-stu-id="1c472-107">Example</span></span>  
 <span data-ttu-id="1c472-108">次の例では CS1706 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="1c472-108">The following example generates CS1706.</span></span>

```csharp  
{  
    public MyAttribute(MyDelegate d) { }

[MyAttribute(delegate{/* anonymous Method in Attribute declaration */})]  // CS1706

{  
}  
```
