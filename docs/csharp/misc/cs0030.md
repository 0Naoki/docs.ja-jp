# <a name="compiler-error-cs0030"></a><span data-ttu-id="ec551-102">コンパイラ エラー CS0030</span><span class="sxs-lookup"><span data-stu-id="ec551-102">Compiler Error CS0030</span></span>
<span data-ttu-id="ec551-103">型 'type' を 'type' に変換できません</span><span class="sxs-lookup"><span data-stu-id="ec551-103">Cannot convert type 'type' to 'type'</span></span>  
  
 <span data-ttu-id="ec551-104">特定の演算子のオーバーロードをサポートするための変換ルーチンを指定する必要があります。</span><span class="sxs-lookup"><span data-stu-id="ec551-104">You must provide conversion routines to support certain operator overloads.</span></span> <span data-ttu-id="ec551-105">詳しくは、「[変換演算子](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md)」をご覧ください。</span><span class="sxs-lookup"><span data-stu-id="ec551-105">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="ec551-106">次の例では CS0030 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="ec551-106">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
