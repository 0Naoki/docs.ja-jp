# <a name="compiler-error-cs0030"></a><span data-ttu-id="3071b-102">コンパイラ エラー CS0030</span><span class="sxs-lookup"><span data-stu-id="3071b-102">Compiler Error CS0030</span></span>
<span data-ttu-id="3071b-103">型 'type' を 'type' に変換できません</span><span class="sxs-lookup"><span data-stu-id="3071b-103">Cannot convert type 'type' to 'type'</span></span>  
  
<span data-ttu-id="3071b-104">型の間に定義済みの変換はありません。</span><span class="sxs-lookup"><span data-stu-id="3071b-104">There is no predefined conversion between types.</span></span> <span data-ttu-id="3071b-105">これらの型の間のカスタム変換を定義できます。</span><span class="sxs-lookup"><span data-stu-id="3071b-105">You can define a custom conversion between those types.</span></span> <span data-ttu-id="3071b-106">詳細については、「[User-defined conversion operators](../language-reference/operators/user-defined-conversion-operators.md)」(ユーザー定義の変換演算子) を参照してください。</span><span class="sxs-lookup"><span data-stu-id="3071b-106">For more information, see [User-defined conversion operators](../language-reference/operators/user-defined-conversion-operators.md).</span></span>  
  
 <span data-ttu-id="3071b-107">次の例では CS0030 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="3071b-107">The following sample generates CS0030:</span></span>  
  
```csharp  
// CS0030.cs  
namespace x  
{  
   public class iii  
   {  
      /*  
      public static implicit operator iii(int aa)  
      {  
         return null;  
      }  
  
      public static implicit operator int(iii aa)  
      {  
         return 0;  
      }  
      */  
  
      public static iii operator ++(iii aa)  
      {  
         return (iii)0;   // CS0030  
         // uncomment the conversion routines to resolve CS0030  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
