# コンパイラ エラー CS1514
{ が必要です。  
  
 左中かっこ \(`{`\) が必要ですが、コンパイラは見つけることができませんでした。  
  
 次の例では CS1514 が生成されます。  
  
```  
// CS1514.cs namespace x // CS1514, no open curly brace  
```
