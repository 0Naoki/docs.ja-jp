# <a name="compiler-error-cs2001"></a>コンパイラ エラー CS2001
ソース ファイル 'file' が見つかりませんでした  
  
 ソース ファイル名がコンパイラに渡されましたが、見つかりませんでした。 ファイル名のスペルとファイルの場所を確認してください。
