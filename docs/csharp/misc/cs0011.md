## <a name="example"></a><span data-ttu-id="27c68-107">例</span><span class="sxs-lookup"><span data-stu-id="27c68-107">Example</span></span>

{  
   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="27c68-108">例</span><span class="sxs-lookup"><span data-stu-id="27c68-108">Example</span></span>  
 <span data-ttu-id="27c68-109">2 つ目のファイルでは、前の例で作成した `C` クラスから派生した `B` クラスを定義する DLL を作成します。</span><span class="sxs-lookup"><span data-stu-id="27c68-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

```
