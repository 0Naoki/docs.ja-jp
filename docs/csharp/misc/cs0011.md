# <a name="compiler-error-cs0011"></a><span data-ttu-id="6b328-102">コンパイラ エラー CS0011</span><span class="sxs-lookup"><span data-stu-id="6b328-102">Compiler Error CS0011</span></span>

 <span data-ttu-id="6b328-104">**/reference**を使用してファイルからインポートされたクラスが、あるクラスの派生クラスであるか、または見つからないインターフェイスを実装しています。</span><span class="sxs-lookup"><span data-stu-id="6b328-104">A class that was imported from a file with **/reference**, is derived from a class or implements an interface that is not found.</span></span> <span data-ttu-id="6b328-105">このエラーは、必要な DLL が **/reference**を使用したコンパイル時にインクルードされていない場合に発生することがあります。</span><span class="sxs-lookup"><span data-stu-id="6b328-105">This can occur if a required DLL is not also included in the compilation with **/reference**.</span></span>  
  
 <span data-ttu-id="6b328-106">詳細については、「[[参照の追加] ダイアログボックス](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager)と[-Reference (C#コンパイラオプション)](../language-reference/compiler-options/reference-compiler-option.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="6b328-106">For more information, see [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) and [-reference (C# Compiler Options)](../language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="6b328-107">例</span><span class="sxs-lookup"><span data-stu-id="6b328-107">Example</span></span>  
  
```csharp  
// CS0011_1.cs  
// compile with: /target:library  
  
public class Outer   
{  
   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="6b328-108">例</span><span class="sxs-lookup"><span data-stu-id="6b328-108">Example</span></span>  
 <span data-ttu-id="6b328-109">2 つ目のファイルでは、前の例で作成した `C` クラスから派生した `B` クラスを定義する DLL を作成します。</span><span class="sxs-lookup"><span data-stu-id="6b328-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

```csharp  
// CS0011_2.cs  
// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll  
public class C : Outer.B {}  
```  
  
## <a name="example"></a><span data-ttu-id="6b328-110">例</span><span class="sxs-lookup"><span data-stu-id="6b328-110">Example</span></span>  
 <span data-ttu-id="6b328-111">3 つ目のファイルでは、最初の手順で作成した DLL を置き換え、内部クラスである `B`の定義を省略しています。</span><span class="sxs-lookup"><span data-stu-id="6b328-111">The third file replaces the DLL created by the first step, and omits the definition of the inner class `B`.</span></span>
