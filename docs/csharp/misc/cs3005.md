# <a name="compiler-warning-level-1-cs3005"></a>コンパイラの警告 (レベル 1) CS3005
大文字、小文字の違いのみの識別子 'identifier' は CLS に準拠していません。  
  
 A[パブリック](../../csharp/language-reference/keywords/public.md)、[保護](../../csharp/language-reference/keywords/protected.md)、または`protected internal`識別子で、他と異なる`public`、 `protected`、または`protected internal`が 1 つまたは複数の文字の場合にのみの識別子準拠していない共通言語仕様 (CLS) にします。 CLS 準拠について詳しくは、「[CLS 準拠コードの記述](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3)」および「[言語への非依存性、および言語非依存コンポーネント](../../../docs/standard/language-independence-and-language-independent-components.md)」をご覧ください。  
  
## <a name="example"></a>例  
 次の例では、CS3003 エラーが生成されます。  
  
```  
// CS3005.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int a1 = 0;  
    public static int A1 = 1;   // CS3005  
  
    public static void Main()  
    {  
        Console.WriteLine(a1);  
        Console.WriteLine(A1);  
    }  
}  
```
