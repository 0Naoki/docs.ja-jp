# コンパイラ エラー CS1556
Main メソッドに指定された 'construct' は有効なクラスか構造体でなければなりません。  
  
 [\/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) コンパイラ オプションには、クラス名でない識別子が渡されました。
