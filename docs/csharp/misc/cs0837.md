## <a name="example"></a><span data-ttu-id="14c22-109">例</span><span class="sxs-lookup"><span data-stu-id="14c22-109">Example</span></span>  
 <span data-ttu-id="14c22-110">次の例では CS0837 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="14c22-110">The following example generates CS0837:</span></span>  
  
```  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
