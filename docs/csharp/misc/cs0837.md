## <a name="to-correct-this-error"></a><span data-ttu-id="8b375-105">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="8b375-105">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="8b375-109">例</span><span class="sxs-lookup"><span data-stu-id="8b375-109">Example</span></span>

<span data-ttu-id="8b375-110">次の例では CS0837 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="8b375-110">The following example generates CS0837:</span></span>

```csharp
// cs0837.cs
namespace TestNamespace
{
    public delegate void Del();

    class Test
    {
        static int Main()
        {
            bool b1 = (() => { }) is Del;   // CS0837
            bool b2 = delegate() { } is Del;// CS0837
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837
            return 1;
        }
    }
}
```
