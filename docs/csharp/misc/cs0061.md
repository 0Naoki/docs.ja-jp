 <span data-ttu-id="5ad3c-104">[パブリック](../../csharp/language-reference/keywords/public.md) コンストラクトは、パブリックにアクセスできるオブジェクトを返す必要があります。</span><span class="sxs-lookup"><span data-stu-id="5ad3c-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="5ad3c-107">次の例では CS0061 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="5ad3c-107">The following sample generates CS0061.</span></span>  
  
```  
// CS0061.cs  
// compile with: /target:library

public interface AA : A {}  // CS0061  
  
// OK

internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
