# <a name="compiler-error-cs1003"></a>コンパイラ エラー CS1003
構文エラーです。'char' が必要です  
  
 いくつかのエラー条件のいずれかに該当する場合、コンパイラはこのエラーを生成します。 コードを確認して、構文エラーを見つけます。  
  
 次の例では CS1003 が生成されます。  
  
```csharp  
// CS1003.cs  
public class b  
{  
   public static void Main()  
   {  
      int[] a;  
      a[);   // CS1003  
   }  
}  
```
