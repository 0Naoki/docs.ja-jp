 <span data-ttu-id="92000-105">このエラーは、インターフェイスのプロパティを実装しようとしたときに、クラス内のプロパティ アクセサーの実装がパブリックではない場合に生じます。</span><span class="sxs-lookup"><span data-stu-id="92000-105">This error occurs when you try to implement a property of an interface, but the implementation of the property accessor in the class is not public.</span></span> <span data-ttu-id="92000-106">インターフェイス メンバーを実装するメソッドには、パブリック アクセシビリティが必要です。</span><span class="sxs-lookup"><span data-stu-id="92000-106">Methods that implement interface members need to have public accessibility.</span></span> <span data-ttu-id="92000-107">解決するには、プロパティ アクセサーのアクセス修飾子を削除します。</span><span class="sxs-lookup"><span data-stu-id="92000-107">To resolve, remove the access modifier on the property accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="92000-108">例</span><span class="sxs-lookup"><span data-stu-id="92000-108">Example</span></span>  
 <span data-ttu-id="92000-109">次の例では、CS0277 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="92000-109">The following example generates CS0277:</span></span>  
  
```csharp  
// CS0277.cs  
public interface MyInterface  
{  
    int Property  
    {  
        get;  
        set;  
    }  
}  
  
public class MyClass : MyInterface   // CS0277  
{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
