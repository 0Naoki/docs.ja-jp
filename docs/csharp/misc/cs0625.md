# <a name="compiler-error-cs0625"></a><span data-ttu-id="72bf7-102">コンパイラ エラー CS0625</span><span class="sxs-lookup"><span data-stu-id="72bf7-102">Compiler Error CS0625</span></span>
<span data-ttu-id="72bf7-103">'field': FieldOffset 属性を StructLayout(LayoutKind.Explicit) でマークされたインスタンス フィールドの型があります。</span><span class="sxs-lookup"><span data-stu-id="72bf7-103">'field': instance field types marked with StructLayout(LayoutKind.Explicit) must have a FieldOffset attribute.</span></span>
  
<span data-ttu-id="72bf7-104">構造体が明示的な **StructLayout** 属性でマークされている場合、その構造体のすべてのフィールドには [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) 属性が必要です。</span><span class="sxs-lookup"><span data-stu-id="72bf7-104">When a struct is marked with an explicit **StructLayout** attribute, all fields in the struct must have the [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute.</span></span> <span data-ttu-id="72bf7-105">詳細については、「 [StructLayoutAttribute クラス](xref:System.Runtime.InteropServices.StructLayoutAttribute)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="72bf7-105">For more information, see [StructLayoutAttribute Class](xref:System.Runtime.InteropServices.StructLayoutAttribute).</span></span> 

<span data-ttu-id="72bf7-106">次の例では CS0625 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="72bf7-106">The following sample generates CS0625:</span></span>  
  
```csharp  
// CS0625.cs  
// compile with: /target:library  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
struct A  
{  
   public int i;   // CS0625 not static; an instance field  
}  
  
// OK  
[StructLayout(LayoutKind.Explicit)]  
struct B  
{  
   [FieldOffset(5)]  
   public int i;  
}  
```
