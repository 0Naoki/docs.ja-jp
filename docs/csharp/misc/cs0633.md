## <a name="example"></a><span data-ttu-id="73c1d-106">例</span><span class="sxs-lookup"><span data-stu-id="73c1d-106">Example</span></span>  
 <span data-ttu-id="73c1d-107"><xref:System.Diagnostics.ConditionalAttribute>を使用した場合の CS0633 の例を次に示します。</span><span class="sxs-lookup"><span data-stu-id="73c1d-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="73c1d-108">次の例では CS0633 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="73c1d-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs  
#define DEBUG  
using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="73c1d-109">例</span><span class="sxs-lookup"><span data-stu-id="73c1d-109">Example</span></span>  
 <span data-ttu-id="73c1d-110"><xref:System.Runtime.CompilerServices.IndexerNameAttribute>を使用した場合の CS0633 の例を次に示します。</span><span class="sxs-lookup"><span data-stu-id="73c1d-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]   
   {   
      get { return i; }   
   }  
}  
```
