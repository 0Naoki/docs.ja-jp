 <span data-ttu-id="d86f5-104">マークされたパラメーター、[アウト](../../csharp/language-reference/keywords/out-parameter-modifier.md)キーワードは、メソッドの本体で値を割り当てられませんでした。</span><span class="sxs-lookup"><span data-stu-id="d86f5-104">A parameter marked with the [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="d86f5-105">詳細については、次を参照してください[パラメーターの引き渡し。](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span><span class="sxs-lookup"><span data-stu-id="d86f5-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)</span></span>

```csharp  
// CS0177.cs

{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;

   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
