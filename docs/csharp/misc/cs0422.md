# <a name="compiler-warning-level-4-cs0422"></a>コンパイラの警告 (レベル 4) CS0422
/incremental オプションは現在サポートされていません。  
  
 インクリメンタル コンパイル (**/incr** または **/incremental**) は、Visual C# 2005 ではサポートされていません。
