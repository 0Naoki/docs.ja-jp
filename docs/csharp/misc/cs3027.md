 <span data-ttu-id="11dc0-106">次の例には、シグニチャに CLS 非準拠の型を使用するメソッドを持ち、型が CLS 非準拠になるインターフェイスが含まれています。</span><span class="sxs-lookup"><span data-stu-id="11dc0-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>
