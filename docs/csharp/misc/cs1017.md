## <a name="example"></a><span data-ttu-id="6735f-106">例</span><span class="sxs-lookup"><span data-stu-id="6735f-106">Example</span></span>  
 <span data-ttu-id="6735f-107">次の例では CS1017 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="6735f-107">The following sample generates CS1017:</span></span>  
  
```csharp  
// CS1017.cs  
using System;  
  
namespace x  
{  
    public class b : Exception  
    {  
    }  
  
    public class a  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
  
            catch   // CS1017, must be last catch  
            {  
            }  
  
            catch(b)  
            {  
                throw;  
            }  
        }  
    }  
}  
```
