## <a name="example"></a><span data-ttu-id="715eb-108">例</span><span class="sxs-lookup"><span data-stu-id="715eb-108">Example</span></span>

```csharp  
// cs0842.cs

using System.Runtime.InteropServices;  
  
namespace TestNamespace  
{  
    [StructLayout(LayoutKind.Explicit)]  
    struct Str  
    {  
        public int Num // CS0842  
        {  
            get;  
            set;  
        }  
  
        static int Main()  
        {  
            return 1;  
        }  
    }  
}  
```
