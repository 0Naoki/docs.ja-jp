# <a name="compiler-error-cs1675"></a>コンパイラ エラー CS1675
Enums に型パラメーターを指定することはできません。  
  
 このエラーを解決するには、 `enum` 宣言から型パラメーターを削除します。  
  
## <a name="example"></a>例  
 次の例では CS1675 が生成されます。  
  
```csharp  
// CS1675.cs  
enum E<T>  // CS1675  
{  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
