# <a name="compiler-error-cs1647"></a>コンパイラ エラー CS1647
式が長すぎるか複雑すぎるため、'code' 付近ではコンパイルできません  
  
 コードを処理するコンパイラでスタック オーバーフローが発生しました。 このエラーを解決するには、コードを簡略化します。 コードが有効な場合にお問い合わせ[製品サポート](http://msdn.microsoft.com/library/77e75b8b-817d-45bf-9c38-458930d873b4)です。
