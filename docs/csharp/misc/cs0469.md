# <a name="compiler-warning-level-2-cs0469"></a><span data-ttu-id="07535-102">コンパイラの警告 (レベル 2) CS0469</span><span class="sxs-lookup"><span data-stu-id="07535-102">Compiler Warning (level 2) CS0469</span></span>
<span data-ttu-id="07535-103">'goto case' 値は型 'type' に暗黙的に変換できません</span><span class="sxs-lookup"><span data-stu-id="07535-103">The 'goto case' value is not implicitly convertible to type 'type'</span></span>  
  
 <span data-ttu-id="07535-104">`goto case`を使用する場合は、goto case の値からスイッチの型への暗黙的な変換が必要です。</span><span class="sxs-lookup"><span data-stu-id="07535-104">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="07535-105">例</span><span class="sxs-lookup"><span data-stu-id="07535-105">Example</span></span>  
 <span data-ttu-id="07535-106">次の例では CS0469 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="07535-106">The following sample generates CS0469.</span></span>  
  
```csharp  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:   
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
