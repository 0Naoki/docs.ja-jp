 <span data-ttu-id="b4f63-104">`goto case`を使用する場合は、goto case の値からスイッチの型への暗黙的な変換が必要です。</span><span class="sxs-lookup"><span data-stu-id="b4f63-104">When you use `goto case`, there must be an implicit conversion from the value of the goto case to the type of the switch.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b4f63-105">例</span><span class="sxs-lookup"><span data-stu-id="b4f63-105">Example</span></span>  
 <span data-ttu-id="b4f63-106">次の例では CS0469 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="b4f63-106">The following sample generates CS0469.</span></span>  
  
```  
// CS0469.cs  
// compile with: /W:2  
class Test  
{  
   static void Main()  
   {  
      char c = (char)180;  
      switch (c)  
      {  
         case (char)127:  
            break;  
  
         case (char)180:   
            goto case 127;   // CS0469  
            // try the following line instead  
            // goto case (char) 127;  
      }  
   }  
}  
```
