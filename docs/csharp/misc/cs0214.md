# <a name="compiler-error-cs0214"></a><span data-ttu-id="05cd2-102">コンパイラ エラー CS0214</span><span class="sxs-lookup"><span data-stu-id="05cd2-102">Compiler Error CS0214</span></span>
<span data-ttu-id="05cd2-103">ポインターおよび固定サイズ バッファーは、unsafe コンテキストでのみ使用することができます。</span><span class="sxs-lookup"><span data-stu-id="05cd2-103">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="05cd2-104">ポインターは [unsafe](../../csharp/language-reference/keywords/unsafe.md) キーワードとのみ使用できます。</span><span class="sxs-lookup"><span data-stu-id="05cd2-104">Pointers can only be used with the [unsafe](../../csharp/language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="05cd2-105">詳細については、「[アンセーフ コードとポインター](../../csharp/programming-guide/unsafe-code-pointers/index.md)」を参照してください。</span><span class="sxs-lookup"><span data-stu-id="05cd2-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="05cd2-106">次の例では CS0214 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="05cd2-106">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
