## <a name="example"></a><span data-ttu-id="81b20-105">例</span><span class="sxs-lookup"><span data-stu-id="81b20-105">Example</span></span>  
 <span data-ttu-id="81b20-106">次の例では、CS0440 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="81b20-106">The following example generates CS0440:</span></span>  
  
```csharp  
// CS0440.cs  
// Compile with: /W:1  
  
using global = MyClass;   // CS0440  
class MyClass  
{  
    static void Main()  
    {  
        // Note how global refers to the global namespace  
        // even though it is redefined above.  
        global::System.Console.WriteLine();  
    }  
}  
```
