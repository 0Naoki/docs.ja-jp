# <a name="compiler-error-cs0534"></a><span data-ttu-id="a13f4-102">コンパイラ エラー CS0534</span><span class="sxs-lookup"><span data-stu-id="a13f4-102">Compiler Error CS0534</span></span>
<span data-ttu-id="a13f4-103">'function1' は継承抽象メンバー 'function2' を実装しません</span><span class="sxs-lookup"><span data-stu-id="a13f4-103">'function1' does not implement inherited abstract member 'function2'</span></span>  
  
 <span data-ttu-id="a13f4-104">抽象クラスでない限り、クラスでは、基底クラスのすべての [抽象](../../csharp/language-reference/keywords/abstract.md) メンバーを実装する必要があります。</span><span class="sxs-lookup"><span data-stu-id="a13f4-104">A class is required to implement all the [abstract](../../csharp/language-reference/keywords/abstract.md) members in the base class, unless the class is also abstract.</span></span>  
  
 <span data-ttu-id="a13f4-105">次の例では CS0534 が生成されます。</span><span class="sxs-lookup"><span data-stu-id="a13f4-105">The following sample generates CS0534:</span></span>  
  
```csharp  
// CS0534.cs  
namespace x  
{  
   abstract public class clx  
   {  
      public abstract void f();  
   }  
  
   public class cly : clx   // CS0534, no override for clx::f  
   {  
      // uncomment the following sample override to resolve CS0534  
      // override public void f()  
      // {  
      // }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
