# <a name="manipulations-and-inertia"></a><span data-ttu-id="52dba-102">操作と慣性</span><span class="sxs-lookup"><span data-stu-id="52dba-102">Manipulations and Inertia</span></span>
<span data-ttu-id="52dba-103">"*操作*" および "*慣性*" プロセッサ クラスは、アプリケーションが自然で直感的な方法で画面上のユーザー インターフェイス (UI) 要素を移動するのを支援します。</span><span class="sxs-lookup"><span data-stu-id="52dba-103">The *manipulations* and *inertia* processor classes assist your application in moving user interface (UI) elements on the screen in a natural and intuitive way.</span></span> <span data-ttu-id="52dba-104">操作では、ユーザーはマウス デバイス、スタイラス、(直接タッチ対応シナリオでの) 指などのマニピュレーターを使って、コンポーネントの移動、回転、サイズ変更を行うことができます。</span><span class="sxs-lookup"><span data-stu-id="52dba-104">Manipulations enable users to move, rotate, and resize components by using manipulators, such as a mouse device, a stylus, or (in a direct touch-enabled scenario) their fingers.</span></span> <span data-ttu-id="52dba-105">"*慣性*" は、要素上で摩擦力をシミュレートして、動いている UI 要素の実際の動作をエミュレートします。</span><span class="sxs-lookup"><span data-stu-id="52dba-105">*Inertia* emulates real-world behavior for UI elements that are in motion by simulating friction forces on the elements.</span></span> <span data-ttu-id="52dba-106">これにより、要素は (直線運動と角運動の両方で) 徐々に動作の速度を遅らせてから停止します。</span><span class="sxs-lookup"><span data-stu-id="52dba-106">This enables elements to gradually slow their movement (both linear and angular) before coming to a stop.</span></span>  
  
 <span data-ttu-id="52dba-107">操作と慣性の処理は、Microsoft Windows Presentation Foundation (WPF) や Microsoft XNA などのさまざまな UI フレームワークで使うことができます。</span><span class="sxs-lookup"><span data-stu-id="52dba-107">Manipulations and inertia processing can be used in a variety of UI frameworks, such as Microsoft Windows Presentation Foundation (WPF) or Microsoft XNA.</span></span>
