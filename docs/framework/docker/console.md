<span data-ttu-id="76686-147">Docker イメージを作成するには、*build.ps1* スクリプトに次のコードを追加します。</span><span class="sxs-lookup"><span data-stu-id="76686-147">In order to create the Docker image, the following code is added to the *build.ps1* script.</span></span> <span data-ttu-id="76686-148">スクリプトを実行すると、「[アプリケーションのビルド](#building-the-application)」セクションで定義した MSBuild からコンパイルされた資産を使用して、`console-random-answer-generator` イメージが作成されます。</span><span class="sxs-lookup"><span data-stu-id="76686-148">When the script is run, the `console-random-answer-generator` image is created using the assets compiled from MSBuild defined in the [Building the application](#building-the-application) section.</span></span>

```powershell
function Invoke-Docker-Build ([string]$ImageName, [string]$ImagePath, [string]$DockerBuildArgs = "") {

```
