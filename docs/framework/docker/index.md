Windows コンテナーとは何か、およびこれを実行するための要件を含む詳細については、「[Windows Containers](https://msdn.microsoft.com/en-us/virtualization/windowscontainers/about/about_overview)」(Windows コンテナー) と「[Getting Started with Docker for Windows](https://docs.docker.com/docker-for-windows/)」(Docker for Windows の概要) の記事を参照してください。
