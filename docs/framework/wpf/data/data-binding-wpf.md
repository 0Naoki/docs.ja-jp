# <a name="data-binding-wpf"></a>データ バインド (WPF)
Windows Presentation Foundation (WPF) のデータ バインディングは、アプリケーションが存在し、データと対話するためのシンプルかつ一貫した方法を提供します。 要素は、さまざまな共通言語ランタイム (CLR) オブジェクトと XML の形式でデータ ソースからデータにバインドできます。
