# リソース (WPF)
リソースはアプリケーション内のさまざまな場所で再利用できるオブジェクトです。  WPF はリソースの種類をサポートします。  これらのリソースはリソース \(主に 2 種類です : XAML リソースとリソース データ ファイル。  XAML リソースにはブラシとスタイルが含まれます。  リソース データ ファイルは非実行可能データ ファイルアプリケーションに必要です。
