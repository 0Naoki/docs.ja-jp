# WPF の XAML
[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] は、宣言的アプリケーション プログラミングで使用するマークアップ言語です。  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] では、XAML プロセッサを実装し、XAML の言語サポートを提供します。  [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] の型は、XAML 表現に必要な型バッキングを提供できるように実装されます。通常、[!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] アプリケーションのほとんどの UI を XAML マークアップで作成できます。
