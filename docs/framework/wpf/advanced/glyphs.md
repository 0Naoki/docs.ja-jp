# グリフ
グリフは、画面上に描画される文字の低レベルの表現です。  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] は、ユーザーが書式設定後にテキストを途中受信して保持できるように、グリフへの直接アクセスをサポートします。
