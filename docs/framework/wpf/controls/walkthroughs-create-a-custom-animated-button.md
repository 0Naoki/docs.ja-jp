# チュートリアル : カスタム アニメーション ボタンの作成
名前のとおり、[!INCLUDE[TLA#tla_wpf](../../../../includes/tlasharptla-wpf-md.md)] は、ユーザーに豊富なプレゼンテーション エクスペリエンスをもたらすのに最適です。  このチュートリアルでは、ボタンの外観や動作 \(アニメーションを含む\) をカスタマイズする方法について説明します。  カスタマイズでは、スタイルやテンプレートを使用するので、このカスタム ボタンをアプリケーションの他のボタンに簡単に適用できます。  これから作成する、カスタマイズされたボタンを次の図に示します。  
  
 ![ユーザーが作成するカスタマイズされたボタン](../../../../docs/framework/wpf/controls/media/custom-button-blend-intro.png "custom\_button\_blend\_Intro")  
  
 ボタンの外観を構成するベクター グラフィックスは、[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] を使用して作成します。  [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] は HTML に似ていますが、より強力で拡張性が高くなっています。  [!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] は、Microsoft Visual Studio やメモ帳を使用して手入力したり、Microsoft Expression Blend などのビジュアル デザイン ツールを使用して入力したりできます。  Expression Blend は、基になる [!INCLUDE[TLA2#tla_xaml](../../../../includes/tla2sharptla-xaml-md.md)] コードを作成することによって機能するため、どちらの方法でも同じグラフィックスが作成されます。
