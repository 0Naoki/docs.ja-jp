# <a name="combobox"></a>ComboBox
<xref:System.Windows.Controls.ComboBox>コントロール オプションの一覧をユーザーに対して表示します。 一覧が表示され、コントロールの展開および折りたたみと非表示になります。 既定の状態で一覧が折りたたまれている、のみ 1 つの選択肢を表示します。 ユーザーがオプションの完全な一覧を表示するボタンをクリックします。
