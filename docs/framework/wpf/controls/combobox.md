# ComboBox
<xref:System.Windows.Controls.ComboBox> コントロールは、ユーザーにオプションの一覧を提供します。  コントロールの展開または折りたたみを行うと、一覧の表示\/非表示が切り替わります。  既定の状態では、一覧は折りたたまれ、選択肢が 1 つだけ表示されています。  ユーザーは、ボタンをクリックして、オプションの完全な一覧を確認します。
