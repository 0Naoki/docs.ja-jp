# RadioButton
<xref:System.Windows.Controls.RadioButton> コントロールは、通常はいくつかを 1 つのグループにまとめて使用し、ユーザーは複数の選択肢の中から 1 つを選択できます。一度に選択できるボタンは 1 つだけです。
