# CheckBox
アプリケーションの[!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] で <xref:System.Windows.Controls.CheckBox> を使用すると、ユーザーが選択または選択解除できるオプションを表示できます。  チェック ボックスは、単独で使用することも、2 つ以上をグループ化することもできます。
