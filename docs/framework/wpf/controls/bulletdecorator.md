# BulletDecorator
<xref:System.Windows.Controls.Primitives.BulletDecorator> には、<xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> および <xref:System.Windows.Controls.Decorator.Child%2A> という 2 つのコンテンツ プロパティがあります。  <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> プロパティは、行頭文字として使用する <xref:System.Windows.UIElement> を定義します。  <xref:System.Windows.Controls.Decorator.Child%2A> プロパティは、行頭文字に合わせて視覚的に配置する <xref:System.Windows.UIElement> を定義します。
