# TabControl
<xref:System.Windows.Controls.TabControl> 要素は、独立した複数のページにコンテンツを表示します。各ページには、それぞれのタブをクリックしてアクセスします。  各タブには、<xref:System.Windows.Controls.TabItem> が含まれます。
