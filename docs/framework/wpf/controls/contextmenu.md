# ContextMenu
コントロールで <xref:System.Windows.Controls.ContextMenu> を使用すると、そのコントロールのコンテキスト固有の <xref:System.Windows.Controls.Menu> を表示できます。  通常、<xref:System.Windows.Controls.ContextMenu> は、マウスの右ボタンまたはキーボードのメニュー ボタンによって、[!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] で公開されます。  
  
 既定の状態と開いている状態の 2 種類の状態の <xref:System.Windows.Controls.ContextMenu> を次の図に示します。  既定の状態では、コントロールは折りたたまれています。  メニューの親の上でマウスの右ボタンが押されると、コントロールが展開してメニュー項目を表示します。
