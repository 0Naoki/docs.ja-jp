# ProgressBar
<xref:System.Windows.Controls.ProgressBar> は、操作の進行状況を示します。  <xref:System.Windows.Controls.ProgressBar> コントロールは、処理の進行状況に合わせてシステムの強調表示色で塗りつぶされていくウィンドウで構成されます。
