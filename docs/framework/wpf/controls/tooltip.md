# ヒント
ツールヒントは、<xref:System.Windows.Controls.Button> などの要素の上にマウス ポインターを置いたときに表示される小さいポップアップ ウィンドウです。  
  
 **\[閉じる\]** <xref:System.Windows.Controls.Button> の上にマウス ポインターを移動し、ボタンの機能を示す <xref:System.Windows.Controls.ToolTip> が表示されている状況を次の図に示します。
