# FlowDocumentPageViewer
<xref:System.Windows.Controls.FlowDocumentPageViewer> コントロールは、1 ページ単位で <xref:System.Windows.Documents.FlowDocument> のコンテンツを表示するために使用します。  これに対して、<xref:System.Windows.Controls.FlowDocumentScrollViewer> は、<xref:System.Windows.Documents.FlowDocument> のコンテンツをスクロール ビューアーで表示します。
