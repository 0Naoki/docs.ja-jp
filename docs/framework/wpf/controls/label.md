# ラベル
通常、<xref:System.Windows.Controls.Label> コントロールは [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] で情報を提供します。  過去には <xref:System.Windows.Controls.Label> に含まれるのはテキストだけでしたが、[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] に付属する <xref:System.Windows.Controls.Label> は <xref:System.Windows.Controls.ContentControl> であることから、テキストまたは <xref:System.Windows.UIElement> を含むことができます。  
  
 <xref:System.Windows.Controls.Label> は、アクセス キーに対する機能上のサポートと表示上のサポートの両方を提供します。  <xref:System.Windows.Controls.TextBox> などのコントロールに対するすばやいキーボード アクセスを実現するために頻繁に使用されます。  <xref:System.Windows.Controls.Label> を <xref:System.Windows.Controls.Control> に割り当てるには、<xref:System.Windows.Controls.Label.Target%2A?displayProperty=fullName> プロパティを、ユーザーがアクセス キーを押したときにフォーカスを取得するコントロールに設定します。  
  
 次のイメージは、<xref:System.Windows.Controls.ComboBox> を表示対象とする <xref:System.Windows.Controls.Label> "テーマ" を示しています。  ユーザーがキーを押した場合、<xref:System.Windows.Controls.ComboBox> はフォーカスを受け取ります。  詳細については、「[How to: Set the Target Property of a Label](http://msdn.microsoft.com/ja-jp/b24c6977-ebcb-4855-a9bb-3fd4435af8f8)」を参照してください。
