# フレーム
<xref:System.Windows.Controls.Frame> コントロールはコンテンツ内でのコンテンツ ナビゲーションをサポートします。  <xref:System.Windows.Controls.Frame> は、<xref:System.Windows.Window>、<xref:System.Windows.Navigation.NavigationWindow>、<xref:System.Windows.Controls.Page>、<xref:System.Windows.Controls.UserControl>、<xref:System.Windows.Documents.FlowDocument> などのルート要素によって、またはルート要素に属するコンテンツ ツリー内のアイランドとしてホストできます。
