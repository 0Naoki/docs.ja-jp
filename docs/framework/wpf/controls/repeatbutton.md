# RepeatButton
<xref:System.Windows.Controls.Primitives.RepeatButton> は、<xref:System.Windows.Controls.Button> に似ています。  ただし、<xref:System.Windows.Controls.Primitives.RepeatButton> 要素を使用することで、<xref:System.Windows.Controls.Primitives.ButtonBase.Click> イベントが発生するタイミングと方法を制御することができます。  
  
 次の図は、Default、PointerFocused、Pressed という、リピート ボタン コントロールの 3 種類の状態を示します。  最初のボタンは、<xref:System.Windows.Controls.Primitives.RepeatButton> の既定の状態を示しています。  2 番目のボタンは、マウス ポインターがボタンの上に置かれ、フォーカスが与えられたときに、ボタンの外観がどのように変化するかを示しています。  最後のボタンは、ユーザーがコントロールの上でマウス ボタンを押したときの、<xref:System.Windows.Controls.Primitives.RepeatButton> の外観を示しています。
