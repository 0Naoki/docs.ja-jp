# メニュー
<xref:System.Windows.Controls.Menu> コントロールを使用すると、コマンドまたはイベント ハンドラーに関連付けられている要素を階層的に編成できます。  各 <xref:System.Windows.Controls.Menu> には、複数の <xref:System.Windows.Controls.MenuItem> コントロールを格納できます。  各 <xref:System.Windows.Controls.MenuItem> は、コマンドまたは `Click` イベント ハンドラーを呼び出すことができます。  また、<xref:System.Windows.Controls.MenuItem> は、複数の <xref:System.Windows.Controls.MenuItem> 要素を子として持つことができ、サブメニューを構成できます。  
  
 次の図は、メニュー コントロールの 3 つの異なる状態を示します。  既定の状態は、マウス ポインターなどのデバイスを <xref:System.Windows.Controls.Menu> 上に置いていない場合です。  フォーカス状態は、マウス ポインターを <xref:System.Windows.Controls.Menu> 上に置いているときに発生し、押された状態は、マウス ボタンを <xref:System.Windows.Controls.Menu> 上でクリックしたときに発生します。
