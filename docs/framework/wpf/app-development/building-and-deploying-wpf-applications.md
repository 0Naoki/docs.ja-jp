# WPF アプリケーションのビルドと配置
次のようなビルドと配置モデルにより、アプリケーションをローカルまたはリモートにビルドおよび配置することができます。  
  
-   MSBuild : Microsoft.Build.Tasks.Windows 名前空間に配置される .NET ビルド システム。  
  
-   リソース : UI リソースの使用。  
  
-   ClickOnce の配置 : .NET の公開および配置システム。
