# WPF アプリケーションのウィンドウ
ユーザーは、ウィンドウを通してアプリケーションと対話します。  ウィンドウの基本目的は、コンテンツをホストして表示することです。  ウィンドウがホストするコンテンツの種類は、アプリケーションが処理するデータの種類によって異なります。このようなデータの種類には、メディア、XAML \(Extensible Application Markup Language\) ページ、Web ページ、ドキュメント、データベースのテーブルやレコード、およびシステム情報などが含まれます。
