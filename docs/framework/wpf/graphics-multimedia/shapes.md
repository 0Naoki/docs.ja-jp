# 図形
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] には、基本的な図形の描画をサポートするための <xref:System.Windows.Shapes.Shape> クラスがあります。  図形オブジェクトは <xref:System.Windows.FrameworkElement> から派生します。したがって、レイアウト システムの一部であり、[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] を使用して簡単に描画できます。
