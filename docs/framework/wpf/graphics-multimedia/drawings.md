# 描画
<xref:System.Windows.Media.Drawing> オブジェクトを使用して、効率的に、図形、イメージ、またはテキストを描画します。  描画は、<xref:System.Windows.Media.DrawingBrush> による塗りつぶし、または <xref:System.Windows.Media.Visual> オブジェクトによるプログラミングで使用します。
