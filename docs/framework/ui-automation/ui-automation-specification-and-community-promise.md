# <a name="ui-automation-specification-and-community-promise"></a>UI Automation Specification および UI Automation Community Promise
UI オートメーションはエンドユーザーに、UI に関する情報を提供する、標準入力以外の方法で UI を操作したり、スクリーン リーダーなどの支援技術製品の有効化のデスクトップで、ほとんどのユーザー インターフェイス (UI) 要素にプログラムでアクセスを提供します.
