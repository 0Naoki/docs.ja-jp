# <a name="install-the-net-framework-on-windows-7-sp1"></a>Windows 7 SP1 への .NET Framework のインストール

Windows 7 SP1 でアプリケーションを実行するために .NET Framework が必要になることがあります。 次の手順を使用して、サポートされている最新バージョンをインストールできます。 必要な .NET Framework のバージョンによって手順は異なります。

## <a name="net-framework-462"></a>.NET Framework 4.6.2

Windows 7 には、[.NET Framework 4.6.2](https://go.microsoft.com/fwlink/?linkid=845529&source=dotnetdocs) 以降をインストールできます。

.NET Framework 4.6.2 は、[Windows 10 Anniversary Update](https://www.microsoft.com/software-download/windows10) に含まれています。

.NET Framework 4.6.2 は、.NET Framework 4.0 以降用に構築されたアプリケーションを実行するために使用できます。

## <a name="net-framework-35"></a>.NET Framework 3.5

.NET Framework 3.5 は、Windows 7 に含まれています。

.NET Framework 3.5 は、.NET Framework 1.0 から 3.5 用に構築されたアプリケーションを実行するために使用できます。

## <a name="help"></a>ヘルプ

インストールされている .NET Framework の正しいバージョンがわからない場合は、[Microsoft にお問い合わせください](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework)。
