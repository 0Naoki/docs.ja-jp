# <a name="installing-the-net-framework-on-windows-10"></a>Windows 10 への .NET Framework のインストール

Windows 10 でアプリケーションを実行するために .NET Framework が必要になることがあります。 このような場合は次の手順を実行します。 必要な .NET Framework のバージョンによって手順は異なります。

## <a name="net-framework-462"></a>.NET Framework 4.6.2

.NET Framework 4.6.2 は、Windows 10 Anniversary Update に含まれています。 

以前のバージョンの Windows 10 を使用している場合、.NET Framework 4.6.2 を入手するには、[Windows 10 Anniversary Update](https://www.microsoft.com/software-download/windows10) にアップグレードする必要があります。 

.NET Framework 4.6.2 は、.NET Framework 4.0 以降用に構築されたアプリケーションを実行するために使用できます。

## <a name="net-framework-35"></a>.NET Framework 3.5

手順に従って [.NET Framework 3.5 を Windows 10 に](dotnet-35-windows-10.md)インストールしてください。 

.NET Framework 3.5 は、.NET Framework 1.0 から 3.5 用に構築されたアプリケーションを実行するために使用できます。

## <a name="help"></a>ヘルプ

インストールされている .NET Framework の正しいバージョンがわからない場合は、[Microsoft にお問い合わせください](mailto:dotnet-install-help@microsoft.com?subject=Help installing .NET Framework)。
