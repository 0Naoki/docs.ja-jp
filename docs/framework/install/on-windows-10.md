# <a name="install-the-net-framework-on-windows-10-and-windows-server-2016"></a>.NET Framework をインストールする Windows 10 および Windows Server 2016

Windows で多くのアプリケーションを実行するには、.NET Framework が必要です。 この記事の手順には、必要な .NET Framework のバージョンをインストールするのに役立ちます。 [.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47)は最新のバージョン。

アプリケーションを実行しようとし、次のようなコンピューターに、ダイアログを表示した後に、このページに到着したが可能性があります。

![このアプリケーションを開始できませんでした。](./media/this-application-could-not-be-started.png)

## <a name="net-framework-471"></a>.NET framework 4.7.1

.NET Framework 4.7.1 は、付属します。

* [Windows 10 秋作成者アップデート (バージョン 1709)](https://www.microsoft.com/software-download/windows10)
* [Windows Server 2016 (バージョン 1709)](https://docs.microsoft.com/windows-server/get-started/get-started-with-1709)

> [!div class="button"]
[.NET Framework 4.7.1 をダウンロードします。](https://www.microsoft.com/net/download/thank-you/net471?utm_source=ms-docs&utm_medium=referral)

[.NET Framework 4.7.1](https://www.microsoft.com/download/details.aspx?id=56115&desc=dotnet47) 4.7.1 経由の .NET Framework 4.0 用にビルドされたアプリケーションを実行するために使用できます。

インストールすることができます、 [.NET Framework 4.7.1](https://www.microsoft.com/en-us/download/details.aspx?id=56115&desc=dotnet47)で。

* Windows 10 の作成者の更新 (バージョン 1703)
* Windows 10 Anniversary 更新 (version 1607)
* Windows Server 2016

は、.NET Framework 4.7.1 はサポートされていません。

* Windows 10 1507
* Windows 10 1511

Windows 10 1507 または 1511 を使用している、.NET Framework 4.7.1 をインストールする場合は、まず、以降の Windows 10 バージョンにアップグレードする必要があります。

## <a name="net-framework-461"></a>.NET Framework 4.6.1

[.NET Framework 4.6.1](https://www.microsoft.com/download/details.aspx?id=49981) Windows 10 1507 および 1511 に最新サポートされている .NET Framework バージョンです。

.NET Framework 4.6.1 は、4.6.1 で .NET Framework 4.0 用にビルドされたアプリをサポートします。

## <a name="net-framework-35"></a>.NET Framework 3.5

手順に従って [.NET Framework 3.5 を Windows 10 に](dotnet-35-windows-10.md)インストールしてください。

.NET Framework 3.5 には、.NET Framework 1.0 3.5 ~ 用にビルドされたアプリがサポートされています。

## <a name="additional-information"></a>追加情報

.NET framework 4.x のバージョンは、以前のバージョンへのインプレース更新です。 次をことを意味します。

- 1 つのバージョンの .NET Framework が持てる 4.x をコンピューターにインストールします。

- 以降のバージョンが既にインストールされている場合、コンピューターに .NET Framework の以前のバージョンをインストールすることはできません。

- 4.x、.NET Framework のバージョンでそのバージョンの .NET Framework 4.0 用にビルドされたアプリケーションの実行に使用できます。 たとえば、4.7 経由の .NET Framework 4.0 用にビルドされたアプリケーションを実行する .NET Framework 4.7 を使用できます。 構築されたアプリケーションはすべて、.NET Framework のバージョン 4.0 以降を実行する、最新バージョン (4.7.1 .NET Framework) を使用できます。

ダウンロード可能な .NET Framework のすべてのバージョンの一覧は、次を参照してください。、 [.NET ダウンロード](https://www.microsoft.com/net/download?utm_source=ms-docs&utm_medium=referral)ページ。
