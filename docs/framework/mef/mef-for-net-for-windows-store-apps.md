# Windows ストア アプリ用 .NET の MEF 名前空間
<xref:System.Composition?displayProperty=fullName> とその子の名前空間には、Managed Extensibility Framework \(MEF\) を使用して拡張可能な [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] アプリを開発するための型が含まれています。  これらの名前空間は、[!INCLUDE[win8](../../../includes/win8-md.md)] オペレーティング システムの [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] のサブセットの一部です。  
  
 これらの名前空間は、.NET Framework で配布されているコア クラス ライブラリの一部ではありません。  これらの名前空間をインストールするには、[!INCLUDE[vs_dev11_long](../../../includes/vs-dev11-long-md.md)] でプロジェクトを開き、**\[プロジェクト\]** メニューの **\[NuGet パッケージの管理\]** を選択し、Microsoft.Composition パッケージをオンラインで検索します。  
  
-   <xref:System.Composition?displayProperty=fullName> は [!INCLUDE[win8_appname_long](../../../includes/win8-appname-long-md.md)] アプリのコア MEF を構成するクラスを提供します。  
  
-   <xref:System.Composition.Convention?displayProperty=fullName> は、規則ベースの構成モデルでの MEF の使用をサポートする型を提供します。  
  
-   <xref:System.Composition.Hosting?displayProperty=fullName> は、ホスト アプリケーションの開発者に役立つ MEF 型を提供します。  
  
-   <xref:System.Composition.Hosting.Core?displayProperty=fullName> は、コンポジション エンジンによって内部的に使用される MEF 型を提供します。  
  
 [!INCLUDE[net_win8_profile](../../../includes/net-win8-profile-md.md)] に関する詳細についてと名前空間を含むリストと型は、Windows [Windows ストア アプリのための .NET はアプリケーションの概要を格納します。](http://go.microsoft.com/fwlink/p/?LinkID=238312) デベロッパー センターで、参照します。
