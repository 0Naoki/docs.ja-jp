# パスの作成および描画
パスとは、1 つの単位として操作および描画できる、連続した複数のグラフィックス プリミティブ \(直線、四角形、曲線、テキストなど\) のことです。  パスは、開いた*図形*または閉じた図形のいずれかに分類できます。  図形には、複数のプリミティブが含まれています。  
  
 パスを描画するには <xref:System.Drawing.Graphics> クラスの <xref:System.Drawing.Graphics.DrawPath%2A> メソッドを呼び出し、パスを塗りつぶすには <xref:System.Drawing.Graphics> クラスの <xref:System.Drawing.Graphics.FillPath%2A> メソッドを呼び出します。
