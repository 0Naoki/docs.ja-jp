# ペンを使用した直線と図形の描画
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] の `Pen` オブジェクトを使用して、線分、曲線、および図形のアウトラインを描画します。  ここでの*線*は、特に線分だけを示すように指定されていない場合は、線分、曲線、および図形のアウトラインのすべてを指します。  ペンのプロパティを設定して、そのペンで描画した線の色、幅、配置、およびスタイルを制御できます。
