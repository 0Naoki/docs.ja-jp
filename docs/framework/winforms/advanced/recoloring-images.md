# イメージの色の変更
色の変更とは、イメージの色を調整するプロセスのことです。  色を変更する例として、ある色から別の色へと変更する、別の色を基準にして色の輝度を相対的に調整する、すべての色の明るさやコントラストを調整する、色をグレー階調に変更する、などがあります。
