### <a name="portable-network-graphics-png"></a><span data-ttu-id="bc3a5-180">ポータブル ネットワーク グラフィックス (PNG)</span><span class="sxs-lookup"><span data-stu-id="bc3a5-180">Portable Network Graphics (PNG)</span></span>

 <span data-ttu-id="bc3a5-186">段階的にイメージを表示するには、その機能の gif png (つまり、として、イメージの向上の推定を表示するが到着したネットワーク接続経由で)。</span><span class="sxs-lookup"><span data-stu-id="bc3a5-186">PNG improves on GIF in its ability to progressively display an image (that is, to display better and better approximations of the image as it arrives over a network connection).</span></span> <span data-ttu-id="bc3a5-187">PNG ファイルは、イメージをディスプレイ デバイスのさまざまなで正確にレンダリングできるように、ガンマ補正と色補正の情報を含めることができます。</span><span class="sxs-lookup"><span data-stu-id="bc3a5-187">PNG files can contain gamma correction and color correction information so that the images can be accurately rendered on a variety of display devices.</span></span>  
  
### <a name="tag-image-file-format-tiff"></a><span data-ttu-id="bc3a5-188">タグの Image File Format (TIFF)</span><span class="sxs-lookup"><span data-stu-id="bc3a5-188">Tag Image File Format (TIFF)</span></span>  
 <span data-ttu-id="bc3a5-189">TIFF は、さまざまなプラットフォームと画像処理アプリケーションでサポートされている柔軟性と拡張性の高い形式です。</span><span class="sxs-lookup"><span data-stu-id="bc3a5-189">TIFF is a flexible and extendable format that is supported by a wide variety of platforms and image-processing applications.</span></span> <span data-ttu-id="bc3a5-190">TIFF ファイルでは、任意の数のピクセルあたりのビットを使用したイメージを格納できる、さまざまな圧縮アルゴリズムを使用できます。</span><span class="sxs-lookup"><span data-stu-id="bc3a5-190">TIFF files can store images with an arbitrary number of bits per pixel and can employ a variety of compression algorithms.</span></span> <span data-ttu-id="bc3a5-191">いくつかのイメージは、1 つ、複数のページの TIFF ファイルに格納できます。</span><span class="sxs-lookup"><span data-stu-id="bc3a5-191">Several images can be stored in a single, multiple-page TIFF file.</span></span> <span data-ttu-id="bc3a5-192">(スキャナーの種類、ホスト コンピューターの種類の圧縮、向き、ピクセルあたりのサンプル) のイメージに関連する情報をファイルに格納し、タグを使用して配置されます。</span><span class="sxs-lookup"><span data-stu-id="bc3a5-192">Information related to the image (scanner make, host computer, type of compression, orientation, samples per pixel, and so on) can be stored in the file and arranged through the use of tags.</span></span> <span data-ttu-id="bc3a5-193">TIFF 形式は、承認して新しいタグの追加、必要に応じて拡張できます。</span><span class="sxs-lookup"><span data-stu-id="bc3a5-193">The TIFF format can be extended as needed by the approval and addition of new tags.</span></span>
