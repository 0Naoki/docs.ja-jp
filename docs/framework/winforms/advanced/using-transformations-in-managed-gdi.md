# マネージ GDI+ での変換の使用
アフィン変換には、回転、スケーリング、反転、傾斜、平行移動などがあります。  [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] では、ベクター図面、イメージ、テキストに対してアフィン変換を実行するための基盤を <xref:System.Drawing.Drawing2D.Matrix> クラスが提供しています。
