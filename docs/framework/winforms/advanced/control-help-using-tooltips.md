# ツールヒントを使用したコントロールのヘルプ
<xref:System.Windows.Forms.ToolTip> コンポーネントを使用することにより、Windows フォーム上の個別のコントロールについて簡単な固有のヘルプ メッセージを表示できます。  <xref:System.Windows.Forms.ToolTip> コンポーネントには、フォーム上の各コントロールに対して表示するテキストを指定するためのプロパティがあります。  このような、<xref:System.Windows.Forms.ToolTip> コンポーネントの操作方法の詳細については、「[方法 : デザイン時に Windows フォームのコントロールにツールヒントを設定する](../../../../docs/framework/winforms/controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md)」を参照してください。  さらに、<xref:System.Windows.Forms.ToolTip> コンポーネントでは、表示するまでの時間を設定できます。  詳細については、「[方法 : Windows フォームの ToolTip コンポーネントの遅延時間を変更する](../../../../docs/framework/winforms/controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md)」を参照してください。
