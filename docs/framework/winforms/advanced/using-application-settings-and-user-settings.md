# アプリケーション設定とユーザー設定の使用
.NET Framework 2.0 以降、アプリケーションの実行セッション間で保持される値を作成してアクセスできます。  これらの値は、*設定*と呼ばれます。  設定は、ユーザー設定を表すことも、アプリケーションが使用する重要な情報を表すこともできます。  たとえば、アプリケーションの配色に対するユーザー設定を格納する一連の設定を作成できます。  または、アプリケーションが使用するデータベースを指定する接続文字列を格納できます。  設定を使用すると、アプリケーションにとって重要な情報のコード外での保持と、個々のユーザーのユーザー設定を保存するプロファイルの作成の両方を実行できます。  
  
 このセクションのトピックでは、設計時と実行時の設定の使用方法について説明します。
