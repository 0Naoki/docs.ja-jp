# Windows フォーム アプリケーションにおけるネットワーク
.NET Framework には、Web ページの表示、Web コンテンツのダウンロード、ファイル転送プロトコル \(FTP\) サイトとのやり取り、および Web サービスの利用のためのクラスが用意されており、ネットワーク機能をアプリケーションに簡単に組み込むことができます。  以下の各項目は、.NET Framework のネットワーク テクノロジと、それを Windows フォームに取り入れる方法を理解するために役立ちます。  
  
## 関連項目  
 <xref:System.Net>  
 ネットワーク接続を処理する、.NET Framework のクラスのルート名前空間。  
  
 <xref:System.Net.WebClient>  
 プログラムで Web コンテンツまたは HTTP ベースのコンテンツを取得するための便利なクラス。  
  
 <xref:System.Net.FtpWebRequest>  
 FTP を使用してファイルを取得および送信するためのクラス。  
  
 <xref:System.Windows.Forms.WebBrowser>  
 Windows に含まれる `WebBrowser` コントロールのマネージ ラッパー クラス。
