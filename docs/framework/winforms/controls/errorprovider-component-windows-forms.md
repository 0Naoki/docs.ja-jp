# ErrorProvider コンポーネント (Windows フォーム)
Windows フォームの `ErrorProvider` コンポーネントを使用すると、処理を中断せずに、ユーザーに問題の発生を通知できます。  通常は、ユーザーがフォームに入力した値の妥当性を検査するとき、またはデータセット内のエラーを表示するときに使用します。
