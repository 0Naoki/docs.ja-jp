# <a name="errorprovider-component-windows-forms"></a>ErrorProvider コンポーネント (Windows フォーム)
Windows フォーム`ErrorProvider`コンポーネントを使用して、問題がある非侵入型の方法でユーザーを表示します。 通常、フォーム上のユーザー入力の検証またはデータセット内のエラーを表示すると組み合わせて使用されます。
