# TabControl コントロール (Windows フォーム)
Windows フォーム `TabControl` は、ノートの仕切りや書類キャビネットのフォルダー セットのラベルに似た、複数のタブを表示します。  タブには画像やその他のコントロールを含めることができます。  プロパティ ページを作成するには、`TabControl` を使用します。
