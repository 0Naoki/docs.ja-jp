# FileDialog クラス
Windows フォームの <xref:System.Windows.Forms.FileDialog> クラスは、<xref:System.Windows.Forms.OpenFileDialog> コンポーネントと <xref:System.Windows.Forms.SaveFileDialog> コンポーネントに共通する基本クラスです。  アプリケーションが実行されている Windows のバージョンによっては、これらのダイアログ ボックスの外観と動作に影響する <xref:System.Windows.Forms.FileDialog> クラスを変更できます。

 ファイル ダイアログ ボックスのスタイルの自動アップグレードを無効にする方法について説明します。

 ファイル ダイアログ ボックスにカスタムのファイルの場所を追加する方法について説明します。  
  
 [ファイル ダイアログ ボックスのカスタム プレイス用既知のフォルダー GUID](../../../../docs/framework/winforms/controls/known-folder-guids-for-file-dialog-custom-places.md)
