# ListView コントロール (Windows フォーム)
Windows フォーム `ListView` コントロールにはアイコン表示で項目の一覧が表示されます。  リスト ビューを使用すると、Windows エクスプローラーの右側のペインのようなユーザー インターフェイスを作成することができます。
