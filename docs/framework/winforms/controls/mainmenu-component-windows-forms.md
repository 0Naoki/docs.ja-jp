# MainMenu コンポーネント (Windows フォーム)
> [!NOTE]
>  `MenuStrip` と `ContextMenuStrip` は、以前のバージョンの `MainMenu` コントロールおよび <xref:System.Windows.Forms.ContextMenu> コントロールに代わると共に追加の機能を提供しますが、`MainMenu` および <xref:System.Windows.Forms.ContextMenu> は、下位互換性を保つ目的および将来使用する目的で、必要に応じて保持できます。  
  
 Windows フォームの `MainMenu` コンポーネントを使用すると、実行時にメニューが表示されます。
