# PageSetupDialog コンポーネント (Windows フォーム)
Windows フォームの <xref:System.Windows.Forms.PageSetupDialog> コンポーネントは、Windows ベースのアプリケーションで印刷時のページ設定に使用する定義済みダイアログ ボックスです。  このコントロールは、独自のダイアログ ボックスを使用せずにページ設定を行うための簡易ソリューションとして、Windows ベースのアプリケーションの中で使用します。  罫線と余白、ヘッダーとフッター、および印刷の向きを  設定できます。  Windows の標準のダイアログ ボックスを使用して、一般的な基本機能を持つアプリケーションを作成できます。
