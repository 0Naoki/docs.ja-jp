# <a name="how-to-copy-controls-between-windows-forms"></a>方法 : Windows フォーム間でコントロールをコピーする
コントロールは、同じフォーム、プロジェクト内で別の形式、またはその他のソリューションで使用するため、クリップボードにコピー可能性があります。  
  
### <a name="to-copy-a-control"></a>コントロールをコピーするには  
  
1.  コントロールを選択してから、**編集**メニューを**コピー**です。  
  
 このコントロールをフォームにコントロールの型を受け入れるようになりました貼り付けできます。 さらに、コントロールがクリップボードに追加されました。
