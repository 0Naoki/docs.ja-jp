# 方法 : Windows フォーム間でコントロールをコピーする
同じフォーム、プロジェクト内の別のフォーム、または他のソリューションで使用するクリップボードにコントロールをコピーできます。  
  
### コントロールをコピーするには  
  
1.  コントロールを選択し、**\[編集\]** メニューの **\[コピー\]** を選択します。  
  
 選択した種類のコントロールを受け入れるフォームに、コントロールを貼り付けます。  このとき、コントロールはクリップボードにも追加されます。
