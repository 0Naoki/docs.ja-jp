# Windows フォーム コントロールのレイアウト
フォーム上にコントロールを正確に配置することは、多くのアプリケーションで重要度の高い処理です。  <xref:System.Windows.Forms?displayProperty=fullName> 名前空間では、これを行うためのさまざまなレイアウト ツールを使用できます。
