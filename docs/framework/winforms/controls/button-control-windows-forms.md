# Button コントロール (Windows フォーム)
Windows フォームの `Button` コントロールを使用すると、ユーザーはそれをクリックしてアクションを実行できます。  `Button` コントロールには、テキストとイメージの両方を表示できます。  ボタンをクリックすると、ボタンを実際に押して離したかのように表示されます。
