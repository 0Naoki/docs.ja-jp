# LinkLabel コントロール (Windows フォーム)
Windows フォームの `LinkLabel` コントロールを使用すると、Web スタイルのリンクを Windows フォーム アプリケーションに追加できます。  `LinkLabel` コントロールは、<xref:System.Windows.Forms.Label> コントロールを使用できるすべての項目に使用できます。テキストの一部をオブジェクトまたは Web ページへのリンクとして設定することもできます。
