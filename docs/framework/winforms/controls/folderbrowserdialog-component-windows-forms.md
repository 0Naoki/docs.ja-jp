# FolderBrowserDialog コンポーネント (Windows フォーム)
Windows フォームの `FolderBrowserDialog` コンポーネントは、ユーザーがフォルダーを参照して選択したり新規フォルダーを作成したりできるインターフェイスを表示します。  このコンポーネントは、ファイルの参照と選択に使用される [OpenFileDialog コンポーネント](../../../../docs/framework/winforms/controls/openfiledialog-component-windows-forms.md) コンポーネントを補完するものです。
