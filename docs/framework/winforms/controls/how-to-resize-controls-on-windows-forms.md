### <a name="to-resize-a-control"></a><span data-ttu-id="1892c-104">コントロールのサイズを変更するには</span><span class="sxs-lookup"><span data-stu-id="1892c-104">To resize a control</span></span>

    > [!NOTE]
    > <span data-ttu-id="1892c-106">コントロールを選択し、SHIFT キーを押しながら方向キーを押すと、コントロールのサイズが一度に1ピクセルずつ変更されます。</span><span class="sxs-lookup"><span data-stu-id="1892c-106">Select the control and press the ARROW keys while holding down the SHIFT key to resize the control one pixel at a time.</span></span> <span data-ttu-id="1892c-107">SHIFT キーまたは CTRL キーを押しながら下方向キーまたは右方向キーを押して、コントロールのサイズを大きくします。</span><span class="sxs-lookup"><span data-stu-id="1892c-107">Press the DOWN or RIGHT arrow keys while holding down the SHIFT and CTRL keys to resize the control in large increments.</span></span>

2. <span data-ttu-id="1892c-111">**[書式]** メニューの **[同じサイズに揃える]** をクリックし、4つのオプションのいずれかを選択します。</span><span class="sxs-lookup"><span data-stu-id="1892c-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="1892c-112">最初の3つのコマンドは、最初に選択されたコントロールに合わせてコントロールのサイズを変更します。</span><span class="sxs-lookup"><span data-stu-id="1892c-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
