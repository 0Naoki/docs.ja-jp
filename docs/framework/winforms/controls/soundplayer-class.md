# <a name="soundplayer-class"></a>SoundPlayer クラス
`SoundPlayer` クラスを使用すると、アプリケーションにサウンドを簡単に組み込むことができます。  
  
 <xref:System.Media.SystemSounds> クラスを使用して、ビープ音を含む、共通のシステム サウンドを再生することもできます。
