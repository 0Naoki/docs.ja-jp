# Windows フォーム DataGridView コントロールでのデータの表示
`DataGridView` コントロールを使用して、さまざまな外部データ ソースのデータを表示できます。  また、このコントロールに行と列を追加し、手動でデータを読み込むこともできます。  
  
 このコントロールをデータ ソースにバインドすると、データ ソースのスキーマに基づいて自動的に列を生成できます。  これらの列が意図したとおりに表示されない場合は、非表示にしたり、削除したり、並べ替えたりできます。  バインドしていない列を追加して、データ ソースのデータではない補足データを表示することもできます。  
  
 また、\(通貨書式などの\) 標準の書式を使用してデータを表示したり、必要な形式に応じて表示書式をカスタマイズしたりできます。たとえば、負の数の背景色を変更したり、文字列値を対応する画像に置き換えたりできます。
