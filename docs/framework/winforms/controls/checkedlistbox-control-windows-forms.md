# CheckedListBox コントロール (Windows フォーム)
Windows フォームの `CheckedListBox` コントロールは、<xref:System.Windows.Forms.ListBox> コントロールと同様に項目のリストを表示します。さらに、リスト内の各項目の横にチェック マークも表示できます。
