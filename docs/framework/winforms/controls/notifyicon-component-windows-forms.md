# NotifyIcon コンポーネント (Windows フォーム)
Windows フォームの `NotifyIcon` コンポーネントは、他にユーザー インターフェイスを持たない、バックグラウンドで動作するプロセスのために、タスク バーの状態通知領域にアイコンを表示します。  例としては、タスクバーの状態通知領域にあるアイコンをクリックしてアクセスできるウイルス防止プログラムなどがあります。
