# ComboBox コントロール (Windows フォーム)
Windows フォーム `ComboBox` コントロールを使用すると、ドロップダウン コンボ ボックスのデータを表示できます。  既定では、`ComboBox` コントロールは 2 つの部分に分かれています。上の部分は、リスト項目を入力できるテキスト ボックスです。  下の部分は、項目のリストが表示されるリスト ボックスであり、このボックスから 1 つの項目を選択できます。
