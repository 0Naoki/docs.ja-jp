# 方法 : ToolStrip に ToolStripItem を配置する
<xref:System.Windows.Forms.ToolStripItem> は <xref:System.Windows.Forms.ToolStrip> の左側または右側に移動したり追加したりできます。  
  
### ToolStripItem を ToolStrip の左側に移動または追加するには  
  
1.  <xref:System.Windows.Forms.ToolStripItem> の <xref:System.Windows.Forms.ToolStripItemAlignment> プロパティを <xref:System.Windows.Forms.ToolStripItemAlignment> に設定します。  
  
### ToolStripItem を ToolStrip の右側に移動または追加するには  
  
1.  <xref:System.Windows.Forms.ToolStripItem> の <xref:System.Windows.Forms.ToolStripItemAlignment> プロパティを <xref:System.Windows.Forms.ToolStripItemAlignment> に設定します。
