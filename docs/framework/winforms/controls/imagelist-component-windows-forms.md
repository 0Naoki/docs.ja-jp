# ImageList コンポーネント (Windows フォーム)
Windows フォーム `ImageList` コンポーネントは、コントロールで表示するイメージの保存に使用します。  イメージ リストでは、一貫性のある 1 つのイメージのカタログのコードを記述することができます。
