# ToolStripStatusLabel コントロールの概要
<xref:System.Windows.Forms.ToolStripStatusLabel> は <xref:System.Windows.Forms.StatusStrip> 用のラベルです。  <xref:System.Windows.Forms.Label> や <xref:System.Windows.Forms.ToolStripLabel> と同様に、<xref:System.Windows.Forms.ToolStripStatusLabel> を使用すると、テキスト、イメージ、またはその両方にクリックできない表示領域を作成できます。  <xref:System.Windows.Forms.ToolStripStatusLabel> は <xref:System.Windows.Forms.StatusStrip> によってホストされます。  
  
### ToolStripStatusLabel の重要なメンバー  
  
|名前|Description|  
|--------|-----------------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|フォームのサイズが変更されたときに、<xref:System.Windows.Forms.ToolStripStatusLabel> が自動的に <xref:System.Windows.Forms.StatusStrip> 上の使用可能な領域を占めるかどうかを表す値を取得または設定します。|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|<xref:System.Windows.Forms.ToolStripStatusLabel> のどちらの側に境界線を表示するかを示す値を取得または設定します。|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|<xref:System.Windows.Forms.ToolStripStatusLabel> の境界線スタイルを取得または設定します。|
