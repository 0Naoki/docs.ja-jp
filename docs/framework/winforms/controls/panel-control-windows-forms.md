# Panel コントロール (Windows フォーム)
Windows フォームの `Panel` コントロールを使用すると、他のコントロールと区別できるようにグループ化できます。  通常は、機能別にフォームを細分化するためにパネルを使用します。  `Panel` コントロールは <xref:System.Windows.Forms.GroupBox> コントロールに似ていますが、スクロール バーを表示できるのは `Panel` コントロールだけであり、キャプションを表示できるのは <xref:System.Windows.Forms.GroupBox> コントロールだけです。
