# TreeView コントロール (Windows フォーム)
Windows フォームの `TreeView` コントロールでは、Windows オペレーティング システムの Windows エクスプローラーの左側のウィンドウにファイルやフォルダーが表示されるのと同じように、ノードの階層を表示することができます。
