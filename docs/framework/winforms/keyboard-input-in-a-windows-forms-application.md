# Windows フォーム アプリケーションにおけるキーボード入力
Windows フォームは標準のキーボード イベントを備えており、これを使用すると、特定のキーの押下に応答できます。また、アプリケーション、フォーム、およびコントロールの各レベルで、キーの押下を受け取ったり、変更したり、処理したりする手段も備えています。

 キーボード イベントの種類と、キーボード イベント ハンドラーが受け取る情報について説明します。

 キー値がコントロールに達する前にキー値を変更する方法を示すコード例です。
