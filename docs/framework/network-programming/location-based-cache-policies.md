 <span data-ttu-id="3d51c-110">有効な要求されたリソースがローカル キャッシュ内にある場合、キャッシュされたリソースが使用されます。それ以外の場合、そのリソースの要求はサーバーに送信されます。</span><span class="sxs-lookup"><span data-stu-id="3d51c-110">If a valid requested resource is in the local cache, the cached resource is used; otherwise, the request for the resource is sent to the server.</span></span> <span data-ttu-id="3d51c-111">要求されたリソースがクライアントとサーバーの間の何らかのキャッシュ内にある場合、中間のキャッシュが要求を満たすことができます。</span><span class="sxs-lookup"><span data-stu-id="3d51c-111">If the requested resource is available in any cache between the client and the server, the request can be satisfied by an intermediate cache.</span></span>  
  
## <a name="cache-only-policy"></a><span data-ttu-id="3d51c-112">"キャッシュのみを使用" ポリシー</span><span class="sxs-lookup"><span data-stu-id="3d51c-112">Cache Only Policy</span></span>  
 <span data-ttu-id="3d51c-113">有効な要求されたリソースがローカル キャッシュ内にある場合、キャッシュされたリソースが使用されます。</span><span class="sxs-lookup"><span data-stu-id="3d51c-113">If a valid requested resource is in the local cache, the cached resource is used.</span></span> <span data-ttu-id="3d51c-114">このキャッシュ ポリシー レベルが指定された場合、項目がローカル キャッシュ内にない場合は <xref:System.Net.WebException> 例外がスローされます。</span><span class="sxs-lookup"><span data-stu-id="3d51c-114">When this cache policy level is specified, a <xref:System.Net.WebException> exception is thrown if the item is not in the local cache.</span></span>  
  
## <a name="cache-or-next-cache-only-policy"></a><span data-ttu-id="3d51c-115">"キャッシュまたは次のキャッシュのみを使用" ポリシー</span><span class="sxs-lookup"><span data-stu-id="3d51c-115">Cache Or Next Cache Only Policy</span></span>

## <a name="no-cache-no-store-policy"></a><span data-ttu-id="3d51c-119">"キャッシュを使用せず格納しない" ポリシー</span><span class="sxs-lookup"><span data-stu-id="3d51c-119">No Cache No Store Policy</span></span>

## <a name="refresh-policy"></a><span data-ttu-id="3d51c-124">更新ポリシー</span><span class="sxs-lookup"><span data-stu-id="3d51c-124">Refresh Policy</span></span>

## <a name="reload-policy"></a><span data-ttu-id="3d51c-128">ポリシーの再読み込み</span><span class="sxs-lookup"><span data-stu-id="3d51c-128">Reload Policy</span></span>
