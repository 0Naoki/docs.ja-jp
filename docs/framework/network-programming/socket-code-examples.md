# ソケットのコード例
次のコード例は、ネットワーク サービスに接続するにはクライアントとクライアントのリモート接続を立てる聴覚にそばサーバーとして <xref:System.Net.Sockets.Socket> クラスを使用する方法を示します。  
  
## このセクションの内容  
 [同期クライアント ソケットの例](../../../docs/framework/network-programming/synchronous-client-socket-example.md)  
 サーバーに接続し、サーバーからのデータを表示する <xref:System.Net.Sockets.Socket> クライアントの同期を実行する方法を示します。  
  
 [同期サーバー ソケットの例](../../../docs/framework/network-programming/synchronous-server-socket-example.md)  
 クライアントから接続を受け入れ、エコーがクライアントから受け取ったデータをサポートする <xref:System.Net.Sockets.Socket> の同期サーバーを実行する方法を示します。  
  
 [非同期クライアント ソケットの例](../../../docs/framework/network-programming/asynchronous-client-socket-example.md)  
 サーバーに接続し、サーバーからのデータを表示する <xref:System.Net.Sockets.Socket> の非同期なクライアントの実行方法を示します。  
  
 [非同期サーバー ソケットの例](../../../docs/framework/network-programming/asynchronous-server-socket-example.md)  
 クライアントから接続を受け入れ、エコーがクライアントから受け取ったデータをサポートする <xref:System.Net.Sockets.Socket> の非同期なサーバーを実行する方法を示します。
