# ホスト
このセクションのトピックでは、サービス ホスティングについて説明します。サービスは、インターネット インフォメーション サービス \(IIS\)、Windows プロセス アクティブ化サービス \(WAS\)、Windows Server AppFabric、Windows サービス、またはマネージ アプリケーションによってホストできます。マネージ アプリケーションによるホストは、一般に、*自己ホスト*と呼ばれます。  
  
 信頼されていないホストからサービスや拡張機能を実行すると、セキュリティが損なわれるので注意してください。
