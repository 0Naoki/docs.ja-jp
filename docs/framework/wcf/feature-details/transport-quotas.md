# <a name="transport-quotas"></a><span data-ttu-id="06923-102">トランスポート クォータ</span><span class="sxs-lookup"><span data-stu-id="06923-102">Transport Quotas</span></span>
<span data-ttu-id="06923-103">トランスポート クォータは、接続がリソースを過剰に消費している時期を特定するポリシー機構です。</span><span class="sxs-lookup"><span data-stu-id="06923-103">Transport quotas are a policy mechanism for deciding when a connection is consuming excessive resources.</span></span> <span data-ttu-id="06923-104">クォータとは、クォータ値を超えた場合に、それ以上のリソースの使用を禁止する確実な制限です。</span><span class="sxs-lookup"><span data-stu-id="06923-104">A quota is a hard limit that prevents the use of additional resources once the quota value is exceeded.</span></span> <span data-ttu-id="06923-105">トランスポート クォータは、悪質な、または意図的でないサービス拒否攻撃を防ぎます。</span><span class="sxs-lookup"><span data-stu-id="06923-105">Transport quotas prevent either malicious or unintentional denial of service attacks.</span></span>  
  
 <span data-ttu-id="06923-106">Windows Communication Foundation (WCF) トランスポートでは、控えめなリソース割り当てに基づく既定クォータ値を指定します。</span><span class="sxs-lookup"><span data-stu-id="06923-106">Windows Communication Foundation (WCF) transports have default quota values that are based on a conservative allocation of resources.</span></span> <span data-ttu-id="06923-107">これらの既定値は開発環境、および小規模のインストール シナリオに適しています。</span><span class="sxs-lookup"><span data-stu-id="06923-107">These default values are suitable for development environments and small installation scenarios.</span></span> <span data-ttu-id="06923-108">インストールでリソースが不足している場合、または追加リソースが使用可能であるにもかかわらず接続が制限されている場合、サービス管理者は、トランスポート クォータをレビューし、個別のクォータ値を調整する必要があります。</span><span class="sxs-lookup"><span data-stu-id="06923-108">Service administrators should review transport quotas and tune individual quota values if an installation is running out of resources or if connections are being limited despite the availability of additional resources.</span></span>  
  
## <a name="types-of-transport-quotas"></a><span data-ttu-id="06923-109">トランスポート クォータの種類</span><span class="sxs-lookup"><span data-stu-id="06923-109">Types of Transport Quotas</span></span>  
 <span data-ttu-id="06923-110">WCF トランスポートでは、次の 3 つの種類のクォータがあります。</span><span class="sxs-lookup"><span data-stu-id="06923-110">WCF transports have three types of quotas:</span></span>  
  
- <span data-ttu-id="06923-111">*タイムアウト*拒否一定の時間のリソースを停滞に依存するサービス拒否攻撃の緩和します。</span><span class="sxs-lookup"><span data-stu-id="06923-111">*Timeouts* mitigate denial of service attacks that rely on tying up resources for an extended period of time.</span></span>  
  
- <span data-ttu-id="06923-112">*メモリ割り当て制限*システム メモリの枯渇し、他の接続にサービスを拒否する 1 つの接続を防止します。</span><span class="sxs-lookup"><span data-stu-id="06923-112">*Memory allocation limits* prevent a single connection from exhausting system memory and denying service to other connections.</span></span>  
  
- <span data-ttu-id="06923-113">*コレクション サイズの制限*間接的にメモリを割り当てまたは供給に制限されているリソースの消費量をバインドします。</span><span class="sxs-lookup"><span data-stu-id="06923-113">*Collection size limits* bound the consumption of resources that indirectly allocate memory or are in limited supply.</span></span>
