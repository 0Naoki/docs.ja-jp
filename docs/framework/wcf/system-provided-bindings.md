<span data-ttu-id="12d4e-103">バインディングにより、エンドポイントとの通信で使用する通信メカニズムが指定され、エンドポイントへの接続方法が示されます。</span><span class="sxs-lookup"><span data-stu-id="12d4e-103">Bindings specify the communication mechanism to use when talking to an endpoint and indicate how to connect to an endpoint.</span></span> <span data-ttu-id="12d4e-104">バインディングには次の要素が含まれます。</span><span class="sxs-lookup"><span data-stu-id="12d4e-104">A binding contains the following elements:</span></span>
