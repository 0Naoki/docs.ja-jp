4. <span data-ttu-id="c843a-122">右クリックし、**サービス**順に選択して**追加**、**新しい項目の**します。</span><span class="sxs-lookup"><span data-stu-id="c843a-122">Right-click the **Service** project and select **Add**, **New Item**.</span></span> <span data-ttu-id="c843a-123">選択**アプリケーション構成ファイル** をクリック**OK**します。</span><span class="sxs-lookup"><span data-stu-id="c843a-123">Select **Application Configuration File** and click **OK**.</span></span>
