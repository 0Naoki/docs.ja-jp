# ServiceModel 登録ツール
ServiceModel 登録ツールは、単一のコンピューターで ServiceModel の登録を管理できるコマンド ライン ツールです。このツールの詳細については、「[ServiceModel 登録ツール \(ServiceModelReg.exe\)](../../../../docs/framework/wcf/servicemodelreg-exe.md)」を参照してください。
