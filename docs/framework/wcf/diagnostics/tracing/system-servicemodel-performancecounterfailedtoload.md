# System.ServiceModel.PerformanceCounterFailedToLoad
System.ServiceModel.PerformanceCounterFailedToLoad  
  
## 説明  
 パフォーマンス カウンターを読み込むことができませんでした。一部のパフォーマンス カウンターは利用できません。
