# System.ServiceModel.Channels.PnrpResolvedAddresses
PNRP リゾルバーは解決操作を完了しました。  
  
## 説明  
 この警告トレースは、PNRP リゾルバーがアドレスを見つけられなかったことを示します。このトレースは、PNRP リゾルバーによる解決操作中に行われます。
