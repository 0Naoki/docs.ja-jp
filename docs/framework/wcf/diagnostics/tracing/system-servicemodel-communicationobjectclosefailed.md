# System.ServiceModel.CommunicationObjectCloseFailed
通信オブジェクトを閉じることができませんでした。  
  
## 説明  
 Close メソッドの実行中に例外がスローされたため、通信オブジェクトを閉じることができませんでした。別の最近のトレースに、例外に関する詳細が含まれている可能性があります。
