# System.ServiceModel.Channels.PrematureDatagramEof
System.ServiceModel.Channels.PrematureDatagramEof  
  
## 説明  
 NULL メッセージ \(チャネルの信号終了\) が、データグラム チャネルから受信されましたが、チャネルはまだ Opened 状態のままです。  これは、データグラム チャネルにエラーがあり、デマルチプレクサー受信ループが途中で止まったことを示しています。
