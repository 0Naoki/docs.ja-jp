# System.ServiceModel.Channels.PeerReceiveMessageAuthenticationFailure
メッセージ ソースを認証できませんでした。  
  
## 説明  
 このトレースは受信メッセージを認証しようとしたときに、そのバインディングでメッセージの認証がオンになっていると発生します。
