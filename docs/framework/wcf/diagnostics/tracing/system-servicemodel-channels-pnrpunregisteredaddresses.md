# System.ServiceModel.Channels.PnrpUnregisteredAddresses
PNRP リゾルバーがローカル ノードのアドレスを登録解除しました。  
  
## 説明  
 このトレースは、PNRP サービスがローカル ノードのアドレスを正常に登録解除すると実行されます。
