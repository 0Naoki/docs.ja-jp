# System.ServiceModel.UnhandledAction
System.ServiceModel.UnhandledAction  
  
## 説明  
 この動作の処理に利用できるサービスがありません。  
  
 パフォーマンス カウンターのすべてのトレースは、パフォーマンス カウンター関連の問題のデバッグに使用できます。
