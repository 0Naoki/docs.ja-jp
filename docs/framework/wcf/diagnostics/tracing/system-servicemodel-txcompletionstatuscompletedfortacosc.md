# System.ServiceModel.TxCompletionStatusCompletedForTACOSC
そのトランザクションは、セッションが終了した際に完了になりました。  
  
## 説明  
 現在のアクティブなセッションが終了して、`TransactionAutoCompleteOnSessionClose` が `true` に設定されると、トレースされます。
