# System.ServiceModel.Channels.PeerServiceOpened
ピア サービス モジュールが開かれ、指定したアドレスでリッスンを開始しました。  
  
## 説明  
 このトレースは、PeerNode のピア サービス モジュールを開くときに行われます。
