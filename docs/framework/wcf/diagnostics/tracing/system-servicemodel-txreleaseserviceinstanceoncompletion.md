# <a name="systemservicemodeltxreleaseserviceinstanceoncompletion"></a>System.ServiceModel.TxReleaseServiceInstanceOnCompletion
ReleaseServiceInstanceOnTransactionComplete の ServiceBehaviorAttribute が true に設定されていたので、トランザクション '{0}' の完了時にサービス インスタンスが解放されました。  
  
## <a name="description"></a>説明  
 現在のアクティブなトランザクションが完了して現在のサービス インスタンスが解放されるか破棄され、ReleaseServiceInstanceOnTransactionComplete が `true` に設定されると、トレースされます。
