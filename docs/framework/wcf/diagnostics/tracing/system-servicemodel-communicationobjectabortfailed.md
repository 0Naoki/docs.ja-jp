# System.ServiceModel.CommunicationObjectAbortFailed
通信オブジェクトを中止できませんでした。  
  
## 説明  
 Abort メソッドの実行中に例外がスローされたため、通信オブジェクトを中止できませんでした。  この付近のトレースに例外に関する詳細が含まれている可能性があります。
