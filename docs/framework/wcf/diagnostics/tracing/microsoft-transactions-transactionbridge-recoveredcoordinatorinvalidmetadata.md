# Microsoft.Transactions.TransactionBridge.RecoveredCoordinatorInvalidMetadata
コーディネーターの回復ログ エントリで見つかったエンドポイント参照は、無効または互換性のないメタデータを含んでいます。  
  
## 説明  
 システムは、復元されたコーディネーター登録リスト用のプロキシを作成できません。  
  
## トラブルシューティング  
 問題が再発する場合は、マイクロソフトに連絡してください。
