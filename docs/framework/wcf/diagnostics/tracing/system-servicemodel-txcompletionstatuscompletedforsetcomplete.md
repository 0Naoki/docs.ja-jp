# System.ServiceModel.TxCompletionStatusCompletedForSetComplete
SetTransactionComplete が呼び出されたため、指定された操作の指定のトランザクションは完了されました。  
  
## 説明  
 OperationContext によって現在のトランザクションを完了することをユーザーが明示的に指定した場合にトレースされます。
