# Microsoft.Transactions.TransactionBridge.CreateTransactionFailure
トランザクションを作成できませんでした。  
  
## 説明  
 このトレースは、MSDTC がトランザクションを作成できないときに生成されます。  原因として、リソース不足、ログ領域の不足、またはその他のエラーが考えられます。  
  
## トラブルシューティング  
 トレース メッセージ内のステータス文字列を調べて、アクション可能な項目が存在するかどうかを確認します。
