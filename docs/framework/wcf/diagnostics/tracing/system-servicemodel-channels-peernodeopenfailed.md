# System.ServiceModel.Channels.PeerNodeOpenFailed
近隣ノードへの接続を開けませんでした。  
  
## 説明  
 このトレースは、近隣ノードとの接続の確立を試みた後に行われます。  原因として、応答しないリゾルバー、不正なバインディング設定、一時的なネットワーク エラーなどが考えられます。  
  
## トラブルシューティング  
 両方のノードが有効なアドレスを持ち、適切に構成されていることを確認してください。
