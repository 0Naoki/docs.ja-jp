# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>説明  
 応答メッセージを受信しましたが、このメッセージを待っている未処理の要求がありません。 Windows Communication Foundation (WCF) ランタイムによって、メッセージを削除中です。  
  
 このトレースは、失われたメッセージのデバッグに使用できます。
