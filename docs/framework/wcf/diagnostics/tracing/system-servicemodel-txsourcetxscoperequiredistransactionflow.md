# System.ServiceModel.TxSourceTxScopeRequiredIsTransactionFlow
指定したトランザクションが指定した操作にフローされました。  
  
## 説明  
 ユーザー コードから利用できるトランザクションが外部ソースからフローされた場合にトレースされます。
