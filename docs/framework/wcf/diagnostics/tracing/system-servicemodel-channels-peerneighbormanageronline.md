# System.ServiceModel.Channels.PeerNeighborManagerOnline
PeerNode が現在オンラインであり、1 つ以上の近隣ノードが接続されていることを示します。  
  
## 説明  
 このトレースは近隣ノードとの正常な接続の最後に実行されます。
