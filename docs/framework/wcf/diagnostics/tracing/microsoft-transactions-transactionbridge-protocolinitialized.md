# Microsoft.Transactions.TransactionBridge.ProtocolInitialized
WS\-AtomicTransaction プロトコル サービスが初期化されました。  
  
## 説明  
 正常に初期化されたプロトコルを示すために、トランザクション マネージャーの起動時にトレースされます。
