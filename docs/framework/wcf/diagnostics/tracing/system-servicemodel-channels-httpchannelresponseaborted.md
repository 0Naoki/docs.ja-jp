# System.ServiceModel.Channels.HttpChannelResponseAborted
HTTP 経由で応答メッセージを送信できませんでした。  
  
## 説明  
 この警告レベルのトレースは、HTTP チャネルの応答が中止されたことを示しています。このトレースの追加情報により、理由、または Abort メソッドに渡されたメッセージがわかります。
