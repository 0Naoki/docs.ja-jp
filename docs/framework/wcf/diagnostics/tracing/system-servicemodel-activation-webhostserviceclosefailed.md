# System.ServiceModel.Activation.WebHostServiceCloseFailed
サービスが正常に閉じられず、中止された場合に発生します。  
  
## 説明  
 このエラー コードはログ ファイルにのみ記録されます。通常は、Abort を既に呼び出した後でサービスを閉じようとした場合などの、プログラミング エラーを示します。  
  
## トラブルシューティング  
 アプリケーションのソース コードをチェックしてください。
