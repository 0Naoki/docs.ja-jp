# Microsoft.Transactions.TransactionBridge.RegisterParticipant
WS\-AtomicTransaction プロトコル サービスは、制御プロトコルの参加要素の登録に成功しました。  
  
## 説明  
 ローカル トランザクション マネージャーが、2pc プロトコルの参加要素からの Register 要求を正常に受信確認したときにトレースされます。
