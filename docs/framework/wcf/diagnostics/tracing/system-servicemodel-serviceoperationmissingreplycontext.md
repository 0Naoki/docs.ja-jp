# System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## 説明  
 ReplyContext は、双方向の操作に対して NULL です。トレース データによって「応答操作には、応答に使用する IReplyContext が含まれていません。」というメッセージが表示されます。  
  
 このトレースは開発中のデバッグに役立ちます。
