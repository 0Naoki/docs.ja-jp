# <a name="wmiputinstancefailed"></a>WmiPutInstanceFailed
Id: 121  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## <a name="description"></a>説明  
 このイベントは、WMI PutInstance 要求が処理されなかったことを示します。 イベントには、WMI オブジェクト、例外、プロセス名、およびプロセス ID が表示されます。
