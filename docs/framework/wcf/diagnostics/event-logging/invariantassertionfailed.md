# InvariantAssertionFailed
ID : 106  
  
 重大度 : エラー  
  
 カテゴリ : トレース  
  
## 説明  
 このイベントは、インバリアント アサーションが false であることを示します。イベントには、メッセージ、プロセス名、およびプロセス ID が表示されます。
