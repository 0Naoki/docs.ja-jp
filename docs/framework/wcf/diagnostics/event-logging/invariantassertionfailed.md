# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
Id: 106  
  
 重大度 : エラー  
  
 カテゴリ : トレース

 このイベントは、インバリアント アサーションが false であることを示します。 イベントには、メッセージ、プロセス名、およびプロセス ID が表示されます。
