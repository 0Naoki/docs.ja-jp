# <a name="wmigetobjectfailed"></a>WmiGetObjectFailed
Id:120  
  
 重大度:Error  
  
 カテゴリ:ServiceModel  
  
## <a name="description"></a>説明  
 このイベントは、WMI GetObject 要求が処理されなかったことを示します。 イベントには、WMI オブジェクト、例外、プロセス名、およびプロセス ID が表示されます。
