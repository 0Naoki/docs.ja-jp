# WmiGetObjectFailed
ID : 120  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## 説明  
 このイベントは、WMI GetObject 要求が処理されなかったことを示します。イベントには、WMI オブジェクト、例外、プロセス名、およびプロセス ID が表示されます。
