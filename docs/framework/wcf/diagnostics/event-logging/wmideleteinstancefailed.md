# WmiDeleteInstanceFailed
ID : 122  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## 説明  
 このイベントは、WMI DeleteInstance 要求が処理されなかったことを示します。イベントには、例外、プロセス名、およびプロセス ID が表示されます。
