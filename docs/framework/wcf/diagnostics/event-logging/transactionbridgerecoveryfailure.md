# TransactionBridgeRecoveryFailure
ID : 146  
  
 重大度 : エラー  
  
 カテゴリ : TransactionBridge  
  
## 説明  
 このイベントは、回復時に MSDTC TransactionBridge でエラーが発生したことを示しています。これは致命的な状況であるため、MSDTC サービスは終了されました。イベントには、例外、プロセス名、およびプロセス ID が表示されます。
