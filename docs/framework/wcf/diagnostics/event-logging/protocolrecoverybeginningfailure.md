# ProtocolRecoveryBeginningFailure
ID : 144  
  
 重大度 : エラー  
  
 カテゴリ : TransactionBridge  
  
## 説明  
 このイベントは、回復の開始時に MSDTC WS\-AtomicTransaction プロトコルでエラーが発生したことを示しています。この結果、WS\-AtomicTransaction 機能は無効になります。イベントには、プロトコル ID、プロトコル名、例外、プロセス名、およびプロセス ID が表示されます。
