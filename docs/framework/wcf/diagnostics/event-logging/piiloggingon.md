# PiiLoggingOn
ID : 107  
  
 重大度 : エラー  
  
 カテゴリ : トレース  
  
## 説明  
 このイベントは、PII ログ記録が有効になっていることを示します。重要な情報がログに記録されます。イベントには、プロセス名とプロセス ID が表示されます。
