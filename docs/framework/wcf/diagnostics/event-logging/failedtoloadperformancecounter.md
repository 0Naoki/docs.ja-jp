# FailedToLoadPerformanceCounter
ID : 118  
  
 重大度 : エラー  
  
 カテゴリ : ServiceModel  
  
## 説明  
 このイベントは、パフォーマンス カウンターが読み込まれなかったことを示します。  イベントには、カテゴリ名、カウンター名、インスタンス名、例外、プロセス名、およびプロセス ID が表示されます。
