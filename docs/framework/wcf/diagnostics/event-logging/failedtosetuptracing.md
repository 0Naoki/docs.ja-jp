# <a name="failedtosetuptracing"></a>FailedToSetupTracing
Id:100  
  
 重大度:Error  
  
 カテゴリ:トレース  
  
## <a name="description"></a>説明  
 トレースを設定できませんでした。 トレースは無効になります。  
  
 イベントには、例外、プロセス名、およびプロセス ID が表示されます。
