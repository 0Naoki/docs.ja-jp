# <a name="complustlbimporterror"></a>ComPlusTLBImportError
Id:132  
  
 重大度:Error  
  
 カテゴリ:ServiceModel  
  
## <a name="description"></a>説明  
 このイベントは、タイプ ライブラリのインポート中にエラーが発生したことを示します。 イベントには、IID、タイプ ライブラリ ID、例外、プロセス名、およびプロセス ID が表示されます。
