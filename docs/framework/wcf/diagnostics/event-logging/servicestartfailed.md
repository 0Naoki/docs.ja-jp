# <a name="servicestartfailed"></a>ServiceStartFailed
Id: 164  
  
 重大度 : エラー  
  
 カテゴリ: SMSvcHost  
  
## <a name="description"></a>説明  
 このイベントは、サービス開始の要求が失敗したことを示します。 イベントには、エラー コード、プロセス名、およびプロセス ID が一覧表示されます。
