# ThumbPrintNotValidated
ID : 153  
  
 重大度 : エラー  
  
 カテゴリ : TransactionBridge  
  
## 説明  
 このイベントは、指定した拇印を持つ ID 証明書を検証できなかったことを示します。  イベントには、プロセス名とプロセス ID が表示されます。
