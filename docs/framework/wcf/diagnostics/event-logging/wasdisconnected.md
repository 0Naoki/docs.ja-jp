# <a name="wasdisconnected"></a>WasDisconnected
Id: 162  
  
 重大度 : エラー  
  
 カテゴリ: SMSvcHost  
  
## <a name="description"></a>説明  
 このイベントは、WAS (Windows Process Activation) が切断されたことを示します。 イベントには、HRESULT、プロセス名、およびプロセス ID が表示されます。
