# <a name="messageloggingon"></a>MessageLoggingOn
Id:116  
  
 重大度:Error  
  
 カテゴリ:ServiceModel  
  
## <a name="description"></a>説明  
 メッセージ ログが有効になりました。 ネットワーク上で暗号化されていても、重要な情報をログに記録できます。 たとえば、メッセージ本文などです。 イベントには、AppID、例外、プロセス名、およびプロセス ID が表示されます。
