# <a name="webhostfailedtolisten"></a>WebHostFailedToListen
Id:112  
  
 重大度:Error  
  
 カテゴリ:ServiceModel  
  
## <a name="description"></a>説明  
 このイベントは、特定の URL をリッスンしようとしたときにエラーが発生したことを示します。 このワーカー プロセスは終了します。 イベントには、例外、プロセス名、およびプロセス ID が表示されます。
