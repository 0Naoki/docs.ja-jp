# <a name="securitynegotiationfailure"></a>SecurityNegotiationFailure
Id:172  
  
 重大度:Error  
  
 カテゴリ:SecurityAudit  
  
## <a name="description"></a>説明  
 このイベントは、セキュリティ ネゴシエーションに失敗したことを示します。 イベントには、サービス、アクション、クライアント ID、アクティビティ ID、およびネゴシエーションが表示されます。
