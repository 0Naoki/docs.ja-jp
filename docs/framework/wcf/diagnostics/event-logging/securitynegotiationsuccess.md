# SecurityNegotiationSuccess
ID : 171  
  
 重大度 : 情報  
  
 カテゴリ : SecurityAudit  
  
## 説明  
 このイベントは、セキュリティ ネゴシエーションが成功したことを示し、サービスの名前、アクション、クライアント ID、アクティビティ ID、およびネゴシエーションを表示します。
