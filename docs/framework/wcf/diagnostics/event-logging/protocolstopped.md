# <a name="protocolstopped"></a>ProtocolStopped
Id:151  
  
 重大度:Error  
  
 カテゴリ:TransactionBridge  
  
## <a name="description"></a>説明  
 このイベントは、WS-AtomicTransaction プロトコル サービスが停止したことを示します。 イベントには、プロトコル ID、プロトコル名、プロセス名、およびプロセス ID が表示されます。 .
