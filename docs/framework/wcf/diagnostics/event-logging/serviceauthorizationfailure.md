# ServiceAuthorizationFailure
ID : 168  
  
 重大度 : エラー  
  
 カテゴリ : SecurityAudit  
  
## 説明  
 このイベントは、サービスの承認に失敗したことを示します。イベントには、サービス、アクション、クライアント ID、承認コンテキスト、アクティビティ ID、および ServiceAuthorizationManager が表示されます。
