# <a name="failedtocreatemessageloggingtracesource"></a>FailedToCreateMessageLoggingTraceSource
Id:115  
  
 重大度:Error  
  
 カテゴリ:ServiceModel  
  
## <a name="description"></a>説明  
 このイベントは、メッセージ ログでトレース ソースを作成できなかったことを示します。 イベントには、例外、プロセス名、およびプロセス ID が表示されます。
