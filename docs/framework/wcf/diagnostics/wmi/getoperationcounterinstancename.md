# GetOperationCounterInstanceName
操作パフォーマンス カウンターのインスタンスの名前を取得します。  
  
## 構文  
  
```  
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## パラメーター  
  
### Operation  
 \[in\] string  
  
## 必要条件  
  
|MOF|Servicemodel.mof で宣言|  
|---------|--------------------------|  
|名前空間|root\\ServiceModel で定義|
