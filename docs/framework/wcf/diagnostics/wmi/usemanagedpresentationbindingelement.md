# <a name="usemanagedpresentationbindingelement"></a>UseManagedPresentationBindingElement
UseManagedPresentationBindingElement  
  
## <a name="syntax"></a>構文  
  
```  
class UseManagedPresentationBindingElement : BindingElement  
{  
};  
```
