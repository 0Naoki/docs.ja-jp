# 動作クラス
## 構文  
  
```  
class Behavior  
{  
  string Type;  
};  
```  
  
## メソッド  
 Behavior クラスで定義されているメソッドはありません。  
  
## プロパティ  
 Behavior クラスには、次のプロパティがあります。  
  
### Type  
 データ型 : string  
  
 アクセスの種類 : 読み取り専用  
  
 動作の種類です。  
  
## 要件  
  
|MOF|Servicemodel.mof にて宣言済み。|  
|---------|------------------------------|  
|名前空間|root\\ServiceModel で定義|
