# <a name="behavior-class"></a>動作クラス
## <a name="syntax"></a>構文  
  
```  
class Behavior  
{  
  string Type;  
};  
```  
  
## <a name="methods"></a>メソッド  
 Behavior クラスで定義されているメソッドはありません。  
  
## <a name="properties"></a>プロパティ  
 Behavior クラスには、次のプロパティがあります。  
  
### <a name="type"></a>型  
 データ型: string  
  
 アクセスの種類 : 読み取り専用  
  
 動作の種類です。

|MOF|Servicemodel.mof にて宣言済み。|  
|---------|-----------------------------------|  
|Namespace|root\ServiceModel で定義|
