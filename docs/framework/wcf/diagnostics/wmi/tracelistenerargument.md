# TraceListenerArgument
TraceListenerArgument

```

{

  string Value;  
};  
```  
  
## メソッド  
 TraceListenerArgument クラスで定義されるメソッドはありません。

 TraceListenerArgument クラスには、次のプロパティがあります。

 データ型: string  
  
 アクセスの種類 : 読み取り専用  
  
 引数の名前。  
  
### 値  
 データ型: string

 引数の値です。
