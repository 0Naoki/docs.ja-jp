# 3801 - RoutingServiceChannelFaulted
## プロパティ  
  
|||  
|-|-|  
|ID|3801|  
|キーワード|RoutingServices|  
|レベル|警告|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、ルーティング サービス クライアントが失敗したときに生成されます。  
  
## メッセージ  
 ルーティング サービスのクライアント '%1' が途中終了しました。
