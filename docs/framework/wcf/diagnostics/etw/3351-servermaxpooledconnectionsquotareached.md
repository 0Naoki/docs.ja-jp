# 3351 - ServerMaxPooledConnectionsQuotaReached
## プロパティ  
  
|||  
|-|-|  
|ID|3351|  
|キーワード|クォータ|  
|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、サーバーのプールされた接続の最大クォータに達したときに生成されます。  
  
## メッセージ  
 サーバーのプールされた接続の最大クォータに達しました。
