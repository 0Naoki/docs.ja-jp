# 3818 - RoutingServiceDuplexCallbackException
## プロパティ  
  
|||  
|-|-|  
|ID|3818|  
|キーワード|RoutingServices|  
|レベル|Error|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、ルーティング サービスで二重コールバックの例外が発生したときに生成されます。  
  
## メッセージ  
 ルーティング サービスのコンポーネント %1 で二重コールバックの例外が発生しました。
