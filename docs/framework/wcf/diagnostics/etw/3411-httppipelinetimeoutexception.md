# 3411 - HttpPipelineTimeoutException
## プロパティ  
  
|||  
|-|-|  
|ID|3411|  
|キーワード|HTTP|  
|レベル|Error|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、WebSocket 接続がタイムアウトしたときに生成されます。  
  
## メッセージ  
 WebSocket の接続がタイムアウトしました。
