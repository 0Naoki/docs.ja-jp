# 3365 - SharedListenerProxyRegisterStop
## プロパティ  
  
|||  
|-|-|  
|ID|3365|  
|キーワード|ActivationServices|  
|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、SharedListenerProxy の登録が停止されたときに生成されます。  
  
## メッセージ  
 SharedListenerProxy の登録を停止します。
