# 3430 - WebSocketUseVersionFromClientWebSocketFactory
## プロパティ  
  
|||  
|-|-|

|キーワード|HTTP|  
|レベル|詳細|

## 説明  
 このイベントは、クライアント WebSocket ファクトリから WebSocketVersion を使用しているときに生成されます。  
  
## メッセージ  
 型 '%1' のクライアント WebSocket ファクトリから WebSocketVersion を使用しています。
