# 3820 - RoutingServiceCreatingTransaction
## プロパティ  
  
|||  
|-|-|  
|ID|3820|  
|キーワード|RoutingServices|  
|レベル|情報|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、ルーティング サービスがメッセージを処理するために新しいトランザクションを作成したときに生成されます。  
  
## メッセージ  
 ルーティング サービスが、メッセージを処理するために ID '%1' の新しいトランザクションを作成しました。
