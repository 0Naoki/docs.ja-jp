# 3362 - HttpChannelResponseAborted
## プロパティ  
  
|||  
|-|-|

## 説明  
 このイベントは、HTTP チャネルの応答が中止されたときに生成されます。  
  
## メッセージ  
 HTTP チャネルの応答が中止されました。
