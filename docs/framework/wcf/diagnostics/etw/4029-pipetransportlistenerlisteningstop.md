# 4029 - PipeTransportListenerListeningStop
## プロパティ  
  
|||  
|-|-|  
|ID|4029|  
|キーワード|ActivationServices|  
|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、パイプ トランスポート リスナーがリッスンを停止したときに生成されます。  
  
## メッセージ  
 パイプ トランスポート リスナーがリッスンを停止します。
