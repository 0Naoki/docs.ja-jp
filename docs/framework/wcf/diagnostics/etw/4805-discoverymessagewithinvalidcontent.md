# 4805 - DiscoveryMessageWithInvalidContent
## プロパティ  
  
|||  
|-|-|

|キーワード|探索|  
|レベル|警告|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、探索メッセージに無効なコンテンツが含まれているときに生成されます。  
  
## メッセージ  
 messageId\='%2' の %1 メッセージは、無効なコンテンツがあったため、ドロップされました。
