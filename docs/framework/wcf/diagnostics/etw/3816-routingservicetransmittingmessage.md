# 3816 - RoutingServiceTransmittingMessage

|||

|レベル|情報|

## 説明  
 このイベントは、ルーティング サービスがメッセージを転送しているときに生成されます。  
  
## メッセージ  
 ルーティング サービスが、ID: '%1' \[operation %2\] のメッセージを '%3' に転送しています。
