# 3823 - RoutingServiceSendingFaultResponse
## プロパティ  
  
|||  
|-|-|  
|ID|3823|  
|キーワード|RoutingServices|  
|レベル|警告|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、ルーティング サービスがエラー応答メッセージを返送しているときに生成されます。  
  
## メッセージ  
 ルーティング サービスが、Action '%1' を含むエラー応答メッセージを返送しています。
