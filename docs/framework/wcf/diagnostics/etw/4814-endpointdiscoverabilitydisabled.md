# 4814 - EndpointDiscoverabilityDisabled
## プロパティ  
  
|||  
|-|-|  
|ID|4814|  
|キーワード|探索|  
|レベル|情報|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、エンドポイントの探索が無効になっているときに生成されます。  
  
## メッセージ  
 EndpointAddress\='%1' および ListenUri\='%2' のエンドポイントの探索が無効になりました。
