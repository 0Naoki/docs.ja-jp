|||  
|-|-|  
|ID|603|  
|キーワード|RoutingServices|  
|レベル|詳細|  
|チャネル|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>説明  
 このイベントは、受信要求が、アドレス %1 の Asp.Net ルートで定義された WCF サービスと一致すると生成されます。

 受信要求が、アドレス %1 の Asp.Net ルートで定義された WCF サービスと一致します。
