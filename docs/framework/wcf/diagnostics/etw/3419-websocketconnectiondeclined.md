# 3419 - WebSocketConnectionDeclined
## プロパティ  
  
|||  
|-|-|  
|ID|3419|  
|キーワード|HTTP|  
|レベル|Error|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、WebSocket 接続が拒否されたときに生成されます。  
  
## メッセージ  
 WebSocket 接続が状態コード '%1' で拒否されました
