# 3827 - RoutingServiceTransmitFailed
## プロパティ  
  
|||  
|-|-|  
|ID|3827|  
|キーワード|RoutingServices|  
|レベル|警告|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、ルーティング サービスの送信が失敗したときに生成されます。  
  
## メッセージ  
 '%1' への送信中にルーティング サービスでエラーが発生しました。
