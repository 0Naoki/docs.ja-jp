# 3429 - WebSocketCloseStatusReceived
## プロパティ  
  
|||  
|-|-|  
|ID|3429|  
|キーワード|HTTP|  
|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、WebSocket の終了ステータスを受信したときに生成されます。  
  
## メッセージ  
 WebSocketId:%1 が、状態 '%2' の接続終了メッセージを受信しました。
