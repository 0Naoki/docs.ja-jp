# 3352 - TcpConnectionTimedOut
## プロパティ  
  
|||  
|-|-|  
|ID|3352|  
|キーワード|TCP|  
|レベル|Error|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、TCP 接続がタイムアウトしたときに生成されます。  
  
## メッセージ  
 リモート アドレス %2 への SocketId:%1 がタイムアウトしました。
