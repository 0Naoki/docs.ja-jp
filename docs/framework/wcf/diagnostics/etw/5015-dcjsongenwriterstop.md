# 5015 - DCJsonGenWriterStop
## プロパティ  
  
|||  
|-|-|  
|ID|5015|  
|キーワード|シリアル化|  
|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、Json で生成されたライターが停止すると生成されます。  
  
## メッセージ  
 Json のライターの生成を停止します。
