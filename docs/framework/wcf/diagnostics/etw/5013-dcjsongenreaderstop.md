# 5013 - DCJsonGenReaderStop
## プロパティ  
  
|||  
|-|-|  
|ID|5013|  
|キーワード|シリアル化|  
|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、Json リーダーの生成を停止すると生成されます。  
  
## メッセージ  
 Json のリーダーの生成を停止します。
