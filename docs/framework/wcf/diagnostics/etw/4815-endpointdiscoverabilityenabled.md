# 4815 - EndpointDiscoverabilityEnabled
## プロパティ  
  
|||  
|-|-|  
|ID|4815|  
|キーワード|探索|  
|レベル|情報|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、エンドポイントの探索が有効になっているときに生成されます。  
  
## メッセージ  
 EndpointAddress\='%1' および ListenUri\='%2' のエンドポイントの探索が有効になりました。
