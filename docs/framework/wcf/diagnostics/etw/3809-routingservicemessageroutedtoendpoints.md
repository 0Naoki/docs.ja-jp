# 3809 - RoutingServiceMessageRoutedToEndpoints

## 説明  
 このイベントは、ルーティング サービスがエンドポイントにルーティングされたときに生成されます。  
  
## メッセージ  
 ルーティング サービスにより、ID: '%1' のメッセージが %2 エンドポイント リストにルーティングされました。
