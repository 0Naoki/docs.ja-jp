# 3426 - WebSocketCloseSent
## プロパティ

|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 このイベントは、WebSocket の終了が送信されると生成されます。  
  
## メッセージ  
 WebSocketId:%1 が、終了ステータス '%3' の終了メッセージを '%2' に送信しています。
