# 3353 - TcpConnectionResetError

## 説明  
 このイベントは、TCP 接続リセット エラーが発生すると生成されます。  
  
## メッセージ  
 リモート アドレス %2 への SocketId:%1 で接続リセット エラーが発生しました。
