# 4012 - MessageQueueRegisterAbort
## プロパティ  
  
|||  
|-|-|  
|ID|4012|  
|キーワード|ActivationServices|  
|レベル|Error|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Analytic|  
  
## 説明  
 このイベントは、メッセージ キューの登録が中止されたときに生成されます。  
  
## メッセージ  
 URI:'%2' のメッセージ キューの登録が状態:'%1' で中止されました。
