# エンドポイント : トランザクション フロー
カウンター名 : トランザクション フロー。  
  
## 説明  
 このエンドポイントでの操作に対して実行されたトランザクションの数です。このカウンターは、エンドポイントに送信されたメッセージにトランザクション ID が付与されている場合は常にインクリメントされます。
