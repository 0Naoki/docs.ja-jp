# <a name="endpoint-calls-failed"></a>エンドポイント:失敗した呼び出し
カウンター名:呼び出しが失敗しました。  
  
## <a name="description"></a>説明  
 このエンドポイントで処理できない例外が発生した呼び出しの数です。 このカウンターは、このエンドポイントで未処理の例外が発生するたびにインクリメントされます。
