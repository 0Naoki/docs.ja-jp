# <a name="calls-failed"></a>失敗した呼び出し
カウンター名:失敗した呼び出し  
  
## <a name="description"></a>説明  
 この操作で未処理の例外が発生した呼び出しの回数です。 このカウンターは、この操作で未処理の例外が発生するたびにインクリメントされます。
