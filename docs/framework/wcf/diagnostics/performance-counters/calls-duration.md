# <a name="calls-duration"></a>呼び出す期間
カウンター名: 呼び出す期間  
  
## <a name="description"></a>説明  
 この操作の呼び出しの平均期間です。 平均期間は、(N1-N0)/(D1-D0) という数式に基づいて計算されます。  
  
> [!WARNING]
>  非同期の WCF サービスで使用する場合の通話合計時間のカウンターは常に-1 を返します。
