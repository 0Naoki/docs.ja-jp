# サービス : 失敗した呼び出し
カウンター名 : 失敗した呼び出し。  
  
## 説明  
 このサービスで未処理の例外が発生した呼び出しの回数です。このカウンターは、このサービスで未処理の例外が発生するたびにインクリメントされます。
