## <a name="in-this-section"></a>このセクションの内容

 [トークンのサポート](../../../../docs/framework/wcf/samples/supporting-tokens.md)

 [トークン認証システム](../../../../docs/framework/wcf/samples/token-authenticator.md)

 [トークン プロバイダー](../../../../docs/framework/wcf/samples/token-provider.md)  
 カスタム トークン プロバイダーを実装する方法を示します。  
  
 [ユーザー名パスワード検証](../../../../docs/framework/wcf/samples/user-name-password-validator.md)

 [カスタム トークン](../../../../docs/framework/wcf/samples/custom-token.md)

 [クライアント検証](../../../../docs/framework/wcf/samples/client-validation.md)
