## <a name="in-this-section"></a>このセクションの内容  
 [CustomDiscoveryMetadata](../../../../docs/framework/wcf/samples/customdiscoverymetadata.md)  
 サービスによって公開される探索可能なエンドポイントの探索メタデータにカスタム XML メタデータを挿入する方法について説明します。
