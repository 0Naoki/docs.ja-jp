# バインディング拡張
このセクションには、[!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] でのカスタム バインディングを示すサンプルが含まれています。  
  
## このセクションの内容  
 <xref:System.ServiceModel.NetHttpBinding>  
 <xref:System.ServiceModel.Channels.BinaryMessageEncodingBindingElement> 上に <xref:System.ServiceModel.Channels.HttpTransportBindingElement> または <xref:System.ServiceModel.Channels.HttpsTransportBindingElement> をスタックするバインディングを実装する方法を示します。  
  
 <xref:System.ServiceModel.WSStreamedHttpBinding>  
 HTTP トランスポート使用時にストリーミングをサポートする目的でデザインされたバインドを作成する方法を示します。
