# WCF システム要件
[!INCLUDE[indigo1](../../../includes/indigo1-md.md)] は、分散アプリケーションの作成に使用される通信インフラストラクチャです。  [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] の実行を可能にするための要件を次に示します。  
  
## システム要件  
 [!INCLUDE[wv](../../../includes/wv-md.md)] には、既定で [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] がインストールされています。  
  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] は、[!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)]、[!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]、または [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] にもインストールできます。  
  
 **メモ** [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] のメッセージ キュー \(MSMQ\) 機能は、[!INCLUDE[wv](../../../includes/wv-md.md)]、[!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]、[!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]、および [!INCLUDE[wxppro](../../../includes/wxppro-md.md)] でのみサポートされます。
