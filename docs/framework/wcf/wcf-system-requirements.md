# <a name="wcf-system-requirements"></a>WCF システム要件
Windows Communication Foundation (WCF) は、分散アプリケーションの作成に使用される通信インフラストラクチャです。 実行する WCF を有効にする要件を次に示します。  
  
## <a name="system-requirements"></a>システム要件  
 WCF がで既定でインストールされている[!INCLUDE[wv](../../../includes/wv-md.md)]です。  
  
 WCF をインストールすることも[!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)]、 [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]、または[!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]です。  
  
 **注**WCF の機能は、メッセージ キュー (MSMQ) がでのみサポート[!INCLUDE[wv](../../../includes/wv-md.md)]、 [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]、 [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)]、および[!INCLUDE[wxppro](../../../includes/wxppro-md.md)]です。
