# <a name="getting-started-entity-framework"></a>はじめに (Entity Framework)
ADO.NET [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)]は、データ中心のアプリケーションとサービスをサポートし、論理リレーショナルレベルから概念レベルまでの抽象化レベルを生成するデータに対してプログラミングするためのプラットフォームを提供します。 開発者は高度に抽象化されたデータを使用できるため、[!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] は特定のデータ ストレージ エンジンまたはリレーショナル スキーマに依存しないコードをサポートします。 詳細については、「 [Entity Framework の概要](overview.md)」を参照してください。  
  
 Entity Framework の最新バージョンの使用をすぐに開始するには、「[はじめ](https://go.microsoft.com/fwlink/?LinkId=235280)に」を参照してください。
