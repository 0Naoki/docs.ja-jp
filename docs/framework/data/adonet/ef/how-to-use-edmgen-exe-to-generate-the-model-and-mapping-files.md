### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="86e4c-116">EdmGen.exe を使用して Visual Basic プロジェクト用の School モデルを生成するには</span><span class="sxs-lookup"><span data-stu-id="86e4c-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>  
  
1.  <span data-ttu-id="86e4c-117">School データベースを作成します。</span><span class="sxs-lookup"><span data-stu-id="86e4c-117">Create the School database.</span></span> <span data-ttu-id="86e4c-118">詳細については、次を参照してください。 [School サンプル データベースを作成する](http://msdn.microsoft.com/en-us/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0)です。</span><span class="sxs-lookup"><span data-stu-id="86e4c-118">For more information, see [Creating the School Sample Database](http://msdn.microsoft.com/en-us/c1bec483-a0ea-4660-aa0b-7b0a8b68fed0).</span></span>  
  
2.  <span data-ttu-id="86e4c-119">コマンド プロンプトで、次のコマンド (改行なし) を実行します。</span><span class="sxs-lookup"><span data-stu-id="86e4c-119">At the command prompt, execute the following command without line breaks:</span></span>  
  
    ```  
    "%windir%\Microsoft.NET\Framework\v4.0.30319\edmgen.exe" /mode:fullgeneration   
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
