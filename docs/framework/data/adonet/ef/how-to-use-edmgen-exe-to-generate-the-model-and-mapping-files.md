2.  <span data-ttu-id="eeb04-119">コマンド プロンプトで、次のコマンド (改行なし) を実行します。</span><span class="sxs-lookup"><span data-stu-id="eeb04-119">At the command prompt, execute the following command without line breaks:</span></span>

