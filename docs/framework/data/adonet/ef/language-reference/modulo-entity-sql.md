# <a name="modulo-entity-sql"></a>(剰余) (Entity SQL)
ある式を別の式で除算した結果の余りを返します。  
  
## <a name="syntax"></a>構文  
  
```  
dividend % divisor  
```  
  
## <a name="arguments"></a>引数  
 `dividend`  
 除算する数値式。 `dividend` は、任意の数値データ型の有効な式です。  
  
 `divisor`  
 被除数を除算する数値式。 `divisor` は、任意の数値データ型の有効な式です。  
  
## <a name="result-types"></a>戻り値の型  
 Edm.Int32
