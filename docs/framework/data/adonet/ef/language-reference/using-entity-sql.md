# <a name="using-entity-sql"></a>USING (Entity SQL)
クエリ式で使用する名前空間を指定します。  
  
## <a name="syntax"></a>構文  
  
```  
USING [ alias = ] namespace  
```  
  
## <a name="arguments"></a>引数  
 `alias`  
 名前空間を修飾する短い別名。  
  
 `namespace`  
 任意の有効な名前空間。
