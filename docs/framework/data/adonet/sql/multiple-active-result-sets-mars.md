# 複数のアクティブな結果セット (MARS)
複数のアクティブな結果セット \(MARS : Multiple Active Result Set\) は、複数のバッチを単一の接続で実行できる機能です。  従来のバージョンでは、単一の接続で同時に実行できるバッチは 1 つだけでした。  MARS で複数のバッチを実行することは、同時に操作を実行することではありません。
