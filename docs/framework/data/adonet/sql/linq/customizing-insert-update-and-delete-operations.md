# <a name="customizing-insert-update-and-delete-operations"></a>挿入、更新、および削除の各操作のカスタマイズ

> [!NOTE]
>  [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] を使用している場合は、[!INCLUDE[vs_ordesigner_long](../../../../../../includes/vs-ordesigner-long-md.md)] を使って挿入、更新、および削除の各操作をカスタマイズできます。
