# Creating the Object Model
既存のデータベースからオブジェクト モデルを作成し、このオブジェクト モデルを既定の状態で使用できます。  オブジェクト モデルの多くの側面と動作をカスタマイズできます。  
  
 [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] を使用している場合は、[!INCLUDE[vs_ordesigner_long](../../../../../../includes/vs-ordesigner-long-md.md)] を使用してオブジェクト モデルを作成できます。
