# Downloading Sample Databases
[!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] のドキュメントに収録されている多数のサンプルとチュートリアルでは、Northwind サンプル データベースと SQL Server Express Edition が使用されています。  これらの製品は、Microsoft のダウンロード サイトから無料でダウンロードできます。  
  
## Northwind データベースのダウンロード  
  
#### SQL Server 用の Northwind サンプル データベースをダウンロードしてインストールするには  
  
1.  Internet Explorer を開始します。  
  
2.  [Northwind および Pubs サンプル データベース](http://go.microsoft.com/fwlink?linkid=64296)の Web サイトにアクセスします。  
  
3.  **\[ダウンロード\]** をクリックします。  
  
4.  **\[ファイルのダウンロード\]** ダイアログ ボックスで、**\[保存\]** をクリックします。  
  
5.  ファイルのダウンロード後に、**Nwind.exe** ファイルをダブルクリックし、データベースをインストールします。  
  
     既定では、このデータベースは *drive*:\\SQL Server 2000 Sample Databases にインストールされます。  
  
## SQL Server Express Edition のダウンロード  
 SQL Server Express Edition は無料で入手でき、アプリケーションと共に再配布できます。  [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)] を使用している場合は、Pro 以上のエディションに SQL Server Express Edition が含まれています。  
  
#### SQL Server Express Edition をダウンロードおよびインストールするには  
  
1.  Internet Explorer を開始します。  
  
2.  [Microsoft ダウンロード センター](http://go.microsoft.com/fwlink?linkid=74602)のWeb サイトにアクセスします。  
  
3.  キーワード ボックスに「SQL Server Express」と入力します。  
  
4.  **\[検索\]** をクリックします。  
  
5.  結果のページで、**Microsoft SQL Server 2005 Express Edition** のダウンロード ページへのリンクをクリックします。  
  
6.  Web サイトに記載されているインストールの指示に従います。  
  
## Management Studio Express のダウンロード  
 ダウンロードしたデータベースを変更する場合は、[!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] 統合開発環境 \(IDE: Integrated Development Environment\) の**サーバー エクスプローラー**からデータベースにアクセスできますが、Microsoft SQL Server Management Studio Express \(SSMSE\) を使用することもできます。  
  
#### Management Studio Express をダウンロードするには  
  
-   [SSMSE サイト](http://go.microsoft.com/fwlink/?LinkId=95933)に記載されている手順に従います。
