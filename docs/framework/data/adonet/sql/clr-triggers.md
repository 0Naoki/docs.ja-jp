# <a name="clr-triggers"></a>CLR トリガー
トリガーは、言語イベントが実行されると自動的に実行される特殊な種類のストアド プロシージャです。 Microsoft SQL Server が .NET Framework 共通言語ランタイム (CLR) と統合されているため、任意の .NET Framework 言語を使用して CLR トリガーを作成できます。
