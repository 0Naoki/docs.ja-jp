# <a name="privacynoticeat"></a>\<privacyNoticeAt >
`wsFederationHttp` バインディングで使用されるプライバシーに関する声明を指定する構成要素を表します。  
  
 \<system.serviceModel>  
\<bindings>  
\<customBinding>  
\<binding>  
\<privacyNotice >  
  
## <a name="syntax"></a>構文  
  
```xml  
<privacyNotice url="String"
               version="Integer" />
```  
  
## <a name="type"></a>型  
 `Type`  
  
## <a name="attributes-and-elements"></a>属性および要素  
 以降のセクションでは、属性、子要素、および親要素について説明します。  
  
### <a name="attributes"></a>属性  
  
|属性|説明|  
|---------------|-----------------|  
|`url`|プライバシーに関する声明の場所を示す URI を指定する文字列。|  
|`version`|このプライバシーに関する声明のバージョンを指定する整数。|  
  
### <a name="child-elements"></a>子要素  
 なし。  
  
### <a name="parent-elements"></a>親要素  
  
|要素|説明|  
|-------------|-----------------|  
|[\<binding>](../../../misc/binding.md)|カスタム バインドのすべてのバインド機能を定義します。|
