# <a name="ltheadersgt-element"></a>&lt;headers&gt; 要素
ローカル発行者を正しくアドレス指定するために必要なアドレス ヘッダーのコレクションが含まれます。 `add` キーワードを使用して、このコレクションにヘッダーを追加できます。
