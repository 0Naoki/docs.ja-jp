# <a name="transports"></a>トランスポート
トランスポート層は、チャネル スタックの最も低いレベルにあります。 Windows Communication Foundation (WCF) を使用する主なトランスポートは、HTTP、HTTPS、TCP、および名前付きパイプです。 このセクションでは、さまざまな種類のトランスポートを構成できるすべての構成要素を示します。
