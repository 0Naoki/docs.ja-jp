# トランスポート
トランスポート層は、チャネル スタックの最も低いレベルにあります。  [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] で使用される主なトランスポートは、HTTP、HTTPS、TCP、および名前付きパイプです。  このセクションでは、さまざまな種類のトランスポートを構成できるすべての構成要素を示します。
