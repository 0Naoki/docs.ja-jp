# .NET Framework の ETW イベント
Windows イベント トレーシング \(ETW: Event Tracing for Windows\) は、パフォーマンスが高くオーバーヘッドの低いスケーラブルなトレース システムで、Windows オペレーティング システムに含まれています。  .NET Framework のプロファイリングとデバッグのサポートを補足する機能として、さまざまなシナリオのトラブルシューティングに使用できます。  ETW に関する一般的な情報については、[ETW のデバッグおよびパフォーマンス調整を改善します。](http://go.microsoft.com/fwlink/?LinkID=161142) MSDN ライブラリを参照してください。  
  
 .NET Framework では、共通言語ランタイム \(CLR: Common Language Runtime\)、[タスク並列ライブラリ](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md)、および [Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md) で ETW を使用できます。
