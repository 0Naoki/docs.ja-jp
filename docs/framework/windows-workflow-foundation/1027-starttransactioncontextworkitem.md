|||  
|-|-|  
|ID|1027|  
|キーワード|WFRuntime|  
|レベル|詳細|

## <a name="description"></a>説明

|InstanceId|xs:string|アクティビティのインスタンス ID。|  
|AppDomain|xs:string|AppDomain.CurrentDomain.FriendlyName で返される文字列。|
