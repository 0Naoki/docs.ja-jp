# 1147 - FlowchartSwitchDefault
## プロパティ  
  
|||  
|-|-|  
|ID|1147|

## 説明  
 フローチャート スイッチで既定の例が選択されたことを示します。  
  
## メッセージ  
 フローチャート '%1'\/FlowSwitch \- 既定の Case が選択されました。
