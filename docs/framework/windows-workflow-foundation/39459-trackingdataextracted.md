# 39459 - TrackingDataExtracted
## プロパティ  
  
|||  
|-|-|  
|ID|39459|  
|キーワード|WFRuntime|  
|レベル|詳細|  
|チャネル|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## 説明  
 追跡データがアクティビティで抽出されたことを示します。  
  
## メッセージ  
 追跡データ %1 がアクティビティ %2 で抽出されました。  
  
## 詳細  
  
|データ項目名|データ項目の型|説明|  
|------------|-------------|--------|  
|データ|xs:string|抽出されたデータ名。|  
|アクティビティ|xs:string|アクティビティの名前。|  
|AppDomain|xs:string|AppDomain.CurrentDomain.FriendlyName で返される文字列。|
