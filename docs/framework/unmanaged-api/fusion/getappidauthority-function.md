# <a name="getappidauthority-function"></a>GetAppIdAuthority 関数

## <a name="syntax"></a>構文  
  
```  
HRESULT GetAppIdAuthority (  
    [out] IAppIdAuthority **ppIAppIdAuthority  
 );
```  
