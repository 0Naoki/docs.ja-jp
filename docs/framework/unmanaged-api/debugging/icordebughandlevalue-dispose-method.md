# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose メソッド
インターフェイス ポインターを明示的に解放せずにこの ICorDebugHandleValue オブジェクトによって参照されるハンドルを解放します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Dispose ();  
```
