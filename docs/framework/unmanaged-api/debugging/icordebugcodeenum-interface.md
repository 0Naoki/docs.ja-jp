# <a name="icordebugcodeenum-interface1"></a>ICorDebugCodeEnum Interface1
"ICorDebugEnum"メソッドを実装して、"ICorDebugCode"配列を列挙します。  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[Next メソッド](../../../../docs/framework/unmanaged-api/debugging/icordebugcodeenum-next-method.md)|指定した数を取得`ICorDebugCode`列挙体の現在位置からのインスタンス。|  
  
## <a name="remarks"></a>コメント  
  
> [!NOTE]
>  このインターフェイスは、コンピューター間またはプロセス間でのリモート呼び出しをサポートしていません。
