# <a name="icordebugtypegetrank-method"></a>ICorDebugType::GetRank メソッド
配列型の次元数を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `pnRank`  
 [out]ディメンションの数へのポインター。
