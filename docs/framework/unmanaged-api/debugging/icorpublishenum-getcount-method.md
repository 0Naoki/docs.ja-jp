# <a name="icorpublishenumgetcount-method"></a>ICorPublishEnum::GetCount メソッド
列挙に含まれる項目の数を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetCount (  
    [out] ULONG   *pcelt  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pcelt`  
 [out]列挙に含まれる項目数へのポインター。
