# <a name="icordebugbreakpointactivate-method"></a>ICorDebugBreakpoint::Activate メソッド
これのアクティブな状態を設定`ICorDebugBreakpoint`です。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Activate (  
    [in] BOOL bActive  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `bActive`  
 [in]この値を設定`true`としてアクティブになった状態を指定するそれ以外の場合、この値を設定`false`です。
