# <a name="icordebugobjectvaluesetfrommanagedcopy-method"></a>ICorDebugObjectValue::SetFromManagedCopy メソッド

## <a name="syntax"></a>構文  
  
```  
HRESULT SetFromManagedCopy (  
    [in] IUnknown            *pObject  
);  
```
