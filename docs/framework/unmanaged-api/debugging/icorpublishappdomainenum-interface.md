# <a name="icorpublishappdomainenum-interface"></a>ICorPublishAppDomainEnum インターフェイス

## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[Next メソッド](../../../../docs/framework/unmanaged-api/debugging/icorpublishappdomainenum-next-method.md)|指定した数を取得`ICorPublishAppDomain`コレクションの現在位置からのインスタンス。|
