# <a name="icordebugbreakpointisactive-method"></a>ICorDebugBreakpoint::IsActive メソッド
示す値を取得するかどうかこの`ICorDebugBreakpoint`がアクティブです。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT IsActive (  
    [out] BOOL *pbActive  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pbActive`  
 [out]`true`このブレークポイントがアクティブ、それ以外の場合は`false`します。
