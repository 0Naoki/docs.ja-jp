# <a name="icordebugarrayvaluegetcount-method"></a>ICorDebugArrayValue::GetCount メソッド
配列の要素の合計数を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetCount (  
    [out] ULONG32 *pnCount  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pnCount`  
 [out]配列内の要素の総数へのポインター。
