# <a name="icorpublishenumskip-method"></a>ICorPublishEnum::Skip メソッド
列挙体の指定数の項目でカーソルを前方に移動します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `celt`  
 [in]カーソルを前方に移動する項目の数。
