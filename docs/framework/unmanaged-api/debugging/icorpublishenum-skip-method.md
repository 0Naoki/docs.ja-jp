# <a name="icorpublishenumskip-method"></a>ICorPublishEnum::Skip メソッド
指定数の項目では、列挙体にカーソルを移動します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `celt`  
 [in]カーソルを前方に移動する項目の数。
