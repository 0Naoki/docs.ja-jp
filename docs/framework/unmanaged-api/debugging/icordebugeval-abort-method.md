# <a name="icordebugevalabort-method"></a>ICorDebugEval::Abort メソッド
この ICorDebugEval オブジェクトが現在実行して、計算を中止します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Abort ();  
```  
  
## <a name="remarks"></a>Remarks  
 評価が入れ子になっており、最新のものでない場合、`Abort`メソッドが失敗する可能性があります。
