# <a name="icordebugfunctionbreakpointgetoffset-method"></a>ICorDebugFunctionBreakpoint::GetOffset メソッド
関数内のブレークポイントのオフセットを取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetOffset (  
    [out] ULONG32  *pnOffset  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pnOffset`  
 [out]ブレークポイントのオフセットへのポインター。
