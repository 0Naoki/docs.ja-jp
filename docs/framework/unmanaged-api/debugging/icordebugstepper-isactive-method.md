# <a name="icordebugstepperisactive-method"></a>ICorDebugStepper::IsActive メソッド

## <a name="syntax"></a>構文  
  
```  
HRESULT IsActive (  
    [out] BOOL   *pbActive

```  
  
#### <a name="parameters"></a>パラメーター  
 `pbActive`
