# <a name="icordebugboxvaluegetobject-method"></a>ICorDebugBoxValue::GetObject メソッド
ボックス化された値を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetObject (  
    [out] ICorDebugObjectValue **ppObject  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `ppObject`  
 [out]ボックス化された値を表す ICorDebugObjectValue オブジェクトのアドレスへのポインター。
