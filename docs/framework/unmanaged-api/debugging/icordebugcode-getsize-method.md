# <a name="icordebugcodegetsize-method"></a>ICorDebugCode::GetSize メソッド
この"ICorDebugCode"で表されるバイナリ コードのバイト単位のサイズを取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetSize (  
    [out] ULONG32    *pcBytes  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pcBytes`  
 [out]このコードのバイナリのバイト単位のサイズへのポインター`ICorDebugCode`オブジェクトを表します。
