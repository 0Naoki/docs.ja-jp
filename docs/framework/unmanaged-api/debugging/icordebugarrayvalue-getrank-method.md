# <a name="icordebugarrayvaluegetrank-method"></a>ICorDebugArrayValue::GetRank メソッド
配列のディメンションの数を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pnRank`  
 [out]このディメンションの数へのポインター`ICorDebugArrayValue`オブジェクト。
