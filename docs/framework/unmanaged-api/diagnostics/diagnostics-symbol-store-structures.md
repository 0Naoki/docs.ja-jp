# <a name="diagnostics-symbol-store-structures"></a>シンボル ストア診断構造体
このトピックでは、シンボル ストア診断 API を使用するアンマネージ構造体について説明します。  
  
## <a name="in-this-section"></a>このセクションの内容  
 [CALL_ID 構造体](../../../../docs/framework/unmanaged-api/diagnostics/call-id-structure.md)  
 呼び出される関数はデバッガーに情報を提供します。  
  
 [SYMLINEDELTA 構造体](../../../../docs/framework/unmanaged-api/diagnostics/symlinedelta-structure.md)  
 シンボル ハンドラーを編集した結果として移動されたメソッドに関する情報を提供します。  
  
 [USER_THREAD 構造体](../../../../docs/framework/unmanaged-api/diagnostics/user-thread-structure.md)  
 デバッガー スレッドに関する情報を提供します。
