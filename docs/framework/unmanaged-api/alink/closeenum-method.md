# <a name="closeenum-method"></a>CloseEnum メソッド
指定の列挙体を終了し、関連付けられているリソースを解放します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT CloseEnum(  
    HALINKENUM hEnum  
) PURE;  
```  
  
## <a name="parameters"></a>パラメーター  
 `hEnum`  
 終了する列挙体のハンドル。  
  
## <a name="return-value"></a>戻り値  
 メソッドが成功した場合は、S_OK を返します。  
  
## <a name="requirements"></a>必要条件  
 Alink.h が必要です。
