# <a name="icorprofilerobjectenumskip-method"></a>ICorProfilerObjectEnum::Skip メソッド
指定した要素数がスキップされるように、現在の位置からこの列挙子のカーソルを進めます。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `celt`  
 [in]スキップする要素の数。  
  
## <a name="remarks"></a>コメント  
 この列挙子のカーソルの新しい位置は: (現在の位置) +`celt`です。
