# <a name="icorprofilerthreadenumgetcount-method"></a>ICorProfilerThreadEnum::GetCount メソッド
アプリケーションで使用されるスレッドの数を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetCount (    [out] ULONG * pcelt  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `celt`  
 [out]アプリケーションで使用されるスレッドの数。
