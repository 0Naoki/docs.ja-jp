# <a name="icatalogservices-interface"></a>ICatalogServices インターフェイス
カタログ サービスのメソッドを提供します。  
  
 このインターフェイスは、.NET Framework インフラストラクチャをサポートします。独自に作成したコードから直接使用するためのものではありません。
