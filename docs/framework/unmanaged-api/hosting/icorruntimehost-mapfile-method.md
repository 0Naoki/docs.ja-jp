# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile メソッド
指定したファイルをメモリにマップします。 このメソッドは、互換性のために残されています。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `hFile`  
 [in]マップするファイルのハンドル。  
  
 `hMapAddress`  
 [out]ファイルのマッピングを開始する位置の開始メモリ アドレス。
