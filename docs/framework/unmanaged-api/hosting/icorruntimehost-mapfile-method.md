# <a name="icorruntimehostmapfile-method"></a>ICorRuntimeHost::MapFile メソッド
メモリに指定されたファイルをマップします。 このメソッドは、互換性のために残されています。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT MapFile(  
    [in]  HANDLE    hFile,  
    [out] HMODULE*  hMapAddress  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `hFile`  
 [in]マップするファイルのハンドル。  
  
 `hMapAddress`  
 [out]ファイルのマッピングを開始する開始メモリ アドレス。
