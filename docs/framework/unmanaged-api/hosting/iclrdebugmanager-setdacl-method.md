# <a name="iclrdebugmanagersetdacl-method"></a>ICLRDebugManager::SetDacl メソッド

## <a name="syntax"></a>構文  
  
```  
HRESULT SetDacl (  
    [in] PACL pacl  
);  
```  
  
#### <a name="parameters"></a>パラメーター
