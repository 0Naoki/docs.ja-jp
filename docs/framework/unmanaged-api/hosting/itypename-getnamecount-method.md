# <a name="itypenamegetnamecount-method"></a>ITypeName::GetNameCount メソッド
このメソッドは、.NET Framework インフラストラクチャをサポートします。独自に作成したコードから直接使用するためのものではありません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetNameCount (  
    [out, retval] DWORD* pCount  
);  
```
