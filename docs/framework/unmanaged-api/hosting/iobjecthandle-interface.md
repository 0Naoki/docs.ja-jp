# <a name="iobjecthandle-interface"></a>IObjectHandle インターフェイス
メソッドをラップ解除の値渡しでマーシャ リング オブジェクトの間接参照を提供します。  
  
## <a name="methods"></a>メソッド  
  
|メソッド|説明|  
|------------|-----------------|  
|[Unwrap メソッド](../../../../docs/framework/unmanaged-api/hosting/iobjecthandle-unwrap-method.md)|間接参照から値渡しでマーシャ リング オブジェクトがアンラップされます。|
