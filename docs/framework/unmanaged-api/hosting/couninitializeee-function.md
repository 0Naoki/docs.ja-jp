# <a name="couninitializeee-function"></a>CoUninitializeEE 関数
`CoUninitializeEE` 廃止され、機能はありません。  
  
## <a name="syntax"></a>構文  
  
```  
void CoUninitializeEE (  
    BOOL fFlags  
);  
```  
  
## <a name="remarks"></a>Remarks
