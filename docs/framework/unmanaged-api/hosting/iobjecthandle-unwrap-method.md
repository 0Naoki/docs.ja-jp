# <a name="iobjecthandleunwrap-method"></a>IObjectHandle::Unwrap メソッド
間接参照から値渡しのマーシャ リングするオブジェクトをアンラップされます。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `ppv`  
 [out]ラップするオブジェクトへのポインター。
