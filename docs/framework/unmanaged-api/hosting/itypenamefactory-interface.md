# <a name="itypenamefactory-interface"></a>ITypeNameFactory インターフェイス
型名を分解するためのメソッドを提供します。  
  
 このインターフェイスは、.NET Framework インフラストラクチャをサポートします。独自に作成したコードから直接使用するためのものではありません。
