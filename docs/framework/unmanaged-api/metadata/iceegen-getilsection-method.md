# <a name="iceegengetilsection-method"></a>ICeeGen::GetIlSection メソッド
指定されたハンドルによって参照される基本の中間言語コードのセクションを取得します。  
  
 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetIlSection (  
    [in] HCEESECTION  *section  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `section`  
 [in]取得するセクションへのハンドル。
