# <a name="iceegengenerateceememoryimage-method"></a>ICeeGen::GenerateCeeMemoryImage メソッド
コード ベースのメモリ内のイメージを生成します。  
  
 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GenerateCeeMemoryImage (  
    [out] void    **ppImage  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `ppImage`  
 [out]生成されたイメージへのポインター。
