# <a name="imetadatafilterunmarkall-method"></a>IMetaDataFilter::UnmarkAll メソッド
現在のメタデータ スコープ内のすべてのトークンからの処理のマークを削除します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT UnmarkAll ();  
```  
  
#### <a name="parameters"></a>パラメーター  
 `UnmarkAll` パラメーターはありません。
