# <a name="coiniticor-enumeration"></a>COINITICOR 列挙型
によって使用される定数を指定[CoInitializeCor](../../../../docs/framework/unmanaged-api/hosting/coinitializecor-function.md)ときに、共通言語ランタイムを初期化します。  
  
## <a name="syntax"></a>構文  
  
```  
typedef enum tagCOINITCOR  
{  
    COINITCOR = 0x0  
} COINITICOR;  
```
