# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken メソッド
現在のメタデータ スコープから、指定されたトークンを削除します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT DeleteToken (   
    [in]  mdToken     tkObj   
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `tkObj`  
 [in]削除するトークンです。
