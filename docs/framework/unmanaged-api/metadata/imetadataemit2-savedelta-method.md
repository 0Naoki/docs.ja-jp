# <a name="imetadataemit2savedelta-method"></a>IMetaDataEmit2::SaveDelta メソッド
エディット コンティニュの現在のセッションからの変更を指定したファイルに保存します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT SaveDelta (  
    [in] LPCWSTR     szFile,   
    [in] DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `szFile`  
 [in]保存先となるファイル名を変更します。  
  
 `dwSaveFlags`  
 [in]予約されています。 ゼロを指定してください。
