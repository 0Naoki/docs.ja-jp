# <a name="imetadatatablesgetnextblob-method"></a>IMetaDataTables::GetNextBlob メソッド
テーブル内には、次のバイナリ ラージ オブジェクト (BLOB) のインデックスを取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetNextBlob (  
    [in]  ULONG   ixBlob,  
    [out] ULONG   *pNext  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `ixBlob`  
 [in]Blob の列から返されるインデックスです。  
  
 `pNext`  
 [out]次の BLOB のインデックスへのポインター。
