# <a name="imetadataemit2resetenclog-method"></a>IMetaDataEmit2::ResetENCLog メソッド
エディット コンティニュは、ログをリセットし、新しいセッションを開始します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT ResetENCLog ();  
```  
  
#### <a name="parameters"></a>パラメーター  
 `ResetENCLog`パラメーターはありません。
