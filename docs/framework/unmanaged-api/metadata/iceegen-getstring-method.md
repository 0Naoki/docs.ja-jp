# <a name="iceegengetstring-method"></a>ICeeGen::GetString メソッド
指定された相対仮想アドレスに格納されている文字列を取得します。  
  
 このメソッドは廃止され、使用する必要があります。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetString (  
    [in]  ULONG      RVA,   
    [out] LPWSTR     *lpString  
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `RVA`  
 [in]返される文字列の相対仮想アドレス。  
  
 `lpString`  
 [out]返される文字列。
