# <a name="imetadatatablesgetnumtables-method"></a>IMetaDataTables::GetNumTables メソッド
現在のスコープ内のテーブルの数を取得`IMetaDataTables`インスタンス。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GetNumTables (  
    [out]  ULONG   *pcTables  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `pcTables`  
 [out]現在のインスタンスのスコープ内のテーブルの数へのポインター。
