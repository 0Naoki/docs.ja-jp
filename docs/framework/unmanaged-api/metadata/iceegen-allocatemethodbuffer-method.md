# <a name="iceegenallocatemethodbuffer-method"></a>ICeeGen::AllocateMethodBuffer メソッド
メソッドで指定されたサイズのバッファーを作成し、メソッドの相対仮想アドレスを取得します。  
  
 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT AllocateMethodBuffer (   
    [in]  ULONG    cchBuffer,   
    [out] UCHAR    **lpBuffer,  
    [out] ULONG    *RVA  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `cchBuffer`  
 [in]作成するためのバッファーの長さ。  
  
 `lpBuffer`  
 [out]返されるバッファー。  
  
 `RVA`  
 [out]メソッドの相対仮想アドレス。
