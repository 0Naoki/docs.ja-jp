# <a name="iceegenemitstring-method"></a>ICeeGen::EmitString メソッド
コード ベースに、指定した文字列を出力します。  
  
 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT EmitString (  
    [in]  LPWSTR    lpString,  
    [out] ULONG     *RVA  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `lpString`  
 [in]出力する文字列。  
  
 `RVA`  
 [out]生成された文字列の相対仮想アドレス。
