# <a name="imetadataimportcountenum-method"></a>IMetaDataImport::CountEnum メソッド
指定された列挙子によって取得された列挙体の要素の数を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT CountEnum (  
   [in]  HCORENUM    hEnum,   
   [out] ULONG       *pulCount  
);  
```  
  
#### <a name="parameters"></a>パラメーター  
 `hEnum`  
 [in]列挙子のハンドル。  
  
 `pulCount`  
 [out]列挙された要素の数。
