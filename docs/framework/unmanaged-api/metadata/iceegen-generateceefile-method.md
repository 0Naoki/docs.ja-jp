# <a name="iceegengenerateceefile-method"></a>ICeeGen::GenerateCeeFile メソッド

 このメソッドは、古いは使用できません。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT GenerateCeeFile ();  
```  
  
#### <a name="parameters"></a>パラメーター  
 `GenerateCeeFile`パラメーターはありません。
