# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA メソッド
指定したトークンによって参照されるフィールドの相対仮想アドレスのグローバル変数の値を設定します。  
  
## <a name="syntax"></a>構文  
  
```  
HRESULT SetFieldRVA (   
    [in]  mdFieldDef  fd,   
    [in]  ULONG       ulRVA   
);  
```  
  
## <a name="parameters"></a>パラメーター  
 `fd`  
 [in]対象フィールドのトークンです。  
  
 `ulRVA`  
 [in]コードまたはデータ領域のアドレス。
