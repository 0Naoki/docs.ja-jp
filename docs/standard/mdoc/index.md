# <a name="building-documentation-with-mdoc"></a>mdoc を使用したドキュメントの構築

mdoc とは、管理対象のリファレンス ドキュメントのドキュメント構築システムです。 mdoc は、ユーザーのアセンブリを反映した、管理対象のリファレンス ドキュメントの基本形式を作成します。これは作成したコンテンツを補完します。
