# ドキュメント フラグメントのコピー
**XmlDocumentFragment** ノードを作成し、その下にノードを追加できます。  **InsertNode** メソッドを使って **XmlDocumentFragment** を挿入する場合、**XmlDocumentFragment** ノードはコピーされず、その子ノードが XML ドキュメント オブジェクト モデル \(DOM\) に挿入されます。
