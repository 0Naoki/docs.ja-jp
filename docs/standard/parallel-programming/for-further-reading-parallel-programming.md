# <a name="for-further-reading-parallel-programming"></a>関連項目 (並列プログラミング)
次のリソースには、.NET Framework の並列プログラミングに関する追加情報が含まれます。  
  
-   MSDN Web サイトの[並列コンピューティングのデベロッパー センター](https://msdn.microsoft.com/vstudio/bb964701)には、.NET Framework 、およびネイティブ C++ の並列プログラミングに関するコンテンツとフォーラム投稿へのリンクがあります。  
  
-   MSDN ブログ サイトの [.NET の並列プログラミングに関するブログ](https://blogs.msdn.microsoft.com/pfxteam/)には、.NET Framework の並列プログラミングに関する多くの詳細な記事が含まれています。  
  
-   MSDN フォーラム サイトの[並列拡張機能に関するフォーラム](https://social.msdn.microsoft.com/Forums/en-US/home?forum=parallelextensions)は、並列プログラミングについて質問したり、回答したりする場所です。  
  
-   MSDN コード ギャラリー Web サイトの[並列拡張機能のサンプル](https://code.msdn.microsoft.com/ParExtSamples) ページには、中度および高度な並列プログラミング手法を示す多くのサンプルが含まれています。
