# For Further Reading (Parallel Programming)
次のリソースには、.NET Framework における並列プログラミングについての追加情報が含まれています。  
  
-   [並列 Developer Center 計算](http://go.microsoft.com/fwlink/?LinkID=160570) MSDN Web サイトの .NET Framework およびネイティブ C\+\+ の並列プログラミングに関する最新のコンテンツおよびフォーラム ポストへのリンクがあります。  
  
-   [.NET のブログとの Parallel Programming](http://go.microsoft.com/fwlink/?LinkID=169627) MSDN ブログ サイトは .NET Framework における並列プログラミングに関する詳しい記事を紹介します。  
  
-   [同時実行ビジュアライザー ブログ](http://go.microsoft.com/fwlink/?LinkID=169630) MSDN ブログ サイトは [!INCLUDE[vs_dev10_long](../../../includes/vs-dev10-long-md.md)]の一部のエディションに付属している新しいパフォーマンス プロファイリング ツールについて説明します。  
  
-   [並列拡張フォーラム](http://go.microsoft.com/fwlink/?LinkID=169628) MSDN フォーラム サイトが並列プログラミングに関する質疑応答またはオンになります。  
  
-   [並列拡張機能の例](http://go.microsoft.com/fwlink/?LinkID=165717) MSDN Code Gallery Web サイトのページには、中級および高度な並列プログラミング技法を紹介する多くの例が含まれています。
