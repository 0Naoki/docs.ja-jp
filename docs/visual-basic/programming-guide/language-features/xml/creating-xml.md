# <a name="creating-xml-in-visual-basic"></a>Visual Basic での XML の作成
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]使用できるように*XML リテラル*コード内で直接します。 XML リテラルの構文を表す[!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)]オブジェクト、およびそれには、XML 1.0 の構文に似ています。 これにより、コード、最終的な XML と同じ構造では、XML 要素、ドキュメント、およびフラグメントをプログラムで作成が簡単にします。
