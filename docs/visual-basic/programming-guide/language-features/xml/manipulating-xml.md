# Manipulating XML in Visual Basic
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

*XML リテラル*を使用して、文字列、ファイル、ストリームなどの外部ソースから XML を読み込むことができます。  その後は、[!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] を使用して XML を操作し、[!INCLUDE[vbteclinqext](../../../../csharp/getting-started/includes/vbteclinqext_md.md)] を使用して XML を照会できます。  
  
## このセクションの内容

 [How to: Transform XML by Using LINQ](../../../../visual-basic/programming-guide/language-features/xml/how-to-transform-xml-by-using-linq.md)
