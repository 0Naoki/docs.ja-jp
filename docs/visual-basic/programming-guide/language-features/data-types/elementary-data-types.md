# <a name="elementary-data-types-visual-basic"></a>基本データ型 (Visual Basic)
Visual Basic では、多くのプログラミング要素に使用できる定義済みのデータ型のセットを提供します。 このセクションでは、これらの型とその使用方法について説明します。  
  
> [!NOTE]
>  Visual Basic の場合は、各基本データ型は構造体は、クラスでサポートされる、<xref:System>名前空間。 コンパイラは、基になる構造体またはクラスのエイリアスとしての各データ型のキーワードを使用します。 予約語を使用して変数を宣言するなど、`Byte`は構造体の完全修飾名を使用して宣言することと同じ<xref:System.Byte?displayProperty=nameWithType>です。
