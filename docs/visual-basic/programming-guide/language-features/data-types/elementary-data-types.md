# Elementary Data Types (Visual Basic)
[!INCLUDE[vs2017banner](../../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] には、多くのプログラミング要素として利用できる、一連の定義済みのデータ型が用意されています。  このセクションでは、これらの型とその使用方法について説明します。  
  
> [!NOTE]
>  Visual Basic のすべての基本データ型は構造体または <xref:System> の名前空間にあるクラスによってサポートされます。  コンパイラでは、基になる構造体またはクラスに対して各データ型のキーワードをエイリアスとして使用します。  たとえば予約語 `Byte` を使用して変数を宣言するとで完全修飾構造体名 <xref:System.Byte?displayProperty=fullName> を使用して宣言することと同じです。  
  
## このセクションの内容  
 [Numeric Data Types](../../../../visual-basic/programming-guide/language-features/data-types/numeric-data-types.md)  
 整数型および非整数型の数値型について説明します。  
  
 [Character Data Types](../../../../visual-basic/programming-guide/language-features/data-types/character-data-types.md)  
 `Char` および `String` について説明します。  
  
 [Miscellaneous Data Types](../../../../visual-basic/programming-guide/language-features/data-types/miscellaneous-data-types.md)  
 ブール型\(`Boolean`\)、日付型 \(`Date`\)、オブジェクト型 \(`Object`\) の各データ型について説明します。
