# <a name="how-to-call-a-procedure-that-does-not-return-a-value-visual-basic"></a>方法: 値を返さないプロシージャを呼び出す (Visual Basic)

### <a name="to-call-a-sub-procedure"></a>Sub プロシージャを呼び出しています  
  
1.  名前を指定、`Sub`プロシージャです。
