## <a name="example"></a><span data-ttu-id="53947-104">例</span><span class="sxs-lookup"><span data-stu-id="53947-104">Example</span></span>  
  
### <a name="description"></a><span data-ttu-id="53947-105">説明</span><span class="sxs-lookup"><span data-stu-id="53947-105">Description</span></span>  
 <span data-ttu-id="53947-106">この例では、2 つの名前空間を宣言します。</span><span class="sxs-lookup"><span data-stu-id="53947-106">This example declares two namespaces.</span></span> <span data-ttu-id="53947-107">指定します、`http://www.adventure-works.com`名前空間のプレフィックス`aw`、および、`www.fourthcoffee.com`名前空間のプレフィックスを持つ`fc`します。</span><span class="sxs-lookup"><span data-stu-id="53947-107">It specifies that the `http://www.adventure-works.com` namespace has the prefix `aw`, and that the `www.fourthcoffee.com` namespace has the prefix of `fc`.</span></span>  
  
### <a name="code"></a><span data-ttu-id="53947-108">コード</span><span class="sxs-lookup"><span data-stu-id="53947-108">Code</span></span>  
  
```vb  
Imports <xmlns:aw="http://www.adventure-works.com">  
Imports <xmlns:fc="www.fourthcoffee.com">  
  
Module Module1  
  
    Sub Main()  
        Dim root As XElement = _  
            <aw:Root>  
                <fc:Child>  
                    <aw:DifferentChild>other content</aw:DifferentChild>  
                </fc:Child>  
                <aw:Child2>c2 content</aw:Child2>  
                <fc:Child3>c3 content</fc:Child3>  
            </aw:Root>  
        Console.WriteLine(root)  
    End Sub  
  
End Module  
```
