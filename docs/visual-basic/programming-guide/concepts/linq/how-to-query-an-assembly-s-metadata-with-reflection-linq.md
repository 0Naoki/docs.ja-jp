# <a name="how-to-query-an-assembly39s-metadata-with-reflection-linq-visual-basic"></a><span data-ttu-id="5f049-102">方法: リフレクション (LINQ) (Visual Basic) を使用してアセンブリのメタデータを照会します。</span><span class="sxs-lookup"><span data-stu-id="5f049-102">How to: Query An Assembly&#39;s Metadata with Reflection (LINQ) (Visual Basic)</span></span>
<span data-ttu-id="5f049-103">次の例では、指定した検索条件に一致するメソッドについてのメタデータを取得する LINQ をリフレクションを使用して使用する方法を示します。</span><span class="sxs-lookup"><span data-stu-id="5f049-103">The following example shows how LINQ can be used with reflection to retrieve specific metadata about methods that match a specified search criterion.</span></span> <span data-ttu-id="5f049-104">ここで、クエリでは、配列などの列挙可能な型を返すアセンブリのすべてのメソッドの名前は見つけします。</span><span class="sxs-lookup"><span data-stu-id="5f049-104">In this case, the query will find the names of all the methods in the assembly that return enumerable types such as arrays.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5f049-105">例</span><span class="sxs-lookup"><span data-stu-id="5f049-105">Example</span></span>  
  
```vb  
Imports System.Reflection  
Imports System.IO  
Imports System.Linq  
Module Module1  
  
    Sub Main()  
        Dim asmbly As Assembly =   
            Assembly.Load("System.Core, Version=3.5.0.0, Culture=neutral, PublicKeyToken= b77a5c561934e089")  
  
        Dim pubTypesQuery = From type In asmbly.GetTypes()   
                            Where type.IsPublic   
                            From method In type.GetMethods()   
                            Where method.ReturnType.IsArray = True   
                            Let name = method.ToString()   
                            Let typeName = type.ToString()   
                            Group name By typeName Into methodNames = Group  
  
        Console.WriteLine("Getting ready to iterate")  
        For Each item In pubTypesQuery  
            Console.WriteLine(item.methodNames)  
  
            For Each type In item.methodNames  
                Console.WriteLine(" " & type)  
            Next  
        Next  
        Console.ReadKey()  
    End Sub  
  
End Module  
```  
  
 <span data-ttu-id="5f049-106">例では、<xref:System.Reflection.Assembly.GetTypes%2A>を指定したアセンブリで型の配列を返すメソッド</xref:System.Reflection.Assembly.GetTypes%2A>。</span><span class="sxs-lookup"><span data-stu-id="5f049-106">The example uses the <xref:System.Reflection.Assembly.GetTypes%2A> method to return an array of types in the specified assembly.</span></span> <span data-ttu-id="5f049-107">[Where 句](../../../../visual-basic/language-reference/queries/where-clause.md)パブリック型のみが返されるようにフィルターを適用します。</span><span class="sxs-lookup"><span data-stu-id="5f049-107">The [Where Clause](../../../../visual-basic/language-reference/queries/where-clause.md) filter is applied so that only public types are returned.</span></span> <span data-ttu-id="5f049-108">使用して各パブリック型のサブクエリが生成される、<xref:System.Reflection.MethodInfo>から返される配列、<xref:System.Type.GetMethods%2A>を呼び出す</xref:System.Type.GetMethods%2A></xref:System.Reflection.MethodInfo>。</span><span class="sxs-lookup"><span data-stu-id="5f049-108">For each public type, a subquery is generated by using the <xref:System.Reflection.MethodInfo> array that is returned from the <xref:System.Type.GetMethods%2A> call.</span></span> <span data-ttu-id="5f049-109">これらの結果は戻り値の型は<xref:System.Collections.Generic.IEnumerable%601>。</xref:System.Collections.Generic.IEnumerable%601>を実装する型か、または配列メソッドのみを返すフィルター処理します。</span><span class="sxs-lookup"><span data-stu-id="5f049-109">These results are filtered to return only those methods whose return type is an array or else a type that implements <xref:System.Collections.Generic.IEnumerable%601>.</span></span> <span data-ttu-id="5f049-110">最後に、これらの結果は、キーとして、型名を使用して、グループ化されます。</span><span class="sxs-lookup"><span data-stu-id="5f049-110">Finally, these results are grouped by using the type name as a key.</span></span>
