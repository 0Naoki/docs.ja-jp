# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML では、.net 統合言語クエリ (LINQ) フレームワークを利用しているメモリ内 XML プログラミング インターフェイスを提供します。 LINQ to XML では、最新の .NET Framework 言語機能を使用および、更新され再設計されたドキュメント オブジェクト モデル (DOM) XML プログラミング インターフェイスに相当します。  
  
 LINQ のファミリのテクノロジは、オブジェクト (LINQ to Objects)、リレーショナル データベース (LINQ to SQL)、および XML (LINQ to XML) の一貫したクエリのエクスペリエンスを提供します。
