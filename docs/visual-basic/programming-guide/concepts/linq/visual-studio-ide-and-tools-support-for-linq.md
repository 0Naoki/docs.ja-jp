# <a name="visual-studio-ide-and-tools-support-for-linq-visual-basic"></a>Visual Studio IDE およびツールの LINQ (Visual Basic) のサポート
Visual Studio 統合開発環境 (IDE) では、LINQ アプリケーションの開発をサポートする次の機能を提供します。  
  
## <a name="object-relational-designer"></a>Object Relational Designer  
 オブジェクト リレーショナル デザイナーで使用できるビジュアル デ ザイン ツールは、 [LINQ to SQL](https://msdn.microsoft.com/library/bb386976)を基になるデータベース内のリレーショナル データを表すクラスを生成する Visual Basic でのアプリケーションです。 詳細については、次を参照してください。 [LINQ to SQL ツール Visual Studio で](https://docs.microsoft.com/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2)します。  
  
## <a name="sqlmetal-command-line-tool"></a>SQLMetal コマンド ライン ツール  
 SQLMetal は、SQL アプリケーションを LINQ で使用する既存のデータベースからクラスを生成するビルド プロセスで使用できるコマンド ライン ツールです。 詳細については、次を参照してください。 [SqlMetal.exe (コード生成ツール)](http://msdn.microsoft.com/library/819e5a96-7646-4fdb-b14b-fe31221b0614)します。  
  
## <a name="linq-aware-code-editor"></a>LINQ 対応コード エディター  
 Visual Basic コード エディターでは、LINQ IntelliSense と書式設定機能を広範囲にサポートしています。  
  
## <a name="visual-studio-debugger-support"></a>Visual Studio デバッガーのサポート  
 Visual Studio デバッガーでは、クエリ式のデバッグをサポートします。 詳細については、次を参照してください。 [LINQ のデバッグ](https://docs.microsoft.com/visualstudio/debugger/debugging-linq)します。
