# <a name="39withevents39-variables-cannot-be-typed-as-arrays"></a>&#39;です。WithEvents &#39;変数を配列として型指定することはできません。
`WithEvents`を使用して、配列を宣言しようとししました。 `WithEvents`を使用して必要な数の個別変数を宣言できますが、そのように配列を宣言できません。  
  
 **エラー ID:** BC30476  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  個別に変数を宣言します。
