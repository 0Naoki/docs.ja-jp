# ソース フォルダーが存在しません
存在しないソース フォルダーを参照する操作を実行しようとしました。 指定したパスが正しくないことが原因となっている可能性があります。  
  
### このエラーを解決するには  
  
-   相対パスを使用している場合は特に、正しいパスを使用していることを確認します。
