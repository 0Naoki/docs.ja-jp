# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39;Sub New&#39;として宣言できません&#39;&lt;修飾子。&gt;&#39;
`Sub New` などの修飾子を使用して `Overrides`を宣言使用としましたが、これは無効です。  
  
 **エラー ID:** BC30364  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   修飾子を削除します。
