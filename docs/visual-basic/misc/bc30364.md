# &#39;Sub New&#39; を &#39;&lt;modifier&gt;&#39; として宣言することはできません。
`Overrides` などの修飾子を使用して `Sub New` を宣言使用としましたが、これは無効です。  
  
 **エラー ID:** BC30364  
  
### このエラーを解決するには  
  
-   修飾子を削除します。
