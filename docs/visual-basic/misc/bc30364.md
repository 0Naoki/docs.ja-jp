# <a name="sub-new-cannot-be-declared-ltmodifiergt"></a>'Sub New' を宣言することはできません '&lt;修飾子&gt;'
`Sub New` などの修飾子を使用して `Overrides`を宣言使用としましたが、これは無効です。  
  
 **エラー ID:** BC30364  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   修飾子を削除します。
