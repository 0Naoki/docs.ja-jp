# <a name="no-accessible-overloaded-methodname-can-be-called-with-these-arguments-without-a-widening-conversion-list"></a>ないオーバー ロードされた '\<methodname >' 拡大変換しないで、これらの引数と共に呼び出すことができます:\<一覧 >
オーバーロードされたメソッドが呼び出されましたが、拡大変換せずに指定された引数のリストと一致するメソッドはありませんでした。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `Option Strict Off`を指定します。  
  
- オーバーロードされたメソッドのシグネチャのいずれかと一致するように、引数を変更します。
