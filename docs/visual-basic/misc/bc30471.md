# 式は、配列またはメソッドではありません。引数リストを指定することはできません。
配列またはメソッドではない式の引数リストを指定しました。 引数リストを指定することができるのは、配列またはメソッドのみです。  
  
 **エラー ID:** BC30471  
  
### このエラーを解決するには  
  
-   引数リストを削除します。
