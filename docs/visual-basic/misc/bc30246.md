# <a name="39ltmodifiergt39-is-not-valid-on-a-local-constant-declaration"></a>&#39;です。&lt;修飾子&gt;&#39; は、ローカル定数宣言では無効
プロシージャ内の `Const` ステートメントに、 `Overloads`のような正しくないキーワードが含まれています。  
  
 **エラー ID:** BC30246  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Const` ステートメントから正しくないキーワードを削除します。
