# &#39;&lt;modifier&gt;&#39; は、ローカル定数宣言では有効ではありません
プロシージャ内の `Const` ステートメントに、`Overloads` のような正しくないキーワードが含まれています。  
  
 **エラー ID:** BC30246  
  
### このエラーを解決するには  
  
1.  `Const` ステートメントから正しくないキーワードを削除します。
