# <a name="39optional39-and-39paramarray39-cannot-be-combined"></a>&#39;以外の場合は省略可能な &#39;です。および &#39;です。ParamArray &#39;組み合わせることができません。
`Optional` 修飾子と `ParamArray` 修飾子は同時に指定できません。  
  
 **エラー ID:** BC30642  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `ParamArray` または `Optional`を指定し、両方は指定しないでください。
