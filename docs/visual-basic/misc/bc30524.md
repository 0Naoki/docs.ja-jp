# <a name="property-39ltpropertynamegt39-is-39writeonly39"></a>プロパティ &#39;&lt;propertyname&gt;&#39; が &#39;です。WriteOnly &#39;
`WriteOnly`で宣言されているプロパティから値を取得しようとしました。  
  
 **エラー ID:** BC30524  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロパティ宣言から `WriteOnly` 指定子を削除します。
