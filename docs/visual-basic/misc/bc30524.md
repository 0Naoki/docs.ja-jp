# <a name="property-propertyname-is-writeonly"></a>プロパティ '\<propertyname >' は 'WriteOnly'
`WriteOnly`で宣言されているプロパティから値を取得しようとしました。  
  
 **エラー ID:** BC30524  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロパティ宣言から `WriteOnly` 指定子を削除します。
