# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39;AddressOf&#39;式は、デバッグ ウィンドウでは有効ではありません
`AddressOf` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30731  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  デバッグ コードから `AddressOf` ステートメントを削除します。
