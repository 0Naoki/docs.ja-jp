# <a name="39option-infer39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;Infer オプション&#39;が続きますのみ&#39;で&#39;または&#39;オフ&#39;
`Option Infer` ステートメントは `On` または `Off`のどちらかを指定する必要があります。 その他の値は使用できません。  
  
 **エラー ID:** BC30979  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Option Infer On` か `Option Infer Off`のどちらかを指定します。
