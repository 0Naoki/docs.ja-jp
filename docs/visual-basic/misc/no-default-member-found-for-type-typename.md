# 既定メンバーが型 &#39;&lt;typename&gt;&#39; に見つかりませんでした
遅延バインディング インデックス `Get` または `Set` を呼び出そうとしました。  
  
### このエラーを解決するには  
  
1.  実行している操作に応じて `Get` または `Set` で既定プロパティがオブジェクトに含まれていることを確認します。
