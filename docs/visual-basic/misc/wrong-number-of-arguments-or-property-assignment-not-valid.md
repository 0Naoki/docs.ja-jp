# 引数の数か、プロパティの割り当てが正しくありません
正しくない割り当てを行おうとしました。  
  
### このエラーを解決するには  
  
1.  指定した引数の数がターゲットに必要な数と一致することを確認します。  
  
2.  プロパティの割り当てを確認します。
