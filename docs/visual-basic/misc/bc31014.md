# DLL &#39;&lt;filename&gt;&#39; にエントリ ポイント &#39;&lt;name&gt;&#39; が見つかりません: &lt;error&gt;
指定した DLL でプロシージャ名が見つかりません。  
  
 **エラー ID:** BC31014  
  
### このエラーを解決するには  
  
1.  プロシージャ名のスペルが正しいことと、DLL 内の関数と大文字と小文字の区別が一致していることを確認します。
