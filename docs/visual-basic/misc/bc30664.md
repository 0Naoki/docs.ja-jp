# <a name="conversions-from-lttypename1gt-to-lttypename2gt-must-be-explicit"></a>変換 '&lt;typename1&gt;'to'&lt;typename2&gt;' 明示的なである必要があります
この変換を暗黙的に行うことはできません。  
  
 **エラー ID:** BC30664  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `CType`または特定の変換関数 ( `CInt` または `CDec`など) を使用します。
