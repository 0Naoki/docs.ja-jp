# <a name="conversions-from-typename1-to-typename2-must-be-explicit"></a>変換 '\<typename1 >' を'\<typename2 >' 明示的なである必要があります
この変換を暗黙的に行うことはできません。  
  
 **エラー ID:** BC30664  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `CType`または特定の変換関数 ( `CInt` または `CDec`など) を使用します。
