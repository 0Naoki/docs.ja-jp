# &#39;&lt;typename1&gt;&#39; から &#39;&lt;typename2&gt;&#39; への変換は、明示的に行う必要があります
この変換を暗黙的に行うことはできません。  
  
 **エラー ID:** BC30664  
  
### このエラーを解決するには  
  
-   `CType` または特定の変換関数 \(`CInt` または `CDec` など\) を使用します。
