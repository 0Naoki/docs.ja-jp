# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>変換 &#39;&lt;typename1&gt;&#39; の &#39;&lt;typename2&gt;&#39; 明示的である必要があります
この変換を暗黙的に行うことはできません。  
  
 **エラー ID:** BC30664  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `CType`または特定の変換関数 ( `CInt` または `CDec`など) を使用します。
