# <a name="39synclock39-statement-must-end-with-a-matching-39end-synclock39"></a>&#39;SyncLock&#39; 、対応するステートメントが終了する必要があります&#39;End SyncLock&#39;
`SyncLock` ブロックが `SyncLock` キーワードから始まり `End SyncLock` コンストラクトで終了します。  
  
 **エラー ID:** BC30675  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `SyncLock` ブロックが `End SyncLock` コンストラクトで終了することを確認します。
