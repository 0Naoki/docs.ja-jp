# <a name="unable-to-apply-security-attributes-to-39ltnamegt39-lterrorgt"></a>セキュリティ属性を適用できません '&lt;名&gt;':&lt;エラー。&gt;
セキュリティ属性を適用するときにエラーが発生しました。 セキュリティ属性の使い方が間違っているか、コンパイラに問題がある可能性があります。  
  
 **エラー ID:** BC31520  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  属性の構文と引数を確認します。  
  
2.  プログラムをもう一度コンパイルし、エラーがまだ発生するかどうか確認します。  
  
3.  エラーが引き続き発生する場合は、作業内容を保存し、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] を再起動します。  
  
4.  エラーが再発する場合は再インストール[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]します。  
  
5.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。
