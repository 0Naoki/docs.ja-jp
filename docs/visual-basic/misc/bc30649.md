# <a name="typename-is-an-unsupported-type"></a>'\<typename>' は、サポートされていない型です。
指定された型は、Visual Studio で表現できません。  
  
 **エラー ID:** BC30649  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   サポートされている型を使用します。  
  
-   新しい型を作成します。
