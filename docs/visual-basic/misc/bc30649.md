# &#39;&lt;typename&gt;&#39; は、サポートされていない型です
指定された型は [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] で表すことができません。  
  
 **エラー ID:** BC30649  
  
### このエラーを解決するには  
  
-   サポートされる型を使用します。  
  
-   新しい型を開発します。
