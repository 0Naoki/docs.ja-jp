# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39;&lt;typename&gt; &#39;は、サポートされていない型です。
指定された型は、Visual Studio によって表されることはできません。  
  
 **エラー ID:** BC30649  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   サポートされる型を使用します。  
  
-   新しい型を開発します。
