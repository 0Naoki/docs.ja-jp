# <a name="expected-closing-39gt39-for-embedded-expression"></a>終わり&#39;%&gt; &#39;埋め込み式の
XML リテラルの埋め込み式の終わりに必要な `%>` 文字がありません。  
  
 **エラー ID:** BC31159  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   埋め込み式の終わりに必要な `%>` 文字を追加します。
