# <a name="expected-closing-gt-for-embedded-expression"></a>終わり '%&gt;' の埋め込み式
XML リテラルの埋め込み式の終わりに必要な `%>` 文字がありません。  
  
 **エラー ID:** BC31159  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   埋め込み式の終わりに必要な `%>` 文字を追加します。
