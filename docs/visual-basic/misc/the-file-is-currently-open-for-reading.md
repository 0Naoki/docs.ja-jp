# <a name="the-file-is-currently-open-for-reading"></a>ファイルは現在読み取り用に開かれています
現在読み取り用に開かれているファイルにアクセスしようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイルを閉じて、操作をやり直します。
