# <a name="39finally39-must-end-with-a-matching-39end-try39"></a>&#39;です。最後に &#39;です。一致する &#39; で終わる必要があります。End Try &#39;
A`Finally`ステートメントが、一致することがなく、コードに表示される`End Try`ステートメントです。 `Finally` ステートメントは、 `End Try` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30442  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Finally` ステートメントを削除します。  
  
2.  ブロックを終了するための `End Try` ステートメントを追加します。
