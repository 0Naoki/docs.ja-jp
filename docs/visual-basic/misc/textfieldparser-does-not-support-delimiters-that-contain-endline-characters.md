# TextFieldParser は行末文字を含む区切り記号をサポートしていません。
行末文字を含む区切り記号が指定されていますが、`TextFieldParse`r は行末文字を含む区切り記号をサポートしていません。  
  
### このエラーを解決するには  
  
-   適切な区切り記号を指定します。
