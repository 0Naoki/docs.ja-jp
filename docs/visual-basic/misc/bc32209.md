# <a name="project-39ltprojectnamegt39-requires-a-reference-to-version-39ltversionnumber1gt39-of-assembly-39ltassemblynamegt39-but-references-version-39ltversionnumber2gt39-of-assembly-39ltassemblynamegt39-visual-basic-error"></a>プロジェクト&#39; &lt;projectname&gt; &#39;バージョンへの参照を必要と&#39; &lt;versionnumber1&gt; &#39;アセンブリの&#39; &lt;assemblyname&gt;&#39;、参照のバージョンでは&#39; &lt;versionnumber2&gt; &#39;アセンブリの&#39; &lt;assemblyname&gt; &#39; (Visual Basic エラー)
プロジェクトは、他の場所で定義されているアセンブリへの間接参照を行いますが、プロジェクトには、それ以降のバージョンのアセンブリへの直接参照も含まれています。  
  
 コンパイラが間接参照を使用するようコードを許可した場合、以前のバージョンではなく以降のバージョンで定義されている型およびプログラミング要素にアクセスすることはできません。  
  
 **エラー ID:** BC32209  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   以前のバージョンのアセンブリへの間接参照を削除し、以降のバージョンへの直接参照を使用します。
