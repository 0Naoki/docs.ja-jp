# オートメーション オブジェクトには既定値がありません
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] が、指定したオブジェクトの既定のメンバーを判断できません。  
  
### このエラーを解決するには  
  
-   オブジェクトのドキュメントを確認し、プロパティまたはメソッドを明示的にご指定ください。
