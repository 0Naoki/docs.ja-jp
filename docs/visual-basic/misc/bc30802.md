# &#39;Type&#39; ステートメントは現在サポートされていません。&#39;Structure&#39; ステートメントを使用してください
ユーザー定義型を定義するために `Type` キーワードを使用することはできません。  
  
 **エラー ID:** BC30802  
  
### このエラーを解決するには  
  
-   新しい型を定義するには、`Structure` ステートメントを使用します。
