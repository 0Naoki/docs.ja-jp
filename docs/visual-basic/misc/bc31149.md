# XML 属性 &#39;attributeName&#39; が重複しています
XML 要素リテラルに指定された属性が、XML 要素リテラルに既に指定されている属性と一致します。 XML 要素リテラルの属性名は一意である必要があります。  
  
 **エラー ID:** BC31149  
  
### このエラーを解決するには  
  
-   属性の名前を XML 要素リテラルに一意な名前に変更します。
