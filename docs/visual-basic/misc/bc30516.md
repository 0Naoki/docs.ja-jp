# <a name="overload-resolution-failed-because-no-accessible-method-accepts-this-number-of-arguments"></a>オーバー ロードの解決が失敗しましたない '\<メソッド >' の引数には、この数を受け付ける
オーバーロードされたメソッドを呼び出しましたが、コンパイラで、対応する引数リストを持つメソッドを検出できません。  
  
 **エラー ID:** BC30516  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定した引数リストを確認します。
