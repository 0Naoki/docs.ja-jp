# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-accepts-this-number-of-arguments"></a>オーバー ロードの解決が失敗しましたない '&lt;メソッド&gt;' この引数の数を受け取る
オーバーロードされたメソッドを呼び出しましたが、コンパイラで、対応する引数リストを持つメソッドを検出できません。  
  
 **エラー ID:** BC30516  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定した引数リストを確認します。
