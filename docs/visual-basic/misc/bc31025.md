# キー ファイル &#39;&lt;filename&gt;&#39; を開けません: &lt;error&gt;
このプロジェクトに指定したキー ファイルを開けませんでした。  
  
 **エラー ID:** BC31025  
  
### このエラーを解決するには  
  
-   キー ファイルが存在し、別のアプリケーションによってロックされていないことを確認します。
