# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39;です。Catch &#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Catch` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30715  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `Catch` ステートメントを削除します。
