# BaudRate は 0 より大きくなければなりません
`My.Computer.Ports.OpenSerialPort` メソッドに指定された `BaudRate` 引数は、0 より大きくなければなりません。  
  
### このエラーを解決するには  
  
-   `BaudRate` 引数の値を正の数値に変更します。
