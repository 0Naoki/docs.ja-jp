# <a name="39ltmodifiergt39-is-not-valid-on-an-event-declaration"></a>&#39;です。&lt;修飾子&gt;&#39; イベント宣言で無効です
`Event` ステートメントに `ReadOnly`などの無効なキーワードが含まれています。  
  
 **エラー ID:** BC30243  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Event` ステートメントから無効なキーワードを削除します。
