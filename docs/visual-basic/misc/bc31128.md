# &#39;RemoveHandler&#39; は既に宣言されています
カスタム イベントの宣言に 2 つ以上 `RemoveHandler` の宣言があります。`RemoveHandler` の宣言では、イベント ハンドラーを削除するために使用するプロシージャを宣言します。  
  
 **エラー ID:** BC31128  
  
### このエラーを解決するには  
  
-   不要な `RemoveHandler` ステートメントを削除します。
