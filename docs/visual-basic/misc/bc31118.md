# &#39;End AddHandler&#39; は、行の最初のステートメントである必要があります
`End AddHandler` ステートメントがコロン \(:\) ステートメント区切り文字の後に続いています。`End AddHandler` はそのソース行でただ 1 つのステートメントである必要があります。  
  
 **エラー ID:** BC31118  
  
### このエラーを解決するには  
  
-   複数のステートメントを別々の行に分けます。
