# Null 許容パラメーターには型を指定しなければなりません
パラメーターに Null 許容型修飾子 \(?\) が含まれていますが、型が指定されていません。  
  
 **エラー ID:** BC36632  
  
### このエラーを解決するには  
  
-   Null 許容パラメーターを宣言するときに、型を指定します。
