# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-the-array-types-have-different-numbers-of-dimensions"></a>型の値 '&lt;type1&gt;'に変換できません'&lt;type2&gt;' 配列型があるディメンションの数が異なるため、
1 つの配列をディメンションの数が異なる別の配列に変換しようとしました。  
  
 **エラー ID:** BC30414  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ディメンションの数が一致するように、片方または両方の配列を変更します。
