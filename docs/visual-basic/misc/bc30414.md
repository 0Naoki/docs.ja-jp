# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>型 &#39; の値&lt;type1&gt;&#39; に変換できません (& m); #39&lt;type2&gt;&#39;配列型の次元数が異なるためです。
1 つの配列をディメンションの数が異なる別の配列に変換しようとしました。  
  
 **エラー ID:** BC30414  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ディメンションの数が一致するように、片方または両方の配列を変更します。
