# 配列型の次元数が異なるため、型 &#39;&lt;type1&gt;&#39; の値を &#39;&lt;type2&gt;&#39; に変換できません。
1 つの配列をディメンションの数が異なる別の配列に変換しようとしました。  
  
 **エラー ID:** BC30414  
  
### このエラーを解決するには  
  
-   ディメンションの数が一致するように、片方または両方の配列を変更します。
