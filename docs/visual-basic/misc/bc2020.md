# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>エラーの数が最大数を超えました。
コンパイラの最大エラー数 100 に達しました。  
  
 **エラー ID:** BC2020  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   各エラーを順にデバッグおよびアドレス指定して、生成されたエラーの数を削減します。
