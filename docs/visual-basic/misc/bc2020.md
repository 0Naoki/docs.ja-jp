# エラーの数が最大数を超えました。
コンパイラの最大エラー数 100 に達しました。  
  
 **エラー ID:** BC2020  
  
### このエラーを解決するには  
  
-   各エラーを順にデバッグおよびアドレス指定して、生成されたエラーの数を削減します。
