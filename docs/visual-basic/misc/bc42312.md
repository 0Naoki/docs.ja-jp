# <a name="xml-comment-block-cannot-be-associated-with-any-language-element-that-supports-the-application-of-xml-documentation-comments"></a>XML コメント ブロックを、XML ドキュメント コメントのアプリケーションをサポートする言語要素と関連付けできません
XML コメント ブロックを、有効な言語要素に関連付けできません。 XML コメントは無視されます。 このエラーは、コメント ブロック内の XML タグに対応する言語要素がない場合にも発生します。  
  
 **エラー ID:** BC42312  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分な XML タグを見つけて削除するか、このタグが記述する説明の言語要素を追加します。
