# <a name="argument-not-specified-for-parameter-39ltparameternamegt39"></a>パラメーター &#39; に対して指定されていない引数&lt;parametername&gt;&#39;です。
必要なパラメーターの引数が指定されていません。  
  
 **エラー ID:** BC36569  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定されたパラメーターの引数を指定します。
