# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>&#39;&#39;ジェネリック型またはメソッドの型引数を指定する場合に必須
ステートメントにより、 [Of](../../visual-basic/language-reference/statements/of-clause.md) 句を使用せずに、ジェネリック型からの型の構築またはジェネリック メソッドの呼び出しが試行されます。  
  
 ジェネリック型の Visual Basic 構文の型パラメーターと型の引数によって発生する可能性を呼び出し、`Of`キーワード。 さらに、型パラメーター リストまたは型引数リストは、かっこで囲まれている必要があります。  
  
 **エラー ID:** BC32093  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   型引数リストの先頭に `Of` キーワードを含めて、リスト全体をかっこで囲みます。
