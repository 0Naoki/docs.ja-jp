# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>ジェネリック型またはメソッドの型引数を指定するときに必要な 'の'
ステートメントでは、ジェネリック型から型を構築または使用しなくてもジェネリック メソッドを呼び出すしよう、[の](../../visual-basic/language-reference/statements/of-clause.md)句。  
  
 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]のジェネリック型の構文の型パラメーターと型の引数によってもたらされるを呼び出し、`Of`キーワードです。 さらに、型パラメーター リストまたは型引数リストは、かっこで囲まれている必要があります。  
  
 **エラー ID:** BC32093  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   型引数リストの先頭に `Of` キーワードを含めて、リスト全体をかっこで囲みます。
