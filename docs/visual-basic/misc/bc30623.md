# &#39;End Namespace&#39; の前には、対応する &#39;Namespace&#39; を指定しなければなりません
名前空間ブロックは `Namespace` ステートメントで始まり、`End Namespace` コンストラクトで終わります。  
  
 **エラー ID:** BC30623  
  
### このエラーを解決するには  
  
-   名前空間ブロックが `Namespace` ステートメントで始まることをご確認ください。
