# <a name="39end-namespace39-must-be-preceded-by-a-matching-39namespace39"></a>&#39;です。End Namespace &#39;一致する &#39; の前が必要Namespace &#39;
名前空間ブロックは `Namespace` ステートメントで始まり、 `End Namespace` コンストラクトで終わります。  
  
 **エラー ID:** BC30623  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   名前空間ブロックが `Namespace` ステートメントで始まることをご確認ください。
