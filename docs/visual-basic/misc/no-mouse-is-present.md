# マウスが存在しません
`My.Computer.Mouse` オブジェクトのいずれかのプロパティが呼び出されましたが、コンピューターにマウスが接続されていないか、マウス ポートが取り付けられていません。  
  
### このエラーを解決するには  
  
-   `My.Computer.Mouse` オブジェクトのプロパティの呼び出しを、`Try...Catch` ブロックで囲みます。  
  
     または  
  
-   マウスをコンピューターに取り付けます。
