# 指定された引数を使って期間を計算することはできません
`NPer` 関数の呼び出しに、すべての必須の引数が含まれていません。  
  
### このエラーを解決するには  
  
-   `Rate`、`Prnt` および `PV` の値を関数呼び出しに含めます。
