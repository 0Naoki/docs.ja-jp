# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39declare-sub39-language-element"></a>XML コメント タグ &#39; を返します、&#39;許可されていません、&#39; sub &#39; の宣言言語要素
XML コメント タグ 'returns' は、'declare sub' 言語要素では許可されていません。 XML コメントは無視されます。  
  
 `Declare Sub` 宣言に対する XML コメントに、 `<`returns`>` タグを含めることはできません。  
  
 **エラー ID:** BC42315  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   サポートされていない `<`returns`>` タグを削除します。
