# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>'\<qualifiedmethod >' はこのコンテキストでアクセスはできません'\<修飾子 >'。
修飾子が適用されているためにこのコンテキストでアクセスできないメソッドにアクセスしようとしました。  
  
 **エラー ID:** BC30390  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   修飾子を削除します。
