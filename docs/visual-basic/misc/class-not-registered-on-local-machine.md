# <a name="class-not-registered-on-local-machine"></a>クラスがローカル コンピューターに登録されていません
このエラーは通常、ファイルが破損している場合に発生します。 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]の再インストールが必要なことがあります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]を再インストールします。
