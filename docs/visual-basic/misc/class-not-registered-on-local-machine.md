# クラスがローカル コンピューターに登録されていません
このエラーは通常、ファイルが破損している場合に発生します。[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] の再インストールが必要なことがあります。  
  
### このエラーを解決するには  
  
-   [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] を再インストールします。
