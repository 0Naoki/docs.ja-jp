# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;です。&lt;mathfunction1&gt;&#39; が宣言されていません
'\<mathfunction1 >' は宣言されていません。 System.Math クラスに移動しましたし、呼ばれていた関数 '\<mathfunction2 >' です。  
  
 組み込まれていたいくつかの関数[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]以前のバージョンに移動されました、<xref:System.Math?displayProperty=nameWithType>名前空間。 これにより、これらの機能をより幅広くすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30819  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   宣言されたメソッドを使用して<xref:System.Math?displayProperty=nameWithType>です。
