# <a name="39ltmathfunction1gt39-is-not-declared"></a>&#39;&lt;mathfunction1&gt; &#39;が宣言されていません
'\<mathfunction1 >' は宣言されていません。 System.Math クラスに移動しましたし、呼ばれていた関数 '\<mathfunction2 >' です。  
  
 以前のバージョンでは、Visual Basic に組み込まれていたいくつかの関数に移動されている、<xref:System.Math?displayProperty=nameWithType>名前空間。 これにより、これらの機能をより一般的にすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30819  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   宣言されたメソッドを使用して<xref:System.Math?displayProperty=nameWithType>です。
