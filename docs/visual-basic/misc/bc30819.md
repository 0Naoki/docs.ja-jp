# <a name="ltmathfunction1gt-is-not-declared"></a>'&lt;mathfunction1&gt;' が宣言されていません
'\<mathfunction1 >' が宣言されていません。 関数、System.Math クラスに移動しと呼ばれる '\<mathfunction2 >'。  
  
 以前のバージョンの Visual Basic に組み込まれていたいくつかの関数に移動されました、<xref:System.Math?displayProperty=nameWithType>名前空間。 これにより、これらの機能をより一般的にすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30819  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   <xref:System.Math?displayProperty=nameWithType>で宣言したメソッドを使用します。
