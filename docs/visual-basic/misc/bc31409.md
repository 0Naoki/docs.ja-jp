# 別の &#39;&lt;methodname2&gt;&#39; が &#39;Overloads&#39; と宣言されているため、&lt;method&gt; &#39;&lt;methodname1&gt;&#39; は &#39;Overloads&#39; と宣言されなければなりません
同じ名前のもう 1 つのプロパティまたはメソッドが `Overloads` 修飾子で宣言されます。  
  
 **エラー ID:** BC31409  
  
### このエラーを解決するには  
  
1.  プロパティまたはメソッドに `Overloads` 修飾子を追加します。  
  
2.  同じ名前のすべてのプロパティまたはメソッドから `Overloads` 修飾子を削除します。
