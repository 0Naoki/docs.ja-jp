# <a name="39end-synclock39-must-be-preceded-by-a-matching-39synclock39"></a>&#39;End SyncLock&#39;前に、対応する必要がある&#39;SyncLock&#39;
`SyncLock` ブロックは `SyncLock` キーワードから始まり `End``SyncLock` コンストラクトで終了します。  
  
 **エラー ID:** BC30674  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `SyncLock` ブロックは必ず `SyncLock` ステートメントで始まるようにしてください。
