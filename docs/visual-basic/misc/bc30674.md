# &#39;End SyncLock&#39; の前には、対応する &#39;SyncLock&#39; が必要です
`SyncLock` ブロックは `SyncLock` キーワードから始まり `End` `SyncLock` コンストラクトで終了します。  
  
 **エラー ID:** BC30674  
  
### このエラーを解決するには  
  
-   `SyncLock` ブロックは必ず `SyncLock` ステートメントで始まるようにしてください。
