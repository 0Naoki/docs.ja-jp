# <a name="expected-closing-39gt39-for-xml-tag"></a>終わり&#39; &gt; &#39;の XML タグ
XML リテラルの XML タグの終わりに必要な '>' 文字がありません。  
  
 **エラー ID:** BC31166  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML リテラルの XML タグの終わりに '>' 文字を追加します。
