# <a name="specifiers-are-not-valid-on-39addhandler39-39removehandler39-and-39raiseevent39-methods"></a>指定子で無効です &#39;AddHandler &#39;、&#39;です。RemoveHandler &#39;および &#39;です。RaiseEvent &#39;メソッド
`AddHandler`、 `RemoveHandler`、および `RaiseEvent` メソッド宣言は、 `Public` や `ReadOnly`などのキーワードを使用して修飾することはできません。 修飾可能な宣言にのみ、これらのキーワードを含めることができます。  
  
 **エラー ID:** BC31135  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   メソッド宣言からキーワードを削除します。
