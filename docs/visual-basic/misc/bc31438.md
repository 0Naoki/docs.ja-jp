# <a name="method-ltmethodname1gt-does-not-have-the-same-generic-constraints-as-the-partial-method-ltmethodname2gt"></a>メソッド '&lt;methodname1&gt;'部分メソッドとして同じジェネリック制約がない'&lt;methodname2&gt;'
部分メソッドの宣言内の制約とは異なるジェネリック制約を持つ部分メソッドの実装を定義しています。 このエラーが発生するコード例を次に示します。  
  
```vb  
Partial Class Class1  
  
    Partial Private Sub Test(Of T As Class)(ByVal arg As T)  
    End Sub  
  
End Class  
  
Partial Class Class1  
  
    '' The error occurs here, for Test.  
    'Private Sub Test(Of T As Structure)(ByVal arg As T)  
    'End Sub  
  
End Class  
```  
  
 **エラー ID:** BC31438
