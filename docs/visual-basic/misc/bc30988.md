# <a name="type-or-with-expected"></a><span data-ttu-id="fa6cc-102">型または 'With' が必要です</span><span class="sxs-lookup"><span data-stu-id="fa6cc-102">Type or 'With' expected</span></span>
<span data-ttu-id="fa6cc-103">クラスのインスタンスを宣言するときには、 `New` キーワードに続いて、型名または `With`を指定しなければなりません。</span><span class="sxs-lookup"><span data-stu-id="fa6cc-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="fa6cc-104">たとえば、次の各ステートメントは `client` が `Customer` クラスのインスタンスとなることを宣言しています。</span><span class="sxs-lookup"><span data-stu-id="fa6cc-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="fa6cc-105">型名 `Customer` が `New`の後に置かれています。</span><span class="sxs-lookup"><span data-stu-id="fa6cc-105">The type name `Customer` follows `New`.</span></span>  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="fa6cc-106">Visual Basic 2008 以降では、できるオブジェクトを宣言する、匿名型のインスタンスである場合データ型を指定しません。</span><span class="sxs-lookup"><span data-stu-id="fa6cc-106">Beginning with Visual Basic 2008, you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="fa6cc-107">匿名型の宣言では、キーワード `With` を `New`の後に置きます。</span><span class="sxs-lookup"><span data-stu-id="fa6cc-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
