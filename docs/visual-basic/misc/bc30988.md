<span data-ttu-id="df9fe-103">クラスのインスタンスを宣言するときには、 `New` キーワードに続いて、型名または `With`を指定しなければなりません。</span><span class="sxs-lookup"><span data-stu-id="df9fe-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="df9fe-104">たとえば、次の各ステートメントは `client` が `Customer` クラスのインスタンスとなることを宣言しています。</span><span class="sxs-lookup"><span data-stu-id="df9fe-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="df9fe-105">型名 `Customer` が `New`の後に置かれています。</span><span class="sxs-lookup"><span data-stu-id="df9fe-105">The type name `Customer` follows `New`.</span></span>

