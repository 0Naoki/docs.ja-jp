# <a name="39ltdeclaration1gt39-cannot-override-39ltdeclaration2gt39-because-it-is-declared-39notoverridable39"></a>&#39;です。&lt;declaration1&gt;&#39; できません上書き &#39;&lt;declaration2&gt;&#39;宣言されているためです。 (& a) #39 です。NotOverridable &#39;
プロシージャまたはプロパティの宣言は同じ名前の継承された要素をオーバーライドしようとしていますが、継承された要素は `NotOverridable`として指定されています。  
  
 **エラー ID:** BC30267  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   継承された要素の宣言から `NotOverridable` キーワードを削除するか、またはオーバーライドする宣言を削除します。
