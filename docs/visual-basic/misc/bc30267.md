# <a name="ltdeclaration1gt-cannot-override-ltdeclaration2gt-because-it-is-declared-notoverridable"></a>'&lt;declaration1&gt;オーバーライドできません'&lt;declaration2&gt;' 'NotOverridable' 宣言されているため、
プロシージャまたはプロパティの宣言は同じ名前の継承された要素をオーバーライドしようとしていますが、継承された要素は `NotOverridable`として指定されています。  
  
 **エラー ID:** BC30267  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   継承された要素の宣言から `NotOverridable` キーワードを削除するか、またはオーバーライドする宣言を削除します。
