# <a name="this-operation-can-only-be-done-when-the-file-is-closed"></a>この操作は、ファイルが閉じられているときにのみ実行することができます
ファイルが開いている場合は、ファイルが閉じられているときにのみ実行できる操作を実行しようとします。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイルを閉じて、操作をやり直します。
