# <a name="39ltstatementnamegt39-statement-requires-an-array"></a>&#39;&lt;statementname&gt; &#39;ステートメントには、配列が必要です。
`Erase` と `ReDim` ステートメントは、配列のデータ型を必要とします。 指定された型が配列ではありません。  
  
 **エラー ID:** BC30049  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   データ型を配列型に変更します。
