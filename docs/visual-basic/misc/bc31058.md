# &lt;type&gt; &#39;&lt;name3&gt;&#39; の &#39;&lt;name2&gt;&#39; に対して暗黙的に作成される &#39;&lt;name1&gt;&#39; と競合します
型メンバーの名前が、暗黙的に作成される別のメンバーと競合します。  
  
 **エラー ID:** BC31058  
  
### このエラーを解決するには  
  
-   明示的に宣言されたメンバーの名前を変更して、名前の競合をなくします。
