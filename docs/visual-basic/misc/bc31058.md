# <a name="conflicts-with-name1-which-is-implicitly-created-for-name2-in-type-name3"></a>競合しています '\<name1 > ', に暗黙的に作成された'\<name2 >' で\<型 >'\<name3 >'
型メンバーの名前が、暗黙的に作成される別のメンバーと競合します。  
  
 **エラー ID:** BC31058  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   明示的に宣言したメンバーの名前を変更して、名前の競合を解決します。
