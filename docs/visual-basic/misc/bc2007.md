# <a name="unrecognized-option-optionname-ignored"></a>認識されないオプション\<optionname >; 無視
コマンド ラインから無効なオプションが指定されています。  
  
 **エラー ID:** BC2007  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   無効なオプションがないか、指定したオプションを確認します。
