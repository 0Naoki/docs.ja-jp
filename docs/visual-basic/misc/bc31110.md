# 属性の引数として使用される配列は、すべての要素の値を明示的に指定する必要があります
属性引数として使用される配列には、指定されていない値を持つ要素があります。  
  
 **エラー ID:** BC31110  
  
### このエラーを解決するには  
  
-   すべての要素の値を指定します。
