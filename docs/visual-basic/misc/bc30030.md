# Try には少なくとも 1 つの &#39;Catch&#39; または &#39;Finally&#39; を指定しなければなりません
`Try` ブロックでは、`End Try` ステートメントの前に、1 つの `Finally` ブロックか、または少なくとも 1 つの `Catch` ブロックを含める必要があります。  
  
 **エラー ID:** BC30030  
  
### このエラーを解決するには  
  
-   `Try` ブロック内に `Catch` ブロックまたは `Finally` ブロックを追加します。
