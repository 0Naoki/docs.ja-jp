# <a name="named-arguments-cannot-match-paramarray-parameters"></a>名前付き引数を ParamArray パラメーターと一致させることはできません。
パラメーター配列は、値で渡される必要があります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  名前を付けることによってではなく、引数 `ByVal`を渡します。
