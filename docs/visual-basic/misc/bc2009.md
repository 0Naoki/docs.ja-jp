# <a name="option-ltoptionnamegt-can-be-followed-only-by-3939-or-39-39"></a>オプション&lt;optionname&gt;が続きますのみ (& a) #39; + (& a) #39; または &#39;-&#39;
オプションに続くテキストが、プラス記号 (+) でもマイナス記号 (-) でもありません。  
  
 **エラー ID:** BC2009  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   オプションに続くテキストをプラス記号 (+) に置き換えて、オプションをオンにします。  
  
     — または —  
  
-   オプションに続くテキストをマイナス記号 (-) に置き換えて、オプションをオフにします。
