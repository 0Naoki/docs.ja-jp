# オプション &lt;optionname&gt; の後には &#39;+&#39; または &#39;-&#39; のみを指定できます
オプションに続くテキストが、プラス記号 \(\+\) でもマイナス記号 \(\-\) でもありません。  
  
 **エラー ID:** BC2009  
  
### このエラーを解決するには  
  
-   オプションに続くテキストをプラス記号 \(\+\) に置き換えて、オプションをオンにします。  
  
     — または —  
  
-   オプションに続くテキストをマイナス記号 \(\-\) に置き換えて、オプションをオフにします。
