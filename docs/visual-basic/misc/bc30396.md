# <a name="specifier-is-not-valid-on-an-enum-declaration"></a>'\<指定子 >'、Enum 宣言では無効です
`Enum` 宣言で無効な指定子が使用されました。  
  
 **エラー ID:** BC30396  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  指定子を削除します。
