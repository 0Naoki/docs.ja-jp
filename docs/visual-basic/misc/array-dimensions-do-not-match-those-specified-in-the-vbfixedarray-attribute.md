# <a name="array-dimensions-do-not-match-those-specified-in-the-39vbfixedarray39-attribute"></a>指定された配列の次元が一致しない、&#39;です。VBFixedArray &#39;属性
配列の次元が、 `VBFixedArray` 属性で指定された次元と一致しません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   配列の `Dim` ステートメントの配列次元が `VBFixedArray` 属性で指定された次元と一致するようにします。
