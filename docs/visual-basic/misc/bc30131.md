# セキュリティ属性 &#39;&lt;attributename&gt;&#39; をモジュールに適用することはできません
共通言語ランタイムでは、モジュールのセキュリティ属性は受け入れられません。  
  
 **エラー ID:** BC30131  
  
### このエラーを解決するには  
  
1.  宣言から属性を削除します。
