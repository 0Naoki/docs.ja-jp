# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>オーバー ロードの解決が失敗しましたない '\<メソッド >' これらの引数と共に呼び出すことができます:\<エラー >
オーバーロードされたメソッドを呼び出しましたが、コンパイラで、対応する引数リストを持つメソッドを検出できません。  
  
 **エラー ID:** BC30518  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定した引数リストを確認します。
