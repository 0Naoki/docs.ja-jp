# &#39;Continue&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Continue` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30780  
  
### このエラーを解決するには  
  
-   `Continue` ステートメントをデバッグ コードから削除します。
