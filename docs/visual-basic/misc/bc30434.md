# <a name="events-in-a-module-cannot-be-declared-ltspecifiergt"></a>モジュールのイベントを宣言することはできません '&lt;指定子&gt;'
`Module` ステートメント内のイベントに対して無効な指定子を使用しました。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30434  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
