# モジュールのイベントを &#39;&lt;specifier&gt;&#39; として宣言できません。
`Module` ステートメント内のイベントに対して無効な指定子を使用しました。 モジュールはインスタンス化されず、継承をサポートしません。また、インターフェイスを実装できません。  
  
 **エラー ID:** BC30434  
  
### このエラーを解決するには  
  
-   指定子を削除します。
