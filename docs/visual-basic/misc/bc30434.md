# <a name="events-in-a-module-cannot-be-declared-specifier"></a>モジュールのイベントを宣言することはできません '\<指定子 >'
`Module` ステートメント内のイベントに対して無効な指定子を使用しました。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30434  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 指定子を削除します。
