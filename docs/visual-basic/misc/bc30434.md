# <a name="events-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>モジュールのイベントを宣言することはできません (& a) #39 です。&lt;指定子&gt;&#39;です。
`Module` ステートメント内のイベントに対して無効な指定子を使用しました。 モジュールはインスタンス化されず、継承をサポートしません。また、インターフェイスを実装できません。  
  
 **エラー ID:** BC30434  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
