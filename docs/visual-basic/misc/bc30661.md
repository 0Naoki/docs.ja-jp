# フィールドまたはプロパティ &#39;&lt;membername&gt;&#39; が見つかりません。
属性について、未定義のフィールドまたはプロパティが参照されています。  
  
 **エラー ID:** BC30661  
  
### このエラーを解決するには  
  
1.  フィールドまたはプロパティのスペルを確認します。  
  
2.  プロジェクトが正しい属性クラスにアクセスできることを確認し、必要に応じて、その属性クラスをプロジェクト参照に追加します。
