# <a name="39systemstathreadattribute39-and-39systemmtathreadattribute39-cannot-both-be-applied-to-39139"></a>&#39;です。'System.mtathreadattribute' &#39;および &#39;です。System.MTAThreadAttribute &#39;両方に適用することはできません (& m); #39 | 1 &#39;
`System.STAThreadAttribute` 属性と `System.MTAThreadAttribute` 属性は相互に排他的です。  
  
 **エラー ID:** BC31513  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `System.MTAThreadAttribute` か `System.STAThreadAttribute`のいずれかを適用し、両方は適用しないでください。
