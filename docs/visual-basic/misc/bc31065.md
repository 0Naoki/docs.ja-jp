# <a name="39set39-parameter-cannot-be-declared-39ltspecifiergt39"></a>&#39;です。セット &#39;パラメーターを宣言することはできません (& a) #39 です。&lt;指定子&gt;&#39;です。
`Set` プロパティ プロシージャのパラメーターに無効な修飾子が指定されました。  
  
 **エラー ID:** BC31065  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `ByRef`、 `ParamArray`、 `Optional`などの無効な修飾子を削除します。
