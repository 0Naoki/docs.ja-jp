# 型 &#39;&lt;typename&gt;&#39; の Loop コントロール変数は &#39;System.IComparable&#39; インターフェイスを実装していません
変数は、`System.IComparable` インターフェイスを実装しない型です。`System.IComparable` は、値型またはクラスで実装される必要があります。  
  
### このエラーを解決するには  
  
1.  変数が値型であることを確認します。
