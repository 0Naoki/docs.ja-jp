# &#39;Try&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Try` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30717  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Try` ステートメントを削除します。
