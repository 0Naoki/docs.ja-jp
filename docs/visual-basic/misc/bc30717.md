# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39;です。Try &#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Try` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30717  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `Try` ステートメントを削除します。
