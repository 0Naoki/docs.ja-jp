# <a name="39structure39-statement-must-end-with-a-matching-39end-structure39"></a>&#39;です。構造体 &#39;ステートメントの終わりに対応する &#39; にする必要があります。End Structure &#39;
構造体は `Structure` ステートメントで始まり、 `End Structure` コンストラクトで終わります。  
  
 **エラー ID:** BC30624  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   構造体ブロックの最後に `End Structure` ステートメントがあることを確認します。
