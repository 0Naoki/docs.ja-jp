# モジュール &#39;&lt;modulename&gt;&#39; は型として使用できません
モジュールを型として使用しようとしました。 モジュールは、`Get` 型として以外は使用できません。  
  
 **エラー ID:** BC30371  
  
### このエラーを解決するには  
  
-   有効な型として使用できるもので、モジュールを置換します。
