# <a name="ltmethodnamegt39-and-39ltmethodnamegt39-cannot-overload-each-because-they-differ-by-39readonly39-or-39writeonly39"></a>&lt;methodname&gt;&#39; と &#39;&lt;methodname&gt;&#39; ことはできませんが異なるため、各オーバー ロード &#39;読み取り専用 &#39;または &#39;です。WriteOnly &#39;
`ReadOnly` および `WriteOnly` 宣言のみが異なる 2 つのメソッドをオーバーロードしようとしました。 引数リスト以外のものを使用して、バージョンを区別することはできません。  
  
 **エラー ID:** BC30366  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   メソッドが `ReadOnly` および `WriteOnly`以外のもので区別されていることを確認します。
