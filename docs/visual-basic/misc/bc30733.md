# <a name="ltmembernamegt-is-already-declared-by-ltprocedurenamegt-which-was-generated-for-this-ltcontainernamegt"></a>'&lt;membername&gt;'は、既にで宣言されて'&lt;procedurename&gt;'、これに対して生成された&lt;containername&gt;
共通言語ランタイムによって宣言されているメンバーと同じ名前のメンバーが宣言されました。  
  
 **エラー ID:** BC30733  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   メンバーの名前を変更して競合を解決します。
