# <a name="xml-dtds-are-not-supported"></a>XML DTD はサポートされていません
XML リテラルとしてドキュメント型定義 (DTD) が指定されています。  
  
 **エラー ID:** BC31175  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   DTD を削除します。 XML ドキュメントのスキーマを指定するには、XML スキーマ定義 (XSD) を使用します。
