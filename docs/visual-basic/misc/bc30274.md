# <a name="parameter-ltparameternamegt-of-ltmethodnamegt-already-has-a-matching-argument"></a>パラメーター '&lt;parametername&gt;'の'&lt;methodname&gt;'、一致する引数が既に
1 つの引数が、同じプロシージャ呼び出しで複数回にわたって名前で渡されています。  
  
 **エラー ID:** BC30274  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数名のいずれかのスペルが誤っていないかどうかを確認します。  
  
-   引数リストから重複している引数を削除します。
