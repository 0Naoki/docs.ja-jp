# オブジェクトはサポートされているイベント受信最大数のイベントを既に発生しているので、オブジェクトのイベントを受け取ることができません
オブジェクトでサポートされているイベント レシーバーの最大数を超えています。  
  
### このエラーを解決するには  
  
-   イベント レシーバーの数を減らします。
