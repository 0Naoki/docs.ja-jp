# <a name="new-constraint-and-structure-constraint-cannot-be-combined"></a><span data-ttu-id="2dbb5-102">'New' 制約と 'Structure' 制約は、組み合わせて使用できません</span><span class="sxs-lookup"><span data-stu-id="2dbb5-102">'New' constraint and 'Structure' constraint cannot be combined</span></span>
<span data-ttu-id="2dbb5-103">制約リストには、両方が含まれて、 [New 演算子](../../visual-basic/language-reference/operators/new-operator.md)制約と[構造体 (Visual Basic)](../../visual-basic/language-reference/statements/structure-statement.md)制約。</span><span class="sxs-lookup"><span data-stu-id="2dbb5-103">A constraint list includes both the [New Operator](../../visual-basic/language-reference/operators/new-operator.md) constraint and the [Structure (Visual Basic)](../../visual-basic/language-reference/statements/structure-statement.md) constraint.</span></span>  
  
 <span data-ttu-id="2dbb5-104">型パラメーターの制約リストには、その型パラメーターに渡される型引数が値型でなければならない ( `Structure` 制約を使用)、または参照型でなければならない ( [Class (Visual Basic)](../../visual-basic/language-reference/statements/class-statement.md) 制約を使用) ことを指定できます。</span><span class="sxs-lookup"><span data-stu-id="2dbb5-104">A constraint list on a type parameter can specify that the type argument passed to that type parameter must be a value type (with the `Structure` constraint) or must be a reference type (with the [Class (Visual Basic)](../../visual-basic/language-reference/statements/class-statement.md) constraint).</span></span> <span data-ttu-id="2dbb5-105">同じ型パラメーターに両方の制約を指定することはできません。また、どちらも複数回指定することはできません。</span><span class="sxs-lookup"><span data-stu-id="2dbb5-105">You cannot specify both constraints on the same type parameter, and you cannot specify either one more than once.</span></span>  
  
 <span data-ttu-id="2dbb5-106">`New` 制約では、型引数は作成元のコードがアクセスできるパラメーターなしのコンストラクターを公開する必要があることを指定します。</span><span class="sxs-lookup"><span data-stu-id="2dbb5-106">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="2dbb5-107">ただし、構造体は非共有のパラメーターなしコンストラクターを持つことができません。</span><span class="sxs-lookup"><span data-stu-id="2dbb5-107">However, a structure cannot have a nonshared parameterless constructor.</span></span> <span data-ttu-id="2dbb5-108">このため、 `New` 制約と `Structure` 制約を同時に使用すると衝突します。</span><span class="sxs-lookup"><span data-stu-id="2dbb5-108">Therefore, the `New` and `Structure` constraints are in conflict.</span></span>  
  
 <span data-ttu-id="2dbb5-109">**エラー ID:** BC32103</span><span class="sxs-lookup"><span data-stu-id="2dbb5-109">**Error ID:** BC32103</span></span>
