# <a name="argument-39access39-is-not-valid-valid-values-for-output-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>引数&#39;アクセス&#39;が無効です。 Output モードの有効な値は&#39;OpenAccess.Write&#39;と&#39;です&#39;
Output モードでアクセスの値が正しくありません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  値を `OpenAccess.Write` または `OpenAccess.Default`のどちらかに変更します。
