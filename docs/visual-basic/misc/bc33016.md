# <a name="operator-ltoperatorgt-must-have-either-one-or-two-parameters"></a>演算子 '&lt;演算子&gt;' いずれか 1 つまたは 2 つのパラメーターが必要
単項演算子または 2 項演算子がパラメーターなしで定義されているか、3 個以上のパラメーターを使用して定義されています。  
  
 単項演算子または 2 項演算子には、1 個または 2 個のパラメーターを指定する必要があります。  
  
 **エラー ID:** BC33016  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   1 個または 2 個のパラメーターを指定するように、定義を調整します。  
  
-   使用する必要がありますパラメーターなし、または 3 つ以上が必要な場合、[関数ステートメント](../../visual-basic/language-reference/statements/function-statement.md)を定義する、`Function`演算子ではなくプロシージャ。
