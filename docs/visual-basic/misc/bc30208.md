# &#39;Compare&#39; が必要です
`Option` ステートメントで `Text` または `Binary` が指定されていますが、`Compare` が指定されていません。  
  
 **エラー ID:** BC30208  
  
### このエラーを解決するには  
  
-   `Compare` キーワードを `Option` ステートメントに追加します。例: `Option Compare Binary`。
