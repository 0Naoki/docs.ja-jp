# クエリ式は、デバッグ ウィンドウでは有効ではありません
デバッグ セッション中に、LINQ クエリが **\[ウォッチ\]** ウィンドウまたは **\[イミディエイト\]** ウィンドウに入力されました。 デバッグ ウィンドウでは、クエリを使用できません。  
  
 **エラー ID:** BC36609
