# <a name="you-must-specify-a-file-name"></a>ファイル名を指定する必要があります
ファイルの移動やファイルのコピーなど、ファイル名を指定する必要がある操作を実行しようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   必要なファイル名を指定します。  
  
-   パスがディレクトリの区切り記号で終わっていないを確認してください (\\)。
