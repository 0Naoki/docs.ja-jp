# <a name="variables-in-modules-cannot-be-declared-39ltspecifiergt39"></a>モジュールの変数を宣言することはできません (& a) #39 です。&lt;指定子&gt;&#39;です。
`MustInherit`ステートメント内の変数に `Module` などの指定子を使用しました。 モジュールはインスタンス化されず、継承をサポートしません。また、インターフェイスを実装できません。  
  
 **エラー ID:** BC30593  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
