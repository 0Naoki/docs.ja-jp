# モジュール内の変数を &#39;&lt;specifier&gt;&#39; として宣言することはできません
`Module` ステートメント内の変数に `MustInherit` などの指定子を使用しました。 モジュールはインスタンス化されず、継承をサポートしません。また、インターフェイスを実装できません。  
  
 **エラー ID:** BC30593  
  
### このエラーを解決するには  
  
-   指定子を削除します。
