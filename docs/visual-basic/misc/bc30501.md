# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-method-declaration"></a>'Shared' と組み合わせて使用できない '&lt;指定子&gt;' メソッドの宣言
メソッド宣言で有効ではない組み合わせで、 `Shared` 修飾子と指定子 ( `Overridable`、 `NotOverridable`、 `MustOverride`など) を組み合わせようとしました。  
  
 **エラー ID:** BC30501  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
