# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-method-declaration"></a>&#39;です。共有 &#39;です。組み合わせて使用できない &#39;です。&lt;指定子&gt;&#39;メソッド宣言でです。
メソッド宣言で有効ではない組み合わせで、 `Shared` 修飾子と指定子 ( `Overridable`、 `NotOverridable`、 `MustOverride`など) を組み合わせようとしました。  
  
 **エラー ID:** BC30501  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
