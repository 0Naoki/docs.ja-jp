# メソッド宣言で、&#39;Shared&#39; を &#39;&lt;specifier&gt;&#39; と組み合わせて使用することはできません
メソッド宣言で有効ではない組み合わせで、`Shared`修飾子と指定子 \(`Overridable`、`NotOverridable`、`MustOverride` など\) を組み合わせようとしました。  
  
 **エラー ID:** BC30501  
  
### このエラーを解決するには  
  
-   指定子を削除します。
