# 区切り記号がないか、または空であるため、区切り記号で分けられたフィールドを読み取れません
`Delimiters` プロパティが `Nothing` に設定されているか、または空であるため、`TextFieldParser` をファイルから読み取ることができません。  
  
### このエラーを解決するには  
  
-   `Delimiters` に対して有効な値を指定します。
