# <a name="unable-to-load-referenced-library-39ltfilenamegt39-lterrorgt"></a>参照先のライブラリを読み込めません '&lt;filename&gt;':&lt;エラー。&gt;
このプロジェクトで参照されるライブラリを読み込むことができませんでした。  
  
 **エラー ID:** BC31011  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  指定した場所にファイルがあることと、ファイルが別のアプリケーションによってロックされていないことを確認します。
