# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39;です。Throw &#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Throw` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30725  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `Throw` ステートメントを削除します。
