# オブジェクトは現在のロケールの設定をサポートしていません (Visual Basic)
アクセスしようとしたオブジェクトは、現在のプロジェクトのロケール設定をサポートしていません。  
  
### このエラーを解決するには  
  
-   オブジェクトでサポートされるロケール設定を確認します。
