# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>参照先のオブジェクト &#39;&lt;objectname&gt;&#39; 値 &#39; には何も &#39;
参照先オブジェクトがまだ設定されていないか、または値 `Nothing`に既に設定されています。  
  
 **エラー ID:** BC30705  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   オブジェクトの名前のスペルが正しいことと、オブジェクトが設定されていることを確認します。
