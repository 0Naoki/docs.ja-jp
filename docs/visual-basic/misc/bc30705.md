# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>参照先オブジェクト '&lt;objectname&gt;' の値を持つ 'Nothing'
参照先オブジェクトがまだ設定されていないか、または値 `Nothing`に既に設定されています。  
  
 **エラー ID:** BC30705  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   オブジェクトの名前のスペルが正しいことと、オブジェクトが設定されていることを確認します。
