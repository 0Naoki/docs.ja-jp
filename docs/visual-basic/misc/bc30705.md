# <a name="referenced-object-39ltobjectnamegt39-has-a-value-of-39nothing39"></a>参照先オブジェクト&#39; &lt;objectname&gt; &#39;の値を持つ&#39;何も行われません&#39;
参照先オブジェクトがまだ設定されていないか、または値 `Nothing`に既に設定されています。  
  
 **エラー ID:** BC30705  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   オブジェクトの名前のスペルが正しいことと、オブジェクトが設定されていることを確認します。
