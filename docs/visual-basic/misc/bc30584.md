# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39;です。&lt;型&gt;&#39; 2 回以上継承することはできません
`Inherits` ステートメントが重複しています。  
  
 **エラー ID:** BC30584  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不要な `Inherits` ステートメントを削除します。
