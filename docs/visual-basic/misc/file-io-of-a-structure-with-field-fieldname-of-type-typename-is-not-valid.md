# <a name="file-io-of-a-structure-with-field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-not-valid"></a>フィールド &#39; を持つ構造体のファイル I/O&lt;fieldname&gt;&#39; の種類 &#39;&lt;typename&gt;&#39; が正しくありません
ファイルへの書き込みを試行している構造体には、サポートされていない型が含まれています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   構造体を確認して、サポートされていない型が検出された場所を判断し、有効な型に変更します。
