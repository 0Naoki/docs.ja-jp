# <a name="file-io-with-type-typename-is-not-valid"></a>この種類の I/O をファイル '\<typename >' が無効です
サポートされていない型をファイルに書き込もうとしています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   構造体を確認して、サポートされていない型が検出された場所を判断し、有効な型に変更します。
