# <a name="file-39ltfilenamegt39-cannot-be-deleted-because-it-is-open"></a>ファイル &#39;&lt;filename&gt;&#39; 開いているために、削除できません
開いているファイルを削除しようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイルを閉じてから削除します。
