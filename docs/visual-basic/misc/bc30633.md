# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;です。セット &#39;ステートメントの終わりに対応する &#39; にする必要があります。終了セット &#39;
`Set` プロパティ プロシージャは、必ず `End Set` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30633  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Set` プロパティ プロシージャが `End Set` コンストラクトで終了していることを確認します。
