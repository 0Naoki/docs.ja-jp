# <a name="39ltprocedurenamegt39-has-a-return-type-that-is-not-supported-or-parameter-types-that-are-not-supported"></a>&#39;&lt;procedurename&gt; &#39;にサポートされていない戻り値の型またはパラメーター型が指定はサポートされていません
プロシージャにパラメーターまたは Visual Studio では表現できない戻り値の型で宣言されています。  
  
 **エラー ID:** BC30657  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   パラメーターまたは戻り値の型を、サポートされている型として定義します。  
  
-   プロシージャによって使用されるデータを記述する、新しい型を作成します。
