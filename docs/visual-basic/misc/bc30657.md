# &#39;&lt;procedurename&gt;&#39; には、サポートされていない戻り値の型か、またはサポートされていないパラメーター型が指定されています
[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] では表現できないパラメーターまたは戻り値の型を指定して、プロシージャが宣言されています。  
  
 **エラー ID:** BC30657  
  
### このエラーを解決するには  
  
-   パラメーターまたは戻り値の型を、サポートされている型として定義します。  
  
-   プロシージャによって使用されるデータを記述する、新しい型を作成します。
