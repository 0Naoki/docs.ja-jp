# <a name="object-initializer-syntax-cannot-be-used-to-initialize-an-instance-of-type-39object39"></a>型 &#39; オブジェクト &#39; のインスタンスを初期化するためにオブジェクト初期化子構文を使用することはできません。
オブジェクト初期化子の構文を使用して、 `Object` のインスタンスを初期化することはできません。 `Object` のインスタンスには値を割り当てられるプロパティやフィールドが含まれていません。オブジェクト初期化子の構文には、このようなプロパティかフィールドが少なくとも 1 つ必要です。  
  
```  
' Not valid.  
' Dim obj1 = New Object With {}  
' Dim obj2 = New Object With {.ToString = <some value>}  
```  
  
 **エラー ID:** BC30994  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Object` 型のインスタンスを、初期化子リストを使用せずに宣言します。  
  
    ```  
    Dim obj3 as Object  
    Dim obj4 as New Object()  
    ```
