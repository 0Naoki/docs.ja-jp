# <a name="withevents-variable-does-not-raise-any-events"></a>'WithEvents' 変数は、イベントを発生させません
`WithEvents` キーワードを使用して宣言されているオブジェクト変数が、イベントを発生させることができるクラス インスタンスを参照していません。  
  
 **エラー ID:** BC30591  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  オブジェクト変数が、イベントを発生させることができるクラス インスタンスを参照していることを確認します。
