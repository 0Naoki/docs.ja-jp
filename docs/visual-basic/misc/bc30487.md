# <a name="operator-39ltoperatornamegt39-is-not-defined-for-type-39lttypenamegt39"></a>演算子 &#39;&lt;operatorname&gt;&#39; の種類 &#39; に対して定義されていない&lt;typename&gt;&#39;です。
`+`などの型に対して単項演算子 ( `-`、 `Not`、または `String`) を使用しようとしましたが、これは定義されていません。  
  
 **エラー ID:** BC30487  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   単項演算子を式の適切な演算子に置き換えます。
