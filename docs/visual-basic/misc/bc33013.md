# <a name="operators-cannot-be-declared-ltkeywordgt"></a>演算子を宣言できません '&lt;キーワード&gt;'
演算子が、演算子の定義に使用できないキーワードを使用して宣言されています。  
  
 すべての演算子を宣言する必要がありますは両方として[パブリック](../../visual-basic/language-reference/modifiers/public.md)と[Shared](../../visual-basic/language-reference/modifiers/shared.md)します。 さらに、変換演算子は [Widening](../../visual-basic/language-reference/modifiers/widening.md) か [Narrowing](../../visual-basic/language-reference/modifiers/narrowing.md)のいずれかを使用して宣言する必要があり、両方を使用することはできません。 演算子の定義には、必要に応じて、 [Overloads](../../visual-basic/language-reference/modifiers/overloads.md) または [Shadows](../../visual-basic/language-reference/modifiers/shadows.md) のキーワードを含めることができます。 その他のキーワードは [Operator Statement](../../visual-basic/language-reference/statements/operator-statement.md)では許可されません。  
  
 **エラー ID:** BC33013  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   使用できないキーワードを演算子の定義から削除します。
