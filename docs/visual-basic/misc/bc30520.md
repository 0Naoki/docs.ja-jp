# <a name="argument-matching-parameter-ltparameternamegt-narrows-from-lttype1gt-to-lttype2gt"></a>引数のパラメーターと一致する '&lt;parametername&gt;'から縮小変換'&lt;type1&gt;'to'&lt;type2&gt;'
オーバーロードされたメソッドを呼び出しましたが、コンパイラで、縮小変換せずに呼び出すことのできるメソッドを検出できません。 縮小変換により、有効値の一部を正確に保持できない可能性のあるデータ型に値が変更されます。  
  
 **エラー ID:** BC30520  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Option Strict Off`を指定します。
