# <a name="39end-set39-must-be-the-first-statement-on-a-line"></a>&#39;End Set&#39;行の最初のステートメントをする必要があります
`End Set` ステートメントがコロン (:) ステートメント区切り文字の後に続いています。 `End Set` はそのソース コード行でただ 1 つのステートメントである必要があります。  
  
 **エラー ID:** BC32034  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   複数のステートメントを別々の行に分けます。
