# <a name="39class39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>&#39;クラス&#39;制約は、同じ型パラメーターに対して複数回指定ことはできません。
制約リストが含まれています、[クラス (Visual Basic)](https://msdn.microsoft.com/library/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) 2 回以上の制約。  
  
 型パラメーターの制約リストがその型パラメーターに渡される型引数は値型である必要がありますを指定することができます (で、[構造体 (Visual Basic)](https://msdn.microsoft.com/library/263ce115-ac36-4c05-8cb7-0e0eead5c6d0)制約) か、参照型である必要があります (で、`Class`制約). 同じ型パラメーターに両方の制約を指定することはできません。また、どちらも複数回指定することはできません。  
  
 エラー ID: BC32101  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   冗長な `Class` キーワードがある場合は削除します。 制約リストには 1 つだけ含める必要があります。
