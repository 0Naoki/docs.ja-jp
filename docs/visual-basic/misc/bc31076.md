# &#39;&lt;modulename&gt;&#39; はモジュールであるため、アセンブリとして参照できません
モジュールをアセンブリとして使用しようとしました。  
  
 **エラー ID:** BC31076  
  
### このエラーを解決するには  
  
-   アセンブリへの参照を変更します。
