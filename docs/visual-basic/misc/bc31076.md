# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>'\<modulename >' モジュールは、アセンブリとして参照できません。
モジュールをアセンブリとして使用しようとしました。  
  
 **エラー ID:** BC31076  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   アセンブリへの参照を変更します。
