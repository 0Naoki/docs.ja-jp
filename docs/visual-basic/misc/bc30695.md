# この名前で別のメンバーが &#39;Shadows&#39; として宣言されているため、&lt;typename&gt; &#39;&lt;membername&gt;&#39; は &#39;Shadows&#39; として宣言されなければなりません。
このプロパティまたはプロシージャには、`Shadows` 修飾子でマークされた別のプロパティまたはプロシージャと同じ名前が指定されています。  
  
 **エラー ID:** BC30695  
  
### このエラーを解決するには  
  
-   `Shadows` 修飾子を使用します。
