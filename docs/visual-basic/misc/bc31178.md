# <a name="expected-closing-3939-for-xml-entity"></a>閉じる &#39; が必要です。&#39;です。XML エンティティの
XML エンティティ参照 (例: `>`など) で最後のセミコロン (;) が見つかりません。  
  
 **エラー ID:** BC31178  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML エンティティ参照の末尾にセミコロンを含めます。
