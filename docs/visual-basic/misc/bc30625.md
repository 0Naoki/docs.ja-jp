# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;モジュール&#39;、対応するステートメントが終了する必要があります&#39;終了モジュール&#39;
モジュール ブロックは、 `End``Module` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30625  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `End Module` ステートメントを `Module` ブロックの最後に追加します。
