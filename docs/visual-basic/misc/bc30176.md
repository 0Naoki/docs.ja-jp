# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>1 つのみ&#39;パブリック&#39;、&#39;プライベート&#39;、 &#39;Protected&#39;、&#39;フレンド&#39;、または&#39;Protected Friend&#39;を指定できます
1 つの要素の宣言内に、競合するアクセシビリティがあります。 アクセシビリティを 1 つだけ指定できます。  
  
 **エラー ID:** BC30176  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   1 つのアクセシビリティを選び、残りのキーワードを宣言から削除します。
