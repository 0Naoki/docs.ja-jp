# <a name="39ltmembergt39-is-already-declared-in-this-structure"></a>&#39;です。&lt;メンバー&gt;&#39; は既にこの構造体で宣言されて
列挙体のメンバーがこの構造体内で 1 回以上宣言されています。  
  
 **エラー ID:** BC31421  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分な宣言を削除します。
