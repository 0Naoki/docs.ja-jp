# &#39;&lt;member&gt;&#39; は既にこの構造体で宣言されています
列挙体のメンバーがこの構造体内で 1 回以上宣言されています。  
  
 **エラー ID:** BC31421  
  
### このエラーを解決するには  
  
-   余分な宣言を削除します。
