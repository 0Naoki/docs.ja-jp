# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>'\<assemblyname >' は、アセンブリとモジュールとして参照できません。
アセンブリをモジュールとして使用しようとしました。  
  
 **エラー ID:** BC31077  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   モジュールへの参照を変更します。
