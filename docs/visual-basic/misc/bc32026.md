# <a name="specifiers-and-attributes-are-not-valid-on-39namespace39-statements"></a>指定子および属性が無効で&#39;Namespace&#39;ステートメント
名前空間は、プログラム ブロックおよびプログラミング要素の階層構造で使用されます。 これらには修飾子または属性を指定できません。  
  
 **エラー ID:** BC32026  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Namespace` ステートメントからすべてのキーワードおよび属性ブロックを削除します。
