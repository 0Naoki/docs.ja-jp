# <a name="response-file-filename-included-multiple-times"></a>応答ファイル\<filename > 複数回含まれました。
応答ファイルが複数回指定されました。 応答ファイルには、コンパイラ オプションおよびコンパイル対象のソース コード ファイルが含まれます。  
  
 **エラー ID:** BC2003  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分な応答ファイル指定子を削除します。
