# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-without-a-narrowing-conversion-lterrorgt"></a>オーバー ロードの解決が失敗しましたない '&lt;メソッド&gt;' 縮小変換せずに呼び出すことができます:&lt;エラー&gt;
オーバーロードされたメソッドを呼び出しましたが、コンパイラで、縮小変換せずに呼び出すことのできるメソッドを検出できません。 縮小変換により、有効値の一部を正確に保持できない可能性のあるデータ型に値が変更されます。  
  
 **エラー ID:** BC30519  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Option Strict Off`を指定します。
