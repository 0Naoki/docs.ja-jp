# <a name="39end-raiseevent39-must-be-preceded-by-a-matching-39raiseevent39-declaration"></a>&#39;End RaiseEvent&#39;前に、対応する必要がある&#39;RaiseEvent&#39;宣言
`End RaiseEvent` ステートメントが出現しましたが、対応する `RaiseEvent` ステートメントがありません。 `End RaiseEvent` の前に、対応する `RaiseEvent` ステートメントが必要です。  
  
 **エラー ID:** BC31126  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   先行する `RaiseEvent` ステートメントが有効であり、スペルが正しいことを確認します。
