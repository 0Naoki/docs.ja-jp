# <a name="named-argument-ltargumentnamegt-specified-multiple-times"></a>名前付き引数 '&lt;argumentname&gt;' 複数回指定されました
名前付き引数が複数回指定されました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   仕様の 1 つ以外はすべて削除します。
