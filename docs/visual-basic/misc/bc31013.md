# <a name="unable-to-load-dll-39ltfilenamegt39-lterrorgt"></a>DLL &#39; を読み込めません。&lt;filename&gt;&#39;:&lt;エラー&gt;

 **エラー ID:** BC31013  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  指定した場所にファイルがあることと、ファイルが別のアプリケーションによってロックされていないことを確認します。
