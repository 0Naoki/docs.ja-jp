# <a name="set-not-supported-read-only-property"></a>Set はサポートされません。読み取り専用のプロパティです。
読み取り専用のプロパティを設定または変更しようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  コードからプロパティへの参照を削除します。  
  
2.  実行時に、プロパティの値のみを返す参照に変更します。
