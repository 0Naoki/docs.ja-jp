# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="82a78-102">配列を 'New' で宣言することはできません</span><span class="sxs-lookup"><span data-stu-id="82a78-102">Arrays cannot be declared with 'New'</span></span>
<span data-ttu-id="82a78-103">`New` キーワードは、配列の宣言の初期化の部分にのみ指定できます。</span><span class="sxs-lookup"><span data-stu-id="82a78-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="82a78-104">つまり、 `New` は等号 (`=`) の右側に配置しなければなりません。このようにして、配列変数に割り当てる新しい配列型を作成できます。</span><span class="sxs-lookup"><span data-stu-id="82a78-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="82a78-105">配列にはクラスの初期化のショートカットがありません。</span><span class="sxs-lookup"><span data-stu-id="82a78-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="82a78-106">次の 2 つのコード行はどちらも正しく、同等です。両方とも同じクラスに基づいてオブジェクトを初期化するためです。</span><span class="sxs-lookup"><span data-stu-id="82a78-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="82a78-107">ただし、配列の初期化では、クラスの初期化と同じショートカットを使用することはできません。</span><span class="sxs-lookup"><span data-stu-id="82a78-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="82a78-108">配列の `New` 句には、かっこ `()`と中かっこ `{}`の両方を含める必要があります。</span><span class="sxs-lookup"><span data-stu-id="82a78-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="82a78-109">かっこは新しい型が配列であることを指定し、中かっこは初期化値を提供します。</span><span class="sxs-lookup"><span data-stu-id="82a78-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="82a78-110">コンパイラでは、中が空でも (つまり、配列値を初期化しない場合でも) 中かっこが必要です。</span><span class="sxs-lookup"><span data-stu-id="82a78-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="82a78-111">**エラー ID:** BC30053</span><span class="sxs-lookup"><span data-stu-id="82a78-111">**Error ID:** BC30053</span></span>
