# <a name="labels-are-not-valid-in-the-immediate-window"></a>ラベルは、イミディエイト ウィンドウでは有効ではありません。
行ラベルは分岐を有効にしますが、デバッグ コンテキストでは使用できません。  
  
 **エラー ID:** BC30134  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   **[イミディ エイト]** ウィンドウでは、行ラベルを入力しないでください。
