# ソース フォルダーとターゲット フォルダーが同じです
ファイルをコピーまたは移動しようとしましたが、指定したソース ディレクトリとターゲット ディレクトリが同じです。 ファイル パスの解析に誤りがある可能性があります。  
  
### このエラーを解決するには  
  
-   適切なソース ディレクトリとターゲット ディレクトリを指定したことを確認してください。
