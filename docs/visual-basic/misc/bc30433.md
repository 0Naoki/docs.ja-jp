# <a name="methods-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>モジュールのメソッドを宣言することはできません&#39;&lt;指定子&gt;&#39;
`Module` ステートメント内のメソッドに対して正しくない指定子を使用しました。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30433  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
