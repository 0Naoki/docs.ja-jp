# <a name="expected-closing--for-xml-cdata-section"></a>終わり '] >' の XML の CDATA セクション
XML CDATA リテラルの終わりに必要な ']]>' 文字がありません。  
  
 **エラー ID:** BC31162  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML CDATA リテラルの終わりに必要な ']]>' 文字を追加します。
