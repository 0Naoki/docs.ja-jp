# &#39;On Error&#39; ステートメントは、&#39;SyncLock&#39; ステートメント内では使用できません
`On Error` ステートメントはスレッドの同期を中断するため、`SyncLock` ブロック内では使用できません。  
  
 **エラー ID:** BC30752  
  
### このエラーを解決するには  
  
1.  `On Error` ステートメントを `SyncLock` ブロック外に配置します。
