# <a name="3939-expected-declaration"></a>&#39; = (& a) #39 です。予期される (宣言)
`Mid` ステートメントまたは `#Const` コンパイル ディレクティブに、値の割り当てが含まれていません。  
  
 **エラー ID:** BC30249  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   等号 (`=`) と割り当てられる値を文字列またはコンパイル定数に追加します。
