# <a name="-expected-declaration"></a>'=' が必要です (宣言)
`Mid` ステートメントや `#Const` コンパイル ディレクティブで値が代入されていません。  
  
 **エラー ID:** BC30249  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- 等号 (`=`) に加えて、文字列やコンパイル定数に代入する値を追加します。
