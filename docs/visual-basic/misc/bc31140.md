# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>&#39;です。カスタム &#39;修飾子は、直前にのみ使用できます、&#39;です。イベント &#39;宣言
カスタム イベントの宣言で `Custom` キーワードは `Event` キーワードの前にしか置くことができません。  
  
 **エラー ID:** BC31140  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Custom` キーワードを `Event` キーワードの前に移動させます。  
  
     または  
  
-   `Custom` キーワードを削除します。
