# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>XML コメント タグ &#39;&lt;タグ&gt;&#39; は許可されていません、&#39;&lt;要素&gt;&#39; 言語要素
XML コメント タグ\<タグ > は許可されていません、'\<要素 >' の要素。 XML コメントは無視されます。  
  
 XML コメント タグは、その適用先の言語要素ではサポートされていません。  
  
 **エラー ID:** BC42306  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   サポートされていない XML コメント タグを削除します。
