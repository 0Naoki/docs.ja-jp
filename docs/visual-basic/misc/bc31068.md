# <a name="delegate-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>インターフェイス内のデリゲートを宣言することはできません (& a) #39 です。&lt;指定子&gt;&#39;です。
インターフェイスで宣言されたデリゲートで、正しくない修飾子が指定されました。  
  
 **エラー ID:** BC31068  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`、 `Overridable`などの修飾子を削除します。
