# <a name="code-resource-not-found"></a>コード リソースが見つかりません
コードのリソース内のプロシージャ呼び出しを試みましたが、コード リソースが見つかりませんでした。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- リソースが使用可能で正しく参照されていることを確認します。
