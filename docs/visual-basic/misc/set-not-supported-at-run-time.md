# Set は実行時にはサポートされません
デザイン時にのみ値を設定できるプロパティを設定または変更しようとしました。  
  
### このエラーを解決するには  
  
1.  コードからプロパティへの参照を削除します。  
  
2.  実行時に、プロパティの値のみを返す参照に変更します。
