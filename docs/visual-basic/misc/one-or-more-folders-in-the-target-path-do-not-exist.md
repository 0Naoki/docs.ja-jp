# ターゲット パスに 1 つまたは複数のフォルダーが存在しません
指定された 1 つ以上のディレクトリがターゲット パスに存在しません。 これは、特に相対パスを指定する場合など、パスが正しく解析されなかった結果である可能性があります。  
  
### このエラーを解決するには  
  
-   正しいターゲット パスを指定していることを確認します。
