# <a name="method-39ltmethodnamegt39-cannot-handle-event-39lteventnamegt39-because-they-do-not-have-the-same-signature"></a>メソッド&#39; &lt;methodname&gt; &#39;イベントを処理できない&#39; &lt;eventname&gt; &#39;同じシグネチャがあるないため
指定されたイベント ハンドラーのパラメーターまたはパラメーターの型が、指定されたイベントに対して定義されたものと一致しません  
  
 **エラー ID:** BC31029  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   パラメーター リストが一致するように、イベントまたはイベント ハンドラーを再定義します。
