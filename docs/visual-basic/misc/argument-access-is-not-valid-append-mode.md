# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>引数 'Access' は有効ではありません Append モードの有効な値は、'OpenAccess.Write' および 'OpenAccess.Default' です。
Append モードでアクセスの値が正しくありません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. 値を `OpenAccess.Write` または `OpenAccess.Default`のどちらかに変更します。
