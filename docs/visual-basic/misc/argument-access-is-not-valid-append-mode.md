# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>引数 &#39;アクセス &#39;正しくありません。 モードは、Append &#39;OpenAccess.Write &#39; の有効な値および &#39;OpenAccess.Default &#39;
Append モードでアクセスの値が正しくありません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  値を `OpenAccess.Write` または `OpenAccess.Default`のどちらかに変更します。
