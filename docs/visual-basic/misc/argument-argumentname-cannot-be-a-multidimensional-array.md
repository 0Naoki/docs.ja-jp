# <a name="argument-ltargumentnamegt-cannot-be-a-multidimensional-array"></a>引数 '&lt;argumentname&gt;' 多次元配列にすることはできません
配列引数のランクが 1 を超えていますが、この配列には 1 次元配列のみを指定できます。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数のランクを 1 に変更します。
