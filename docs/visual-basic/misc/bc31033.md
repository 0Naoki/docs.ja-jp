# <a name="interface-39ltinterfacenamegt39-can-be-implemented-only-once-by-this-type"></a>インターフェイス&#39; &lt;interfacename&gt; &#39;この型によって 1 回のみ実装できます
指定したインターフェイスはこの型によって既に実装されており、再実装できません。  
  
 **エラー ID:** BC31033  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   この型を 1 回だけ使用してこのインターフェイスを実装します。
