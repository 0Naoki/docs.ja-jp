# 現在の区切り記号を使用して、行 &lt;number&gt; を解析できません
指定したもの以外の区切り記号を使用しているために、指定した行を解析できません。  
  
### このエラーを解決するには  
  
-   行が正しく解析できるように `Delimiters` を調整するか、またはその行を処理するために例外処理コードを挿入します。
