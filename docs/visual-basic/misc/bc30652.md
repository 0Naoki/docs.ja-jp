# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>参照アセンブリが必要です '&lt;assemblyname&gt;'の型を含む'&lt;membername&gt;'
参照アセンブリが必要です '\<assemblyname >' の型を含む'\<membername >' です。 プロジェクトに参照を追加してください。  
  
 プロジェクトには、プロジェクト参照に含まれていないアセンブリに属する別の型に依存する型が含まれています。  
  
 **エラー ID:** BC30652  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   依存する型のプロジェクト参照をプロジェクトに追加します。
