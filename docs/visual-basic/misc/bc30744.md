# <a name="referenced-array-element-has-a-value-of-39nothing39"></a>参照先の配列要素には、値は &#39;何も &#39;
参照されている配列要素が設定されていないか、または以前に `Nothing`に設定されました。  
  
 **エラー ID:** BC30744  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   配列内の値が正しく設定されていることを確認します。
