# <a name="property-ltpropertynamegt-is-of-an-unsupported-type"></a>プロパティ '&lt;propertyname&gt;' がサポートされていない型
参照先アセンブリのプロパティには、表現できない型が含まれています。  
  
 **エラー ID:** BC30643  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   アプリケーションで使用できる型を指定します。
