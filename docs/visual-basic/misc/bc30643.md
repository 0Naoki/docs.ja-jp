# プロパティ &#39;&lt;propertyname&gt;&#39; はサポートされていない型です
参照先アセンブリのプロパティには、表現できない型が含まれています。  
  
 **エラー ID:** BC30643  
  
### このエラーを解決するには  
  
-   アプリケーションで使用できる型を指定します。
