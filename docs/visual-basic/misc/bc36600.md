# 範囲変数 &lt;variable&gt; は既に宣言されています
`Select` 句、または `Aggregate` や `Group By` や `Group Join` 句の `Into` 部分に指定された範囲変数の名前が、クエリ句に既に指定されている範囲変数の名前と重複しています。  
  
 **エラー ID:** BC36600  
  
### このエラーを解決するには  
  
1.  クエリ句内のすべての範囲変数の名前が一意であることを確認します。
