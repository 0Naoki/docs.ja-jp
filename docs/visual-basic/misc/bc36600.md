# <a name="range-variable-ltvariablegt-is-already-declared"></a>範囲変数&lt;変数&gt;は既に宣言されています
`Select` 句、または `Into` や `Aggregate`や `Group By`句の `Group Join` 部分に指定された範囲変数の名前が、クエリ句に既に指定されている範囲変数の名前と重複しています。  
  
 **エラー ID:** BC36600  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  クエリ句内のすべての範囲変数の名前が一意であることを確認します。
