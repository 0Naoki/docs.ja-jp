# ディスクの空き容量が不足しています (Visual Basic)
`Print`、`Write`、または `FileClose` 操作を完了するか、または必要なファイルを作成するために十分なディスク領域がありません  
  
### このエラーを解決するには  
  
-   一部のファイルを別のディスクに移動するか削除して、ディスク上の空き領域を増やします。
