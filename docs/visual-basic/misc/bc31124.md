# <a name="39end-addhandler39-must-be-preceded-by-a-matching-39addhandler39-declaration"></a>&#39;です。End AddHandler &#39;一致する &#39; の前が必要AddHandler &#39;宣言
`End AddHandler` ステートメントが発生しましたが、対応する `AddHandler` ステートメントがありません。 `End AddHandler` の前に、対応する `AddHandler` ステートメントが必要です。  
  
 **エラー ID:** BC31124  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   先行する `AddHandler` ステートメントが有効であり、スペルが正しいことを確認します。
