# <a name="argument-matching-parameter-ltparameternamegt-narrows-to-lttypenamegt"></a>引数のパラメーターと一致する '&lt;parametername&gt;'へ縮小変換'&lt;typename&gt;'
オーバーロードされたメソッドを呼び出しましたが、コンパイラは縮小変換せずに呼び出すことができるメソッドを検出できません。 *縮小変換* により、有効値の一部を保持できない可能性のあるデータ型に値が変更されます。  
  
 **エラー ID:** BC36711
