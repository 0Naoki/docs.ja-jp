# &#39;Microsoft.VisualBasic.ComClassAttribute&#39; と &#39;&lt;attribute&gt;&#39; の両方を同じクラスに適用することはできません
`COMClassAttribute` 属性ブロックが、COM オブジェクトに適用されない属性と組み合わせて使用されています。 原因の 1 つとして、[!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] 属性と COM 属性が混在していることが考えられます。  
  
 **エラー ID:** BC32501  
  
### このエラーを解決するには  
  
-   `COMClassAttribute` 属性ブロック、または COM に適用されない属性のいずれかを削除します。
