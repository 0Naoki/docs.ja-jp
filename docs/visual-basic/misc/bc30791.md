# <a name="declare-statements-in-a-structure-cannot-be-declared-ltspecifiergt"></a>'Declare' ステートメントの構造体を宣言することはできません '&lt;指定子&gt;'
`Declare` 宣言に、 `Structure` 宣言内で無効な指定子があります。  
  
 **エラー ID:** BC30791  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
