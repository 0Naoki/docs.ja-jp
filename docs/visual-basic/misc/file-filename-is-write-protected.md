# <a name="file-filename-is-write-protected"></a>ファイル '\<filename >' は書き込み禁止
書き込もうとしているファイルは書き込み禁止であるため、変更できません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ファイルのアクセス許可を読み取り/書き込みに変更するか、別のファイルに書き込みます。
