# 文字のシーケンス &#39;--&#39; は、XML コメントでは使用できません
XML コメント リテラルに、サポートされていない文字のシーケンス '\-\-' が含まれています。  
  
 **エラー ID:** BC31171  
  
### このエラーを解決するには  
  
-   XML コメント リテラルから、文字のシーケンス '\-\-' を削除します。
