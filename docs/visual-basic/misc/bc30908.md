<span data-ttu-id="6085d-103">クラスまたはインターフェイスの定義に含まれる、[ステートメントの継承](../../visual-basic/language-reference/statements/inherits-statement.md)内に入れ子になった型を指定します。</span><span class="sxs-lookup"><span data-stu-id="6085d-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>

    End Class
