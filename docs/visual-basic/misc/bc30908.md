<span data-ttu-id="5f553-103">クラスまたはインターフェイスの定義に、その内に入れ子にされた型を指定する [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) が含まれています。</span><span class="sxs-lookup"><span data-stu-id="5f553-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>  
  
 <span data-ttu-id="5f553-104">継承は、循環ではなく線形である必要があります。</span><span class="sxs-lookup"><span data-stu-id="5f553-104">Inheritance must be linear, not circular.</span></span> <span data-ttu-id="5f553-105">型は、その型を継承する型から継承することはできません。</span><span class="sxs-lookup"><span data-stu-id="5f553-105">A type cannot inherit from a type that inherits from it.</span></span>  
  
 <span data-ttu-id="5f553-106">関連する制限として、型は未定義の型から継承することはできません。</span><span class="sxs-lookup"><span data-stu-id="5f553-106">A related restriction is that a type cannot inherit from a type that is not yet defined.</span></span> <span data-ttu-id="5f553-107">継承には基底クラスのメンバーを再利用する機能が関与するため、これらのメンバーが定義されている必要があります。</span><span class="sxs-lookup"><span data-stu-id="5f553-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="5f553-108">したがって、Visual Basic は、次の例のようなコードをコンパイルすることはできません。</span><span class="sxs-lookup"><span data-stu-id="5f553-108">Therefore, Visual Basic cannot compile code such as the following example.</span></span>

```  
    Public Class innerClass  
        Public Sub doSomething()  
        End Sub  
    End Class  
End Class  
```
