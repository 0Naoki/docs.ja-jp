<span data-ttu-id="356c8-103">クラスまたはインターフェイスの定義に、その内に入れ子にされた型を指定する [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) が含まれています。</span><span class="sxs-lookup"><span data-stu-id="356c8-103">A class or interface definition includes an [Inherits Statement](../../visual-basic/language-reference/statements/inherits-statement.md) that specifies a type nested within it.</span></span>
