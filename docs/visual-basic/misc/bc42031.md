# <a name="39catch39-block-never-reached-ltexceptiongt-handled-above-in-the-same-39try39-statement"></a>&#39;です。Catch &#39;ブロックに到達しません。&lt;例外&gt;同じ &#39; で処理しました。Try &#39;ステートメント
先行する `Catch` ブロックで処理されるため、コードの `Try` ブロックには到達できません。  
  
 既定では、このメッセージは警告です。 警告を非表示や、警告をエラーとして扱う方法の詳細についてを参照してください[Visual Basic での警告の構成](/visualstudio/ide/configuring-warnings-in-visual-basic)  
  
 **エラー ID:** BC42031  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不要なステートメントを削除します。
