# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttypename1gt39-and-39lttypename2gt39"></a>演算子&#39; &lt;operatorname&gt; &#39;型に対して定義されていない&#39; &lt;typename1&gt; &#39;と&#39; &lt;typename2&gt;&#39;
演算子 '\<operatorname >' の型に対して定義されていない\<typename1 >' と'\<typename2 >' です。 'Is' 演算子を使用すると、2 つの参照型を比較できます。  
  
 指定した型に対して不適切な方法で、演算子を使用しようとしました。 このエラーは、 `Is` 演算子ではなく "=" 演算子を使用して、2 つのオブジェクトを比較すると発生する可能性があります。  
  
 **エラー ID:** BC31080  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Is` 演算子を使用して、2 つの参照型を比較します。
