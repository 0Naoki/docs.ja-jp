 <span data-ttu-id="00125-105">ジェネリック拡張メソッドを呼び出すときに、型の推定を使用して型パラメーターの型を決定しようとしました。</span><span class="sxs-lookup"><span data-stu-id="00125-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="00125-106">コンパイラによって 1 つ以上の型パラメーターについて複数のデータ型の候補が検出されると、このエラーが報告されます。</span><span class="sxs-lookup"><span data-stu-id="00125-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

 <span data-ttu-id="00125-108">次のコードはエラーを示しています。</span><span class="sxs-lookup"><span data-stu-id="00125-108">The following code demonstrates the error.</span></span>

Option Strict Off  
Imports System.Runtime.CompilerServices  
Module Module1  
    Sub Main()  
  
        Dim caller As New Class1  
        '' Not valid.  
        'caller.targetExtension(1, "2")  
  
    End Sub  
  
    <Extension()> _  
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)

    Class Class1  
    End Class
