 <span data-ttu-id="3509c-108">次のコードにこのエラーを示します。</span><span class="sxs-lookup"><span data-stu-id="3509c-108">The following code demonstrates the error.</span></span>

    Sub Main()  
  
        Dim caller As New Class1  
        '' Not valid.
