# <a name="data-types-of-the-type-parameters-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-cannot-be-inferred-from-these-arguments-because-more-than-one-type-is-possible"></a><span data-ttu-id="cf26d-102">拡張メソッド内の型パラメーターのデータ型を '&lt;methodname&gt;'で定義されている'&lt;typename&gt;' は、複数の型が考えられるために、これらの引数から推論することはできません</span><span class="sxs-lookup"><span data-stu-id="cf26d-102">Data type(s) of the type parameter(s) in extension method '&lt;methodname&gt;' defined in '&lt;typename&gt;' cannot be inferred from these arguments because more than one type is possible</span></span>
<span data-ttu-id="cf26d-103">拡張メソッド内の型パラメーターのデータ型を '\<methodname >' で定義されている'\<typename >' は、複数の型が考えられるために、これらの引数から推論することはできません。</span><span class="sxs-lookup"><span data-stu-id="cf26d-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in '\<typename>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="cf26d-104">データ型を明示的に指定すると、このエラーが修正される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="cf26d-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="cf26d-105">ジェネリック拡張メソッドを呼び出すときに、型の推定を使用して型パラメーターの型を決定しようとしました。</span><span class="sxs-lookup"><span data-stu-id="cf26d-105">An attempt has been made to use type inference to determine the type (or types) of the type parameter (or parameters) in a call to a generic extension method.</span></span> <span data-ttu-id="cf26d-106">コンパイラによって 1 つ以上の型パラメーターについて複数のデータ型の候補が検出されると、このエラーが報告されます。</span><span class="sxs-lookup"><span data-stu-id="cf26d-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="cf26d-107">引数の指定がオプションではない場合 (たとえば、クエリ式内のクエリ演算子など)、エラー メッセージの 2 つ目の文は表示されません。</span><span class="sxs-lookup"><span data-stu-id="cf26d-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="cf26d-108">次のコードはエラーを示しています。</span><span class="sxs-lookup"><span data-stu-id="cf26d-108">The following code demonstrates the error.</span></span>  
  
```vb  
Option Strict Off  
Imports System.Runtime.CompilerServices  
Module Module1  
    Sub Main()  
  
        Dim caller As New Class1  
        '' Not valid.  
        'caller.targetExtension(1, "2")  
  
    End Sub  
  
    <Extension()> _  
    Sub targetExtension(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)  
    End Sub  
  
    Class Class1  
    End Class  
  
End Module  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="cf26d-110">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="cf26d-110">To correct this error</span></span>
