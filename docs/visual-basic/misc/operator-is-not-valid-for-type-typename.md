# <a name="operator-is-not-valid-for-type-lttypenamegt"></a>演算子で型に対して有効でない '&lt;typename&gt;'
演算子は、型に適切ではありません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   演算子が演算対象の型に適していることを確認します。
