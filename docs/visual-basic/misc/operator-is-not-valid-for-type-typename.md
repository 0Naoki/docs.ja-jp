# <a name="operator-is-not-valid-for-type-39lttypenamegt39"></a>演算子は型に対して無効&#39; &lt;typename&gt;&#39;
演算子は、型に適切ではありません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   演算子が演算対象の型に適していることを確認します。
