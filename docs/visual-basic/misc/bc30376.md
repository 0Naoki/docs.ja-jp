# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39;です。新しい &#39;です。クラス &#39; では使用できません。&lt;classname&gt;&#39; が含まれています、&#39;MustOverride &#39;オーバーライドされていないメンバー
オーバーライドされていない `New` メンバーを含むクラスで `MustOverride` の使用を試みました。  
  
 **エラー ID:** BC30376  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `New` ステートメントを削除します。
