# <a name="evaluation-of-expression-or-statement-stopped"></a>式またはステートメントの評価が停止されました
**[評価の停止]** メニュー項目が選択されたため、評価が停止されました。  
  
 **エラー ID:** BC30721  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- **[評価の停止]** を選択しなければ、引き続き実行できます。
