# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39;終了&#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Exit` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30713  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `Exit` ステートメントを削除します。
