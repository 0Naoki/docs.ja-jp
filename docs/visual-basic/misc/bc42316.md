# XML ドキュメントの解析エラーです。開始タグ &#39;&lt;tag&gt;&#39; に対応する終了タグがありません
XML ドキュメントの解析エラーです。開始タグ \<tag\> に対応する終了タグがありません。 XML コメントは無視されます。  
  
 XML コメントに開始タグがありますが、対応する終了タグがありません。  
  
 **エラー ID:** BC42316  
  
### このエラーを解決するには  
  
-   開始タグに対応する終了タグを追加します。  
  
     または  
  
-   [\<seealso\>](../../visual-basic/language-reference/xmldoc/seealso.md) など、タグに内部テキストがない場合は、終わり山かっこの前にスラッシュを指定します。
