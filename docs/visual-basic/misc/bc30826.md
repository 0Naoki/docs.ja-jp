# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39;です。EndIf &#39;ステートメントがサポートされていません。使用して &#39;終了 (& a) #39 です。その代わりに
`EndIf` ステートメントはサポートされなくなりました。  
  
 **エラー ID:** BC30826  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `End If` ステートメントを終了するには、 `If` を使用します。
