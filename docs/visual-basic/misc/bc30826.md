# <a name="endif-statements-are-no-longer-supported-use-end-if-instead"></a>'EndIf' ステートメントは現在サポートされていません。'End If' を使用してください
`EndIf` ステートメントはサポートされなくなりました。  
  
 **エラー ID:** BC30826  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- `End If` ステートメントを終了するには、 `If` を使用します。
