# <a name="replacements-too-long"></a>置換後の文字列が長すぎます
許容される長さを超える置換文字列を指定しました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  長さの制限については、コンポーネントのドキュメントを参照してください。
