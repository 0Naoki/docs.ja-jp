# <a name="39variant39-is-no-longer-a-supported-type-use-the-39object39-type-instead"></a>&#39;Variant&#39;が不要になったサポートされる型です。使用して、&#39;オブジェクト&#39;代わりを入力
`Object` データ型を `Variant`の代わりに使用します。  
  
 **エラー ID:** BC30804  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   アイテムを `Object`として宣言します。
