# <a name="39region39-statement-must-end-with-a-matching-39end-region39"></a>'#Region' ステートメントは、一致する ' #End Region' で終わる必要があります。
使用して、`#Region`ディレクティブで使用する展開したり折りたたんだりのアウトライン表示機能を使用する場合のコードのブロックを指定する、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]コード エディターです。 `#Region` ステートメントの開始と終了は同じコード ブロック内にある必要があります。  
  
 **エラー ID:** BC30681  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  コードの `#End Region` ステートメントの後の適切な場所に `#Region` を挿入します。
