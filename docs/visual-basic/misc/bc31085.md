# <a name="date-constant-is-not-valid"></a>日付定数が有効ではありません
日付定数が無効な日付で宣言されています。  
  
 **エラー ID:** BC31085  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   定数で使用される日付を修正します。
