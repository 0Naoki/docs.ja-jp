# 要素に終了タグがありません。
XML 要素リテラルには、XML の終了タグまたは空のタグ識別子 '\/' \(`<tagName />` など\) がありません。  
  
 **エラー ID:** BC31151  
  
### このエラーを解決するには  
  
-   XML タグの終了タグを追加するか、または空のタグ識別子 '\/' を XML タグに追加します。
