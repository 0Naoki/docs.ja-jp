# プロジェクト レベルの条件付きコンパイル定数 &#39;&lt;constantname&gt;&#39; の形式が有効ではありません: &lt;error&gt;
無効な条件付きコンパイル オプションが、プロジェクトに指定されています。  
  
 **エラー ID:** BC31030  
  
### このエラーを解決するには  
  
1.  このプロジェクトで使用されている条件付きコンパイル定数を確認します。
