# <a name="project-level-conditional-compilation-constant-39ltconstantnamegt39-is-not-valid-lterrorgt"></a>プロジェクト レベルの条件付きコンパイル定数&#39; &lt;constantname&gt; &#39;が正しくありません:&lt;エラー&gt;
無効な条件付きコンパイル オプションが、プロジェクトに指定されています。  
  
 **エラー ID:** BC31030  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  このプロジェクトで使用されている条件付きコンパイル定数を確認します。
