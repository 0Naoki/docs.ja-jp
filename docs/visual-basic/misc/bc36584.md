# <a name="parameter-ltparameternamegt-in-extension-method-ltmethodnamegt-defined-in-lttypenamegt-already-has-a-matching-argument"></a>パラメーター '&lt;parametername&gt;'で拡張メソッド'&lt;methodname&gt;'で定義されている'&lt;typename&gt;'、一致する引数が既に
1 つの引数が、同じプロシージャ呼び出しで複数回にわたって名前で渡されています。  
  
 **エラー ID:** BC36584  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数名のいずれかのスペルが誤っていないかどうかを確認します。  
  
-   引数リストから重複している引数を削除します。
