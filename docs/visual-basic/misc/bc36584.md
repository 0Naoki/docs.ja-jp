# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>パラメーター '\<parametername >' で拡張メソッド'\<methodname >' で定義されている '\<typename >'、一致する引数が既に
1 つの引数が、同じプロシージャ呼び出しで複数回にわたって名前で渡されています。  
  
 **エラー ID:** BC36584  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数名のいずれかのスペルが誤っていないかどうかを確認します。  
  
-   引数リストから重複している引数を削除します。
