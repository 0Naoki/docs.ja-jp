# <a name="imports-alias-ltnamegt-conflicts-with-ltname2gt-declared-in-the-root-namespace"></a>Imports エイリアス&lt;名前&gt;'と競合する'&lt;name2&gt;' ルート名前空間で宣言されています。
`Imports` ステートメントで使用されるエイリアスは、ルート名前空間で宣言されている別の名前と競合しています。  
  
 **エラー ID:** BC31403  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Imports` ステートメントで使用されるエイリアスに一意の名前を選択します。
