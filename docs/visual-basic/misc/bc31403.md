# <a name="imports-alias-39ltnamegt39-conflicts-with-39ltname2gt39-declared-in-the-root-namespace"></a>Imports エイリアス &#39;&lt;名前&gt;&#39; と競合しています &#39;&lt;name2&gt;&#39; ルート名前空間で宣言されています。
`Imports` ステートメントで使用されるエイリアスは、ルート名前空間で宣言されている別の名前と競合しています。  
  
 **エラー ID:** BC31403  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Imports` ステートメントで使用されるエイリアスに一意の名前を選択します。
