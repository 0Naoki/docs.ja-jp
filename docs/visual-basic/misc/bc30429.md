# &#39;End Sub&#39; の前には、対応する &#39;Sub&#39; を指定しなければなりません。
前に一致する `Sub` プロシージャ定義のない `End Sub` ステートメントがコードに出現します。  
  
 **エラー ID:** BC30429  
  
### このエラーを解決するには  
  
-   `End Sub` ステートメントが重複している場合は、削除します。  
  
-   `Sub` プロシージャが不足している場合は、指定します。  
  
-   `End Sub` をコード内の適切な場所に移動します。
