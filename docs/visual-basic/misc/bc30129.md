# アセンブリ属性 &#39;&lt;attributename&gt;&#39; が正しくありません: &lt;error&gt;
共通言語ランタイムでは、アセンブリの属性は受け入れられません。  
  
 **エラー ID:** BC30129  
  
### このエラーを解決するには  
  
1.  宣言から属性を削除するか、コンテキストに有効な属性で置き換えます。  
  
2.  引用符で囲まれたエラー メッセージを確認し、適切な処理を行います。
