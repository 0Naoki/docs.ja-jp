# <a name="attribute-constant-39ltconstantnamegt39-cannot-be-the-target-of-an-assignment"></a>属性定数 &#39;&lt;constantname&gt;&#39; 代入のターゲットにすることはできません
属性内で宣言された定数に値を代入しようとしました。  
  
 **エラー ID:** BC31510  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   間違っている代入を削除します。  
  
-   独自に開発したカスタム属性を使用する場合は、属性内のメンバーをフィールドとして再定義します。
