# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>型文字 &#39;&lt;文字&gt;&#39; 明示的な型を含む宣言では使用できません
データ型を明示的に指定する際に、 `$`などの型文字が使用されています。  
  
 **エラー ID:** BC30302  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   宣言から型文字を削除するか、 `As <Type>` 句を削除します。
