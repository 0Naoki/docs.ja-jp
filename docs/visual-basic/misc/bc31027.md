# <a name="unable-to-open-file-filename-error"></a>ファイルを開けません '\<filename >':\<エラー >。
このプロジェクトでファイルを開くときにエラーが発生しました。  
  
 **エラー ID:** BC31027  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ファイルが存在し、破損していたり別のアプリケーションによってロックされたりしていないことを確認します。
