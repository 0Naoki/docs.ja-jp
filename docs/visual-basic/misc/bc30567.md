# <a name="array-initializer-is-missing-ltnumbergt-elements"></a>配列初期化子がありません&lt;数&gt;要素
配列の初期化子で指定した要素が少なすぎます。  
  
 **エラー ID:** BC30567  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不足している引数を指定します。  
  
-   `ReDim` を使って配列のサイズを変更します。
