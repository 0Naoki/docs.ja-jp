# <a name="39end-raiseevent39-must-be-the-first-statement-on-a-line"></a>&#39;です。End RaiseEvent &#39;行の最初のステートメントをする必要があります。
`End RaiseEvent` ステートメントの前にコロン (:) ステートメント区切り記号があります。 `End RaiseEvent` は、そのソース行で唯一のステートメントである必要があります。  
  
 **エラー ID:** BC31120  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   複数のステートメントを別々の行に分けます。
