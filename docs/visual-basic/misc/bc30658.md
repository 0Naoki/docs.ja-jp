# パラメーターなしのプロパティ &#39;&lt;propertyname&gt;&#39; が見つかりません
使用中の属性には、1 つまたは複数のパラメーターを必要とするプロパティがあります。  
  
 **エラー ID:** BC30658  
  
### このエラーを解決するには  
  
1.  プロパティのパラメーターを指定します。
