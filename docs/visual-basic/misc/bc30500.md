# 定数 &#39;&lt;constantname&gt;&#39; が、それ自体の値に依存することはできません
定数がそれ自体の値に依存する循環依存をコードに作成しました。 たとえば、`Const a = Const b; Const b = Const a` のようにします。  
  
 **エラー ID:** BC30500  
  
### このエラーを解決するには  
  
1.  コードを調べて定数が評価されている場所を確認し、必要に応じて変更します。
