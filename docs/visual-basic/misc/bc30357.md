# <a name="39inherits39-statements-must-precede-all-declarations-in-an-interface"></a>&#39;継承&#39;ステートメントは、インターフェイス内のすべての宣言を付ける必要があります
`Inherits` ステートメントは、クラス定義の最初の空行でもコメント行でもない行に記述されている必要があります。  
  
 **エラー ID:** BC30357  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   すべての宣言の前に来るように `Inherits` ステートメントを移動します。
