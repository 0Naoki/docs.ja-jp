# &#39;&lt;filename&gt;&#39; はアセンブリでないため、参照できません
.NET アセンブリではないファイルが参照されました。  
  
 **エラー ID:** BC31519  
  
### このエラーを解決するには  
  
1.  アセンブリの場所と名前を確認します。  
  
2.  アセンブリが破損していないことを確認します。
