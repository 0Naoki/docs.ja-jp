# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;名前&gt;&#39;プロジェクトで宣言された&#39; &lt;projectname1&gt;&#39;、プロジェクトによって参照されていません&#39; &lt;projectname2&gt;&#39;
プログラミング要素が、現在のプロジェクトに参照が含まれていないプロジェクトで定義されています。  
  
 **エラー ID:** BC32004  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   現在のプロジェクトで、目的の要素が入っているプロジェクトへの参照を追加します。
