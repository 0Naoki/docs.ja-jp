# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;です。&lt;名前&gt;&#39; プロジェクト &#39; で宣言されている&lt;projectname1&gt;&#39;、プロジェクト &#39; によって参照されていません&lt;projectname2&gt;&#39;です。
プログラミング要素が、現在のプロジェクトに参照が含まれていないプロジェクトで定義されています。  
  
 **エラー ID:** BC32004  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   現在のプロジェクトで、目的の要素が入っているプロジェクトへの参照を追加します。
