# <a name="ltnamegt-is-declared-in-project-ltprojectname1gt-which-is-not-referenced-by-project-ltprojectname2gt"></a>'&lt;名前&gt;'プロジェクトで宣言された'&lt;projectname1&gt;'、プロジェクトによって参照されていません'&lt;projectname2&gt;'
プログラミング要素が、現在のプロジェクトに参照が含まれていないプロジェクトで定義されています。  
  
 **エラー ID:** BC32004  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   現在のプロジェクトで、目的の要素が入っているプロジェクトへの参照を追加します。
