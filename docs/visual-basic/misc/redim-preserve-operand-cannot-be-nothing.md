# <a name="39redim39-preserve-operand-cannot-be-nothing"></a>&#39;です。ReDim &#39;Preserve オペランドには、何もすることはできません。
`ReDim` ステートメントが `Preserve` キーワードを使用して、最後のディメンションではない配列のディメンションを変更しようとしましたが、そのオペランドに対して正しい値を指定していません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Preserve` オペランドを正しい値に変更します。
