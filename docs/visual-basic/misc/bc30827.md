# &#39;D&#39; は指数を表す文字として使用できません。&#39;E&#39; を使用してください。
'D' 文字を使用して指数演算を示すことはできません。  
  
 **エラー ID:** BC30827  
  
### このエラーを解決するには  
  
-   指数部があることを示すには、`^` 演算子または `E+` 文字を使用します。 例:  
  
    ```  
    Const Mole = 6.02E+23 ' Same as 6.02D23 Const Mole2 = 6.02 * 10 ^ 23 ' Same as 6.02D23  
    ```
