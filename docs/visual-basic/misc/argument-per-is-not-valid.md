# <a name="argument-per-is-not-valid"></a>引数 'Per' は使用できません
`Per`に指定された引数は、1 ～ `Double` の範囲で支払い期間を指定する `NPer`である必要があるため、使用できません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   引数が正しい範囲内にあることを確認します。
