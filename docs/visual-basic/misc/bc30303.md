# <a name="type-character-cannot-be-used-in-a-39sub39-declaration-because-a-39sub39-doesn39t-return-a-value"></a>型文字は使用できません、 &#39;Sub&#39;宣言のため、 &#39;Sub&#39;しません&#39;値を返さない
`Sub` プロシージャは、 `Function` プロシージャと同様に、引数を受け取り、一連のステートメントを実行できる個別のプロシージャです。 `Function` プロシージャと異なり、 `Sub` は値を返さないため、型宣言を含めることはできません。  
  
 **エラー ID:** BC30303  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Sub` プロシージャを `Function` プロシージャに変更します。
