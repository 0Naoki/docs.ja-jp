# &#39;Sub&#39; は値を返さないため、&#39;Sub&#39; 宣言で型文字を使用することはできません
`Sub` プロシージャは、`Function` プロシージャと同様に、引数を受け取り、一連のステートメントを実行できる個別のプロシージャです。`Function` プロシージャと異なり、`Sub` は値を返さないため、型宣言を含めることはできません。  
  
 **エラー ID:** BC30303  
  
### このエラーを解決するには  
  
-   `Sub` プロシージャを `Function` プロシージャに変更します。
