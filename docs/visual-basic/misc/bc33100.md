# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>変数とその型の両方で、null 許容修飾子を指定することはできません
null 許容型修飾子 (?) は、宣言された変数と、その宣言された型の両方に含まれています。  
  
 **エラー ID:** BC33100  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  null 許容型修飾子 (?) を変数または宣言された型から削除します。
