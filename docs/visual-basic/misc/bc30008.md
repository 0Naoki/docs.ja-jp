# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-base-class-39ltclassnamegt39"></a>モジュールに必要な参照 '&lt;modulename&gt;'基本クラスを含む'&lt;classname&gt;'
モジュールに必要な参照 '\<modulename >' 基本クラスを含む\<classname >' です。 プロジェクトに参照を追加してください。  
  
 プロジェクト内で直接参照されないモジュールでクラスが定義されています。 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]コンパイラ クラスが&1; つ以上のモジュールで定義されている場合に備えて、あいまいさを避けるために参照が必要です。  
  
 **エラー ID:** BC30008  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   参照されていないモジュールの名前をプロジェクト参照に含めます。
