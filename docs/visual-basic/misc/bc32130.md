# 現在のバージョンの &#39;mscorlib.dll&#39; には型 &#39;Nullable(of T)&#39; が定義されていないため、Null 許容型はサポートされていません
使用しているフレームワークのバージョンは、Null 許容型をサポートしていません。  
  
 **エラー ID:** BC32130
