# &#39;&lt;specifier1&gt;&#39; と &#39;&lt;specifier2&gt;&#39; を組み合わせることはできません
`Shadows` と `Overrides` など、2 つの互換性のない指定子が一緒に使用されています。  
  
 **エラー ID:** BC31408  
  
### このエラーを解決するには  
  
-   いずれかの指定子を削除します。
