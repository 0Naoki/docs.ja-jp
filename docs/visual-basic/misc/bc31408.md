# <a name="ltspecifier1gt-and-ltspecifier2gt-cannot-be-combined"></a>'&lt;specifier1&gt;'と'&lt;specifier2&gt;' 組み合わせることはできません
`Shadows` と `Overrides` など、2 つの互換性のない指定子が一緒に使用されています。  
  
 **エラー ID:** BC31408  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   いずれかの指定子を削除します。
