# <a name="39ltspecifier1gt39-and-39ltspecifier2gt39-cannot-be-combined"></a>&#39;です。&lt;specifier1&gt;&#39; と &#39;&lt;specifier2&gt;&#39; 組み合わせることができません
`Shadows` と `Overrides` など、2 つの互換性のない指定子が一緒に使用されています。  
  
 **エラー ID:** BC31408  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   いずれかの指定子を削除します。
