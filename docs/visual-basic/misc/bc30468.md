# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>型宣言文字は、このコンテキストでは有効ではありません。
$ など、このコンテキストでは有効でない文字を使用しようとしました。  
  
 **エラー ID:** BC30468  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  該当する文字を削除します。
