# <a name="attribute-value-is-not-valid-expecting-39value139-or-39value239"></a>属性の値が無効です。指定してください&#39;value1&#39;または&#39;value2&#39;
XML ドキュメント リテラルに、無効な属性の値が指定されています。 XML ドキュメント リテラルの有効な属性値は、XML 1.0 仕様に基づきます。  
  
 **エラー ID:** BC31182  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   属性値を XML 1.0 仕様で有効な値に変更します。
