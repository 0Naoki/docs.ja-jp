# ファイルが存在しない場合、ファイル情報を照会することはできません
存在しないファイルの情報に対して、オブジェクトの <xref:System.IO.FileInfo> のクエリが試行されました。 通常これは、ファイル名を指定するときの誤りが原因です。  
  
### このエラーを解決するには  
  
-   正しいファイル名を指定していることを確認します。
