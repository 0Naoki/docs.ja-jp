# リソース ファイル &#39;&lt;filename&gt;&#39; を開けません: &lt;error&gt;
指定したリソース ファイルを開くときにエラーが発生しました。  
  
 **エラー ID:** BC31509  
  
### このエラーを解決するには  
  
1.  指定した名前と場所が正しいことを確認します。  
  
2.  ファイルが破損していないことを確認します。
