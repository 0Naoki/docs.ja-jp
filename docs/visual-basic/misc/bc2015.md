# ファイル &#39;&lt;filename&gt;&#39; はテキスト ファイルではありません
指定したファイルはバイナリ ファイルのため、書き込みができません。  
  
 **エラー ID:** BC2015  
  
### このエラーを解決するには  
  
-   正しいファイル名とパスを指定していることを確認します。
