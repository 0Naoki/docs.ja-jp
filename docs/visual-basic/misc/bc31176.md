# 文字 &#39;character&#39; (0x&lt;hexadecimal&gt;) は、XML リテラルでは使用できません
XML リテラルでサポートされていない文字を使用しています。  
  
 **エラー ID:** BC31176  
  
### このエラーを解決するには  
  
-   サポートされていない文字を削除します。
