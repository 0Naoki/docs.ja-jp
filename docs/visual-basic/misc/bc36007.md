# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;使用して終了&#39;前に、対応する必要がある&#39;を使用します。&#39;
一致する `End Using` 宣言を先に記述することなく、 `Using` ステートメントを記述しています。  
  
 **エラー ID:** BC36007  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `End Using` ステートメントが重複している場合は、削除します。  
  
-   不足[Using ステートメント](../../visual-basic/language-reference/statements/using-statement.md)場合は 1 つが見つかりません。  
  
-   `End Using` ステートメントをコード内の適切な場所に移動します。
