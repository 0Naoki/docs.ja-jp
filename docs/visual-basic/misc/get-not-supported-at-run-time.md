# Get は実行時にはサポートされません
設計時のみアクセスできるプロパティを実行時に読み取ろうとしました。  
  
### このエラーを解決するには  
  
1.  プロパティを調べて、どのような条件でプロパティを設定できるかを確認します。  
  
2.  プロパティへの参照を削除します。
