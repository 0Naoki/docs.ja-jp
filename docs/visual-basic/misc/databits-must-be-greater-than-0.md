# データビットは 0 より大きくなければなりません
`My.Computer.Ports.OpenSerialPort` メソッドに指定された `DataBits` 引数は、0 より大きくなければなりません。  
  
### このエラーを解決するには  
  
-   `DataBits` 引数の値を正の整数に変更します。
