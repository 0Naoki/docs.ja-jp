# <a name="argument-argumentname-cannot-be-converted-to-type-date"></a>引数 '\<argumentname >' 型 'Date' に変換できません
変数を型を `Date`に変更する変換が試みられましたが、値が有効な日付として認識されません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   有効な変換を検索します。
