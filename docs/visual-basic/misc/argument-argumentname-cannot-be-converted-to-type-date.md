# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-type-39date39"></a>引数 &#39;&lt;argumentname&gt;&#39; の種類 &#39; 日付 &#39; に変換できません
変数を型を `Date`に変更する変換が試みられましたが、値が有効な日付として認識されません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   有効な変換を検索します。
