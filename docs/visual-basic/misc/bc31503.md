# &#39;&lt;typename&gt;&#39; はクラスでないため、属性として使用できません
クラスではない型を属性として使用しようとしました。  
  
 **エラー ID:** BC31503  
  
### このエラーを解決するには  
  
1.  `System.Attribute` から派生するクラスとしてカスタム属性を定義します。
