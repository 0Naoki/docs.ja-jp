# <a name="referenced-39ltmembernamegt39-has-a-value-of-39nothing39"></a>参照されている&#39; &lt;membername&gt; &#39;の値を持つ&#39;何も行われません&#39;
参照されている項目に値が割り当てられていないか、値 `Nothing`が割り当てられました。  
  
 **エラー ID:** BC30759  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   項目名のスペルが正しいこと、および正しく設定されていることを確認します。
