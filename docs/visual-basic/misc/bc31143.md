<span data-ttu-id="2143c-103">このエラーは、メソッドとデリゲート間で必要な変換ができない場合に生じます。</span><span class="sxs-lookup"><span data-stu-id="2143c-103">This error occurs when a conversion is required between a method and a delegate that is not possible.</span></span> <span data-ttu-id="2143c-104">エラーの原因としては、パラメーターの間の変換か、戻り値の変換 (メソッドとデリゲートが関数の場合) が考えられます。</span><span class="sxs-lookup"><span data-stu-id="2143c-104">The cause of the error can be conversion between parameters or, when the method and delegate are functions, conversion in the return values.</span></span>  
  
 <span data-ttu-id="2143c-105">次のコードは、こうした障害が発生する変換を示しています。</span><span class="sxs-lookup"><span data-stu-id="2143c-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="2143c-106">デリゲートは `FunDel`です。</span><span class="sxs-lookup"><span data-stu-id="2143c-106">The delegate is `FunDel`.</span></span>  
  
```vb

```  
  
 <span data-ttu-id="2143c-107">次の関数は、このエラーを生じさせる原因が `FunDel` とはそれぞれ異なります。</span><span class="sxs-lookup"><span data-stu-id="2143c-107">The following functions each differ from `FunDel` in a way that will cause this error.</span></span>

```vb  
End Function  
  
Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date  
End Function  
```  
  
 <span data-ttu-id="2143c-108">次のそれぞれの代入ステートメントでもエラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="2143c-108">Each of the following assignment statements causes the error.</span></span>  
  
```vb

    ' are not compatible.  
    'Dim d1 As FunDel = AddressOf ExampleMethod1  
  
    ' The return types of FunDel and ExampleMethod2, Integer and Date,  
    ' are not compatible.  
    'Dim d2 As FunDel = AddressOf ExampleMethod2

```
