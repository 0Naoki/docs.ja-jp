# <a name="39char39-values-cannot-be-converted-to-39lttypenamegt39"></a>&#39;です。Char &#39;値に変換できません (& a) #39 です。&lt;typename&gt;&#39;です。
'Char' 値に変換できません '\<typename >' です。 文字を Unicode 値として扱うには Microsoft.VisualBasic.AscW を、数字として扱うには Microsoft.VisualBasic.Val を使用します。  
  
 式が `Char` 値を `String` または `Object`以外のデータ型に変換しようとしました。  
  
 **エラー ID:** BC32006  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `AscW` 値を Unicode 文字コードとして扱うには `Char` 関数を、数字として扱うには `Val` 関数を使用します。
