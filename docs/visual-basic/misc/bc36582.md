# <a name="too-many-arguments-to-extension-method-methodname-defined-in-typename"></a>拡張メソッドに引数が多すぎます '\<methodname >' で定義されている'\<typename >'
拡張メソッドの呼び出しで指定した引数の数が、メソッドについて定義されている引数の数を超えています。  
  
 **エラー ID:** BC36582  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   拡張メソッドの定義を調べ、余分な引数を呼び出しから削除します。
