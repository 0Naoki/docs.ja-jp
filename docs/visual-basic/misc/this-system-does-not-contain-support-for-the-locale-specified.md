# このシステムには、指定されたロケールのサポートは含まれていません。
`VbStrConv` 列挙型を使用しようとしましたが、システムは指定したロケールをサポートしていません。  
  
### このエラーを解決するには  
  
1.  ロケールをサポートするようにコンピューターをアップグレードするか、ロケールをサポートするコンピューターでアプリケーションを実行します。
