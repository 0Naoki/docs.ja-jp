# &#39;&lt;name&gt;&#39; は、イベントの基になるデリゲート型 &#39;&lt;delegatetype&gt;&#39; を公開できません。&lt;specifier&gt; &#39;&lt;type&gt;&#39; 経由でプロジェクトの外側に実装しています。
このコードにより、パブリック クラスの外部でプライベート型が公開されます。  
  
 **エラー ID:** BC30915  
  
### このエラーを解決するには  
  
-   型 `Public` を宣言します。  
  
     または  
  
-   別の型を使用します。
