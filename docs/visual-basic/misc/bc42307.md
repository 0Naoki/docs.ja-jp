# <a name="xml-comment-parameter-ltparametergt-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML コメント パラメーター '&lt;パラメーター&gt;' に対応するパラメーターと一致しません&lt;キーワード&gt;ステートメント
XML コメント パラメーター\<パラメーター > に対応するパラメーターと一致しません\<キーワード > ステートメント。 XML コメントは無視されます。  
  
 XML コメント タグが、メソッドの一部の (すべてのではない) パラメーターに対して指定されています。  
  
 **エラー ID:** BC42307  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML コメント タグを、メソッドのすべてのパラメーターに指定します。
