# <a name="xml-comment-parameter-39ltparametergt39-does-not-match-a-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML コメント パラメーター &#39;&lt;パラメーター&gt;&#39; で、対応するパラメーターと一致しません&lt;キーワード&gt;ステートメント
XML コメント パラメーター\<パラメーター > は、対応するパラメーターと一致しません\<キーワード > ステートメントです。 XML コメントは無視されます。  
  
 XML コメント タグが、メソッドの一部の (すべてのではない) パラメーターに対して指定されています。  
  
 **エラー ID:** BC42307  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML コメント タグを、メソッドのすべてのパラメーターに指定します。
