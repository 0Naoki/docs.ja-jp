# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>メソッドは、両方を含めることはできません、&#39;再試行&#39;ステートメントおよび&#39;On Error&#39;または&#39;再開&#39;ステートメント
`Try` ステートメントと `On Error` または `Resume`を組み合わせました。  
  
 **エラー ID:** BC30544  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `On Error` または `Resume`を削除します。
