# <a name="39lttypename1gt39-cannot-be-converted-to-39lttypename2gt39"></a>&#39;です。&lt;typename1&gt;&#39; に変換できません (& m); #39&lt;typename2&gt;&#39;です。
2 つの互換性のない型の間で型変換が試行されました。  
  
 **エラー ID:** BC30741  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   キャストされている型を確認します。
