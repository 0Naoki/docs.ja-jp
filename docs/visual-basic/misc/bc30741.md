# <a name="lttypename1gt-cannot-be-converted-to-lttypename2gt"></a>'&lt;typename1&gt;'に変換できません'&lt;typename2&gt;'
2 つの互換性のない型の間で型変換が試行されました。  
  
 **エラー ID:** BC30741  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   キャストされている型を確認します。
