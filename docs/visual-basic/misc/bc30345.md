# &#39;&lt;method1&gt;&#39; と &#39;&lt;method2&gt;&#39; の違いは、パラメーターが &#39;ByRef&#39; と &#39;ByVal&#39; のどちらに宣言されているかのみのため、お互いにオーバーロードすることはできません
メソッドを、`ByRef` または `ByVal` として宣言されたパラメーターだけが異なるメソッドでオーバーロードしようとしました。  
  
 **エラー ID:** BC30345  
  
### このエラーを解決するには  
  
-   `ByRef` または `ByVal` のパラメーター名以外の部分も異なるメソッドでオーバーロードします。
