# <a name="nothing-cannot-be-evaluated"></a>'Nothing' を評価できません
`Nothing` キーワードがデバッグ ウィンドウで使用されました。  
  
 **エラー ID:** BC30701  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- デバッグ中に `Nothing` を使用しないでください。
