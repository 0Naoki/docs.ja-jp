# &#39;&lt;projectname&gt;&#39; は、直接的または間接的に &#39;&lt;projectname&gt;&#39; を参照しているため、プロジェクト &#39;&lt;projectname&gt;&#39; でプロジェクト &#39;&lt;projectname&gt;&#39; を参照できません。
プロジェクトには、2 番目のプロジェクトへの参照が含まれています。 さらに、2 番目のプロジェクトには、それが参照されているプロジェクトへの参照が含まれています。  
  
 **エラー ID:** BC30761  
  
### このエラーを解決するには  
  
-   いずれかの参照を削除します。
