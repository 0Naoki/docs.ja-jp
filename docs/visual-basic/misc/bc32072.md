```  
Public Interface iFace2(Of t)  
    Sub testSub(ByVal arg As t)  
End Interface

    Implements iFace1, iFace2(Of String)  
End Class  
```  
  
 <span data-ttu-id="a5736-105">`iFace2` が `String`を使って作成されるため、 `testClass` はシグネチャの同じ 2 つのバージョンの `testSub` を実装する必要があります。</span><span class="sxs-lookup"><span data-stu-id="a5736-105">Because `iFace2` is constructed using `String`, `testClass` must implement two versions of `testSub` with identical signatures.</span></span> <span data-ttu-id="a5736-106">この場合、どちらのバージョンにアクセスするかがあいまいになります。</span><span class="sxs-lookup"><span data-stu-id="a5736-106">Doing so would produce an ambiguity about which version to access.</span></span>
