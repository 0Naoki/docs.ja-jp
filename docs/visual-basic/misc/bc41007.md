# 属性 &#39;Conditional&#39; は &#39;Sub&#39; 宣言でのみ有効です
<xref:System.Diagnostics.ConditionalAttribute> 属性が `Sub` プロシージャ以外のメソッドの宣言に適用されています。  
  
 **エラー ID:** BC41007  
  
### このエラーを解決するには  
  
1.  無効な宣言から <xref:System.Diagnostics.ConditionalAttribute> を削除します。
