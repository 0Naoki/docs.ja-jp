# <a name="39end39-statements-are-not-valid-in-the-immediate-window"></a>'End' ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Stop` ステートメントおよび `End` ステートメントは実行を中断するため、デバッグ コンテキストでは使用できません。  
  
 **エラー ID:** BC30123  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `End` [イミディエイト] `Stop` ウィンドウで **または** のステートメントを実行しないようにします。
