# <a name="readonly-attribute-property-ltpropertyfieldgt-cannot-be-the-target-of-an-assignment"></a>'ReadOnly' 属性プロパティ '&lt;propertyfield&gt;' を代入式のターゲットにすることはできません
属性の `ReadOnly` プロパティに値を代入しようとしました。  
  
 **エラー ID:** BC31501  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  プロパティの代入ステートメントを削除します。  
  
2.  独自に作成したプロパティを使用している場合は、属性プロパティから `ReadOnly` 修飾子または `Shared` 修飾子を削除します。
