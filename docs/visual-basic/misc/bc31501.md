# &#39;ReadOnly&#39; 属性プロパティ &#39;&lt;propertyfield&gt;&#39; を代入のターゲットにすることはできません
属性の `ReadOnly` プロパティに値を代入しようとしました。  
  
 **エラー ID:** BC31501  
  
### このエラーを解決するには  
  
1.  プロパティの代入ステートメントを削除します。  
  
2.  独自に作成したプロパティを使用している場合は、属性プロパティから `ReadOnly` 修飾子または `Shared` 修飾子を削除します。
