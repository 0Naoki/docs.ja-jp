# <a name="39ltpropertyname1gt39-cannot-override-39ltpropertyname2gt39-because-they-differ-by-39readonly39-or-39writeonly39"></a>&#39;です。&lt;propertyname1&gt;&#39; できません上書き &#39;&lt;propertyname2&gt;&#39;が異なるためです。 (& a) #39 です。読み取り専用 &#39;または &#39;です。WriteOnly &#39;
1 番目のプロパティを、 `ReadOnly` か `WriteOnly`かの違いしかない 2 番目のプロパティでオーバーライドしようとしました。  
  
 **エラー ID:** BC30362  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロパティどうしの間に `ReadOnly` と `WriteOnly`以外の違いがあるようにします。
