# <a name="ltpropertyname1gt-cannot-override-ltpropertyname2gt-because-they-differ-by-readonly-or-writeonly"></a>'&lt;propertyname1&gt;オーバーライドできません'&lt;propertyname2&gt;' 'ReadOnly' または 'WriteOnly' が異なるため、
1 番目のプロパティを、 `ReadOnly` か `WriteOnly`かの違いしかない 2 番目のプロパティでオーバーライドしようとしました。  
  
 **エラー ID:** BC30362  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロパティどうしの間に `ReadOnly` と `WriteOnly`以外の違いがあるようにします。
