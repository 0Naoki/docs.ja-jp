# <a name="xml-comment-tag-include-must-have-a-ltattributegt-attribute"></a>XML のコメント タグ 'include' にする必要がありますが、'&lt;属性&gt;' 属性
XML のコメント タグ 'include' にする必要がありますが、\<属性 > 属性。 XML コメントは無視されます。  
  
 `file` include `path` タグに渡される `<`または`>` 属性がないか、不完全でした。  
  
 **エラー ID:** BC42310  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   追加または完了に必要な属性、\<含める > タグです。
