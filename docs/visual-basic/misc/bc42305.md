# <a name="xml-comment-tag-39lttaggt39-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>XML コメント タグ&#39;&lt;タグ&gt;&#39;同じ XML コメント ブロックに 2 回以上同じ属性を持つが表示されます
XML コメント タグ\<タグ > 同じ XML コメント ブロックに 2 回以上同じ属性を持つが表示されます。 XML コメントは無視されます。  
  
 XML コメントに、同じ属性を持つ複数のタグが含まれています。  
  
 **エラー ID:** BC42305  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   重複しているタグを削除します。
