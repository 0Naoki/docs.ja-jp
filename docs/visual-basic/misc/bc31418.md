# <a name="39handles39-in-modules-must-specify-a-39withevents39-variable-qualified-with-a-single-identifier"></a>&#39;です。ハンドル &#39;モジュールは指定する必要があります、&#39;です。WithEvents &#39;単一の識別子で限定された変数
イベント ハンドラーを指定するには、 `Handles` キーワードで宣言されたオブジェクト変数を `WithEvents` ステートメントに指定する必要があります。  
  
 **エラー ID:** BC31418  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `WithEvents` 修飾子を使用して、 `Handles` ステートメントで使用する変数を宣言します。
