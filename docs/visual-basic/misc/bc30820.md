# <a name="39ltlsetstmtgt39-is-not-declared"></a>&#39;です。&lt;lsetstmt&gt;&#39; が宣言されていません
'\<lsetstmt >' は宣言されていません。 LSet ステートメントは現在サポートされていません。Microsoft.VisualBasic.LSet を使用してください。  
  
 組み込まれていたいくつかの関数[!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]以前のバージョンに移動されました、<xref:Microsoft.VisualBasic?displayProperty=nameWithType>名前空間。 これにより、これらの機能をより一般的にすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30820  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `LSet` 名前空間の `Microsoft.VisualBasic` 関数を代わりに使用します。
