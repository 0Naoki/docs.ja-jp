# <a name="enum-ltenumnamegt-must-contain-at-least-one-member"></a>列挙型 '&lt;enumname&gt;' には少なくとも 1 つのメンバーを含める必要があります
空の列挙型があります。 少なくとも 1 つのメンバーを `Enum` ステートメントと `End Enum` ステートメントの間で宣言する必要があります。  
  
 **エラー ID:** BC30280  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  列挙体にメンバーの宣言を追加します。
