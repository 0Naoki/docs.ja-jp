# <a name="object-doesn39t-support-this-property-or-method-visual-basic"></a>オブジェクトではありません &#39; このプロパティまたはメソッド (Visual Basic) をサポートしません。
このオートメーション オブジェクトには存在しないメソッドまたはプロパティを指定しました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   オブジェクトのドキュメントでオブジェクトの詳細を参照し、プロパティとメソッドのスペルを確認します。
