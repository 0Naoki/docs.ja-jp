# 型 &#39;&lt;type1&gt;&#39; の開始値、型 &#39;&lt;type2&gt;&#39; の制限値、または型 &#39;&lt;type3&gt;&#39; のステップ値を共通の型に変換できません
指定された `For...Next` コントロールの値は、異なる型であるため、共通の型に変換できないので無効です。  
  
### このエラーを解決するには  
  
1.  すべてを共通の型に変換できるように、値の 1 つ以上の型を変更します。
