# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>型の開始値に変換できません&#39; &lt;type1&gt;&#39;の値を制限する&#39; &lt;type2&gt; &#39;ステップの値と&#39; &lt;type3&gt; &#39;に、共通の型
指定された `For...Next` コントロールの値は、異なる型であるため、共通の型に変換できないので無効です。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  すべてを共通の型に変換できるように、値の 1 つ以上の型を変更します。
