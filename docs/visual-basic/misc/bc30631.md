# <a name="39get39-statement-must-end-with-a-matching-39end-get39"></a>&#39;です。Get &#39;ステートメントの終わりに対応する &#39; にする必要があります。Get &#39; を終了します。
`Get` プロパティ プロシージャは、必ず `End Get` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30631  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Get` プロパティ プロシージャが `End Get` コンストラクトで終了していることを確認します。
