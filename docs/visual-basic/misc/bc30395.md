# <a name="ltspecifiergt-is-not-valid-on-a-structure-declaration"></a>'&lt;指定子&gt;' 構造体の宣言では無効です
`Dim` または `Override`などの指定子を使用していますが、 `Structure` 宣言では使用できません。  
  
 **エラー ID:** BC30395  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
