# <a name="39ltspecifiergt39-is-not-valid-on-a-structure-declaration"></a>&#39;です。&lt;指定子&gt;&#39; は無効です構造体の宣言。
`Dim` または `Override`などの指定子を使用していますが、 `Structure` 宣言では使用できません。  
  
 **エラー ID:** BC30395  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
