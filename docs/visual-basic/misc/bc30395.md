# <a name="specifier-is-not-valid-on-a-structure-declaration"></a>'\<指定子 >'、Structure 宣言では無効です
`Dim` または `Override`などの指定子を使用していますが、 `Structure` 宣言では使用できません。  
  
 **エラー ID:** BC30395  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
