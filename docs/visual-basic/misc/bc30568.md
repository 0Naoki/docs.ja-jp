# <a name="array-initializer-has-ltnumbergt-too-many-elements"></a>配列初期化子が&lt;数&gt;要素が多すぎます
配列初期化子に含まれている要素が多すぎます。  
  
 **エラー ID:** BC30568  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `ReDim` を使って配列のサイズを変更します。
