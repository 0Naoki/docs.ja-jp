# 配列初期化子に &lt;number&gt; 個の要素が余分に含まれています
配列初期化子に含まれている要素が多すぎます。  
  
 **エラー ID:** BC30568  
  
### このエラーを解決するには  
  
-   `ReDim` を使って配列のサイズを変更します。
