# &#39;&lt;name1&gt;&#39; は、&#39;&lt;name2&gt;&#39; で定義された同じ名前によって &lt;type&gt; と競合しています
型の名前と競合する名前が検出されました。  
  
 **エラー ID:** BC31073  
  
### このエラーを解決するには  
  
1.  一意の名前を宣言します。  
  
2.  競合しているメンバー名を角かっこで囲みます。
