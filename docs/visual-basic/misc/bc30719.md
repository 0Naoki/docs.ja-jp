# &#39;Case&#39; ステートメントは、イミディエイト ウィンドウでは使用できません
`Case` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30719  
  
### このエラーを解決するには  
  
-   **イミディエイト** ウィンドウから `Case` ステートメントを削除します。
