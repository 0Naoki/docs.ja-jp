# <a name="paramarray-parameters-must-be-declared-39byval39"></a>ParamArray パラメーターを宣言する必要があります (& a) #39 です。ByVal &#39;
`ParamArray` パラメーターは `ByRef` 修飾子を使用できません。  
  
 **エラー ID:** BC30667  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `ParamArray` 修飾子を使用して `ByVal` パラメーターを宣言します。
