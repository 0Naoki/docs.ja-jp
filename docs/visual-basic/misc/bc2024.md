# <a name="the-default-assembly-reference-ltreferencegt-has-already-been-automatically-added-ignored"></a>既定のアセンブリ参照 '&lt;参照&gt;' は既に自動的に追加されています無視されます。
既定のアセンブリ参照を追加しようとしましたが、アセンブリが既に追加されているため、無視されます。  
  
 **エラー ID:** BC2024  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分なコードを削除します。
