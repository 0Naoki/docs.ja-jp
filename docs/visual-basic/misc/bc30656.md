# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>フィールド '&lt;fieldname&gt;' はサポートされていない型
フィールドが、表現できない型として宣言されて[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]します。  
  
 **エラー ID:** BC30656  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  サポートされている型としてフィールドを定義します。  
  
2.  フィールド内のデータを記述する新しい型を作成します。
