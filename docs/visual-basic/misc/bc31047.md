# Protected 型はクラスの内部でのみ宣言できます
モジュール内にある型が `Protected` として宣言されています。  
  
 **エラー ID:** BC31047  
  
### このエラーを解決するには  
  
1.  クラス内で型を宣言します。  
  
2.  `Protected` 修飾子を削除します。
