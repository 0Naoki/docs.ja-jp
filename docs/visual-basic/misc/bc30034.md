# <a name="bracketed-identifier-is-missing-closing-3939"></a>かっこで囲まれた識別子は、閉じる &#39; がありません]&#39;です。
エスケープされた名前内の角かっこは、対になっている必要があります。  
  
 **エラー ID:** BC30034  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   エスケープされた名前の最後に右角かっこを配置します。
