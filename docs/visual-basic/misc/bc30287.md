# <a name="-expected"></a>'.' が必要です
コードには、感嘆符 ( `Handles` ) が含まれている`!`句があります。  
  
 **エラー ID:** BC30287  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. `Handles` 句がオブジェクト内のイベントを参照している場合は、ピリオド (`.`) を使用してオブジェクトとイベントを分離してください。
