# &#39;.&#39; が必要です
コードには、感嘆符 \(`!`\) が含まれている `Handles` 句があります。  
  
 **エラー ID:** BC30287  
  
### このエラーを解決するには  
  
1.  `Handles` 句がオブジェクト内のイベントを参照している場合は、ピリオド \(`.`\) を使用してオブジェクトとイベントを分離してください。
