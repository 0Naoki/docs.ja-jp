# <a name="39end-sub39-must-be-the-first-statement-on-a-line"></a>&#39;です。End Sub &#39;行の最初のステートメントをする必要があります。
`End Sub` ステートメントがコロン (:) ステートメント区切り文字の後に続いています。 `End Sub` はそのソース行でただ 1 つのステートメントである必要があります。  
  
 **エラー ID:** BC32031  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   複数のステートメントを別々の行に分けます。
