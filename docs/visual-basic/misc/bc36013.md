# <a name="39on-error39-statements-are-not-valid-within-39using39-statements"></a>&#39;エラー&#39;ステートメントは内で有効ではありません&#39;Using&#39;ステートメント。
`On Error` ステートメントが `Using` ステートメント内で使用されていますが、このコンテキストには使用できません。  
  
 **エラー ID:** BC36013  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Try…Catch` ブロックなどの構造化エラー処理を `On Error` ステートメントの代わりに使用します。
