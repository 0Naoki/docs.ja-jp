# <a name="class-39ltclassnamegt39-does-not-implement-the-systemcollectionsicollection-interface"></a>クラス &#39;&lt;classname&gt;&#39; System.Collections.ICollection インターフェイスを実装しません
クラスに、すべてのコレクションのサイズ、列挙子、および同期メソッドを定義する `System.Collections.ICollection` インターフェイスが実装されていません。 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]を再インストールする必要がある場合があります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]を再インストールします。
