# クラス &#39;&lt;classname&gt;&#39; は System.Collections.ICollection インターフェイスを実装していません。
クラスに、すべてのコレクションのサイズ、列挙子、および同期メソッドを定義する `System.Collections.ICollection` インターフェイスが実装されていません。[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] を再インストールする必要がある場合があります。  
  
### このエラーを解決するには  
  
1.  [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] を再インストールします。
