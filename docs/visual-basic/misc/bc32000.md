# <a name="local-variable-ltvariablenamegt-cannot-be-referred-to-before-it-is-declared"></a>ローカル変数 '&lt;variablename&gt;' は宣言されていない参照できません
プロシージャ内の変数が、宣言される前に使用されています。  
  
 **エラー ID:** BC32000  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   宣言を、変数が最初に参照される前に移動します。
