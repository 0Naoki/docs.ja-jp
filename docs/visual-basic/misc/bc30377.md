# <a name="method-ltmethodnamegt-is-already-declared-in-interface-ltinterfacenamegt"></a>メソッド '&lt;methodname&gt;'は、既にインターフェイスで宣言された'&lt;interfacename&gt;'
重複してインターフェイス メソッドの 2 回目の宣言を行いました。  
  
 **エラー ID:** BC30377  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   重複している宣言を削除します。
