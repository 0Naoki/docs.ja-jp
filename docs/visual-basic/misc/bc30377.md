# <a name="method-methodname-is-already-declared-in-interface-interfacename"></a>メソッド '\<methodname >' はインターフェイスで既に宣言されて'\<interfacename >'
重複してインターフェイス メソッドの 2 回目の宣言を行いました。  
  
 **エラー ID:** BC30377  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   重複している宣言を削除します。
