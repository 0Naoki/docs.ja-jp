# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-declared-39mustinherit39"></a>&#39;&lt;typename&gt; &#39;宣言されているため、属性として使用することはできません&#39;MustInherit&#39;
カスタム属性クラスは `MustInherit`として宣言できません。  
  
 **エラー ID:** BC31506  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   カスタム属性から `MustInherit` 修飾子を削除します。
