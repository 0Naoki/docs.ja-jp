# <a name="class-classname-cannot-be-indexed-because-it-has-no-default-property"></a>クラス\<classname >' 既定のプロパティが存在しないインデックスを作成できません
クラスの既定のプロパティが指定されていません。  
  
 **エラー ID:** BC30367  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   1 つのプロパティの宣言を `Default` キーワードから始めることにより、そのプロパティを既定のプロパティとして指定します。
