# <a name="ltrsetstmtgt-is-not-declared"></a>'&lt;rsetstmt&gt;' が宣言されていません
'\<rsetstmt >' が宣言されていません。 RSet ステートメントは現在サポートされていません。Microsoft.VisualBasic.RSet を使用してください。  
  
 以前のバージョンの Visual Basic に組み込まれていたいくつかの関数に移動されました、<xref:Microsoft.VisualBasic?displayProperty=nameWithType>名前空間。 これにより、これらの機能をより一般的にすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30821  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `RSet` 名前空間の `Microsoft.VisualBasic` 関数を代わりに使用します。
