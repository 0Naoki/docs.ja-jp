# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;rsetstmt&gt; &#39;が宣言されていません
'\<rsetstmt >' は宣言されていません。 RSet ステートメントは現在サポートされていません。Microsoft.VisualBasic.RSet を使用してください。  
  
 以前のバージョンでは、Visual Basic に組み込まれていたいくつかの関数に移動されている、<xref:Microsoft.VisualBasic?displayProperty=nameWithType>名前空間。 これにより、これらの機能をより一般的にすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30821  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `RSet` 名前空間にある `Microsoft.VisualBasic` 関数を代わりに使用します。
