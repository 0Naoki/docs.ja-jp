# <a name="expected-beginning-lt-for-an-xml-tag"></a>先頭が必要です '&lt;' の XML タグ
XML リテラルの XML タグの先頭に必要な '<' 文字がありません。  
  
 **エラー ID:** BC31165  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML リテラルの XML タグの先頭に '<' 文字を追加します。
