# <a name="39option-explicit39-can-be-followed-only-by-39on39-or-39off39"></a>&#39;オプションの Explicit&#39;が続きますのみ&#39;で&#39;または&#39;オフ&#39;
`Option Explicit` ステートメントは `On` または `Off`のどちらかを指定する必要があります。 その他のすべての値は無効です。  
  
 **エラー ID:** BC30640  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Option Explicit On` か `Option Explicit Off`のどちらかを指定します。
