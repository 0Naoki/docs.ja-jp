# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;です。MyBase &#39;メソッド &#39; では使用できません。&lt;methodname&gt;&#39;宣言されているためです。 (& a) #39 です。MustOverride &#39;
`MyBase` が宣言されているメソッドと共に `MustOverride`を使用しようとしました。  
  
 **エラー ID:** BC30399  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `MustOverride` 宣言を削除します。
