# ロケール ID &#39;&lt;name&gt;&#39; はこのシステムでサポートされていません
`VbStrConv` を使用しようとしましたが、システムは指定したロケールをサポートしていません。  
  
### このエラーを解決するには  
  
1.  ロケールをサポートするためにコンピューターをアップグレードするか、または指定したロケールをサポートするコンピューターでアプリケーションを実行します。
