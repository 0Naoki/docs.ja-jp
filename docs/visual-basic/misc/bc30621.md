# <a name="39end-structure39-must-be-preceded-by-a-matching-39structure39"></a>&#39;End Structure&#39;前に、対応する必要がある&#39;構造体&#39;
`End Structure` ステートメントは、 `Structure` ステートメントで始まる構造体ブロックの最後にのみ使用できます。  
  
 **エラー ID:** BC30621  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   現在の構造体ブロックが `Structure` ステートメントで始まることを確認します。
