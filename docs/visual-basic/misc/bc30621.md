# &#39;End Structure&#39; の前には、対応する &#39;Structure&#39; を指定しなければなりません
`End Structure` ステートメントは、`Structure` ステートメントで始まる構造体ブロックの最後にのみ使用できます。  
  
 **エラー ID:** BC30621  
  
### このエラーを解決するには  
  
-   現在の構造体ブロックが `Structure` ステートメントで始まることを確認します。
