# <a name="user-interrupt-occurred"></a>ユーザーによる割り込みが発生しました
ユーザーが Ctrl + Break キーを押したか、別の中断キーを押しました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   操作をやり直します。
