# <a name="file-information-cannot-be-queried-while-open-for-writing"></a>書き込みのためにファイルが開いている間は、ファイル情報を照会することはできません
書き込み用に現在開いているファイルに関する情報を取得しようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイルを閉じて、操作をやり直します。
