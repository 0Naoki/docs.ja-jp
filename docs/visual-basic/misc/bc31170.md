# 文字 &#39;character&#39; (0x&lt;hexadecimal&gt;) は、XML 名では使用できません。
サポートされていない文字が、XML リテラルまたは XML 軸プロパティの名前に含まれています。  
  
 **エラー ID:** BC31170  
  
### このエラーを解決するには  
  
-   XML リテラルまたは XML 軸プロパティの名前から、サポートされていない文字を削除します。
