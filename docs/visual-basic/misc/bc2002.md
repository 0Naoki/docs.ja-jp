# <a name="source-file-filename-specified-multiple-times"></a>ソース ファイル\<filename > 複数回指定されました
ファイルが複数回指定されました。  
  
 **エラー ID:** BC2002  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分なファイル指定子を削除します。
