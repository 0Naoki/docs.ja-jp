# <a name="properties-cannot-be-declared-ltmodifiernamegt"></a>プロパティを宣言できません '&lt;modifiername&gt;'。
`Property` 宣言で使用されている構文にエラーがあります。 通常、このエラーは、 `WithEvents`などの指定されたキーワードを `Property` 宣言で使用できないことを意味します。  
  
 **エラー ID:** BC30639  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Property` 宣言の構文を確認します。
