# <a name="properties-cannot-be-declared-39ltmodifiernamegt39"></a>プロパティを宣言することはできません (& a) #39 です。&lt;modifiername&gt;&#39;です。
`Property` 宣言で使用されている構文にエラーがあります。 通常、このエラーは、 `WithEvents`などの指定されたキーワードを `Property` 宣言で使用できないことを意味します。  
  
 **エラー ID:** BC30639  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Property` 宣言の構文を確認します。
