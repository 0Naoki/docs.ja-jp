# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>警告番号&#39;&lt;数&gt;&#39;オプション&#39; &lt;optionname&gt; &#39;は構成できないか、有効ではありません
警告 ID が無効です。 ID は 0 より大きい整数値である必要があります。  
  
 **エラー ID:** BC2026  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   表示された警告 ID が有効かどうかを確認します。
