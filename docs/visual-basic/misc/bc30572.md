# エイリアス &#39;&lt;namespacename&gt;&#39; は、既に宣言されています
`Imports` を使用して、既に宣言されている名前空間をインポートしようとしました。  
  
 **エラー ID:** BC30572  
  
### このエラーを解決するには  
  
-   インポートしようとしている名前空間を確認します。
