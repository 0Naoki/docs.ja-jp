# <a name="alias-namespacename-is-already-declared"></a>エイリアスの\<namespacename >' は既に宣言されています
`Imports` を使用して、既に宣言されている名前空間をインポートしようとしました。  
  
 **エラー ID:** BC30572  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- インポートしようとしている名前空間を確認します。
