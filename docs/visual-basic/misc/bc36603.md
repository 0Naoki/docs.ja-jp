<span data-ttu-id="8f489-103">ほとんどの式は、式ツリーに変換できますが、多次元配列の場合は変換できません。</span><span class="sxs-lookup"><span data-stu-id="8f489-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="8f489-104">たとえば、次のコードでは、このエラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="8f489-104">For example, the following code causes this error:</span></span>  
  
```vb

    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}

End Module
```  
