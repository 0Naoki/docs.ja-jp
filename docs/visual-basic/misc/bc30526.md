# <a name="property-39ltpropertynamegt39-is-39readonly39"></a>プロパティ&#39; &lt;propertyname&gt; &#39;は&#39;読み取り専用&#39;
`ReadOnly`で宣言されているプロパティに値を代入しようとしました。  
  
 **エラー ID:** BC30526  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロパティ宣言から `ReadOnly` 指定子を削除します。
