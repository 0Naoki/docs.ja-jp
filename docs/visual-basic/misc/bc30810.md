# メソッドを &#39;Static&#39; として宣言することはできません
`Static` 修飾子は、プロシージャ全体に適用できません。  
  
 **エラー ID:** BC30810  
  
### このエラーを解決するには  
  
-   `Static` 修飾子を使用して、プロシージャ内の個々の項目を static として宣言します。
