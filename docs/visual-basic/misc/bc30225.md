# <a name="39option-ltspecifiergt39-statement-can-only-appear-once-per-file"></a>&#39;オプション&lt;指定子&gt;&#39;ステートメントできますのみ 1 回ファイルあたり
`Option Compare`ステートメント、 `Option Explicit`ステートメント、 `Option Strict` ステートメントは、同じソース ファイル内でそれぞれ一度だけ使用できます。  
  
 **エラー ID:** BC30225  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   重複している `Option` ステートメントを削除します。
