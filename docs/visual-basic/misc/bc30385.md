# &#39;&lt;modifier&gt;&#39; は、デリゲート宣言では有効ではありません
デリゲート宣言に対して `Shared` などの修飾子を使用しようとしました。 デリゲートは、オブジェクト メソッドの仕様を渡されるコンストラクターを定義することで、他のオブジェクトのメソッドを呼び出すために使用するオブジェクトです。 デリゲート宣言に対して修飾子を指定することはできません。  
  
 **エラー ID:** BC30385  
  
### このエラーを解決するには  
  
1.  修飾子を削除します。
