# <a name="39ltmodifiergt39-is-not-valid-on-a-local-variable-declaration"></a>&#39;です。&lt;修飾子&gt;&#39; は、ローカル変数宣言では無効
プロシージャ内の `Dim` ステートメントに、 `Overloads`のような正しくないキーワードが含まれています。  
  
 **エラー ID:** BC30247  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Dim` ステートメントから無効なキーワードを削除します。
