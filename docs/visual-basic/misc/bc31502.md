# リソース名 &#39;&lt;resourcename&gt;&#39; を 2 回以上使用することはできません
ダイナミックリンク ライブラリまたは実行可能ファイルで指定されたリソースの名前が複数回使用されました。  
  
 **エラー ID:** BC31502  
  
### このエラーを解決するには  
  
-   リソースの名前を一意に付けます。
