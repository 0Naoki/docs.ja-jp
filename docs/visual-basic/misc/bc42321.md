# XML の形式が正しくありません。 ファイル &#39;&lt;commentfile&gt;&#39; を含めることはできません。
XML コメントのいずれかの `<include>` タグが、存在しないファイルまたは XML の形式が正しくないファイルを参照しています。  
  
 **エラー ID:** BC42321  
  
### このエラーを解決するには  
  
1.  ファイルが存在することを確認します。  
  
2.  ファイル内の XML が有効であることを確認します。
