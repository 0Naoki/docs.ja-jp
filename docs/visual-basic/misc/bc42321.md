# <a name="badly-formed-xml-file-39ltcommentfilegt39-cannot-be-included"></a>XML の形式が正しくありません。 ファイル &#39;&lt;commentfile&gt;&#39; 含まれることはできません
XML コメントのいずれかの `<include>` タグが、存在しないファイルまたは XML の形式が正しくないファイルを参照しています。  
  
 **エラー ID:** BC42321  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ファイルが存在することを確認します。  
  
2.  ファイル内の XML が正しいことを確認します。
