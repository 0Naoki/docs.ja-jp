# このシステムには、繁体字中国語ロケールのサポートは含まれていません
`VbStrConv.TraditionalChinese` 列挙体メンバーの使用が試行されましたが、システムは繁体字中国語ロケールをサポートしていません。  
  
### このエラーを解決するには  
  
1.  繁体字中国語ロケールをサポートするためにコンピューターをアップグレードするか、またはロケールをサポートするコンピューターでアプリケーションを実行します。
