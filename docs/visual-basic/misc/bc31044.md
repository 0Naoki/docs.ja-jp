# &#39;As&#39; 句を伴って宣言されたイベントは、デリゲート型を指定しなければなりません。
デリゲートではない型が指定されています。  
  
 **エラー ID:** BC31044  
  
### このエラーを解決するには  
  
1.  `As` 句を削除します。  
  
2.  `As` 句を持つデリゲートを指定します。
