# <a name="events-declared-with-an-39as39-clause-must-have-a-delegate-type"></a>'As' 句を使用して宣言されたイベントにデリゲート型を持つ必要があります。
デリゲートではない型が指定されています。  
  
 **エラー ID:** BC31044  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `As` 句を削除します。  
  
2.  `As` 句を持つデリゲートを指定します。
