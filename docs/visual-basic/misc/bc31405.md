# 省略可能なパラメーターが、構造体型を持つことはできません
省略可能なパラメーターを構造体型にすることはできません。  
  
 **エラー ID:** BC31405  
  
### このエラーを解決するには  
  
1.  必要に応じて構造体型パラメーターを宣言します。  
  
2.  構造体の代わりに静的クラス メンバーを宣言します。
