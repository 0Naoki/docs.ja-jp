# <a name="39inherits39-can-appear-only-once-within-a-39class39-statement-and-can-only-specify-one-class"></a>&#39; 継承 &#39;です。内で 1 回だけ出現できます、&#39;です。クラス &#39;ステートメントと、1 つのクラスにのみ指定できます
同じクラスに複数の `Inherits` ステートメントを記述しているか、または `Inherits` ステートメントに複数のクラスを指定しています。 クラスは、1 つの基底クラスのみから継承できます。  
  
 **エラー ID:** BC30121  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分な `Inherits` ステートメントを削除し、残りの `Inherits` ステートメントに基底クラスを 1 つのみ指定していることを確認します。
