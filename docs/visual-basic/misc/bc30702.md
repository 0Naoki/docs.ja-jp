# &#39;&lt;dimensionnumber&gt;&#39; 次元のインデックス &#39;&lt;indexnumber&gt;&#39; が範囲外です
定義されている範囲を超えているインデックスがあります。  
  
 **エラー ID:** BC30702  
  
### このエラーを解決するには  
  
-   配列の境界を越えないようにプロシージャのロジックを変更します。  
  
-   配列を再定義して値の範囲を広げます。
