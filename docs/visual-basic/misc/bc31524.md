# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39;です。System.Runtime.InteropServices.DllImportAttribute &#39;適用できません、&#39;です。Get &#39;または &#39;です。セット &#39;
`DllImportAttribute` 属性が `Get` または `Set` プロパティ プロシージャに適用されました。  
  
 **エラー ID:** BC31524  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `DllImportAttribute` および `Get` プロパティ プロシージャから `Set` を削除します。
