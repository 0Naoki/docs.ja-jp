# <a name="properties-declared-39writeonly39-cannot-have-a-39get39"></a>宣言されたプロパティ &#39;WriteOnly &#39;できません、&#39;です。Get &#39;
`Get` プロシージャはプロパティの値を読み取ります。 `WriteOnly` プロパティを読み取ることはできません。  
  
 **エラー ID:** BC30023  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `WriteOnly` キーワードを `Property` ステートメントから削除するか、 `Get` プロシージャをプロパティ本体から削除します。
