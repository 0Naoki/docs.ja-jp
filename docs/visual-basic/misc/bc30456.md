# &#39;&lt;name&gt;&#39; は &#39;&lt;classname&gt;&#39; のメンバーではありません。
指定したメンバーは、クラスのメンバーではありません。  
  
 **エラー ID:** BC30456  
  
### このエラーを解決するには  
  
1.  メンバーの名前が正確であることを確認します。  
  
2.  クラスの実際のメンバーを使用します。
