# &#39;Stop&#39; ステートメントは、イミディエイト ウィンドウでは使用できません。
`Stop` ステートメントおよび `End` ステートメントは実行を中断するため、デバッグ コンテキストでは使用できません。  
  
 **エラー ID:** BC30122  
  
### このエラーを解決するには  
  
-   **イミディエイト** ウィンドウでは、`Stop` ステートメントや `End` ステートメントを実行しないようにします。
