# <a name="39stop39-statements-are-not-valid-in-the-immediate-window"></a>&#39;です。Stop &#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Stop` ステートメントおよび `End` ステートメントは実行を中断するため、デバッグ コンテキストでは使用できません。  
  
 **エラー ID:** BC30122  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Stop` イミディエイト `End` ウィンドウでは、 **ステートメントや** ステートメントを実行しないようにします。
