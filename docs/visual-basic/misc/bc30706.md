# 式またはステートメントは、デバッグ ウィンドウでは有効ではありません
`Try...Catch...Finally` ステートメントは、デバッグ コンテキストでは使用できません。  
  
 **エラー ID:** BC30706  
  
### このエラーを解決するには  
  
1.  デバッグ コードから `Try...Catch...Finally` ステートメントを削除します。
