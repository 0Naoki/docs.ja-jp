# <a name="namespace-or-type-name-has-already-been-imported"></a>Namespace 型または '\<名 >' は既にインポートされています
名前空間を複数回インポートしようとしました。  
  
 **エラー ID:** BC31051  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不要な `Imports` ステートメントを削除します。
