# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>.NET ランタイム インターフェイスを作成できません:&lt;エラー&gt;
インターフェイスの作成中、またはインターフェイスのメソッド呼び出しが失敗したときに、内部コンパイラ エラーが発生しました。  
  
 **エラー ID:** BC31024  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  作業内容を保存し、再起動[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]します。  
  
2.  エラーがまだ発生する場合は、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] を再インストールします。  
  
3.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。
