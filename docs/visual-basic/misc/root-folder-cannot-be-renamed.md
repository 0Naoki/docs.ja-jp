# <a name="root-folder-cannot-be-renamed"></a>ルート フォルダーの名前は変更できません。
ルート ディレクトリの名前を変更しようとしました。 正しく解析されなかったファイル パスが原因の可能性があります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイル パスが正しいことを確認します。
