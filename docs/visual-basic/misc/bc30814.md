# &#39;GoSub&#39; ステートメントは現在サポートされていません
`GoSub` を使用してプロシージャを呼び出すにことはできません。  
  
 **エラー ID:** BC30814  
  
### このエラーを解決するには  
  
-   `GoSub` を使用せずにプロシージャを直接呼び出します。次に例を示します。  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
