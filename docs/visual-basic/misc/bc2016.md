# <a name="codepage-ltnamegt-is-invalid-or-not-installed"></a>コードページ '&lt;名前&gt;' は無効か、インストールされていません。
指定されたコードページは無効か、見つかりません。  
  
 **エラー ID:** BC2016  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   コードをチェックして、特定の状況に適したソリューションを判断します
