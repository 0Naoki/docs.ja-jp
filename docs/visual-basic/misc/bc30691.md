# <a name="lttypenamegt-is-a-type-in-ltcontainernamegt-and-cannot-be-used-as-an-expression"></a>'&lt;typename&gt;'内の型は'&lt;containername&gt;'、式として使用できません
使用中の名前が型の名前として既に宣言されているため、式では使用できません。  
  
 **エラー ID:** BC30691  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   メンバーに一意の名前を付けます。  
  
-   メンバーの名前を角かっこで囲みます。
