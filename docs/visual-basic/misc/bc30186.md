# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>&#39;です。使用して &#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Using` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30186  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Using` [イミディエイト] **ウィンドウから、** ステートメントを削除します。
