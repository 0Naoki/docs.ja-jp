# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-definition-for-event-39lteventnamegt39"></a>モジュールに必要な参照 '&lt;modulename&gt;'イベントの定義を含む'&lt;eventname&gt;'
モジュールに必要な参照 '`modulename`>' イベントの定義を含む'`eventname`>' です。 プロジェクトに参照を追加してください。  
  
 プロジェクト内で直接参照されないモジュールでイベントが定義されています。 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]コンパイラには、イベントが&1; つ以上のモジュールで定義されている場合に、あいまいさを避けるへの参照が必要です。  
  
 **エラー ID:** BC30006  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   参照されていないモジュールの名前をプロジェクト参照に含めます。
