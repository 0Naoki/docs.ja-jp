# <a name="reference-required-to-module-ltmodulenamegt-containing-the-definition-for-event-lteventnamegt"></a>モジュールに必要な参照 '&lt;modulename&gt;'イベントの定義を含む'&lt;eventname&gt;'
イベント '<`eventname`>' の定義を含むモジュール '<`modulename`>' への参照が必要です。 プロジェクトに参照を追加してください。  
  
 プロジェクト内で直接参照されないモジュールでイベントが定義されています。 Visual Basic コンパイラでは、イベントが 1 つ以上のモジュールで定義されている場合に備えて、あいまいさを避けるためへの参照が必要です。  
  
 **エラー ID:** BC30006  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   参照されていないモジュールの名前をプロジェクト参照に含めます。
