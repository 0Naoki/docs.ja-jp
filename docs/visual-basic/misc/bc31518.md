# <a name="assembly-assemblyname-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>アセンブリ '\<assemblyname >' のパスが 259 文字より長いために作成できません
このアセンブリのパスが 259 文字の最大長を超えています。  
  
 **エラー ID:** BC31518  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   短い名前を使用するか浅いファイル階層を選択して、短いパスを作成します。
