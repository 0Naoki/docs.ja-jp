# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>モジュールのプロパティを宣言することはできません&#39;&lt;指定子&gt;&#39;
モジュール内のプロパティに `MustOverride`、 `NotOverridable`、 `Overridable`などの指定子を使用しようとしました。  
  
 **エラー ID:** BC30503  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
