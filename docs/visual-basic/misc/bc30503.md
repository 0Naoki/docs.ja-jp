# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>モジュールのプロパティを宣言することはできません (& a) #39 です。&lt;指定子&gt;&#39;です。
モジュール内のプロパティに `MustOverride`、 `NotOverridable`、 `Overridable`などの指定子を使用しようとしました。  
  
 **エラー ID:** BC30503  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
