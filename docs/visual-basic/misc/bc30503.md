# <a name="properties-in-a-module-cannot-be-declared-ltspecifiergt"></a>モジュールのプロパティを宣言することはできません '&lt;指定子&gt;'
モジュール内のプロパティに `MustOverride`、 `NotOverridable`、 `Overridable`などの指定子を使用しようとしました。  
  
 **エラー ID:** BC30503  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
