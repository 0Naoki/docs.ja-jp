# &#39;Overrides&#39; として宣言されているメソッドを &#39;Overridable&#39; として宣言することはできません。これらのメソッドは暗黙的にオーバーライド可能です。
ほとんどのメソッドとは異なり、`Overrides` 修飾子でマークされたメソッドは既定でオーバーライド可能です。  
  
 **エラー ID:** BC30730  
  
### このエラーを解決するには  
  
-   `Overrides` 修飾子でマークされたメソッドから `Overridable` 修飾子を省略します。
