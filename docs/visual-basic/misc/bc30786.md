# <a name="declare-statements-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>モジュール内のステートメントを宣言できません。 宣言 &#39;です。&lt;指定子&gt;&#39;です。
`Declare` 宣言に、 `Module` 宣言内で無効な指定子があります。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30786  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
