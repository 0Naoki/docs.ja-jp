# モジュール内の Declare ステートメントを &#39;&lt;specifier&gt;&#39; と宣言することはできません
`Declare` 宣言に、`Module` 宣言内で無効な指定子があります。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30786  
  
### このエラーを解決するには  
  
-   指定子を削除します。
