# <a name="declare-statements-in-a-module-cannot-be-declared-ltspecifiergt"></a>モジュール内のステートメントを宣言することはできませんの宣言 '&lt;指定子&gt;'
`Declare` 宣言に、 `Module` 宣言内で無効な指定子があります。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30786  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
