# <a name="expression-ltnamegt-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>式 '&lt;名前&gt;'、プロシージャではありませんが、プロシージャ呼び出しのターゲットとして発生します
プロシージャではない式に対してプロシージャ呼び出しを行いました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  プロシージャの名前のスペルを間違えていないか確認します。  
  
2.  <`name`> によって呼び出されるプロシージャを指定します。
