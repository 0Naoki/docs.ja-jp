# <a name="expression-name-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>式 '\<名 >'、プロシージャではありませんが、プロシージャ呼び出しのターゲットとして発生します
プロシージャではない式に対してプロシージャ呼び出しを行いました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  プロシージャの名前のスペルを間違えていないか確認します。  
  
2.  プロシージャを指定する <`name`> の呼び出し。
