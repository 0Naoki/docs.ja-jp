# <a name="sub-main-is-declared-more-than-once-in-ltnamespacegt-ltmessagegt"></a>複数回宣言は 'sub Main' '&lt;名前空間&gt;':&lt;メッセージ&gt;
`Sub Main` は、名前空間の内部で 1 回だけ宣言できます。  
  
 **エラー ID:** BC30738  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロジェクト内に `Sub Main` プロシージャがただ 1 つのみ存在することを確認します。
