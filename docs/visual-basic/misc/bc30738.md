# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;です。Sub Main &#39;回を超える &#39; が宣言されています。&lt;名前空間&gt;&#39;:&lt;メッセージ&gt;
`Sub Main` は、名前空間の内部で 1 回だけ宣言できます。  
  
 **エラー ID:** BC30738  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   プロジェクト内に `Sub Main` プロシージャがただ 1 つのみ存在することを確認します。
