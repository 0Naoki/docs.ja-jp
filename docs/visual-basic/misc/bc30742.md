# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>値 '\<valuename1 >' に変換できません'\<valuename2 >'。
2 つの互換性のない型どうしで型変換を行おうとしました。  
  
 **エラー ID:** BC30742  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- キャストされている型を確認します。
