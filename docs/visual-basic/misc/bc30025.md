# Property に &#39;End Property&#39; がありません。
`End` ステートメントは、終了するブロックと一致する必要があります。`Property` 定義が正しく終了していません。  
  
 **エラー ID:** BC30025  
  
### このエラーを解決するには  
  
-   `End Property` ステートメントをプロパティ定義の最後に追加します。
