# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>XML 名前空間プレフィックス &#39; プレフィックス &#39;XML と、名前空間 URI を変更できませんで使用するために予約されています
`xmlns`などの予約済み XML 名前空間プレフィックスが XML リテラルで使用されています。  
  
 **エラー ID:** BC31152  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML 名前空間プレフィックスを予約されていない値に変更します。
