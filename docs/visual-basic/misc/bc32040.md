# <a name="the-ltkeywordgt-keyword-is-used-to-overload-inherited-members-do-not-use-the-ltkeywordgt-keyword-when-overloading-sub-new"></a>'&lt;キーワード&gt;' は継承されたオーバー ロード メンバーにキーワードが使用されます使用しないでください、'&lt;キーワード&gt;'キーワード' Sub New' をオーバー ロードと。
コンストラクターが `Overloads` キーワードを使用して宣言されています。  
  
 Visual Basic は、コンストラクターの継承またはオーバーロードをサポートしていません。  
  
 **エラー ID:** BC32040  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   すべてのコンストラクターの宣言から `Overloads` キーワードを削除します。
