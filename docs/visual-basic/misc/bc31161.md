# <a name="expected-closing-39--gt39-for-xml-comment"></a>閉じる &#39; が必要です--&gt;&#39;XML コメント用です。
XML コメント リテラルの終わりに必要な '-->' 文字がありません。  
  
 **エラー ID:** BC31161  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML コメント リテラルの終わりに必要な '-->' 文字を追加します。
