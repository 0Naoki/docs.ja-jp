# XML コメントの終わりに &#39;--&gt;&#39; が必要です
XML コメント リテラルの終わりに必要な '\-\-\>' 文字がありません。  
  
 **エラー ID:** BC31161  
  
### このエラーを解決するには  
  
-   XML コメント リテラルの終わりに必要な '\-\-\>' 文字を追加します。
