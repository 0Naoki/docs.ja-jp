# クラスを &#39;&lt;specifier&gt;&#39; として宣言できません
`Class` または `Module` を、無効な指定子を使用して宣言しようとしました。  
  
 **エラー ID:** BC30461  
  
### このエラーを解決するには  
  
-   指定子を削除します。
