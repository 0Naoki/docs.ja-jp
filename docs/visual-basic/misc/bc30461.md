# <a name="classes-cannot-be-declared-ltspecifiergt"></a>クラスを宣言できません '&lt;指定子&gt;'。
`Class` または `Module` を、無効な指定子を使用して宣言しようとしました。  
  
 **エラー ID:** BC30461  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
