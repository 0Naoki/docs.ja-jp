# <a name="39ltmethodnamegt39-cannot-override-a-method-that-has-been-shadowed"></a>&#39;&lt;methodname&gt; &#39;シャドウされたメソッドをオーバーライドすることはできません
メソッドは、別のメンバーをシャドウするメソッドをオーバーライドしようとしました。  
  
 **エラー ID:** BC31406  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Shadows` 修飾子を使用して、シャドウされたメンバーをオーバーライドします。
