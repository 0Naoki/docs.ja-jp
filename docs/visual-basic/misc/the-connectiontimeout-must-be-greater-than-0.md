# <a name="the-connectiontimeout-must-be-greater-than-0"></a>ConnectionTimeout は 0 より大きくなければなりません。
アップロードとダウンロード ファイルが、 [My.Computer.Network オブジェクト](../../visual-basic/language-reference/objects/my-computer-network-object.md)を指定する必要があります、`connectionTimeout`より大きい`0`です。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `connectionTimeout` より大きい `0`をご指定ください。
