# <a name="cannot-convert-start-value-of-type-39lttype1gt39-and-step-value-of-type-39lttype2gt39-to-a-common-type"></a>型の開始値を変換できません '&lt;type1&gt;'と型の値をステップ イン'&lt;type2&gt;' 共通の型
指定された `For...Next` コントロールの値は、型が異なり、共通の型に変換できないため、無効です。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  すべての値を共通の型に変換できるように、1 つ以上の値の型を変更します。
