# <a name="cannot-convert-start-value-of-type-39lttype1gt39-and-step-value-of-type-39lttype2gt39-to-a-common-type"></a>型 &#39; の開始値を変換できません。&lt;type1&gt;&#39; ステップのおよび値型 &#39;&lt;type2&gt;&#39;を共通型です。
指定された `For...Next` コントロールの値は、異なる型であるため、共通の型に変換できないので無効です。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  すべての値を共通の型に変換できるように、1 つ以上の値の型を変更します。
