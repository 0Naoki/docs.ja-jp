# 型 &#39;&lt;type1&gt;&#39; の引数を &#39;&lt;argumentname&gt;&#39; を型 &#39;&lt;type2&gt;&#39; に変換することはできません
変換操作は、引数を互換性のない型に変換しようとしています。  
  
### このエラーを解決するには  
  
1.  引数の変換を試みている型が有効であることを確認します。
