# <a name="39ltnamegt39-necessary-for-compiling-this-construct-cannot-be-found"></a>&#39;&lt;名前&gt;&#39;このコンストラクトをコンパイルするが見つからないために必要な
コンパイルに必要な要素が見つかりません。  
  
 **エラー ID:** BC30930  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  必要なヘルパー要素を指定します。  
  
2.  コードが有効なマネージ コードであることを確認します。
