# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39;通貨&#39;が不要になったサポートされる型です。使用して、 &#39;Decimal&#39;代わりを入力
`Currency` データ型は `Decimal` データ型に置き換えられました。  
  
 **エラー ID:** BC30803  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Decimal` の代わりに `Currency`データ型を使用します。
