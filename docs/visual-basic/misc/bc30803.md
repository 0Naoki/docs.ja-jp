# &#39;Currency&#39; 型は現在サポートされていません。&#39;Decimal&#39; 型を使用してください
`Currency` データ型は `Decimal` データ型に置き換えられました。  
  
 **エラー ID:** BC30803  
  
### このエラーを解決するには  
  
-   `Currency` の代わりに `Decimal` データ型を使用します。
