# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;です。最後に &#39;です。ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`Finally` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30716  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `Finally` ステートメントを削除します。
