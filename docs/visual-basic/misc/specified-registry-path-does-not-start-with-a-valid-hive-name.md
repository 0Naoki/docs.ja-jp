# 指定されたレジストリ パスには、先頭に有効なハイブ名がありません
指定されたレジストリ パスには、先頭に正しいハイブ名がありません。 正しいハイブ名は、以下のとおりです。  
  
-   `HKEY_CLASSES_ROOT`  
  
-   `HKEY_CURRENT_CONFIG`  
  
-   `HKEY_CURRENT_USER`  
  
-   `HKEY_DYNDATA`  
  
-   `HKEY_LOCAL_MACHINE`  
  
-   `HKEY_PERFORMANCE_DATA`  
  
-   `HKEY_USERS`  
  
### このエラーを解決するには  
  
-   正しいハイブ名を指定していることを確認します。
