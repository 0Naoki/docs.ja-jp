# ファイルのオープン モードが有効な値に設定されていませんでした。
ファイルのオープン モードに指定された値が無効でした。 次の表に、<xref:Microsoft.VisualBasic.OpenMode> 列挙型の有効な値を示します。  
  
|値|モード|  
|-------|---------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
### このエラーを解決するには  
  
-   ファイルのオープン モードに指定された値を確認します。
