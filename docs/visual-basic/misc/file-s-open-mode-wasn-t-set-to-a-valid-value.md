# <a name="file39s-open-mode-wasn39t-set-to-a-valid-value"></a>ファイル &#39; s オープン モードされなかった &#39; 有効な値に設定できません。
ファイルのオープン モードに指定された値が無効でした。 次の表に、 <xref:Microsoft.VisualBasic.OpenMode> 列挙型の有効な値を示します。  
  
|値|モード|  
|-----------|----------|  
|1|`OpenMode.Input`|  
|2|`OpenMode.Output`|  
|4|`OpenMode.Random`|  
|8|`OpenMode.Append`|  
|32|`OpenMode.Binary`|  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイルのオープン モードに指定された値を確認します。
