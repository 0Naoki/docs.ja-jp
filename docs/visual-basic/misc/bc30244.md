# <a name="ltmodifiergt-is-not-valid-on-a-declare"></a>'&lt;修飾子&gt;'、Declare では無効です
`Declare` ステートメントに `ReadOnly`などの使用できないキーワードが含まれています。  
  
 **エラー ID:** BC30244  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Declare` ステートメントから使用できないキーワードを削除します。
