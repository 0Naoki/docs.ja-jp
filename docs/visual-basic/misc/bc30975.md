# &#39;For&#39; ループ コントロール変数は、それを囲む &#39;For&#39; ループによって既に使用されています
入れ子になったすべての `For` ループが一意のコントロール変数を使用する必要があります。  
  
 **エラー ID:** BC30975  
  
### このエラーを解決するには  
  
-   コントロールの変数名のいずれかを変更します。
