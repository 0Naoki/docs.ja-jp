# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;&#39;ループ コントロール変数が、それを囲むによって既に使用されて&#39;の&#39;ループ
入れ子になったすべての `For` ループが一意のコントロール変数を使用する必要があります。  
  
 **エラー ID:** BC30975  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   コントロールの変数名のいずれかを変更します。
