# このシステムに、日本語ロケールのサポートは含まれていません
`VbStrConv.Japanese` 列挙型メンバーの使用が試行されましたが、システムは日本語のロケールをサポートしていません。  
  
### このエラーを解決するには  
  
1.  日本語ロケールをサポートするためにコンピューターをアップグレードするか、またはロケールをサポートするコンピューターでアプリケーションを実行します。
