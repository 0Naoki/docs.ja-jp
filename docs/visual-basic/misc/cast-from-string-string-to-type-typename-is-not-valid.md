# 文字列 &quot;&lt;string&gt;&quot; から型 &#39;&lt;typename&gt;&#39; へのキャストは有効ではありません
文字列変数を別の型に変換するのは無効です。  
  
### このエラーを解決するには  
  
1.  変換に関係する型をチェックし、意味のある型に変換しようとしていることを確認してください。
