# オブジェクトは名前付き引数をサポートしていません
メソッドが、名前付き引数をサポートしていないオブジェクトにアクセスしようとしました。  
  
### このエラーを解決するには  
  
-   このオブジェクトのメソッドにアクセスするときは、引数の位置を指定します。 引数の位置と型の詳細については、オブジェクトのドキュメントを参照してください。
