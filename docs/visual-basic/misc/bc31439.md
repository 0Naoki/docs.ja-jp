# <a name="partial-method-methodname-cannot-use-the-implements-keyword"></a>部分メソッド '\<methodname >'、'Implements' キーワードを使用することはできません
部分メソッド定義では、インターフェイスまたはインターフェイス メンバーを実装できません。  
  
 **エラー ID:** BC31439  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  部分メソッド定義から `Implements` ステートメントを削除します。
