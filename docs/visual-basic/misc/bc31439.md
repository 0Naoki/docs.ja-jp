# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>部分メソッド &#39;&lt;methodname&gt;&#39; は使用できません、&#39; Implements &#39; キーワード
部分メソッド定義では、インターフェイスまたはインターフェイス メンバーを実装できません。  
  
 **エラー ID:** BC31439  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  部分メソッド定義から `Implements` ステートメントを削除します。
