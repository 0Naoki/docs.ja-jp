# <a name="partial-method-ltmethodnamegt-cannot-use-the-implements-keyword"></a>部分メソッド '&lt;methodname&gt;'、'Implements' キーワードを使用することはできません
部分メソッド定義では、インターフェイスまたはインターフェイス メンバーを実装できません。  
  
 **エラー ID:** BC31439  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  部分メソッド定義から `Implements` ステートメントを削除します。
