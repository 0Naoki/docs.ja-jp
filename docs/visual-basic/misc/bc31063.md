# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39;です。セット &#39;メソッドは、1 つ以上のパラメーターを持つことはできません。
`Set` プロパティ プロシージャについて複数のパラメーターが定義されています。  
  
 **エラー ID:** BC31063  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Set` プロパティ プロシージャから余分なパラメータを削除します。
