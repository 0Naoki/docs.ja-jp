# &#39;Set&#39; メソッドに 2 つ以上のパラメーターを指定することはできません
`Set` プロパティ プロシージャについて複数のパラメーターが定義されています。  
  
 **エラー ID:** BC31063  
  
### このエラーを解決するには  
  
-   `Set` プロパティ プロシージャから余分なパラメータを削除します。
