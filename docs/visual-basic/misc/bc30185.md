# <a name="39enum39-must-end-with-a-matching-39end-enum39"></a>&#39;です。Enum &#39;一致する &#39; で終わる必要があります。End Enum &#39;
`Enum` ステートメントが発生していますが、対応する `End Enum` ステートメントがありません。 `End Enum` ステートメントを使用して、列挙を終了する必要があります。  
  
 **エラー ID:** BC30185  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Enum` メンバーの書式設定が正しいことを確認します。  
  
2.  `End Enum` ステートメントを列挙の最後に追加します。
