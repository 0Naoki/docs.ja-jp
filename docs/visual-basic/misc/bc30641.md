# &#39;ByVal&#39; および &#39;ByRef&#39; を組み合わせて使用することはできません
`ByVal` キーワードと `ByRef` キーワードは相互に排他的です。  
  
 **エラー ID:** BC30641  
  
### このエラーを解決するには  
  
-   `ByVal` または `ByRef` を指定し、両方は指定しないでください。
