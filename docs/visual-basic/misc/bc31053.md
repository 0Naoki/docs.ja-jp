<span data-ttu-id="0f219-103">`Implements` ステートメントを無効な場所に記述しました。</span><span class="sxs-lookup"><span data-stu-id="0f219-103">An `Implements` statement was encountered in an invalid location.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="0f219-105">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="0f219-105">To correct this error</span></span>
