# <a name="method-39ltmethodnamegt39-has-no-parameter-named-39ltparameternamegt39"></a>メソッド&#39; &lt;methodname&gt; &#39;という名前のパラメーターを持たない&#39; &lt;parametername&gt;&#39;
指定された名前付き引数が対象メソッドの引数のいずれにも一致しません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  名前付き引数が正しく指定されていることを確認します。
