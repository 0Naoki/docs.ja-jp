# XML 終了タグには &#39;/&#39; が必要です
XML リテラルの XML 終了タグに必要な '\/' 文字がありません。  
  
 **エラー ID:** BC31167  
  
### このエラーを解決するには  
  
-   XML リテラルの XML 終了タグの先頭に '\/' 文字を追加します。
