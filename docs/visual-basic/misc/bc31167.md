# <a name="expected-3939-for-xml-end-tag"></a>予想 &#39;/(& a) #39 です。XML 終了タグの
XML リテラルの XML 終了タグに必要な '/' 文字がありません。  
  
 **エラー ID:** BC31167  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML リテラルの XML 終了タグの先頭に '/' 文字を追加します。
