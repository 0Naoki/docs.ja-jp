# <a name="expected-closing-39gt39-for-xml-processor-instruction"></a>予期された終了 &#39; しますか。&gt;&#39;XML プロセッサ命令のです。
XML 処理命令リテラルの終わりに必要な `?>` 文字がありません。  
  
 **エラー ID:** BC31160  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML 処理命令リテラルの終わりに必要な `?>` 文字を追加します。
