# <a name="cannot-convert-to-39lttypegt39"></a>変換できません&#39;&lt;型。&gt;&#39;
無効な変換を行おうとしました。  
  
 **エラー ID:** BC30748  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   互換性のある変換の種類を選択します。
