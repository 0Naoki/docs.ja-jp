# <a name="the-file-is-already-open"></a>ファイルは既に開いています
既に開いているファイルを開こうとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ファイルを開こうとする前にファイルを閉じます。  
  
     - または -  
  
-   不要な `FileOpen` ステートメントをコードから削除します。
