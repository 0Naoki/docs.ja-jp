# 名前空間の型の完全修飾名には、&lt;number&gt; 文字を超える名前を指定することはできません
型の完全修飾名は、指定された文字数より長くできません。  
  
 **エラー ID:** BC30031  
  
### このエラーを解決するには  
  
-   型の名前を短くします。
