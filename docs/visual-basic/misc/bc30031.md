# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-ltnumbergt-characters"></a>名前空間の型の完全修飾名をより長くすることはできません&lt;数&gt;文字
型の完全修飾名は、指定された文字数より長くできません。  
  
 **エラー ID:** BC30031  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   型の名前を短くします。
