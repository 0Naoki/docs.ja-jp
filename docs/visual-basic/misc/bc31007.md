# モジュール ファイル &#39;&lt;filename&gt;&#39; を開けません: &lt;error&gt;
このプロジェクトのモジュールを含むファイルを開くことができませんでした。  
  
 **エラー ID:** BC31007  
  
### このエラーを解決するには  
  
1.  指定した場所にファイルがあることと、ファイルが別のアプリケーションによってロックされていないことを確認します。
