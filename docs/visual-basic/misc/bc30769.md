# <a name="the-targeted-version-of-the-net-compact-framework-does-not-support-the-39end39-statement"></a>.NET Compact Framework の対象となるバージョンがサポートしていない、&#39;です。End &#39;ステートメント
使用している .NET Compact Framework のバージョンは、 `End` ステートメントをサポートしていません。  
  
 **エラー ID:** BC30769  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ステートメントを削除します。
