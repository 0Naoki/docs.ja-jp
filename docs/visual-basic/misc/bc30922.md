<span data-ttu-id="d91fa-103">派生クラスまたはインターフェイスが、基底クラスまたはインターフェイスの型引数として使用することで、制限がある型のアクセス レベルを拡張しようとしています。</span><span class="sxs-lookup"><span data-stu-id="d91fa-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="d91fa-104">次のコードでは、このエラーが生成される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="d91fa-104">The following code can generate this error.</span></span>  
  
```

End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="d91fa-105">アセンブリの外部のコードが `restrictedStructure`にアクセスすることはできません。</span><span class="sxs-lookup"><span data-stu-id="d91fa-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="d91fa-106">ただし、 `derivedClass` には、それを参照できるすべてのコードからアクセスできます。</span><span class="sxs-lookup"><span data-stu-id="d91fa-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="d91fa-107">そのため、任意のアセンブリのコードに `derivedClass` が公開される可能性があるので、 `baseClass` を型引数として使用する場合、 `restrictedStructure` は `restrictedStructure` を継承できません。</span><span class="sxs-lookup"><span data-stu-id="d91fa-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="d91fa-108">**エラー ID:** BC30922</span><span class="sxs-lookup"><span data-stu-id="d91fa-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d91fa-109">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="d91fa-109">To correct this error</span></span>
