# コンパイル エラーまたは削除により、オブジェクトは現在存在しません
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] コンパイラ内でコード要素のアクセス エラーが発生しました。  
  
 **エラー ID:** BC32300  
  
### このエラーを解決するには  
  
1.  プログラムをもう一度コンパイルし、エラーがまだ発生するかどうか確認します。  
  
2.  エラーがまだ発生する場合は、[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] コンパイラを再インストールします。  
  
3.  再インストールした後もエラーが再発する場合は、状況に関する情報を収集し、マイクロソフト製品サポート サービスに通知してください。
