# <a name="end-removehandler-must-be-preceded-by-a-matching-removehandler-declaration"></a>'End RemoveHandler' の前には、対応する 'RemoveHandler' 宣言を指定しなければなりません
`End RemoveHandler` ステートメントが出現しましたが、対応する `RemoveHandler` ステートメントがありません。 `End RemoveHandler` の前に、対応する `RemoveHandler` ステートメントが必要です。  
  
 **エラー ID:** BC31125  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   先行する `RemoveHandler` ステートメントが有効であり、スペルが正しいことを確認します。
