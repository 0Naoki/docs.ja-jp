# <a name="39myclass39-cannot-be-used-outside-of-a-class"></a>&#39;MyClass&#39;クラスの外部で使用することはできません
クラス構造の外部で `MyClass` キーワードを使用しました。  
  
 **エラー ID:** BC30470  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `MyClass` 宣言をクラス構造の中に移動します。
