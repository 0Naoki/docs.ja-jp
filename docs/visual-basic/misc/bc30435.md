# 構造体のメンバーを &#39;&lt;specifier&gt;&#39; として宣言できません
`Structure` ステートメント内のメンバーに対して無効な指定子を使用しました  
  
 **エラー ID:** BC30435  
  
### このエラーを解決するには  
  
-   指定子を削除します。
