# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>'&lt;名前&gt;' 型を公開できません&lt;type1&gt;で&lt;指定子&gt; &lt;type2&gt; '&lt;typename&gt;'経由。&lt;名&gt;
パブリック クラス `MyClass`の外部でプライベート型を公開しようとしました。  
  
 **エラー ID:** BC30508  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   型 `Public`を宣言します。
