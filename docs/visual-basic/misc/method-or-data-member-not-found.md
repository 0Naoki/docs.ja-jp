# <a name="method-or-data-member-not-found"></a>メソッドまたはデータ メンバーが見つかりません
アクセスしようとしていたメソッドまたはデータ メンバーは見つかりませんでした。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
- メソッドまたはデータ メンバーの名前のスペルが正しいことを確認します。
