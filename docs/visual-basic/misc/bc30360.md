# Property &#39;&lt;propertyname&gt;&#39; は、ベース &#39;&lt;name&gt;&#39; の既定プロパティをオーバーライドするため、&#39;Default&#39; として宣言しなければなりません
コード内の `Property` ステートメントは、基底クラス、構造体、またはインターフェイスの `Default` プロパティをオーバーライドするため、`Default` と宣言する必要があります。  
  
 **エラー ID:** BC30360  
  
### このエラーを解決するには  
  
-   プロパティを `Default` と宣言します。
