# <a name="error-number-must-be-within-the-range-0-and-65535"></a>エラー番号は 0 から 65535 の範囲でなければなりません
エラー番号が 0 から 65535 までの使用可能な範囲の外側にあります。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   番号が有効な範囲内にあることを確認します。
