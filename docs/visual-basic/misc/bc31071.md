# インターフェイス内の構造体を &#39;&lt;specifier&gt;&#39; と宣言することはできません
インターフェイスで宣言された構造体で、無効な修飾子が指定されました。  
  
 **エラー ID:** BC31071  
  
### このエラーを解決するには  
  
-   `Public`、`Private`、`Shared`、`Friend`、`Protected`、`Protected Friend`、`Overridable` などの修飾子を削除します。
