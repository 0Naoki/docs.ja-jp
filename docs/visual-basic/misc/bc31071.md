# <a name="structure-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>インターフェイス内の構造体を宣言できません&#39;&lt;指定子。&gt;&#39;
インターフェイスで宣言された構造体で、無効な修飾子が指定されました。  
  
 **エラー ID:** BC31071  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`、または `Overridable`などの修飾子を削除します。
