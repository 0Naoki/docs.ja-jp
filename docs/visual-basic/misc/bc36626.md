# &#39;&lt;typename&gt;&#39; は &#39;MustInherit&#39; として宣言されていて作成することができないので、ラムダ式を &#39;&lt;typename&gt;&#39; に変換することはできません。
ステートメントがラムダ式を型に変換しようとしていますが、その型は基底クラスにすることのみができ、インスタンスの作成には使用できません。  
  
 **エラー ID:** BC36626
