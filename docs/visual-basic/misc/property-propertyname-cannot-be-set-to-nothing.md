# プロパティ &lt;propertyname&gt; は、Nothing に設定することはできません。
プロパティを `Nothing` 以外の値に設定する必要があります。 値の計算に誤りがある可能性があります。  
  
### このエラーを解決するには  
  
-   有効な値を指定します。
