# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>&#39;エラー&#39;ステートメントは、イミディ エイト ウィンドウでは有効ではありません。
`On Error` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30720  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `On Error` ステートメントを削除します。
