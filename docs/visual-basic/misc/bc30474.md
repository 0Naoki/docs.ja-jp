# <a name="method1-cannot-override-method2-because-it-is-a-declare-statement"></a>'\<method1 >' をオーバーライドできません'\<method2 >'、'Declare' ステートメントであります。
`Declare` ステートメントで宣言された基底クラス名のデリゲートをオーバーライドしようとしました。  
  
 **エラー ID:** BC30474  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1. `Declare` ステートメント以外になるように、オーバーライドされるメンバーを変更します。  
  
2. このメソッドをオーバーライドしないでください。
