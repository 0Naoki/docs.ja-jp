# <a name="39ltmethod1gt39-cannot-override-39ltmethod2gt39-because-it-is-a-39declare39-statement"></a>&#39;です。&lt;method1&gt;&#39; できません上書き &#39;&lt;method2&gt;&#39;はなので、&#39; Declare &#39; ステートメント
`Declare` ステートメントで宣言された基底クラス名のデリゲートをオーバーライドしようとしました。  
  
 **エラー ID:** BC30474  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  `Declare` ステートメント以外になるように、オーバーライドされるメンバーを変更します。  
  
2.  このメソッドをオーバーライドしないでください。
