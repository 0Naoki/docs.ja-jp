# <a name="reference-required-to-assembly-ltassemblynamegt-containing-the-implemented-interface-ltinterfacenamegt"></a>参照アセンブリが必要です '&lt;assemblyname&gt;'実装されたインターフェイスを含む'&lt;interfacename&gt;'
参照アセンブリが必要です '\<assemblyname >' 実装インターフェイスを含む'\<interfacename >'。 プロジェクトに参照を追加してください。  
  
 プロジェクト内で直接参照されないダイナミック リンク ライブラリ (DLL) またはアセンブリでインターフェイスが定義されています。 Visual Basic コンパイラでは、インターフェイスが 1 つ以上の DLL またはアセンブリで定義されている場合に備えて、あいまいさを避けるためへの参照が必要です。  
  
 **エラー ID:** BC30009  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   参照されない DLL またはアセンブリの名前をプロジェクト参照に含めます。
