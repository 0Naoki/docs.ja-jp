# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>アセンブリに必要な参照&#39; &lt;assemblyname&gt; &#39;実装インターフェイスを含む&#39; &lt;interfacename&gt;&#39;
参照アセンブリが必要です '\<assemblyname >' を含む実装されたインターフェイス'\<interfacename >' です。 プロジェクトに参照を追加してください。  
  
 プロジェクト内で直接参照されないダイナミック リンク ライブラリ (DLL) またはアセンブリでインターフェイスが定義されています。 Visual Basic コンパイラでは、インターフェイスが 2 つ以上の DLL またはアセンブリで定義されている場合に備えて、あいまいさを避けるために参照が必要です。  
  
 **エラー ID:** BC30009  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   参照されない DLL またはアセンブリの名前をプロジェクト参照に含めます。
