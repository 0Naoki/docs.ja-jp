# 非組み込み型名は条件付きコンパイル式で許可されていません
条件付きコンパイル式が非組み込み型を参照していますが、許可されているのは組み込み型のみです。  
  
 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] の組み込み型の詳細については、「[Data Types Summary](../../visual-basic/language-reference/keywords/data-types-summary.md)」を参照してください。  
  
 **エラー ID:** BC31426  
  
### このエラーを解決するには  
  
-   非組み込み型を組み込み型で置き換えます。
