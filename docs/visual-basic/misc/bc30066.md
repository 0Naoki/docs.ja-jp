# <a name="39exit-property39-is-not-valid-in-a-function-or-sub"></a>&#39;Exit Property&#39; 、Function または Sub では無効です
`Exit Property` は `Function` プロシージャまたは `Sub` プロシージャに出現します。 `Exit` ステートメントは、それが出現するブロックと一致する必要があります。  
  
 **エラー ID:** BC30066  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   必要に応じて、 `Exit Property` を `Exit Function` または `Exit Sub` ステートメントに置き換えます。
