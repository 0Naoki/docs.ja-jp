# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-property-declaration"></a>'Shared' と組み合わせて使用できない '&lt;指定子&gt;' プロパティ宣言で
プロパティ宣言で有効ではない組み合わせで、 `Shared` 修飾子と指定子を組み合わせようとしました。  
  
 **エラー ID:** BC30502  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
