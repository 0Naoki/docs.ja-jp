# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39;共有&#39;と組み合わせて使用できない&#39;&lt;指定子&gt;&#39;プロパティ宣言で
プロパティ宣言で有効ではない組み合わせで、 `Shared` 修飾子と指定子を組み合わせようとしました。  
  
 **エラー ID:** BC30502  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
