# <a name="shared-cannot-be-combined-with-specifier-on-a-property-declaration"></a>'Shared' と組み合わせて使用できない '\<指定子 >' のプロパティの宣言
プロパティ宣言で有効ではない組み合わせで、 `Shared` 修飾子と指定子を組み合わせようとしました。  
  
 **エラー ID:** BC30502  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
