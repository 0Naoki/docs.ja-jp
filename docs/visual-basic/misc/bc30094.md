# <a name="label-39ltlabelnamegt39-is-already-defined-in-the-current-methodmultiline-lambda"></a>ラベル&#39; &lt;labelname&gt; &#39;は、現在のメソッドや複数行ラムダで既に定義
ラベル名がプロシージャ内で重複しています。 プロシージャ内のすべての名前は一意である必要があります。  
  
 **エラー ID:** BC30094  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  重複したラベルの名前を変更します。
