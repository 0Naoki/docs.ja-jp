# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-to-ltregiongt-ltregionnamegt"></a><span data-ttu-id="66ceb-102">'&lt;derivedtypename&gt;' から継承できません&lt;型&gt;'&lt;constructedbasetypename&gt;'' 型へのアクセスを展開するので&lt;internaltypename&gt;'に&lt;リージョン&gt;'&lt;regionname&gt;'</span><span class="sxs-lookup"><span data-stu-id="66ceb-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' to &lt;region&gt; '&lt;regionname&gt;'</span></span>
<span data-ttu-id="66ceb-103">派生クラスまたはインターフェイスは、基底クラスまたはインターフェイスの型引数として使用することで、内部の型のアクセス レベルを拡張しようとしています。</span><span class="sxs-lookup"><span data-stu-id="66ceb-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="66ceb-104">次のコードでは、このエラーが生成される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="66ceb-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="66ceb-105">`containingClass` の外部のコードは `internalStructure`にアクセスできません。</span><span class="sxs-lookup"><span data-stu-id="66ceb-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="66ceb-106">ただし、 `derivedClass` は、同じアセンブリ内のコードからアクセスできます。</span><span class="sxs-lookup"><span data-stu-id="66ceb-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="66ceb-107">そのため、定義コード領域全体で `derivedClass` が公開される可能性があるので、 `baseClass` を型引数として使用する場合、 `internalStructure` は `internalStructure` を継承できません。</span><span class="sxs-lookup"><span data-stu-id="66ceb-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="66ceb-108">**エラー ID:** BC30921</span><span class="sxs-lookup"><span data-stu-id="66ceb-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="66ceb-109">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="66ceb-109">To correct this error</span></span>  
  
-   <span data-ttu-id="66ceb-110">派生型が内部の型のアクセス レベルを拡張しないように、クラスまたはインターフェイスのアクセス レベルを調整します。</span><span class="sxs-lookup"><span data-stu-id="66ceb-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="66ceb-111">または</span><span class="sxs-lookup"><span data-stu-id="66ceb-111">-or-</span></span>  
  
-   <span data-ttu-id="66ceb-112">アクセス レベルを調整できない場合、基底クラスまたはインターフェイスを構築するときに、型引数として内部の型を使用しないでください。</span><span class="sxs-lookup"><span data-stu-id="66ceb-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
