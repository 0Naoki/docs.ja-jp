# <a name="derivedtypename-cannot-inherit-from-type-constructedbasetypename-because-it-expands-the-access-of-type-internaltypename-to-region-regionname"></a><span data-ttu-id="c0093-102">'\<derivedtypename >' から継承できません\<型 >'\<constructedbasetypename >' 型へのアクセスを展開するので、'\<internaltypename >' に\<リージョン > '\<regionname >'</span><span class="sxs-lookup"><span data-stu-id="c0093-102">'\<derivedtypename>' cannot inherit from \<type> '\<constructedbasetypename>' because it expands the access of type '\<internaltypename>' to \<region> '\<regionname>'</span></span>
<span data-ttu-id="c0093-103">派生クラスまたはインターフェイスは、基底クラスまたはインターフェイスの型引数として使用することで、内部の型のアクセス レベルを拡張しようとしています。</span><span class="sxs-lookup"><span data-stu-id="c0093-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="c0093-104">次のコードでは、このエラーが生成される可能性があります。</span><span class="sxs-lookup"><span data-stu-id="c0093-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="c0093-105">`containingClass` の外部のコードは `internalStructure`にアクセスできません。</span><span class="sxs-lookup"><span data-stu-id="c0093-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="c0093-106">ただし、 `derivedClass` は、同じアセンブリ内のコードからアクセスできます。</span><span class="sxs-lookup"><span data-stu-id="c0093-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="c0093-107">そのため、定義コード領域全体で `derivedClass` が公開される可能性があるので、 `baseClass` を型引数として使用する場合、 `internalStructure` は `internalStructure` を継承できません。</span><span class="sxs-lookup"><span data-stu-id="c0093-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="c0093-108">**エラー ID:** BC30921</span><span class="sxs-lookup"><span data-stu-id="c0093-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="c0093-109">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="c0093-109">To correct this error</span></span>
