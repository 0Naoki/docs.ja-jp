# <a name="type-typename-cannot-be-used-as-an-attribute-because-its-container-containername-is-not-declared-public"></a>型 '\<typename >' ため、属性として使用することはできません、コンテナー'\<containername >' は 'Public' と宣言できません
この属性が定義されているクラスまたはモジュールが `Public` 修飾子を使用して宣言されていません。 アクセス修飾子が指定されていないクラスとモジュールは、既定で `Friend` として宣言されます。  
  
 **エラー ID:** BC31517  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  この属性が定義されているクラスまたはモジュールに `Public` 修飾子を追加します。
