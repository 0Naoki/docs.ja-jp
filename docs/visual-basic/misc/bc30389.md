`Private`で宣言されているために、このコンテキストでアクセスできないメソッドにアクセスしようとしました。 このエラーの考えられる原因、Visual Basic コンパイラとインポートされているクラスのすべてのメンバーは、大文字と小文字によってのみ区別されます。 この名前が衝突する可能性がありますので、大文字と小文字です。
