# <a name="39catch39-block-never-reached-because-39ltname1gt39-inherits-from-39ltname2gt39"></a>&#39;キャッチ&#39;ブロックに到達しません、ため&#39; &lt;name1&gt; &#39;継承&#39; &lt;name2&gt;&#39;
コード内の `Catch` ブロックに到達することはありません。  
  
 既定では、このメッセージは警告です。 警告を非表示または警告をエラーとして扱う方法の詳細については、次を参照してください。 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)します。  
  
 **エラー ID:** BC42029  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   余分な `Catch` ブロックを削除します。
