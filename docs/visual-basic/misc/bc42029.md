# &#39;&lt;name1&gt;&#39; は &#39;&lt;name2&gt;&#39; から継承するため、&#39;Catch&#39; ブロックに到達しませんでした
コード内の `Catch` ブロックに到達することはありません。  
  
 既定では、このメッセージは警告です。 警告を非表示にする方法や、警告をエラーとして扱う方法の詳細については、「[Visual Basic での警告の構成](/visual-studio/ide/configuring-warnings-in-visual-basic)」を参照してください。  
  
 **エラー ID:** BC42029  
  
### このエラーを解決するには  
  
-   余分な `Catch` ブロックを削除します。
