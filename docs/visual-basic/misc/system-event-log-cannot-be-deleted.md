# システム イベント ログを削除できません
システム イベント ログを削除しようとしましたが、削除できません。 システム ログでは、システムの起動やハードウェアの障害などのシステム イベントを追跡します。  
  
### このエラーを解決するには  
  
-   システム イベント ログではなく、アプリケーションまたはカスタムのログに書き込むことを検討してください。  
  
-   システム イベント ログを削除しようとしないでください。
