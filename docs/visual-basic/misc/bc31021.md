# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic コンパイラは、次のエラーから回復できません:&lt;エラー&gt;
Visual Basic コンパイラは、次のエラーから回復できません: <`error`>。  
  
 一部の IDE 機能が無効になっています。  
  
 Visual Basic コンパイラで内部エラーが発生しました。  
  
 **エラー ID:** BC31021  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  作業内容を保存し、 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]を再起動します。  
  
2.  エラーがまだ発生する場合は、 [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)]を再インストールします。  
  
3.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。
