# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Visual Basic コンパイラは、次のエラーから回復できません:&lt;エラー&gt;
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]コンパイラでは、次のエラーから回復できない: `error`> します。  
  
 一部の IDE 機能が無効になっています。  
  
 内部エラーが発生しました、[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]コンパイラです。  
  
 **エラー ID:** BC31021  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  作業内容を保存し、再起動[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)]します。  
  
2.  エラーがまだ発生する場合は、[!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)] を再インストールします。  
  
3.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。
