# 共有 &#39;Sub New&#39; を &#39;&lt;specifier&gt;&#39; として宣言することはできません。
`Shared Sub New` 宣言で有効ではない指定子を使用しようとしました。  
  
 **エラー ID:** BC30480  
  
### このエラーを解決するには  
  
-   指定子を削除します。
