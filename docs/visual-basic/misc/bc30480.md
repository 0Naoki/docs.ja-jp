# <a name="shared-sub-new-cannot-be-declared-ltspecifiergt"></a>共有 'Sub New' として宣言できません'&lt;指定子&gt;'
`Shared Sub New` 宣言で有効ではない指定子を使用しようとしました。  
  
 **エラー ID:** BC30480  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定子を削除します。
