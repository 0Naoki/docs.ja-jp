# &lt;type1&gt; &#39;&lt;typename&gt;&#39; および &lt;type2&gt; &#39;&lt;typename&gt;&#39; は &lt;namespace&gt; &#39;&lt;namespacename&gt;&#39; で競合しています
クラスやモジュールなどの 2 つの宣言された要素が、同じ名前空間内で同じ名前を持っています。  
  
 **エラー ID:** BC30179  
  
### このエラーを解決するには  
  
-   いずれかの要素の宣言を別の名前空間に移動します。  
  
-   いずれかの要素の名前を変更します。
