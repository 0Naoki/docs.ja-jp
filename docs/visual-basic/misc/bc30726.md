# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;&#39;コンテキストおよびステートメントには使えないデバッグ ウィンドウ
`With` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30726  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバッグ コードから `With` ステートメントを削除します。
