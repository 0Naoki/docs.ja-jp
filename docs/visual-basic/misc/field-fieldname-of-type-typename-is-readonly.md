# <a name="field-ltfieldnamegt-of-type-lttypenamegt-is-readonly"></a>フィールド '&lt;fieldname&gt;'type' の&lt;typename&gt;' は 'ReadOnly'
`ReadOnly`に宣言されているフィールドに書き込もうとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `ReadOnly`に宣言されないようにフィールドを変更します。  
  
-   または  
  
-   このフィールドに書き込まないでください。
