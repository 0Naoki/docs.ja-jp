# <a name="field-39ltfieldnamegt39-of-type-39lttypenamegt39-is-39readonly39"></a>フィールド &#39;&lt;fieldname&gt;&#39; の種類 &#39;&lt;typename&gt;&#39; が &#39;です。読み取り専用 &#39;
`ReadOnly`に宣言されているフィールドに書き込もうとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `ReadOnly`に宣言されないようにフィールドを変更します。  
  
-   または  
  
-   このフィールドに書き込まないでください。
