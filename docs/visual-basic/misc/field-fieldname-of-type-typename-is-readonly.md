# 型 &#39;&lt;typename&gt;&#39; のフィールド &#39;&lt;fieldname&gt;&#39; は &#39;ReadOnly&#39; です
`ReadOnly` に宣言されているフィールドに書き込もうとしました。  
  
### このエラーを解決するには  
  
-   `ReadOnly` に宣言されないようにフィールドを変更します。  
  
-   または  
  
-   このフィールドに書き込まないでください。
