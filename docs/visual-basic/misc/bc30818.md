# <a name="ltfunctiongt-is-not-declared"></a>'&lt;関数&gt;' が宣言されていません
'\<関数 >' が宣言されていません。 関数は Microsoft.VisualBasic 名前空間に移動しました。  
  
 **エラー ID:** BC30818  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   適切な関数を `Microsoft.VisualBasic` 名前空間から使用します。
