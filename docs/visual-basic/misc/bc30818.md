# &#39;&lt;function&gt;&#39; が宣言されていません
'\<function\>' が宣言されていません。 関数は Microsoft.VisualBasic 名前空間に移動しました。  
  
 **エラー ID:** BC30818  
  
### このエラーを解決するには  
  
-   適切な関数を `Microsoft.VisualBasic` 名前空間から使用します。
