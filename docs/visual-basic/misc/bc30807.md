# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39;です。Let &#39;および &#39;です。セット &#39;代入ステートメントはサポートされていません
`Let` および `Set` は代入で使用できません。  
  
 **エラー ID:** BC30807  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   代入ステートメントから `Let` および `Set` を削除します。次に例を示します。  
  
     `X = 2`
