# <a name="xml-declaration-does-not-allow-attribute-39attributename39"></a>属性 &#39; attributeName &#39; XML 宣言が許可しません。
XML ドキュメント リテラルに無効な属性が指定されています。 XML ドキュメント リテラルの有効な属性値は、XML 1.0 仕様に基づきます。  
  
 **エラー ID:** BC31154  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   XML ドキュメント リテラルから無効な属性を削除します。
