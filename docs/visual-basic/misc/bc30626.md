# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39;Namespace&#39; 、対応するステートメントが終了する必要があります&#39;End Namespace&#39;
`Namespace` ブロックは、 `End Namespace` ステートメントで終了する必要があります。  
  
 **エラー ID:** BC30626  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   `End Namespace` ステートメントを `Namespace` ブロックの最後に追加します。
