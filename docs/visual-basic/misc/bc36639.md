# <a name="39byref39-parameter-39ltparameternamegt39-cannot-be-used-in-a-lambda-expression"></a>&#39;です。ByRef &#39;パラメーター &#39;&lt;parametername&gt;&#39; ラムダ式では使用できません
`Sub` または関数の中で宣言したラムダ式では、その `ByRef` または関数の `Sub` パラメーターを使用できません。 たとえば、以下のコードでは、ラムダ式で `ByRef` パラメーター `n` を使用しているので、このエラーが発生します。  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 **エラー ID:** BC36639  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   次のコードに示すように、 `ByRef` パラメーターをローカル変数に割り当てて、そのローカル変数をラムダ式で使用します。  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
