# <a name="byref-parameter-ltparameternamegt-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="34fc8-102">'ByRef' パラメーター '&lt;parametername&gt;' ラムダ式では使用できません</span><span class="sxs-lookup"><span data-stu-id="34fc8-102">'ByRef' parameter '&lt;parametername&gt;' cannot be used in a lambda expression</span></span>
<span data-ttu-id="34fc8-103">`Sub` または関数の中で宣言したラムダ式では、その `ByRef` または関数の `Sub` パラメーターを使用できません。</span><span class="sxs-lookup"><span data-stu-id="34fc8-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="34fc8-104">たとえば、以下のコードでは、ラムダ式で `ByRef` パラメーター `n` を使用しているので、このエラーが発生します。</span><span class="sxs-lookup"><span data-stu-id="34fc8-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```  
'' Not valid.   
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="34fc8-105">**エラー ID:** BC36639</span><span class="sxs-lookup"><span data-stu-id="34fc8-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="34fc8-106">このエラーを解決するには</span><span class="sxs-lookup"><span data-stu-id="34fc8-106">To correct this error</span></span>  
  
-   <span data-ttu-id="34fc8-107">次のコードに示すように、 `ByRef` パラメーターをローカル変数に割り当てて、そのローカル変数をラムダ式で使用します。</span><span class="sxs-lookup"><span data-stu-id="34fc8-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>  
  
    ```  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
