# <a name="picture-is-not-valid"></a>ピクチャが有効ではありません
フォームまたはコントロールの `Picture` プロパティに、ビットマップ、アイコン、または Windows メタファイル以外のグラフィック形式を割り当てようとしました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ファイルに読み込むことを確認、`Picture`プロパティでサポートされている有効な画像ファイルは、[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]です。
