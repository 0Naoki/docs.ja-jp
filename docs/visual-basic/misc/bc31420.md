# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>&#39;&lt;メンバー&gt; &#39;すべての enum で暗黙的に宣言されている、同じ名前の予約メンバーと競合しています
型メンバーの名前が、すべての列挙体で暗黙的に宣言されている名前と競合します。  
  
 Visual Basic コンパイラでは、宣言されている特定のプログラミング要素に対応する暗黙的なメンバーを作成します。 列挙体は暗黙的にメンバー `value__member`を宣言します。  
  
 **エラー ID:** BC31420  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   メンバーの名前を変更します。
