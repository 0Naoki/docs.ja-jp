# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>'&lt;メンバー&gt;' のすべての列挙型に暗黙的に宣言されているこの名前の予約済みのメンバーと競合
型メンバーの名前が、すべての列挙体で暗黙的に宣言されている名前と競合します。  
  
 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]コンパイラが暗黙的なメンバーを宣言する特定のプログラミング要素に対応するを作成します。 列挙体は暗黙的にメンバー `value__member`を宣言します。  
  
 **エラー ID:** BC31420  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   メンバーの名前を変更します。
