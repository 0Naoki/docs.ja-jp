# &#39;Interface&#39; の終わりには、対応する &#39;End Interface&#39; を指定しなければなりません。
`Interface` ステートメントが発生していますが、対応する `End Interface` ステートメントがありません。`End Interface` ステートメントを使用して、インターフェイス定義を終了する必要があります。  
  
 **エラー ID:** BC30253  
  
### このエラーを解決するには  
  
1.  `Interface` メンバーの書式設定が正しいことを確認します。  
  
2.  `End Interface` ステートメントをインターフェイス定義の最後に追加します。
