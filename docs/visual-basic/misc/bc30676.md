# <a name="lteventnamegt-is-not-an-event-of-ltcontainernamegt"></a>'&lt;eventname&gt;'は、イベントのではありません'&lt;containername&gt;'
指定されたイベントは、オブジェクトに対して宣言されていません。  
  
 **エラー ID:** BC30676  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  イベント名のスペルを確認します。  
  
2.  正しいオブジェクトにアクセスしていることを確認します。 オブジェクトへの完全修飾参照を使用するか、 `Imports` ステートメントを使用して必要に応じて適切な名前空間をインポートします。
