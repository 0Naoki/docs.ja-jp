# &#39;&lt;typename&gt;&#39; で定義された拡張メソッド &#39;&lt;methodname&gt;&#39; のパラメーター &#39;&lt;parametername&gt;&#39; に対して引数が指定されていません
拡張メソッドの必須パラメーターの引数が指定されていません。  
  
 **エラー ID:** BC36586  
  
### このエラーを解決するには  
  
-   指定されたパラメーターに引数を指定します。
