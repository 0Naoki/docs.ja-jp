# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>引数のパラメーターに指定されていない '&lt;parametername&gt;'の拡張メソッド'&lt;methodname&gt;'で定義されている'&lt;typename&gt;'
拡張メソッドの必須パラメーターの引数が指定されていません。  
  
 **エラー ID:** BC36586  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定されたパラメーターの引数を指定します。
