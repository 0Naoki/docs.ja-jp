# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>パラメーターに指定されていない引数&#39; &lt;parametername&gt; &#39;拡張メソッドの&#39; &lt;methodname&gt; &#39;で定義されている&#39; &lt;typename&gt;&#39;
拡張メソッドの必須パラメーターの引数が指定されていません。  
  
 **エラー ID:** BC36586  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定されたパラメーターの引数を指定します。
