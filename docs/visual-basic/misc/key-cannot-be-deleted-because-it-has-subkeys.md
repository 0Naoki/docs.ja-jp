# <a name="key-cannot-be-deleted-because-it-has-subkeys"></a>サブキーがあるためキーを削除できません
キーにはサブキーがあるため削除できません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   この操作を実行する前に、指定したキーのサブキーを特定し、削除します。
