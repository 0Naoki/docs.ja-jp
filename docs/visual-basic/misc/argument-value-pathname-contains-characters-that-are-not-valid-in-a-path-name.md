# <a name="argument-value-ltpathnamegt-contains-characters-that-are-not-valid-in-a-path-name"></a>引数の値 '&lt;pathname&gt;' パス名に無効な文字が含まれています
パス名に無効な文字が 1 つ以上含まれています。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   パス名から無効な文字を削除します。
