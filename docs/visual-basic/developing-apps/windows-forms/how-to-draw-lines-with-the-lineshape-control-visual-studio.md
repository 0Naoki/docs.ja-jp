使用することができます、<xref:Microsoft.VisualBasic.PowerPacks.LineShape>デザイン時および実行時の両方のフォームまたはコンテナー、水平、垂直、または斜めの線を描画するコントロール。

2.  サイズ変更をドラッグし、サイズし、位置の行へのハンドルを移動します。
