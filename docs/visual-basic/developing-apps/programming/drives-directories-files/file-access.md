# File Access with Visual Basic
[!INCLUDE[vs2017banner](../../../../visual-basic/developing-apps/includes/vs2017banner.md)]

`My.Computer.FileSystem` オブジェクトは、ファイルとフォルダーを扱うためのツールです。  そのプロパティ、メソッド、およびイベントを使用すると、ファイルとフォルダーを作成、コピー、移動、調査、および削除できます。  `My.Computer.FileSystem` は、下位互換性のために [!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] に用意されているレガシ関数 \(`FileOpen`、`FileClose`、`Input`、`InputString`、`LineInput` など\) よりもパフォーマンスが優れています。
