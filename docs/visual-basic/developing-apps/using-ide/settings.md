# <a name="visual-basic-settings"></a>Visual Basic の設定
Visual Basic の設定は、ユーザー エクスペリエンスを最適化して、Visual Basic 開発者の生産性を最大化に設計されています。 一部では、 [Visual Studio での開発設定のカスタマイズ](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3)機能で、カスタマイズしで、Visual Studio 統合開発環境 (IDE) の設定を保存することができます。  
  
 移植したり、他のコンピューター上の IDE 設定を再読み込みできます。 詳細については、次を参照してください。[方法: 設定をコンピューター間で共有](http://msdn.microsoft.com/en-us/1131fb10-35c1-42da-9cd8-91aa3235b882)します。  
  
## <a name="windows-and-views"></a>ウィンドウとビュー  
  
|特性|既定で表示されますか。|ノート|  
|---|---|---|    
|[クラス ビュー](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|いいえ|非表示に**型およびメンバーを非表示**|  
|[コマンド ウィンドウ](https://docs.microsoft.com/visualstudio/ide/reference/command-window)|いいえ||  
|**ダイナミック ヘルプ**ウィンドウ|いいえ|F1 キーを押したときに表示されません。|  
|[イミディエイト ウィンドウ](https://docs.microsoft.com/visualstudio/ide/reference/immediate-window)|はい、デバッグの開始時です。||  
|[オブジェクト ブラウザー](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|いいえ|既定で表示します。<br /><br /> 名前空間<br />-パブリック メンバー<br />-継承されたメンバー|  
|[[出力] ウィンドウ](https://docs.microsoft.com/visualstudio/ide/reference/output-window)|はい、ビルド開始時にします。||  
|[[プロパティ] ウィンドウ](https://docs.microsoft.com/visualstudio/ide/reference/properties-window)|はい||  
|**ソリューション エクスプローラー**|はい||  
|**サーバー エクスプ ローラー**|いいえ|データ接続と使用可能なシステム サービスが表示されます。|  
|[スタート ページ](https://docs.microsoft.com/visualstudio/ide/customizing-the-start-page-for-visual-studio)|はい|IDE を起動する最初の時間が表示されます。 カスタマイズすることができます。|  
|**タスク一覧**|はい、コンパイル エラーまたは警告がある場合します。||  
|[チーム エクスプローラー](http://msdn.microsoft.com/library/1372e56c-b34f-42c2-b72c-94b57620c75c)|はい|チーム エクスプ ローラーを使用して、チーム プロジェクトにアクセスし、Team Foundation のアプリケーション ライフ サイクル管理機能を使用することができます。 バージョン管理、ビルド、プロジェクト管理、統合のレポートおよびダッシュ ボードが含まれます。|  
|[ツールボックス](https://docs.microsoft.com/visualstudio/ide/reference/toolbox)|はい|コントロールおよびコンポーネントがアルファベット順に表示されます。|  
  
 さらに、次の要素にある特定の動作を選択すると有効になっている**Visual Basic 開発設定**します。  
  
## <a name="dialog-boxes"></a>ダイアログ ボックス  
  
|特性|動作|
|---|---|  
|**[新しいプロジェクト**] ダイアログ ボックス|コンピューターには、その他のプラットフォームがインストールされている、対象となるプラットフォームを指定することができるようにするコンボ ボックスが有効にします。|  
|[[オプション] ダイアログ ボックス](https://docs.microsoft.com/visualstudio/ide/reference/options-dialog-box-visual-studio)|Visual Basic により簡略**オプション**ページです。 クリックして、**すべての設定を表示する**オプションの一覧を表示する チェック ボックスです。|  
  
## <a name="keyboard"></a>キーボード  
  
|特性|動作|  
|---|---|  
|Visual Basic 6.0 の既定のショートカット キー|Visual Basic 6 のキーボード マッピングの一般的なサポートしています。|
