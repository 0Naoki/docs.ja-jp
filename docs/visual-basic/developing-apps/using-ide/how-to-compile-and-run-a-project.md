# <a name="how-to-compile-and-run-a-project-in-visual-basic"></a>方法 : Visual Basic でプロジェクトをコンパイルして実行する
[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] 統合開発環境 (IDE) では、プロジェクトをコンパイルし、生成されたアプリケーションを実行するのが簡単です。  
  
 IDE の組み込みのデバッガーを使用すると、プロジェクトを実行時にデバッグできます。 デバッグの詳細については、次を参照してください。 [Visual Basic アプリケーションのデバッグ](../../../visual-basic/developing-apps/debugging.md)と[エディット コンティニュ (Visual Basic)](https://docs.microsoft.com/visualstudio/debugger/edit-and-continue-visual-basic)します。  

[!INCLUDE[note_settings_general](../../../csharp/language-reference/compiler-messages/includes/note_settings_general_md.md)]  
  
### <a name="to-compile-and-run-the-current-project"></a>現在のプロジェクトをコンパイルして実行するには  
  
-   [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] 統合開発環境 (IDE) で F5 キーを押します。  
  
     プロジェクトがコンパイルされ、[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] デバッガー内でアプリケーションが実行されます。  
  
### <a name="to-compile-and-run-the-current-project-from-the-menu"></a>現在のプロジェクトをメニューからコンパイルして実行するには  
  
-   **デバッグ**のメニューを開き、 [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] IDE 選択**[デバッグ開始]**します。  
  
     プロジェクトがコンパイルされ、[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] デバッガー内でアプリケーションが実行されます。  
  
### <a name="to-compile-and-run-the-current-project-without-debugging"></a>デバッグせずに現在のプロジェクトをコンパイルして実行するには  
  
-   [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)] IDE で Ctrl + F5 キーを押します。  
  
     プロジェクトがコンパイルされ、アプリケーションが実行されます。  
  
 コンパイルおよび実行する方法について[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]コマンド ライン コンパイラを使用するコードは、「[コマンドラインからのビルド](../../../visual-basic/reference/command-line-compiler/building-from-the-command-line.md)します。
