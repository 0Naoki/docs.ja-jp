# Using the Visual Basic Development Environment
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Visual Studio に用意されている基本的なユーザー インターフェイス機能のほかに、Visual Basic の統合開発環境 \(IDE: Integrated Development Environment\) には、生産性と操作性を向上させるための機能が追加されています。  
  
## このセクションの内容  
 [Visual Basic Settings](../../../visual-basic/developing-apps/using-ide/settings.md)  
 開発環境における操作性を最適化する方法について説明します。  
  
 [Visual Basic IntelliSense Code Snippets](../../../visual-basic/developing-apps/using-ide/intellisense-code-snippets.md)  
 開発環境から利用できるコードのライブラリを紹介します。  
  
 [リファクタリングと \[名前の変更\] ダイアログ ボックス](../../../visual-basic/developing-apps/using-ide/refactoring-and-rename-dialog-box.md)  
 コード内の識別子の名前を変更する際のしくみについて説明します。  
  
 [Help for Event Handlers in Visual Basic Code](../../../visual-basic/developing-apps/using-ide/help-for-event-handlers.md)  
 イベント ハンドラーから F1 ヘルプを取得する方法について説明します。  
  
 [How to: Compile and Run a Project](../../../visual-basic/developing-apps/using-ide/how-to-compile-and-run-a-project.md)  
 開発環境内からのコンパイルおよび実行に使用するオプションについて説明します。  
  
## 関連項目  
 [既定のキーボード ショートカット](/visual-studio/ide/default-keyboard-shortcuts-in-visual-studio)  
 マウスを使用せずにアクションを実行する場合に使用できる既定のキーボード ショートカット キーについて説明します。  
  
 [エディット コンティニュ \(Visual Basic\)](/visual-studio/debugger/edit-and-continue-visual-basic)  
 中断モードでの実行中に、コードを変更することを可能にするデバッグ機能について説明します。  
  
 [Visual Basic での警告の構成](/visual-studio/ide/configuring-warnings-in-visual-basic)  
 警告を無効にしたり、警告をエラー メッセージとして扱う方法について説明します。
