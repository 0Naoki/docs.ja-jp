# Customizing Projects and Extending My with Visual Basic
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

プロジェクト テンプレートをカスタマイズして、追加的な `My` オブジェクトを提供できます。  こうすると、自分の作成したオブジェクトを他の開発者が見つけたり使用したりしやすくなります。  
  
## このセクションの内容  
 [Extending the My Namespace in Visual Basic](../../../visual-basic/developing-apps/customizing-extending-my/extending-the-my-namespace.md)  
 Visual Basic の `My` 名前空間にカスタム メンバーと値を追加する方法を説明します。  
  
 [Packaging and Deploying Custom My Extensions](../../../visual-basic/developing-apps/customizing-extending-my/packaging-and-deploying-custom-my-extensions.md)  
 Visual Studio テンプレートを使用して、カスタム `My` 名前空間の拡張機能を公開する方法を説明します。  
  
 [Extending the Visual Basic Application Model](../../../visual-basic/developing-apps/customizing-extending-my/extending-the-visual-basic-application-model.md)  
 <xref:Microsoft.VisualBasic.ApplicationServices.WindowsFormsApplicationBase> クラスのメンバーをオーバーライドすることによって、アプリケーション モデルに対して独自の拡張機能を指定する方法を説明します。  
  
 [Customizing Which Objects are Available in My](../../../visual-basic/developing-apps/customizing-extending-my/customizing-which-objects-are-available-in-my.md)  
 プロジェクトの \_MYTYPE 条件付きコンパイル定数を設定することによって、有効にする `My` オブジェクトを制御する方法を説明します。
