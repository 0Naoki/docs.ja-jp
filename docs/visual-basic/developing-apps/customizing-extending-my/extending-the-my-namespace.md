##  <span data-ttu-id="ddb36-167"><a name="design"></a>デザインのガイドライン</span><span class="sxs-lookup"><span data-stu-id="ddb36-167"><a name="design"></a> Design Guidelines</span></span>

##  <span data-ttu-id="ddb36-180"><a name="designing"></a>用のクラス ライブラリをデザイン、</span><span class="sxs-lookup"><span data-stu-id="ddb36-180"><a name="designing"></a> Designing Class Libraries for My</span></span>
