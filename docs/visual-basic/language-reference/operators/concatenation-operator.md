# <a name="amp-operator-visual-basic"></a>&amp;演算子 (Visual Basic)

## <a name="syntax"></a>構文  
  
```  
result = expression1 & expression2  
```  
  
## <a name="parts"></a>指定項目  
 `result`  
 必須です。 どの`String`または`Object`変数。  
  
 `expression1`  
 必須です。 データ型に拡大変換を持つ任意の式`String`です。  
  
 `expression2`  
 必須です。 データ型に拡大変換を持つ任意の式`String`です。
