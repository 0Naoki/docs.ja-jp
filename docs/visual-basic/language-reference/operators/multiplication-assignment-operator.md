# <a name="-operator-visual-basic"></a>*= 演算子 (Visual Basic)
式の値で変数またはプロパティの値を乗算し、結果を変数またはプロパティに代入します。  
  
## <a name="syntax"></a>構文  
  
```  
variableorproperty *= expression  
```  
  
## <a name="parts"></a>指定項目  
 `variableorproperty`  
 必須です。 任意の数値型の変数またはプロパティ。  
  
 `expression`  
 必須です。 任意の数式。
