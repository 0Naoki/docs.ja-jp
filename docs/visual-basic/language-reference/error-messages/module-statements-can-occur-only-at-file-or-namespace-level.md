# <a name="39module39-statements-can-occur-only-at-file-or-namespace-level"></a>&#39;です。モジュール &#39;ステートメントは、ファイルまたは名前空間レベルでのみ発生することができます。
`Module`ステートメントが、ソース ファイルの上部に表示する必要がありますの直後に`Option`と`Imports`ステートメント、グローバル属性、および名前空間宣言が、その他のすべての宣言の前にします。  
  
 **エラー ID:** BC30617  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   移動、`Module`名前空間の宣言またはソース ファイルの先頭にステートメントです。
