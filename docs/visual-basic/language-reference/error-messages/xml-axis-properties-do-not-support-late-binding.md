# <a name="xml-axis-properties-do-not-support-late-binding"></a>XML 軸のプロパティは遅延バインディングをサポートしていません
XML 軸プロパティが型指定されていないオブジェクトで参照されています。  
  
 **エラー ID:** BC31168  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   オブジェクトでは、厳密に型指定されたことを確認してください。 <xref:System.Xml.Linq.XElement> XML 軸プロパティを参照する前にオブジェクト。
