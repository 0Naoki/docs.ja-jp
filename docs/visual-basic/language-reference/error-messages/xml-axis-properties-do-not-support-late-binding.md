# XML axis properties do not support late binding
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

型指定されていないオブジェクトで XML 軸のプロパティが参照されています。  
  
 **エラー ID:** BC31168  
  
### このエラーを解決するには  
  
-   XML 軸のプロパティを参照する前に、厳密に型指定された <xref:System.Xml.Linq.XElement> オブジェクトであることを確認します。
