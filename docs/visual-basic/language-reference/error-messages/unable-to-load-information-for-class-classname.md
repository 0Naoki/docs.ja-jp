# Unable to load information for class &#39;&lt;classname&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

使用できないクラスが参照されました。  
  
 **Error ID:** BC30712  
  
### このエラーを解決するには  
  
1.  クラスが定義されていること、およびクラス名のスペルが正しいことを確認します。  
  
2.  モジュール内で宣言されているいずれかのメンバーにアクセスしてみます。  場合によっては、メンバーが宣言されているモジュールが読み込まれていないために、デバッグ環境でメンバーを特定できないことがあります。
