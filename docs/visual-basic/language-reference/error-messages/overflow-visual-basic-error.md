# Overflow (Visual Basic Error)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

リテラルの値が、代入先のデータ型の制限を超えています。  
  
 **Error ID:** BC30036  
  
### このエラーを解決するには  
  
-   代入先のデータ型の値の範囲を調べ、その範囲内に収まるようにリテラルを書き直します。
