# <a name="overflow-visual-basic-error"></a>オーバーフローしました。(Visual Basic エラー)
リテラルは、割り当てられるデータ型の制限の範囲外の値を表します。  
  
 **エラー ID:** BC30036  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   データ型で、その範囲に準拠するように、そのリテラルを書き直してください。 ターゲットの値の範囲を参照してください。
