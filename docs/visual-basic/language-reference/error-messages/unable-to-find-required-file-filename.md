# <a name="unable-to-find-required-file-filename"></a>必要なファイルが見つかりません '\<filename >'
Visual Studio で必要とされるファイルがないか、破損しています。  
  
 **エラー ID:** BC30655  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   Visual Studio を再インストールしてください。
