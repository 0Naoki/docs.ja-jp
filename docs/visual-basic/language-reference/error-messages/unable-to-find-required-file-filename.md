# Unable to find required file &#39;&lt;filename&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Visual Studio で必要とされるファイルがないか、破損しています。  
  
 **Error ID:** BC30655  
  
### このエラーを解決するには  
  
-   Visual Studio を再インストールしてください。
