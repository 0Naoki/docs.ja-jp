# This key is already associated with an element of this collection
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

既にコレクションの別のメンバーを指定しているキーを指定しました。  キーは、`Add` メソッドで指定される、コレクションの特定のメンバーを一意に識別する文字列です。  
  
### このエラーを解決するには  
  
-   このメンバーに別のキーを指定します。
