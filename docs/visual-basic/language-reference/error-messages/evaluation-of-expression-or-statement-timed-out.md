# Evaluation of expression or statement timed out
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

式の評価が時間内に完了しませんでした。  
  
 **Error ID:** BC30722  
  
### このエラーを解決するには  
  
1.  入力したコードが正しいことを確認してください。  
  
2.  実行時間を短くするために、式を単純化します。
