# <a name="evaluation-of-expression-or-statement-timed-out"></a>式またはステートメントの評価がタイムアウトしました。
式の評価は、適切なタイミングで完了しませんでした。  
  
 **エラー ID:** BC30722  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  入力したコードが正しいことを確認します。  
  
2.  実行時間を短くなるように、式を簡略化します。
