# Line is too long
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

ソース テキストの 1 行の文字数は、65,535 文字以内に制限されています。  
  
 **Error ID:** BC30494  
  
### このエラーを解決するには  
  
-   行の長さを 65,535 文字以内にします。
