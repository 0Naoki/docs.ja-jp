# <a name="identifier-expected"></a>識別子が必要です。
宣言された要素を認識可能な名前が異なるプログラミング要素は、コンテキストが要素名は、必要とする場合に発生します。 1 つの考えられる原因は、属性が指定されている場所以外のステートメントの先頭にします。  
  
 **エラー ID:** BC30203  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   ステートメントで属性がすべてに配置される先頭にあることを確認します。  
  
-   ステートメント内のすべての要素名のスペルが正しいことを確認します。
