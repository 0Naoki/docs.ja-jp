# <a name="39sub-main39-was-not-found-in-39ltnamegt39"></a>&#39;Sub Main&#39;に見つからなかった&#39;&lt;名&gt;&#39;
`Sub Main` が存在しないか、間違った場所が指定されています。  
  
 **エラー ID:** BC30420  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  不足している `Sub Main` ステートメントを指定するか、存在する場合はコード内の適切な位置に移動します。 詳細については`Sub Main`を参照してください[Visual Basic の Main プロシージャ](../../../visual-basic/programming-guide/program-structure/main-procedure.md)です。  
  
2.  内のプロジェクトのスタートアップ オブジェクトの場所の指定、**スタートアップ フォーム**のボックス、**プロジェクト デザイナー**です。
