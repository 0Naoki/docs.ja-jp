# <a name="array-subscript-expression-missing"></a>配列インデックス式が見つかりません。
配列の初期化が 1 つ以上の配列の境界を定義する添字のままにします。 たとえば、ステートメントは、式を含めることが`myArray (5,5,,10)`、3 番目の添字を残っています。  
  
 **エラー ID:** BC30306  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   不足しているインデックスを指定します。
