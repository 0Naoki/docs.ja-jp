# <a name="constant-expression-not-representable-in-type-typename"></a>型で表現できない定数式 '\<typename >'
対象の型に収まりません定数を評価しようとして通常は範囲をオーバーフローしていることが原因です。  
  
 **エラー ID:** BC30439  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  定数を処理できるには、対象の型を変更します。
