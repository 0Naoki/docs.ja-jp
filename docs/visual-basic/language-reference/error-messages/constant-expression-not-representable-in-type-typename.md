# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>型 &#39; で表現できない定数式&lt;typename&gt;&#39;です。
対象の型に適合しない定数を評価しようとする通常の範囲をオーバーフローしていることが原因です。  
  
 **エラー ID:** BC30439  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  定数を処理するには、対象の型を変更します。
