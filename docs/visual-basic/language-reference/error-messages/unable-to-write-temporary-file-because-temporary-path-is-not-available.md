# Unable to write temporary file because temporary path is not available
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] で、一時ファイルが格納されているディレクトリのパスを判断できませんでした。  
  
 **Error ID:** BC30698  
  
### このエラーを解決するには  
  
1.  [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] を再起動します。  
  
2.  問題が再発する場合は、[!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)] を再インストールします。
