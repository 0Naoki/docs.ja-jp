指定されたキー ファイルから、厳密な名前のアセンブリを作成できませんでした。  
  
 **エラー ID:** BC31026  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  適切なキー ファイルが指定されていると、別のアプリケーションによってロックされていないことを確認します。
