# <a name="unable-to-create-strong-named-assembly-from-key-file-39ltfilenamegt39-lterrorgt"></a>キー ファイルから厳密な名前付きのアセンブリを作成できません&#39; &lt;filename&gt;&#39;:&lt;エラー。&gt;
指定されたキー ファイルから厳密な名前のアセンブリを作成できませんでした。  
  
 **エラー ID:** BC31026  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  適切なキー ファイルが指定されていると、別のアプリケーションによってロックされていないことを確認します。
