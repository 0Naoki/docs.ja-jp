# <a name="some-subkeys-cannot-be-deleted"></a>一部のサブキーが削除できません
レジストリ キーの削除が試行されましたが、一部のサブキーを削除できないために、操作が失敗しました。 通常のアクセス許可がないのためにです。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   指定したサブキーを削除するための十分なアクセス許可があることを確認します。
