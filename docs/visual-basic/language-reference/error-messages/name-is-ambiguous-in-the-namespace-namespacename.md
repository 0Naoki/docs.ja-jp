# <a name="39ltnamegt39-is-ambiguous-in-the-namespace-39ltnamespacenamegt39"></a>&#39;&lt;名前&gt;&#39;名前空間のあいまいな&#39; &lt;namespacename&gt;&#39;
あいまいな名前を指定したため、別の名前と競合しています。 Visual Basic コンパイラが、競合解決規則です。名は、自分で明確する必要があります。  
  
 **エラー ID:** BC30560  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   名前を完全修飾します。
