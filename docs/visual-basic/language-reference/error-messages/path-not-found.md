# Path not found
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

ファイル アクセス操作またはディスク アクセス操作で、指定されているパスが見つかりませんでした。  ファイルへのパスには、ファイルがある場所のドライブ、ディレクトリ、およびサブディレクトリを指定します。  相対パスと絶対パスのどちらでも指定できます。  
  
### このエラーを解決するには  
  
-   パスを確認し、指定し直します。
