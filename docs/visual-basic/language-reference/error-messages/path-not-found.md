# <a name="path-not-found"></a>パスが見つかりません。
ファイルへのアクセスまたはディスク アクセスの操作中に、オペレーティング システムは、指定されたパスを検索できませんでした。 ファイルへのパスには、ドライブの指定に加えて、ディレクトリ、ファイルを検索するために走査する必要がありますサブディレクトリが含まれています。 相対ファイル パスと絶対ファイル パスのどちらでも構いません。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   確認し、パスを再度指定します。
