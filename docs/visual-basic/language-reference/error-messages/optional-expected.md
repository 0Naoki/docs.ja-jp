# <a name="39optional39-expected"></a>&#39;以外の場合は省略可能な &#39;です。必要です。
プロシージャの宣言で省略可能な引数には、必須の引数が続きます。 各引数は次のオプションの引数も省略可能な場合があります。  
  
 **エラー ID:** BC30202  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  引数が必要になるとしている場合は、引数リストの最初の省略可能な引数の前に移動します。  
  
2.  引数は省略可能としている場合を使用して、`Optional`キーワード。
