# <a name="can39t-create-necessary-temporary-file"></a>できます&#39;t 必要な一時ファイルを作成します。
ドライブがいっぱい、TEMP 環境変数で指定されたディレクトリを格納しているか、TEMP 環境変数は、無効であるか、読み取り専用のドライブまたはディレクトリを指定します。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  いっぱいの場合は、ドライブからファイルを削除します。  
  
2.  TEMP 環境変数を別のドライブを指定します。  
  
3.  TEMP 環境変数の有効なドライブを指定します。  
  
4.  現在指定されているドライブまたはディレクトリからの読み取り専用の制限を削除します。
