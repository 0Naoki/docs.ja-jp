# Device I/O error
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

プログラムでプリンターやディスク ドライブなどのデバイスを使用しているときに、入力エラーまたは出力エラーが発生しました。  
  
### このエラーを解決するには  
  
-   デバイスが正常に動作していることを確認してから、操作をやり直します。
