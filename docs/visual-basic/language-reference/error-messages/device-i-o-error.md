# <a name="device-io-error"></a>デバイス I/O エラーです。
プログラムは、プリンターやディスク ドライブなどのデバイスを使用していたときに、入力または出力エラーが発生しました。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   デバイスが正常かどうかを確認し、操作を再試行してください。
