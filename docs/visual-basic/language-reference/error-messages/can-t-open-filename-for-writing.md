# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>&#39; 開いている t &#39;&lt;filename&gt;&#39;書き込み用です。
作成するため、おそらく既に開かれているために、指定したファイルを開くことができません。  
  
 **エラー ID:** BC2012  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  ファイルを閉じて、再度開きます。  
  
2.  ファイルのアクセス許可を確認してください。
