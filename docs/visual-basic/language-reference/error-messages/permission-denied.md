# <a name="permission-denied-visual-basic"></a>アクセス許可は拒否されました。(Visual Basic)
試みましたを書き込み可能ディスクに書き込む、またはロックされたファイルにアクセスします。  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
1.  読み取り専用のファイルを開くには、ファイルの書き込み禁止属性を変更します。  
  
2.  別のプロセスが、ファイルをロックしていないことを確認し、待ってから、他のプロセスが解放されるまで、ファイルを開きます。  
  
3.  レジストリにアクセスするには、ユーザーのアクセス許可がこの種類のレジストリへのアクセスを含めることを確認します。
