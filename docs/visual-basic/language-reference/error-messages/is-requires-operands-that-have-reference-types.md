# <a name="39is39-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-39lttypenamegt39"></a>&#39; が &#39;です。オペランドが必要です参照型を持つことがこのオペランドの値の型 &#39;&lt;typename&gt;&#39;です。
`Is`比較演算子は、2 つのオブジェクト変数が、同じインスタンスを参照しているかどうかを判定します。 この比較は、値型に対して定義されていません。  
  
 **エラー ID:** BC30020  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   適切な算術比較演算子を使用して、または`Like`2 つの値の型を比較する演算子です。
