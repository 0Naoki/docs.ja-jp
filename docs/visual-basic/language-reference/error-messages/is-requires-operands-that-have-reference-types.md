# <a name="39is39-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-39lttypenamegt39"></a>&#39;&#39;オペランドが必要です、参照型がこのオペランドの値の型は&#39; &lt;typename&gt;&#39;
`Is`比較演算子は、同じインスタンスに 2 つのオブジェクト変数が参照しているかどうかを判断します。 この比較値の型が定義されていません。  
  
 **エラー ID:** BC30020  
  
## <a name="to-correct-this-error"></a>このエラーを解決するには  
  
-   適切な算術比較演算子を使用して、または`Like`2 つの値の型を比較する演算子。
