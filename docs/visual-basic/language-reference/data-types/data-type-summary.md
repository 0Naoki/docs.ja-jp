-   <span data-ttu-id="c07ec-184">**オーバーヘッドです。**</span><span class="sxs-lookup"><span data-stu-id="c07ec-184">**Overhead.**</span></span> <span data-ttu-id="c07ec-185">一部の複合型では、追加のメモリ要件があります。</span><span class="sxs-lookup"><span data-stu-id="c07ec-185">Some composite types have additional memory requirements.</span></span> <span data-ttu-id="c07ec-186">たとえば、配列は配列自体および各ディメンションに余分なメモリを使用します。</span><span class="sxs-lookup"><span data-stu-id="c07ec-186">For example, an array uses extra memory for the array itself and also for each dimension.</span></span> <span data-ttu-id="c07ec-187">32 ビット プラットフォームでは、このオーバーヘッドは現在、12 バイト + ディメンションごとに 8 バイトです。</span><span class="sxs-lookup"><span data-stu-id="c07ec-187">On a 32-bit platform, this overhead is currently 12 bytes plus 8 bytes for each dimension.</span></span> <span data-ttu-id="c07ec-188">64 ビット プラットフォーム上でこの要件は 2 倍になります。</span><span class="sxs-lookup"><span data-stu-id="c07ec-188">On a 64-bit platform this requirement is doubled.</span></span>  
  
-   <span data-ttu-id="c07ec-189">**記憶域のレイアウトです。**</span><span class="sxs-lookup"><span data-stu-id="c07ec-189">**Storage Layout.**</span></span> <span data-ttu-id="c07ec-190">メモリ内に格納される順序は、宣言の順序と同じ安全に想定することはできません。</span><span class="sxs-lookup"><span data-stu-id="c07ec-190">You cannot safely assume that the order of storage in memory is the same as your order of declaration.</span></span> <span data-ttu-id="c07ec-191">2 バイトまたは 4 バイト境界などのバイトのアラインメントについて想定することもできません。</span><span class="sxs-lookup"><span data-stu-id="c07ec-191">You cannot even make assumptions about byte alignment, such as a 2-byte or 4-byte boundary.</span></span> <span data-ttu-id="c07ec-192">クラスまたは構造体を定義してそのメンバーのストレージ レイアウトを制御する必要があります、適用することができる場合、<xref:System.Runtime.InteropServices.StructLayoutAttribute>属性をクラスまたは構造体。</span><span class="sxs-lookup"><span data-stu-id="c07ec-192">If you are defining a class or structure and you need to control the storage layout of its members, you can apply the <xref:System.Runtime.InteropServices.StructLayoutAttribute> attribute to the class or structure.</span></span>  
  
### <a name="object-overhead"></a><span data-ttu-id="c07ec-193">オブジェクトのオーバーヘッド</span><span class="sxs-lookup"><span data-stu-id="c07ec-193">Object Overhead</span></span>  
 <span data-ttu-id="c07ec-194">`Object`を参照する、基本または複合データ型を使用してデータ型に含まれているデータだけでなく 4 バイト。</span><span class="sxs-lookup"><span data-stu-id="c07ec-194">An `Object` referring to any elementary or composite data type uses 4 bytes in addition to the data contained in the data type.</span></span>
