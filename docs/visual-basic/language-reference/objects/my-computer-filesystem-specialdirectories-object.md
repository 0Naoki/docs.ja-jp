# My.Computer.FileSystem.SpecialDirectories Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

頻繁に参照されるディレクトリへのアクセスに使用するプロパティを提供します。  
  
## 解説  
 `My.Computer.FileSystem.SpecialDirectories` のメソッドとプロパティについては、 <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>にオブジェクト。、 " " を参照してください。  
  
 詳細については、「[How to: Retrieve the Contents of the My Documents Directory](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md)」を参照してください。  
  
## 要件  
 **名前空間**: <xref:Microsoft.VisualBasic.MyServices>  
  
 **クラス:** <xref:Microsoft.VisualBasic.MyServices.SpecialDirectoriesProxy> （ <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>へのアクセスを提供します）
