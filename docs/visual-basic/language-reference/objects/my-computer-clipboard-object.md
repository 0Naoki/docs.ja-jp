# My.Computer.Clipboard Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

クリップボードを操作するためのメソッドを提供します。  
  
## 解説  
 `My.Computer.Clipboard` のメソッドとプロパティについては、 <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>にオブジェクト。、 " " を参照してください。  
  
 詳細については、「[Storing Data to and Reading from the Clipboard](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md)」を参照してください。  
  
> [!NOTE]
>  また、クリップボードの処理に <xref:System.Windows.Forms.Clipboard?displayProperty=fullName> のクラスのメソッドを使用できます。  
  
## 要件  
 **名前空間**: <xref:Microsoft.VisualBasic.MyServices>  
  
 **クラス:** <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy> （ <xref:System.Windows.Forms.Clipboard>へのアクセスを提供します）
