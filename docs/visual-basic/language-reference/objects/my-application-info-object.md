# My.Application.Info Object
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

バージョン番号、説明、読み込まれているアセンブリなど、アプリケーションに関する情報を取得するためのプロパティを提供します。  
  
## 解説  
 `My.Application.Info` のメソッドとプロパティについては、 <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>にオブジェクト。、 " " を参照してください。  
  
> [!NOTE]
>  ディスク上のファイルに関する情報を取得するために <xref:System.Diagnostics.FileVersionInfo?displayProperty=fullName> クラスのプロパティを使用できます。
