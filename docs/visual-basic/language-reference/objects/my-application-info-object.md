# <a name="myapplicationinfo-object"></a>My.Application.Info オブジェクト
バージョン番号、説明、読み込まれたアセンブリなど、アプリケーションに関する情報を取得するためには、プロパティを提供します。  
  
## <a name="remarks"></a>コメント  
 メソッドとプロパティについては、`My.Application.Info`オブジェクトを参照してください<xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>です。

>  プロパティを使用することができます、<xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType>クラス ディスク上のファイルに関する情報を取得します。
