# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
クラスを基底クラスとして使用できないことを指定します。  
  
## <a name="remarks"></a>コメント  
 **別の用語**です。 クラスは継承できませんと呼ぶことが、*シール*クラスです。  
  
 `NotInheritable`修飾子は、このコンテキストで使用できます。
