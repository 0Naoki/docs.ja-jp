# <a name="special-characters-visual-basic-reference"></a>特殊文字 (Visual Basic リファレンス)

特殊文字は、定義済み、コンテキストの文字が付加されるプログラム要素を変更します。 Visual Basic では、次の特殊文字がサポートされています。 

- [$](interpolated.md): 挿入文字列の文字。
