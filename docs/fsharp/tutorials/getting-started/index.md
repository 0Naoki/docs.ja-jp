# <a name="getting-started-with-f"></a>F の概要# #

.NET を使い始めるには主に 3 つの方法があります。

Windows の場合は、「[Getting Started with F# in Visual Studio](getting-started-visual-studio.md)」 (Visual Studio での F# の概要) を参照してください。

Windows、Linux、または macOS 上の軽量の IDE でのコード作成に興味がある場合は「[Getting Started with F# in Visual Studio Code with Ionide](getting-started-vscode.md)」 (Ionide と Visual Studio Code を使用した F# の概要) を参照してください。

いずれかの OS でコマンドラインを使用する場合は、「[Getting Started with F# with Command-line Tools](getting-started-command-line.md)」(コマンド ライン ツールでの F# の概要) を参照してください。
