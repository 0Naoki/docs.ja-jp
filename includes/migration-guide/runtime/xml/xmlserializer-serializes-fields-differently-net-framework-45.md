### <a name="xmlserializer-serializes-fields-differently-in-net-framework-45"></a>.NET Framework 4.5 で XmlSerializer によるフィールドのシリアル化が異なる

|   |   |
|---|---|
|説明|.NET Framework 4.5 で <xref:System.Xml.Serialization.XmlSerializer?displayProperty=name> を変更すると、シリアル化された XML でフィールドが異なる書式になります。|
|提案される解決策|この動作は、.NET Framework 4.5 のサービス更新プログラムで修正されました。 この問題を修正するには、.NET Framework 4.5 を .NET Framework 4.5.1 以降に更新してください。 または、次の構成の設定で 4.0 <xref:System.Xml.Serialization.XmlSerializer?displayProperty=name> 動作に戻されます。<pre><code>&lt;system.xml.serialization&gt;&#13;&#10;&lt;xmlSerializer useLegacySerializerGeneration=&quot;true&quot; /&gt;&#13;&#10;&lt;/system.xml.serialization&gt;&#13;&#10;</code></pre>|
