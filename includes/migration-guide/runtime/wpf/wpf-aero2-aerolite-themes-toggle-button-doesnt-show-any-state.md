### <a name="wpf-aero2-and-aerolite-themes-toggle-button-doesnt-show-any-state"></a>WPF Aero2 と AeroLite テーマのトグル ボタンに状態が表示されない

|   |   |
|---|---|
|説明|.NET Framework 4.5 の WPF&#39;s Aero2 と AeroLite テーマでは、トグル ボタンの &#39;on&#39; と &#39;off&#39; の状態の違いが表示されません。|
|提案される解決策|この問題は、サービス更新プログラムで修正されました。 この問題を修正するには、.NET Framework 4.5 を .NET Framework 4.5.1 以降に更新してください。 または、この問題を回避するために、トグル ボタンのスタイルを明示的に設定できます。|
|スコープ|Major|
|バージョン|4.5|
|型|ランタイム|
